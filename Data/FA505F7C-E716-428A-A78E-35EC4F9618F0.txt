一、中文摘要 
相對於票價較高的傳統航空公司(FSC, 
Full-Service Carriers)，低成本航空公司(LCC, 
Low-Cost Carriers)以其低廉的票價，已在世
界各地吸引了廣大的短程旅客。雖然台灣地
區目前並無國籍低成本航空公司營運，但我
國的地理條件和社會經濟條件皆顯示台灣地
區極具有發展低成本航空的潛力。鄰近東南
亞地區營運中的低成本航空公司都發展得非
常快速，可推測採用低成本營運方式經營我
國飛往亞洲區域的短程航線可能會是個很好
的選擇。 
航空業者所追求的不外乎能夠服務最多
的旅客、得到最大的利潤，而航線網路的規
劃則為影響這些目標是否能夠達成的重要因
素。過去的航線規劃方法大多是憑藉決策者
本身的經驗和主觀，雖然有經驗的決策者能
夠挑選出適合的航線，但對於航線網路的整
體表現卻通常無法在實際營運之前便加以考
量。本研究使用多目標規劃的概念，判斷利
潤最大化、載客人數最大化時，應開行哪些
航點才能達到最佳的整體效益。並以飛往鄰
近之東南亞區域為例，使用本研究所建構之
航點評選模式規劃我國籍低成本型態航空公
司營運飛往東南亞地區之最佳航點組合。 
 
關鍵詞：低成本航空、航線網路規劃、東南
亞地區 
 
Abstract 
Point-to-point network lets air passengers 
arrive the destinations directly, which is also a 
common network style for regional short-haul 
routes. If there are several direct routes that all 
start from the same airport and connect the 
central airport to various satellite airports, then 
the network is radiant. A radiant network can 
be considered as the basic component of a 
complex point-to-point network. In this study, a 
model for radiant network design has been built 
based on the concept of multi-objective goal 
programming (MOGP), which considers how 
to make the network that has the lowest 
operating cost and the highest revenue, and 
services the most passengers. The model then 
shows decision makers how to select satellite 
airports for operation to obtain optimal overall 
performance. A radiant network from Taipei to 
Southeast Asia region was designed in the 
study as an experiment, to validate the model’s 
practicability. The result demonstrates that the 
model can be used for radiant network design. 
 
Keywords: Airport selection, Radiant network 
design, Multi-Objective Goal 
Programming

(二) 設置廉價航廈之影響 
東南亞地區已有許多專門為低成本航空
所設置的廉價航廈，收費比一般航廈便宜。
為避免有任何有廉價航廈的機場因運量不足
或其他原因被過濾掉，同樣不設定任何初步
篩選門檻。評選的結果變為開航曼谷、胡志
明市、吉隆坡、馬尼拉和河內五個航點時為
最佳航線組合。開航不同航點數目時的整體
績效值變化如圖 4。 
 比較有無區分廉價航廈的兩種情況在各
個運算迴圈內刪除航點的順序(如表 2)，愈早
被刪除的航點表示個別績效愈差。由表 2 中
可以發現有廉價航廈的四個航點(曼谷、新加
坡、吉隆坡、亞庇)中，當不區分有無廉價航
廈時，新加坡因為競爭激烈，收入相對較低，
雖然運量大但卻是最先被刪除的。興建廉價
航廈的效益在亞庇更為明顯，當不考慮是否
設置廉價航廈時，亞庇因為運量少，是第 7
個被刪除的。但是如果將廉價航廈的影響加
入，就變成第 11 個被刪除。吉隆坡的運量在
各個航點中算是中上，而競爭也未特別激
烈，因此在兩個情境時都是最後一個被刪除
的。由這樣的結果，應可以推測廉價航廈的
設置可做為原本的機場條件不利時的補償，
應能有效提升機場與其他航線的競爭力。

名明顯提升，因此，廉價航廈的設置應
能有效提升機場之競爭力。 
 
(二)建議 
1. 因機場的使用成本被航空公司視為商業
機密，本研究用於進行實證分析的航點
成本資料為根據 Air Business 資料庫所
蒐集之航空公司類型、區域平均值加以
假設。此外，航空公司的機場使用成本
亦可能會隨著班次數的增減而有所改
變。本研究中所建構之航線網路規劃模
式在實際使用時仍必須先進行各航點使
用成本的估計，甚至在每一次運算迴圈
中都重新修正，才能得到最準確的結果。 
2. 本研究實證分析中將每條航線的承載率
都設為 70%。然而在實際營運時，各條
航線的承載率可能是不相同的。此外，
本研究中所設定之運量轉移規則也非絕
對。當兩航點的距離接近到一定程度
時，旅客願意轉乘的比例應會提高。在
實際運用本模式進行航線網路規劃時，
應依據實際的情形加以調整，以使評選
的結果更為精確。 
3. 本研究中假設當我國低成本航空公司不
開航某一航點時，若此航點已有傳統航
空公司直航，則會有 50%的旅客選擇直
航的傳統航空公司，另外 50%的運量則
轉移至低成本航空公司的鄰近航點。事
實上，因國籍傳統航空公司飛往東南亞
區域某些地區的班次極少，可能會有一
些旅客會因班表的緣故，被迫選擇低成
本航空公司。此外，航空公司的班表如
果能夠配合旅客的需求，或者是增加班
次數，可能也會刺激運量成長。 
4. 本研究實證分析使用之運量資料為推
估，尚需要更精準的運量預測才能夠得
到更符合實際的結果。此外，對於決定
不開航的航點，假設旅客選擇「轉乘」
與「搭乘傳統航空公司的直航航線」的
比例為各半，未來若能對旅客的轉乘行
為有更進一步的研究，也可使規劃的結
果更準確。本研究之模式未來亦可套用
至兩岸直航或其他區域之航線規劃，提
供我國航空業者之參考。
