1.摘要                                                 
    本計畫 AZ31 鎂合金滾軋板材經
ECAE多重降溫擠型 10道次後，晶粒
細化至 0.46 µm，basal面從原始滾軋板
材 Y面旋轉至與 Z面平行，ECAE後
再退火處理得到平均晶粒尺寸為 0.62
至 3.22 µm的試片，退火前後的織構並
未明顯產生變化。 
極細晶鎂合金在室溫至 125℃以
3×10-5至 6×10-2 s-1的應變率進行拉伸
試驗，結果顯示晶粒細化後在室溫的
降伏強度獲得很大的改善，在應變率
3×10-3 s-1 的拉伸降伏強度高達 394 
MPa。隨著溫度的升高，極細晶鎂合金
試片的應變率靈敏度從室溫的 0.024
增至 125℃的 0.321。與拉伸強度及晶
粒尺寸有關的 Hall-Petch equation, σ = 
σ0 + kd-1/2中的斜率 k值隨著溫度的升
高及應變率的降低而下降。在應變率
3×10-5 s-1，在 75℃及 100℃的時 k值皆
呈現負值。在室溫不同方向壓縮變
形，應變集中的滑移帶形成於 X、Y
及 X45Z方向的試片。不同方向壓縮試
片變形機構的不同，造成 Hall-Petch k
值的差異。當晶粒尺寸小到 0.46 µm
時，雙晶變形仍然出現，拉伸/壓縮降
伏非對稱性仍然存在。 
關鍵詞：等徑轉角擠型、鎂合金、晶
粒細化、極細晶、滑移帶, ECAE, Mg 
alloy, Grain refinement, Ultra-Fined 
Grain, Shear Band. 
 
2.前言 
    ECAE 廣泛應用於金屬材料的晶
粒細化與性質改變的研究；但對鎂合
金而言，因為受到滑移系統的限制，
在室溫時由於加工延展性不佳，通常
必須以較高溫度才能順利的進行擠製

擠製溫度大部分都在高於 200℃或在

2-2鎂合金 ECAE晶粒細化後的機械  
   性質 
晶粒在 ECAE 細化之後對材料機
械性質的影響，無論是在非鎂合金或
鎂合金部分在過去的文獻報告 [36, 
48-50]，都可以發現材料的機械性質明
顯的大幅提高，很顯然的藉由 ECAE
擠製的方式，可以使晶粒細化，並大
幅提高材料的強度。 
相較於晶粒細化對機構性質的影
響，由於晶粒在細化之後的 texture 發
生變化，因此在過去的文獻報告中亦
呈現 texture 對機構性質相當明顯的影
響[31, 36, 51-52]，像 Ding等人[36]將
起始晶粒尺寸 7 µm的鎂合金 AZ31以
ECAE不同的擠製路徑(A、BC及 C)，
在φ角 1200，ψ角 00的模具及擠製溫度
200℃的條件下，分別擠製四個道次，
得到的平均晶粒尺寸約 2.0 µm。在
texture 量測與拉伸試驗的結果，他們
發現由路徑BC與路徑C擠製所得到的
材料強度卻比未經 ECAE 擠製的材料
強度低，而由路徑 A 擠製所得到的強
度則明顯的增加 56%。於是 Ding等人
認為，材料在未經 ECAE 擠製時的
basal plane平行於拉伸軸方向，在經路
徑 BC與 C擠製四道之後，已旋轉至與
拉伸軸成 450方向，而路徑 A 擠製四
道之後的 basal plane，則幾乎維持與拉
伸軸平行的方向，因此他們認為 texture
對機械性質的影響高過於晶粒細化的
關係。除此之外，Mukai 等人[51]、
Agnew等人[31]以及Kim等人[52]對鎂
合金的機械性質實驗結果，也認為材

    經由上述文獻對極細晶材料的研
究報告，可以發現極細晶材料呈現高
強度及傾向低延展性的特性，探究其
原因，部分是因為材料低應變硬化速
率導致局部應變及早發生，部分是因
為材料在大量塑性變形時的調適能力
降低，而基本的變形機制則是造成極
細晶材料一些特殊行為的主因

研究，如 Carsley 等人[66]在探討晶粒
尺寸45 nm到1.7 µm範圍的Fe-10%Cu
壓縮變形行為時，他們發現經壓縮之
後的試片表面呈現 shear banding的局
部變形，且 shear bands寬度隨著晶粒
尺寸的增加而增加，並且與負載軸之
間的夾角接近 490，但 SB所形成的角
度事實上與晶粒尺寸是無關的，於是
Carsley等人認為 shear banding是極細
晶 Fe-10%Cu 唯一的變形機制。Jia 等
人[67]在觀察晶粒尺寸 80 nm至 20 µm
範圍純鐵的變形行為時，他們發現晶
粒降至極細晶粒尺寸範圍時，材料的
變形模式產生相當的變化。當他們觀
察晶粒尺寸為 20 µm-Fe及 980 nm-Fe
時，均呈現均勻的壓縮變形，但在微
觀組織的觀察下並未發現試片表面有
任何的 shear bands 及 twins 的組織產
生，於是他們觀察晶粒尺寸小於 300 
nm-Fe時，發現在塑性變形發生時，試
片表面很快的產生 shear bands，而且
shear bands 除沿著剪平面隨應變量的
增加而增長變寬之外，並產生新的
shear bands，經量測每一種晶粒尺寸試
片的 SB 方向時，他們發現大部分的
SB方向與負載軸夾角在 41-460，顯示
SB 方向對晶粒尺寸的反應是不敏感
的。Jia等人認為 shear bands僅在材料
有足夠小的晶粒產生，當晶粒尺寸低
於某個晶粒尺寸時，材料明顯的由均
勻的變形轉變成非均勻的變形，因此
shear bands成為明顯的塑性變形模式。 
     
2-4鎂合金的塑性變形與機械性質 
  2-4-1鎂合金的塑性變形行為 
    相較於前面所述一些非鎂合金的
變形機制，差排滑移及變形雙晶則為
鎂合金的主要變形機制。過去文獻曾
對單晶鎂的研究報告指出 non-basal滑
移與 basal滑移在室溫時的臨界分解剪
應 力 (critical resolved shear stress, 
CRSS)相比較，non-basal滑移有著相當
高的 CRSS值，這表示 basal滑移在室
溫鎂及鎂合金中比 non-basal滑移更容
易被啟動，而隨著溫度的升高，
non-basal 滑移的 CRSS 值降的很快，
因此，當 basal滑移在鎂室溫佔滑移模
式主要的地位時，non-basal 滑移在溫
度升高之後的啟動也漸增加[69]。近來
也有學者觀察到 non-basal滑移出現在
室溫鎂合金 AZ31的塑性變形當中，並
得到較佳的延展性[70-72]。由於鎂在
室溫時僅能提供有限的滑移系統，進
而導致延展性不佳的結果，因此需要
其他變形機制的啟動才能使鎂產生任
意形狀的變化。雙晶(twin)在鎂的室溫
變形也是重要的變形機制之一，可分
為壓縮雙晶及拉伸雙晶，當鎂受到的
7 
變形量時(after fracture)雙晶呈現窄小
帶狀的形貌，這些雙晶型態經 EBSD
分析之後，確認它們分別是晶粒沿
<c>-axis 拉伸的 { }2110 、與晶粒沿
<c>-axis 壓縮的 { }1110 以及 { }2330 等雙
晶。另外 Koike等人以 ECAE的 AZ31
進行室溫拉伸試驗，拉伸方向幾乎垂
直 <c>-axis ， 他 們 除 了 觀 察 到
{ }1110 twins之外，亦發現到{ }2110 twins
的出現。類似的現象，也曾出現在對
單晶鎂及多晶純鎂的<c>-axis 方向的
壓縮的觀察報告[69,74]，他們推測這是
在卸除負荷時，因非等向塑性
(anisotropic plasticity)因素造成的殘留
應力所引起的{ }2110 twins。{ }1110 twins
與{ }2110 twins 明顯的差異在於形成時
所需要的 CRSS 值不同， { }2110 twins
比{ }1110 twins的 CRSS值小很多，通常
僅需要 2至 3 MPa。因此，Koike等人
認為{ }2110 twins 為了調適差排滑移所
造成的應力集中，很容易發生在初期
的變形，而<c>-axis壓縮雙晶的形成則
會引起局部變形(shear localization)及
破壞發生。也有一些文獻研究報告指
出[75-76]，雙晶在小晶粒鎂合金室溫
變形中很難形成，Lahaie等人[75]觀察
晶粒尺寸 1.2 µm的鎂合金 AZ91的變
形行為，並未觀察到雙晶的產生，而
Yang等人[76]對於晶粒尺寸 1.4 µm的
極細晶鎂合金 AZ31 在室溫壓縮變形
的觀察，他們同樣也未觀察到雙晶的
產生。 
 
2-4-2鎂合金的機械性質 
    至於影響鎂合金塑性變形的一些
因素，從過去一些文獻研究報告[31, 
77-79, 81-85]可以發現這些因素，包括
晶粒大小、應變率、溫度及織構(texture)
等幾個因素。 
    在晶粒大小對機械性質的影響研
究部分，Barnett等人[77]以平均晶粒尺
寸 3-22 µm 的 AZ31 鎂合金試片觀察
晶粒大小對壓縮變形的影響。他們以
EBSD 分別量測在擠型溫度 220℃及
450℃得到 3 µm及 16 µm晶粒組織的
織構，量得兩者的 c-axis與擠型方向幾
近垂直。從室溫至 200℃的應力-應變
曲線，隨著應變量的增加，他們發現
室溫的變形曲線在一個明顯的降伏點
之後產生應變硬化直到最高應力(peak 
stress)，但在高溫較高應變量時，因塑
性不穩定的發生而產生明顯的應變軟
化現象。Barnett等人亦發現 22 µm晶
粒的室溫壓縮應變硬化程度大於拉伸
應變硬化程度，根據過去文獻研究報
告，這差異是因為 c-axis 受到壓縮時
{ }2110  twins的形成所造成。而 Barnett
等人分析變形曲線時，發現室溫壓縮
的變形曲線全部呈現凹的曲線
(concave)特徵，拉伸曲線則呈現凸的
曲線(convex)特徵，這與Kelley等人[78]
在觀察單晶鎂試片經歷 { }2110 twins 變
形時發現凹曲線的特徵相同。在晶粒
尺寸與 0.2%偏距降伏應力的關係，當
變形機制由雙晶控制轉變成滑移控制
時，Barnett 等人亦發現兩個之間存在
9 
低於再結晶溫度時，除了雙晶的變形
之外，亦包含差排在 basal 面及
non-basal 面的滑移變形，而隨著變形
溫度的增加，再結晶晶粒產生，明顯
的動態再結晶發生在變形的過程。 
    在鎂合金受力方向與 texture 影響
的研究，早期的像 Kelley等人[78]觀察
鎂在室溫的變形模式為代表，他們選
用單晶鎂，並以七種不同方位
(orientation)進行平面應變壓縮實驗
(plane-strain compression)，藉以確認變
形系統的啟動及評估啟動它們所需的
CRSS 值。Kelley 等人在 4%壓縮應變
時，觀察到 basal slip的痕跡，他們認
為在壓縮軸與 c-axis 之間只要 0.40或
更大的角度的偏差量導致 basal slip的
啟動 (代表只要很小的 CRSS 值就足
以啟動它)。當應力高於 40 ksi時，出
現細的 bands與{ }1110 twins出現且相互
平行，像這樣的 bands也曾被其他學者
們研究指出，認為這些 bands 是由
{ }1110 twins 所造成並且緊跟著這些
{ }1110 twins 很 快 的 再 發 展 出
{ }2110 retwins 的二次雙晶以及在二次
雙晶內發生 basal slip 的變形機制
[69]。在壓縮方向垂直於受限制的
c-axis 時，出現了 { }1110 及 { }2110 twins
所組成的變形機制， Kelley等人反而
沒 有 發 現 任 何 prismatic slip 及
pyramidal slip 等痕跡出現，對於這樣
的結果 kelley 等人認為可能是因為不
完美的平面應變壓縮因素所導致。     
在壓縮方向垂直於未受限制的 c-axis
時，結果促進了{ }2110 twins 變形至大
約 6%的應變量，kelley 等人認為
{ }2110 twins 系統促使晶體產生旋轉，
使整個雙晶變形產生較高的破壞強

basal slip，促使產生較低的應力及較佳
的延展性。 
    Agnew 等人 [31]則觀察 ECAE 
AZ31 鎂合金在室溫不同方位的拉伸
延展性，他們根據 ECAE 擠製試棒與
拉伸試片的幾何位置關係切取不同方
位的拉伸試片進行實驗。結果他們發
現這些不同方向試片得到的機械性質
有顯著的差異。Agnew 等人認為拉伸
試片的方位與起始 texture 的關係對於
決定降伏應力比 Hall-Petch 對降伏應
力的影響更重要。而Figueiredo等人[85]
將 ECAE 擠製得到之不同晶粒尺寸的
AZ31 鎂合金試棒切取三個相互垂直
方向(X-Y-Z)的試片進行室溫壓縮試
驗。他們發現材料的應力-應變曲線圖
因為不同的加工過程及壓縮方向造成

    過去一些學者曾對 ECAE 鎂合金

先旋轉 1800，當降低溫度擠製時，擠
製速度必須跟著降低，否則試棒會破
裂。而在非鎂金屬的擠製速度高低並
不影響擠製的結果[23]，但對鎂合金及
其他所謂”不易擠製”的材料，如鈦

成，加上 X方向拉伸試片的 basal面與
拉伸方向平行，此時差排在 basal面的
分解剪應力等於 0，使得差排無法在
basal面上產生滑移，因此形成以 prism 
slip為主的<a>差排滑移。對 Z方向試
片的壓縮變形而言，由於壓縮軸沿試
片晶粒的 c-axis方向，同樣因壓縮雙晶
較不易形成，加上壓縮軸與 basal面垂
直， prism slip 有關的 Schmid factor
值會等於 0，因此可能的差排滑移為
pyramidal slip 加上可能少數 Schmid 
factor不為零的 basal slip。對 X45Z方
向試片的壓縮變形而言，因為壓縮軸
與試片晶粒的 c-axis夾約 450，Schmid 
factor值為最大，因此差排非常容易在
basal面滑移。至於 Y方向試片的壓縮
變形，因為與 X 方向試片具有相同的
texture，所以同樣以{ }2110 拉伸雙晶變
形為主。由於極細晶粒 0.46 µm的 SBs
微觀組織無法以 SEM觀察，經由圖十
七(a)的 s-s curves觀察可以發現到不同
晶粒尺寸所呈現的形狀相類似，代表
著它們具有類似的變形機制，因此在
極細晶試片裡的 SBs，我們推論它也應
該具有相同的微觀結構。從圖二十六
較大晶粒尺寸的 SBs 的微觀結構，可
清楚看出在 SBs 區域內呈現均勻分佈
的雙晶組織，在 SBs 區域外，即基材
區域並未發現任何的雙晶產生。另
外，經由圖二十七的 TEM觀察到極細
晶 0.46 µm 試片經壓縮後有變形的存
在，顯示即使在次微米細晶中仍有變
形雙晶出現。我們以 TEM diffraction 
pattern 分析方式來確認 0.46 µm 極細
晶內部的變形雙晶型態，如圖二十九
的 TEM 照片及電子束同樣來自 [ ]1021
方向的兩個 SAED patterns，其 g = 0001 
vector與試片的(0001)面垂直，經量測
結果，發現這兩個由雙晶邊界將晶粒
分成上半部與下半部的 SAED patterns
的 g = 0001繞著 [ ]1021 旋轉了 870，此
結果說明在 4.4%應變量的變形雙晶為
{ }2110 拉伸雙晶型態[91]。另外，從
SEM、TEM及應力-應變曲線形狀的觀
察結果，證實了沿 X 軸壓縮時雙晶變
形是室溫變形開始階段，形成 SBs 的
主因。文獻 [66-67]分別觀察極細晶
Fe-10%Cu及純鐵壓縮變形行為，他們
在材料變形的初期都觀察到 SBs 產生
在試片的表面，因此他們都認為這些
SBs 是極細晶材料塑性變形的主要機
制。但事實上形成這些 SBs 的主因，
依本實驗所觀察到的 SBs 內部微觀組
織是由變形雙晶所組成，完全不同於
文獻[65]對極細晶商業純鐵所觀察的
SBs 是因差排變形使晶粒被拉長所形
成。而過去一些文獻[75-76]研究報告
亦曾指出，當鎂合金晶粒很小時變形
雙晶根本無法生成，因為它所需的生
21 
圍，因而產生偏離 Hall-Petch關係的現
象，Yu 等人[63]認為因為晶粒尺寸的
降低造成可移動差排的不足，而且晶
界動態回復速度增加及應變硬化速率
降低，因而產生高降伏強度及過早頸
縮現象，以致次微米細晶範圍的
Hall-Petch關係與大於 1 µm晶粒尺寸
的 Hall-Petch關係不同。但從本實驗所
研究的晶粒尺寸範圍 0.46-3.22 µm，只
有最大晶粒 3.22 µm 試片均略低於
Hall-Petch關係式的預測強度。我們認
為可能的原因是在 3.22 µm 的試片退
火後晶粒較不均勻，粗大晶粒開始出
現，因此造成上述的結果。文獻[38, 77]
對 AZ31 鎂合金室溫變形行為的研究
結果都得到降伏強度隨著晶粒尺寸降
低而增加的 Hall-Petch關係，而且發現
拉伸與壓縮強度呈現非對稱性的降伏
行為，是因為{ }2110 雙晶造成的原因[77, 
80]。從變形機構來看，拉伸與壓縮降
伏非對稱性的發生是因當鎂合金沿
basal面壓縮時，變形會以{ }2110 拉伸雙
晶方式進行，而沿 basal面拉伸時，若
以變形雙晶方式進行，只有 { }1110 或
{ }3110 壓縮雙晶可以滿足變形，但因形
成壓縮雙晶所需的應力太大，因此試
片會以 prism slip的方式進行而無法形
成壓縮雙晶。因為拉伸雙晶所需的應
力很小，因此通常沿 basal面壓縮強度
會比沿 basal面拉伸強度來的小。 
以雙晶變形為主的變形所得的
Hall-Petch k值會大於以差排滑移為主
的變形所得的 Hall-Petch k值[77]，因
此當晶粒變小時，雙晶變形所造成強
度的上升會比差排滑移所造成的強度
上升來的快。因此，在大晶粒時，拉
伸雙晶變形所需的應力較小，但當晶
粒小至某一大小時，拉伸雙晶變形所
需的應力便會大到與差排變形相同而
不再以拉伸雙晶方式進行變形。本實
驗的結果顯示即使晶粒小到 0.46 
µm，拉伸雙晶仍然形成，因此壓縮/
拉伸非對稱性的現象仍然存在。 
根據Armstrong[92]由 pile-up理論
所導出 Hall-Petch 的 k 值的公式是由
晶界旁啟動變形所需的分解剪應力τc
及方向因子 m (orientation factor)所決
定。圖三十為鎂合金 X、Y及 X45Z不
同方向試片在室溫的壓縮後所得的 k
值。在不同方向試片所得不同的 k 值
乃由不同變形機構所造成。在 X 方
向，由於壓縮軸與試片晶粒的 c-axis
垂直，因此 k值中的τc由{ }2110 拉伸雙
晶變形所決定。在 X45Z方向，由於壓
縮軸與試片晶粒的 c-axis夾約 450，因
此 k值中的τc由 basal slip所決定，所
以 k值在此三方向中是最小的。至於 Z
方向，由於壓縮軸沿試片晶粒的
c-axis，若要用雙晶變形便是用壓縮雙
晶來進行，但因壓縮雙晶不易產生，
而且此時prismatic slip的Schmid factor
值為 0，因此變形可能以 <a+c> 
non-basal 差排滑移進行，因此 k值中
的τc便較 X45Z時的τc來的大，因為 Z
方向的 pyramidal slip的 m值與 X45Z
時的 m值相差不大，因此，Z方向的 k
值之所以大於 X45Z，主要應是不同
Burgers vector之差排變形所致。 
圖三十一顯示當Hall-Petch k值隨
溫度的上升而下降，在應變率 3×10-5 
s-1，在 75℃及 100℃ k 值已為負值，

在 X、Y、Z方向及 2.53 µm在 X45Z

陸 結論 
1.AZ31鎂合金的 ECAE晶粒細化機制
主要為動態再結晶，與傳統 ECAE
擠製的非鎂金屬由低邊界角度隨著
擠製道次增加而逐漸轉變成高邊界
角度的細化機制是不同。 
2.AZ31經ECAE三個階段降溫擠型 10
道次後，晶粒尺寸由 6.91 µm細化至
0.46 µm，basal面織構從與 Y面平行
轉至與 Z面平行。 
3.沿 X 方向拉伸以差排滑移變形為
主，其拉伸降伏強度隨晶粒的細化
與 擠 型 道 次 的 增 加 而 增 加 。
Hall-Petch關係式中的 k值隨著溫度
的升高及應變率的降低而下降。在
應變率為 3×10-5 s-1，在 75℃及 100℃
得到的 Hall-Petch斜率為負的 k值。 
4.沿 X 方向壓縮以拉伸雙晶變形為
主，其 Hall-Petch k值大於以差排滑
移變形為主的 Hall-Petch k值。 
5.在晶粒小到 0.46 µm的範圍，拉伸雙
晶仍然形成，壓縮強度/拉伸強度之
非對稱性仍然存在。 
6.不同方向試片的壓縮變形行為產生
一些差異性，主要是變形機構的不
同造成，其中垂直 c 軸壓縮時(X 與
Y方向)，變形以{ }2110  tension twins
為主。當壓縮平行 c軸時(Z方向)，
可能的變形為 pyramidal slip，且可
能有少數的 basal slip。而壓縮軸與 c
軸夾 45 度時(X45Z 方向)，變形以
basal slip變形為主。 
7. 0.46 µm至 3.22 µm的晶粒尺寸範圍
在 X、Y及 X45Z方向壓縮試片表面
皆形成應變集中的 shear bands變形
特徵。在 Z方向壓縮則無 shear bands
的形成。
