摘要 
影像向量化，是工程圖管理與應用領域的重要課題。要從由大量的線條及字元所組成
的工程圖影像，重建線段與字元向量資訊，必須進行：雜訊去除、傾斜校正、影像二值化、
影像細化、直線偵測、文字識別…等各種影像處理技術，以獲取完整的向量成果。在一連
串的處理程序中，字元的萃取與辨識，一直是最關鍵與困難的自動化項目。 
因此，本研究的重點在於如何進行 2D 工程圖的分析，將線條及文字所組成的工程圖，
對其影像進行字元萃取。首先，方法為先將影像二值化，經過二值化處理可把背景
(Background)與前景(Foreground)分離，再根據此影像中的工程圖所含特徵，利用點與點
之間的串接找出影像中線條與文字的所有可能群聚區域。接著以形態學影像處理的兩種形
態運算：膨脹（Dilation）與侵蝕（Erosion）來重建影像。實驗結果顯示本文所用的方
法可以萃取出工程圖影像字元的位置，本研究計畫研究成果可以應用於商業用途。 
關鍵詞：字元萃取、工程圖影像、二值化、形態學。 
 
一、人才培育成果說明： 
本研究計畫共有四位研究生實際參與，另有 7位大學專題生參與相關的車牌辨識的研
究，參與學生除熟練 Java 程式語言外，並瞭解開發工具 Eclipse3 整合開發環境，實際瞭
解系統開發的過程；而在學術研究上，參與者可以對於二值化、形態學的膨脹與侵蝕、字
元萃取等相關技術有更深入的瞭解。另外大學生參與研究計畫，可提早瞭解研究過程並認
識產業的研發需求。 
 
二、技術研發成果說明： 
本計畫實現一個工程圖字元萃取的雛型，所開發的成果軟體，將慢慢投入商業應用，
因此本計畫程式設計將採用目前普遍使用的 C#程式語言，C#除了在網路上有非常豐富的資
源，物件導向(Object-oriented)的特性可使系統有彈性易於維護。所使用的所有工程圖影
像由光特資訊科技股份有限公司所提供，並在軟體技術上提供支援及配合。 
本研究所使用測試影像為光特資訊科技股份有限公司所提供的三張工程圖影像，三張
解析度皆為 640＊480。實作平台使用 Processor：Intel Core 2 Duo CPU  E8400 @ 3000 MHz，
Physical Memory：4096 MB(2 x 2048 DDR2-DRAM)，Hard Disk：500 GB(7200 RPM)。本計
畫所開發的成果軟體，將為投入商業應用雛型，因此，程式設計將採用目前普遍使用的 C#
程式語言，C#除了在網路上有非常豐富的資源，物件導向(Object-oriented)的特性可使系
統有彈性易於維護。開發工具為 Microsoft Visual Studio 2008。
