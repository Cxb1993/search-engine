研究摘要(500 字以內)： 
 
 資料探勘 (Data Mining) 技術於近幾年來開始蓬勃的發展。其目的是要從資料庫中找
出隱含於其中的有用資訊。其中關聯規則探勘 (Association Rule Mining) 的發展是為了挖
掘出具有高度關聯性的規則並加以應用。然而我們發現關聯規則探勘在某些應用上會發生
不適合的情況。例如，決策者可能希望針對特定消費項目的客戶群進行電話行銷，所以必
須找出消費者特徵與消費項目之間的關聯性。可是傳統的關聯規則探勘可能無法正確地找
出其中的關聯性。原因有二，第一、資料內部可能存在大量不平衡的消費項目，例如日常
用品的消費次數比起珠寶金飾等，就有很大的差距。像這樣的情況，很難產生出消費項目
是珠寶金飾的規則。另外由於消費日用品的次數太多，導致無法找到日用品真正的消費特
徵。第二、在資料庫中，某一類型之消費特徵的人特別多，導致此消費特徵在每一類的消
費項目可能出現很多次，使得其可能成為很多消費項目的主要消費特徵，造成此消費特徵
在任何消費項目上，均不具代表性。因此，本計畫將研發特徵規則探勘 (Characteristic Rule 
Mining) 的系統，引用卡方分析法 (x2 Analysis) 來評估資料之間的關聯性，並改良關聯規
則中支持度 (Support) 的計算方式，幫助決策者了解隱含於資料內的訊息，進而改善管理
或制定新的行銷策略。 
 
  另一方面，資料探勘技術的發展上，針對實務研究的例子甚少。其中包括從探勘計畫、
資料來源整合、資料前處理、評估探勘結果、…等以及完整地討論與提供整體解決方法。
其原因有二，第一、取得大量實際資料的困難。第二、發展出來的探勘演算法並未與資料
庫管理系統緊密結合，以致於在現實環境下探勘系統的可用度不高。在本計畫中，我們將
針對上述問題提出解決方案，實作一套以 Microsoft SQL Server 為後盾的資料探勘系統。整
合了前處理、邊界值過濾、視覺化工具、動態量化、…等功能，並且與聯邦網通合作，取
得實際信用卡消費的紀錄作為實驗的資料來源，期望發展具良好穩定性與挖掘能力的資料
探勘系統。

 在說明問題前，我們先介紹一些名詞與定義，一筆實例 (Instance) I 是包含 N 個特徵屬
性(Characteristic attributes) 的屬性值加上 1 個類別屬性 (Class attribute) 的屬性值所形成的
一筆紀錄；在特徵屬性的屬性值集合 X 與類別屬性的屬性值 Y 之間的規則以 X→Y 表示，
一實例中的特徵屬性值若包含規則 X→Y 中的 X，則稱此實例符合規則 X→Y 的前提；一
實例中的特徵屬性值若包含規則 X→Y 中的 X，且實例中的類別屬性值等於 Y，則稱此實
例符合規則 X→Y。規則 X→Y 的支持度 (Support) S 為在所有的實例中，符合規則 X→Y
的比率值，即 S = (符合規則 X→Y 的實例數量)／(總實例數量)，而規則 X→Y 的信賴度 
(Confidence) C 為在符合前提的實例集合中，也同時符合規則的比率值，即 C ＝ (符合規則
X→Y 的數量)／(符合規則 X→Y 前提的實例數量) 。 
 
  傳統上分類問題要處裡的是從具有Ｋ個屬性並已被分類到Ｍ個已知類別的實例集合
中，找出各類的特徵描述或分類規則，這一類的問題已經被大量的研究過。然而傳統的分
類演算法無法產生類別不互斥的規則，原因有兩個：第一、分類演算法基本上是針對類別
互斥的資料所設計，例如分類人種的問題上，一個人被分類到是白種人，就不會是黃種人，
不可能又是白種人又是黃種人，然而在分類消費項目與消費者特徵的問題上，一個人可能
被分類到 “消費日用品” 但同時也被分類到 “消費旅遊”，因為這個人在這兩個項目都有消
費，如果整個資料集合都是這樣的情況，以傳統分類問題的角度來說，資料內充滿雜訊，
而分類演算法基本上是對雜訊相當敏感的，以致產生的規則不精確，事實上這樣的雜訊其
實並非雜訊。第二、資料內部可能存在嚴重地不平衡類別分佈。例如分類禿頭與否問題上，
雖然具有類別互斥的特性，但是禿頭在人口比例裡相當地少，所以可能無法挖掘出類別為
禿頭的規則。 
 
針對上述的傳統分類演算法的問題，我們認為使用關聯規則探勘 (Association Rule 
Mining)能大幅改善第一個問題，原因是關聯規則原本就是設計找出所有項目 (屬性值) 間
的關聯性，所以若一個人時常消費日用品項目但同時也時常消費旅遊項目，關聯規則探勘
仍能找出什麼樣的特徵與日用品項目和旅遊項目會常常一起出現。不幸地，上述的第二個
問題，利用關聯規則探勘，仍然無法解決不平衡類別分佈的問題，若某一類別的實例個數
很多，會造成不論其特徵為何，信賴度都會偏高，例如日用品人人都會買，所以任何一種
消費特徵的規則裡，類別為日用品的規則，信賴度都會偏高。若某一類特徵的實例個數很
多，使得此特徵在不同類別出現的次數都很高，會造成很多類別裡，都有此特徵的規則。
