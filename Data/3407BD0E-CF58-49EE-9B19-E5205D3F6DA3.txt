1. 簡介 
 全細胞醣化系統(whole cell glycosylation system) 是利用細胞內
所含的多種醣化酵素，對不同基質來進行醣化反應，而左旋維他命 C
是本研究所選用的基質。左旋維他命 C 其構造是α-ketolactone，為一
種水溶性極高，且對光、熱高度敏感的化合物，在生理酸、鹼值下以
單價氫氧根陰離子存在，易氧化形成 dehydro-L-ascorbic acid 
(DAAD)。本研究的目的在於利用糖化反應，使左旋維他命 C 生成糖
苷，以改善左旋維他命 C 不穩定性質，圖 1 簡要的表示出其中兩種
L-維他命 C 糖苷之結構式。為有效的反應生成 L-維他命 C 糖苷，所
採用的方法有微生物轉化法與酵素轉化法兩種類型；表 1 為微生物
轉化維他命 C 醣苷的文獻比較，從表中可以觀察到不同微生物對於
醣化的轉化效果。在本研究中亦比較不同菌種 Xanthomonas 
campestris pv.、Bacillus subtilis、Geobacillus stearothermophilus、
Aspergillus niger、Leuconostoc mesenteroides 的糖化 (glucosidation)與
轉糖 (transglucosidation)生成 L-維他命 C 糖苷效果，並挑選具高維他
命 C 醣苷產率的菌種 Aspergillus niger，深入探討全細胞糖化與轉糖
酵素對維他命 C 醣苷生成之活性高低的條件，使之配合全細胞酵素
固定法來生產維他命 C 醣苷。此方法的優點在於無需使用高單價之
特定酵素來進行轉化反應，並利用微生物體內的多種酵素來提升醣苷
2. 材料與方法 
 2.1 菌種 Aspergillus niger 
2.2 培養基條件 
培養基條件採用Shiow-Ling’s 培養基，培養基成分為 (g/L) maltose, 
4.5; yeast extract, 6.6; K2HPO4, 0.5，於 30  ℃ 及 200 rpm 下在 100 mL
有溝錐形瓶作前培養 (v/v)進行 6 天的培養。 
2.3 全細胞的醣苷轉化率測定 
取不同時間點的菌體細胞 0.1 克，令其與反應基質維他命 C 及麥芽
糖(1/5 毫莫耳)，反應液 2 mL 0.02 M Sodium acetate (pH 5.5)，在 30℃、
200 rpm 的條件下培養 6 小時後，以 HPLC 直接測定維生素 C 與維生
素 C 醣苷的濃度，用以計算轉化率與比較轉化效率之差異。 
2.4 細胞固定化 
取 0.7 % 乾菌重與 4 % (w/v) calcium alginate 於 0.25 M CaCl2 
solution 中，形成 4 mm 的 固定化 Beads，置放於 4℃中的環境中 12
小時，使之進行固定化細胞的程序，並以此作為固定化酵素的方式，
並比較不同菌體固定量對於醣化效果的影響。
