報告內容 
一、 前言 
近年來，節能減碳已成為各種電器產品運作時必須考慮的重要因素，對於能源的妥善運用以及新
能源的開發皆成為一刻不容緩的議題。目前用於照明設備的電能約佔 21%的總電能，而電能又佔總能
源使用量的一半，所以現今耗用於照明的能源約佔能源總使用量的 10%。氮化鎵（GaN）等具有對稱
六方晶系（wurtzite）結構之 III-V 族半導體材料，屬於直接能隙（direct band-gap）半導體，相當適合
作為發光元件的材料。藉由化合物成分的多寡，可調變出能隙從 6.5 eV 到 0.7 eV 的三元 ( ternary ) 或
四元 ( quaternary ) 化合物半導體，所對應之發光波長涵蓋深紫外光到遠紅外光波段範圍。利用此一可
任意調變所需發光波長的特點，使氮化鎵系列半導體材料被廣泛的運用在發光二極體(Light-Emitting 
Diode, LED)與雷射二極體( Laser Diode, LD )等光電元件上。早期由於氮化鎵晶體與成長基板的晶格常
數不匹配，使得氮化鎵系列藍/綠光發光二極體的磊晶品質相較於砷化鎵 ( GaAs ) 系列紅/黃光發光二
極體的磊晶品質相差甚遠。直到日本日亞 (Nichia) 公司成功地將氮化鎵藍/綠光發光二極體結構成長
於 ( 0001 ) 藍寶石 ( sapphire ) 基板上，並且不斷的提升其發光效率，使得人類擁有全彩化之發光二
極體的夢想得以實現。然而此突破不僅帶給顯示科技一個新的趨勢與方向，也使固態照明產業有機會
取代現有的傳統照明設備，並逐漸展露出 LED 成為下一世代新照明光源的潛力。LED 晶片在能量的
利用效率上，相較起以熱輻射發光之白熾燈(Incandescence Lamp)或傳統日光燈(Fluorescence Lamp)，
白熾燈之效率約為 10 lm/W，能源轉換效率大約只有 5%，螢光燈之效率為 60~80 lm/W，能源轉換效
率大約只有 25%，LED 應具有更高比例的能源轉換效率，故 LED 實為一具有極大應用潛能之新世代
照明光源。同時，LED 更具有輕巧、耗電量少、壽命長、耐震、無汞、安全性高、色彩多樣性等眾多
優點，尤其是其節能與潔能的特性，更是符合當前世界潮流之省能利用與環保需求的發展趨勢，使得
白光 LED 儼然成為未來照明市場中最耀眼的明星產業，也是世界各先進國家積極發展固態照明的重點
之一。早期在車用方面，第三煞車燈、車尾燈與面板燈源，皆已被發光二極體所取代；在交通號誌上，
利用紅、黃、綠三色發光二極體代替傳統燈泡，可有效的省下電力與提高行車安全；在顯示科技領域，
利用紅、綠、藍發光二極體當作背光源，可有效的縮小面板的體積與重量，也可以使得色彩飽和度提
升至 120%以上。根據美國 OIDA(Optoelectronic Industry Development Association)的分析報告指出 [1]，
至 2020 年白光 LED 的發光效率將可達到 200 lm/W 以上，如能全面取代日常生活中室內與戶外之照明
器具，預估屆時將可節省 50%的照明電力與 10%的總電力，並減少 2 億噸二氧化碳的排放，及減少傳
統燈具中毒化物的使用，對地球環境的保護有著多重優點。另外台灣有很好的 LED 產業上下游結構，
技術落後先進國家不多，故政府把 LED 列為優先發展之能源產業。 
LED的發光效率進步非常快速，2010 年 OSRAM 開發出在 350 mA操作條件下，可得到 117 lm
光通量的產品 [2]。隨即，在2011年，CREE 則發表了效率可達 231 lm/W 的產品 [3]。加上現今以發
展高功率LED為主，如果未能有效的散熱，就會提高p-n接面(junction)及晶片整體之溫度，而過高的LED
溫度將造成下列的影響： 
(a) LED 晶片的失效：高階 LED 晶片 p-n 接面所能承受極限溫度最大值為 185˚C (LUXEON K2, 
Philipslumileds) [4]，而一般設計工作溫度不超過 120˚C，但 LED 的理想工作溫度約 60˚C。 
(b) 降低發光效率：圖一為 LED 晶片的發光效率對於 p-n 接面溫度(junction temperature)的關係圖 [5]，
可知 LED 晶片之發光效率受到其接面溫度的影響非常顯著。 
(c) 造成中心波長的漂移：溫度越高，中心波長越往短波長移動，這也造成使用 LED 之產品開機時，
色彩變化之問題。 
(d) 壽命及光輸出的下降：壽命及光輸出隨著 p-n 接面溫度的增加而下降。 
(e) 封裝高分子材料劣化。

三、 文獻探討 
在 LED 晶片電極型式對電流分佈的影響之分析方面 Laser lift-off (LLO)法 [6-8] 應用於高功率
LED 封裝結構上是一個相當理想的方式。LLO LED 以銅或矽等高導熱係數材料取代低導熱係數之基板
材料。LLO LED 具有垂直結構使其可提供較佳之電流分布、較有效率的散熱條件及優越的可靠度
[9-11]。經過 LLO 製程，GaN 結構變成僅有一薄膜貼附在一個轉換晶圓；因此藉由 LLO 製程所生產的
LED 稱為 ThinGaN LED。雖然 ThinGaN LED 可提供較佳的電與熱特性，但電流擴散的問題仍然受到
輸入功率的影響。而電極的幾何形狀會對電流擴散的狀況造成影響，一個好的電極設計可藉由較均勻
的電流分布而提供 LED 電特性的改善[12]。故一些評估電流擴散與電極分布間關係的解析模型已被提
出[13,14]。許多不同的電極形狀的設計在過去也被提出並製作來找尋出較佳性能的 LED [15,16]。 
一般 LED 之研究多為晶片結構方面，對於晶片內部散熱及封裝結構之熱分析大約自 2004 年起
Arika[17]等人發表之相關研究較為完整。其藉由計算流體力學軟體的使用，將 LED 產品作為模擬分析
之模型，分析出其封裝方式、材料、散熱裝置等對於 LED 晶片層溫度分布之影響，並歸納出封裝條件
中影響 junction 溫度的因素，從而提出改善 LED 散熱的方式。另外針對 LED 封裝後之外型進行散熱分
析時，可參考 McAdams[18]及 Coulson[19]等學者提出水平板之對流熱傳遞係數估算經驗關係式，可以
推算出自然對流的熱傳係數。其他還有部分學者[20-22]針對不同形狀以實驗結果定義出圓柱或球體之
自然對流熱對流係數。在 LED 封裝元件的放置角度變化方面，1975 年 Charchill[21]等發表關於垂直等
溫平板的流動情況，估算自然對流的熱傳遞係數。此外也有其他文獻對於平板在不同擺置角度時引發
自然對流的效應進行探討[18,19]。 
前述相關文獻大多數係以大尺寸外型( > 0.1 m)之物體作傳統熱對流之研究，其相關經驗公式對於
LED 封裝後等小尺寸元件( < 0.05 m)則可能無法適用。Ellison[23]於 1984 年對於小型電子元件之熱對流
係數提出合理估算方式。另外為增強自然對流，Yu[24]等學者提出在元件上進行開孔設計所造成之影
響，並以數值模擬搭配實驗來觀察電子元件在各種流場及輸入功率等條件下之散熱情形，得到相當重
要之結論。但由於傳統電子元件之操作功率皆不至於太高，而 CPU 等較高功率之電子元件則以風扇或
熱管來進行強制對流散熱。LED 因為要應用於照明及背光模組領域，近期對於高亮度 LED(HB LED)
的發展相當快速，同時其在單位體積內所產生的熱也大幅提升；但 HB LED 應用於室內照明或背光模
組時，尚無法接受搭配風扇等散熱方式，故 LED 封裝及燈具對於以自然對流的散熱方式來為 HB LED
散熱視為一大考驗及努力的方向。Yung 等[25]在 2010 年對在 PCB 板上陣列式封裝的 HB LED 進行數
值模擬與實驗的散熱分析，溫度量測係以熱像儀拍攝，探討 FR4 PCB 與 MCPCB 對 LED 陣列之散熱
影響。

之經驗公式，以利相關產業設計之應用。 
五、 結果與討論（含建議） 
本研究所設計的 n 電極中，最大的光輸出功率可被提高 11%，如圖六所示。LED 晶片的 I-V 曲線和
光功率表現都可觀察到較佳的電流分布之效應。由本研究之結果可以確認計畫中所提出之三維數值模型
對於 LED 晶片之設計有相當程度之準確性，將有助於 LED 業界在晶片之先期設計階段的應用。

六、結論 
本計畫之研究成果的一部分已發表於 Solid-State Electronics (SCI)，對於 ThinGaN LED 晶片內
部的電流分布情形受 n 電極的影響進行深入的探討。所建立之數值模型經與實驗比對後有相當之正
確性。另外對於封裝元件散熱的機制可說是有進一步之深入研究與發現，並與過去一般認知散熱方
式有所差異。藉由本研究開發之數值模擬模型可繼續對於 LED 封裝元件外部流場進行更細部之分
析，若搭配實驗驗證將可推導出適用於 LED 封裝元件之新散熱公式，相關研究成果將會持續整理
並發表。 
本計畫所使用之 LED 晶片已可精確模擬垂直式磊晶結構之電特性。由於 100 年度仍承蒙國科
會對計畫主持人所研提之 LED 相關研究計畫的支持，使前期研究成果將可得以繼續深入，持續進
行數值模型的精進，使 LED 晶片各種所需計算與呈現之物理特性皆可經由數值模型計算出正確結
果。如：光學特性、熱應力等，並可藉由電熱光等特性之耦合進行更深入的研究，且透過熱應力的
計算結果可建立 LED 之失效分析模型。
