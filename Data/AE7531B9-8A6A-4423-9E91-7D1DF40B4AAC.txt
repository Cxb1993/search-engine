Abstract 
In a traditional network, information is transmitted by 
store-and-forward approach. On the other hand, network coding 
techniques allow an intermediate node to encode data from its 
input links before sending the data to its output links. With 
network coding, the bandwidth efficiency of network is 
increased and the information transmission is more reliable. 
In 2006, Dong Nguyen, etc., proposed some schemes with 
network coding techniques for wireless broadcast applications. 
The schemes combine different lost packets in a fixed time 
interval from different receivers such that multiple receivers can 
recover their lost packets with one transmission from the source. 
By such a way, the retransmission times can be reduced and the 
bandwidth efficiency is increased. 
In this report, we use a more flexible way to choose the interval 
for collecting the lost packets, which we called floating interval 
scheme. By such a way, the lost packets are combined more 
efficiently and the throughput is more increased. In another 
scheme, we pre -allocate some buffers in receivers for coding 
packets usage. Each time when receivers get a correct packet, the 
coding packet is updated and the received packet is sent to the 
upper application if it is the expected next one. By such a way, 
the average length of receiver buffers is reduced and the memory 
usage is more efficient.
