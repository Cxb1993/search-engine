2Abstract
隨著行動網路及無線網路環境的發展，使用者可於許多地方在任何時候都可藉由有線
或無線上網的機制來學習。因著無線網路的日益普及、以可攜式設備學習例如：筆記型電
腦、PDA 及手機，可說是把握時間學習的最好選擇。由於圖形人機介面、超鏈結、超媒體
的技術快速發展，均使得移動式學習系統愈來愈能具有親和性及容易操作的功能。然而，
由於這些學習教材的內容必須在小小的筆記型電腦、PDA 或是手機之螢幕中展示，如何讓
使用者不會因為各種媒體的呈現而眼花撩亂，學習內容的設計就顯得更加重要了。此外，
多媒體技術也改變了資料庫的儲存結構與儲存內容，本計畫提出之以推理法則來管理移動
式學習環境，就是將學習者學習的動作、喜好進行分析並記錄在使用者的學習檔案中，當
學習者下一次進到此學習環境中，藉由學習系統之專業類型的分析與推理法則，可將適合
學習者近似程度的學習內容，顯示給學習者繼續進行學習。藉此可以提高使用者的學習效
能達到事半功倍，並能讓移動式學習環境，學習使用者的習慣及程度，以建立一套客制化
的移動式學習系統。
Keywords：貝氏網路 (Bayesian network), 行動網路 ( Mobile Network), 無線網路
( Wireless Network), 理解規則 (Reasoning Rule)
1. Introduction
近年來，由於個人電腦的普及以及網際網路的盛行，教學方式產生了巨大的變革，
從以往面對面教學的傳統授課方式逐漸轉變為數位學習的教學方式，因此，人們能不再
受到時間、地點的限制來進行學習。透過網路學習並不僅侷限於有線網路環境，無線網
路學習機制更可讓學生於任一地點、任何時間透過網路學習，是隨時學習的一個新的型
態。傳統在課堂上教學，透過教師與學生的互動，教師能夠依照學生的能力和興趣調整
教學內容，依學生的程度調整教材的難易度，但透過網路學習，變成學生與電腦間的互
動，雖然許多數位學習系統能提供豐富的教材內容，但是無法依照學生的興趣與能力提
供適當的教材內容給學生，亦即無法達到因材施教及能力適性的目的。本計畫利用貝氏
網路推論各個變數之間的因果關係，來分析不同背景學生的學習風格，找出每位學生他
們在學習過程中不同的偏好及習慣，並且分析學習者的學習模式及學習成效低落之原
因。當分析出學習者的學習模式之後，便可依照系統所判斷學習者的程度及喜好，提供
適合的教材，以期提高學習效率，並且達到客製化教學的目的。
為了完成此一目的，本計畫亦將同時進行以貝氏網路為基礎的互動式回饋查詢系統
及資料庫技術之應用及研發，以專業領域的推理演算法來彌補一般使用關鍵字查詢
(Keyword Querying)或內容式查詢(Content-based Querying)所無法達成的效果。利用先進
的網頁製作技術，XML 與資料庫技術的結合，完成更友善且便利的使用者介面，讓使
用者與網頁的互動增加以達到建立一個客制化學習網站的目的。並希望讓學習系統在無
形之中輔助使用者學習的狀態，利用貝氏網路所制定的內部規範提供適當的教學內容來
幫助使用者學習，藉此改變傳統的學習者自行資料搜尋學習方式，進而將知識領域推至
另一高峰。

4. Implementation
本計畫在實作部分使用 MSBNX(Microsoft Bayesian Networks) 和 Hugin Lite 7.1 來
建構貝氏網路模型，Web Server 採用 Microsoft IIS，資料庫使用 MS SQL Server，並
使用 ASP.NET搭配 HTML、JavaScript及 XML 製作互動式自學系統。
將使用者與在使用系統時與系統互動的數據收集起來，再透過行為模式推理子系統
中的動態決策樹計算出的機率值，推測出使用者可能會有興趣的課程，將該課程以較大
圖示顯示，以吸引使用者的目光，圖 4 為系統畫面範例。
