 
人才培育成果說明： 
 
1. 對於學術研究、國家發展及其他應用方面貢獻 
本計畫預期將研究成果發表於國際性會議及學術性論文期刊，同時部分內容將做
技術移轉及專利申請，研究成果將可提昇我國學術研究水準，以及相關產業技術提升，
並引發國內對相關產業的重視與發展，進而提高我國在國際間相關產業之地位，促進
國內與國外學者和研究機構進行更多的學術交流。藉此計畫，也讓我們對於光碟儲存
媒體的特性以及製造方法設計有更完整且深入之瞭解，同時對於國內相關資訊領域技
術整合之提昇亦將有所助益。 
其次，由於本計畫中，系統發展係由最基礎層面起步，進而開發完成一套具實用
性之系統，因此參與本計畫之人員將可藉由此計畫之執行獲得各項紮實的基礎訓練，
對於國家資訊相關技術人才培訓具有正面意義，並進而促進產業發展，以縮短產業人
才需求及學界人才培育的距離。 
2. 對於參與之工作人員，獲得之訓練成果 
本計畫主要係提出提出一個應用於光碟壓製之數位影音前處理及整合服務加值技
術研究，因此，本計畫除了研究新的數位影音前處理及整合服務加值研究方法外，同
時將完成一套圖形化視窗使用者介面之應用系統，以期能落實學術研究技術之發展與
實務相結合之目標。因此參與本計畫之人員將可獲得以下內容之訓練成果： 
(1) 熟悉數位影像之特性及重要之影像處理方法。 
(2) 熟悉數位影音資料格式與數位化資訊系統等技術。 
(3) 資料蒐集、分析、研究等能力之建立。 
(4) 培養瞭解數位影像對比度加強、影像分割、影像比對等理論與推導能力。 
(5) 培養數位影像理論技術發展與相關產業資訊結合之實務經驗。 
(6) 熟悉影像處理、系統程式之撰寫、系統軟體建置及整合等所需之相關技術。 
(7) 參予研究計畫之相關人員皆可對實作與分析有充分的了解，並能培養彼此
互助合作觀念與團隊研發能力。 
 
技術研發成果說明： 
 
本計畫主要提出一個應用於光碟壓製之數位影音前處理及整合服務加值技術研究，針
對光碟壓製生產過程中對於數位影音內容的前處理，以及整合加值服務技術上，提出有效
解決問題的方案及技術。首先針對 DVD 下一代光碟格式藍光光碟(Blu-ray Disc, BD)規格，
有效地建構出語法分析樹，也可用來協助使用者進行更進一步的分析，以有效達成 BD 字
幕語法建構之目的。 
 
2. 即時視頻預覽時間碼編修技術 
 
在 DVD 電影製作過程中裡，Time Code 被用來使影像和聲音較容易同步，或是在影像
被播放時能夠同步帶動音樂。而在後製作的過程中，Time Code 廣泛被應用在影像剪輯以及
使聲音、畫面、音效與音樂等不同元素能夠同步。而目前的做法大部分是透過 Betacam 機
器播放母帶，加上一併播放聲音效果部份，結合另一個具備 Time Code 能力的錄製錄影帶
的錄影機，進行錄影及同步工作。 
由於目前 DVD 光碟視頻部分主要以 MPEG-1 及 MPEG-2 為主，而新一代的藍光光碟
規格在視頻壓縮技術上則支援 VC-1、MPEG-2、MPEG-4 AVC 等幾種格式，而這些格式目
前在播放時均須透過編解碼器(codec)進行解譯後始能還原成影像畫面內容後進行播放，而
在建立時間碼時則僅能透過聲音部分，或是僅憑視頻部分配合鍵入時間碼資訊，缺乏一個
可以在同時播放影像畫面及音效場景真實情境下的時間碼鍵入及編修系統，若能有效提供
此一功能，將能大幅提升時間碼編輯時之效能及準確度。 
因此，本計畫針對前述特性提出一個結合視頻及音訊編解碼環境，將字幕結合播放時
之實際情境提供一個即時的時間碼編輯及修改系統，同時可以即時檢查鍵入或修改之時間
碼是否符合時間碼規則及順序、時間碼差異調整參數設定等，並提供即時預覽之播放效果
與字幕配合結果，期能獲得最佳同步效果之時間碼建立資訊。 
 
3. 不同畫面播放率時間碼轉換 
 
時間碼(Time Code)原本是由 NASA(美國太空總署)為了記錄太空任務的黑盒子而發明
的，然後被 ECCO 所採用，隨後接續發展出許多系統，而在 1969 年由 SMPTE 協會正式訂
下標準，同年 EBU 協會也採用此標準，故 Time Code 又常稱為 SMPTE/EBU。 
Time Code 是一個簡單的訊息，其中一部份資料是以小時：分：秒：格(Frame)來表示，
同時以 24 小時制表示，Frame 即是 Time Code 的最小單位，它切割一秒為許多小單位，從
00 開始，最大的數值可能是 23/24、24/25 或是 29/30...等，主要端看其格式而定，而在 SMPTE
時間碼有五種格式，分別是: 24 fps、29.97fps non drop frame、29.97fps drop frame、30 fps 
non-drop frame 及 30 fps drop frame。而 EBU 時間碼則只有一種格式-25 fps。這六種格式它
們都有特定的使用時機，24 fps 現在通常較少用，除非是在某些的動畫應用上。25 fps 是歐
洲通用的影像、動畫時間碼格式，30 fps 則是使用在大多數的映畫與音聲應用上，而現在
的電視、錄放影系統仍存在著許多不同的系統，像 NTSC、PAL、SECAM 等，而不同系統
在影像比對與檢索方面已有許多學者提出相關的研究，從最早的 QBIC(query by image 
content ) 影像檢索系統[4]，透過影像關鍵字及指定查詢影像 (query image)的顏色、影像像
素出現的機率以及形狀進行檢索，以及之後的 CBIR(content-based image retrieval)系統直接
萃取影像特徵，進一步進行特徵比對以找出相似影像。影像比對與檢索主要可以分為以區
域基礎的檢索方式，主要依據顏色、紋理、形狀[5]等特徵進行分類，再針對區域進行比對
檢索；另一種為全域影像比對檢索，即為整張影像進行比對檢索。影像比對的特徵，最早
出現的為接近人類感觀的顏色[6]-[9]、形狀[10]、紋理[11]-[13]，及考慮到顏色分佈情況發
展出的顏色空間的特徵[7]。 
由於海報內容均是由較小的圖案所結合而成，因此此一檢測步驟主要方法將偏重於影
像比對部份，特別是部分影像比對(partial image matching)的方式。同時由於影像內容具有
不同之特性，因此本計畫分成全域(global region)及部分(partial)兩種比對方式做介紹。影像
比對主要的觀念是利用顏色(color)、輪廓外型(shape)、顏色空間關係(spatial)及紋理(texture)
特徵作為比對的特徵，以達到外觀與本質均精確且可靠的比對結果。 
在全域比對方面，主要透過 K-mean 分群方法取得整張影像的顏色特徵，結合顏色空
間關係與紋理特性進行比對，紋理特徵部份則採用 co-occurrence 矩陣特徵，透過前述特徵
值的取得，計算兩張影像間的特徵值差異來達到全域影像比對的目的。而部分比對部份則
加上輪廓外型特徵如慣性軸(moment of inertia)、空間顏色關係特徵，並對整張影像做漸進
式掃描比對的方式進行。 
 
技術特點說明： 
 
隨著資訊科技技術的快速發展，各國均積極致力將多元化的社會、優質的文化、豐富
的生活形態、創新的環境等淺質特性與電子資訊技術充分結合，在數位內容（Digital Content）
產業與技術發展上提供具有高度附加產值的各項服務。數位內容主要係指將圖片、文字、
影像、語音等運用資訊科技加以數位化並整合運用之產品或服務。而其中伴隨著光碟儲存
媒體技術的發展，使得數位內容與相關產品的發展及技術昇級，對促進媒體展現/播放相關
技術之開發將具有關鍵地位，並成為當前重要課題。 
而由於數位電視逐漸普及的驅使下，已加速 DVD 下一代光碟格式藍光（Blu-ray Disc, 
BD）新世代來臨，而由於藍光播放機在美國及日本的熱銷，已更進一步將 BD 影片與硬體
的需求成長大幅提高，將有助於國內光碟產業的推升。就以往 DVD 碟片發展歷史軌跡觀
察，新世代產品崛起時，預錄的生產廠商將最先受到重視，也因為如此，發展出藍光光碟
相關影音技術及整合服務加值技術將是一項非常重要的課題。 
本計畫主要提出一個應用於光碟壓製之數位影音前處理及整合服務加值技術研究，針
式藍光（BD）新世代來臨，預錄的生產廠商將更受到重視。未來更可結合數位內容（Digital 
Content）產業與技術發展上提供具有高度附加產值的各項服務，伴隨著光碟儲存媒體技術
的發展，使得數位內容與相關產品的發展及技術昇級，結合大量網際網路媒體的運用，對
促進媒體展現/播放相關技術之開發將具有關鍵地位，在媒體儲存傳播產業與數位內容加值
應用服務的發展上，都將能獲致最大的成效，將此一領域之發展在國際上佔有重要的一席
之地。

技術說明
(中文) 本計畫主要提出一個應用於光碟壓製之數位影音前處理及整合服務加值技術研究，
針對光碟壓製生產過程中對於數位影音內容的前處理，以及整合加值服務技術上，
提出有效解決問題的方案及技術，以達到協助此一產業掌握技術優勢及擴大整合
上下游產業價值鏈的目的，為未來數位內容相關產業技術及數位寬頻資訊傳播時
代提供更為有效的技術與解決方案，提升整體產業的競爭力。 
本計畫主要分成四個部份，首先針對目前最新的 DVD 下一代光碟格式藍光光碟
(Blu-ray Disc, BD)規格，提供藍光光碟(BD)製作過程中，一個符合藍光光碟
(BD)製作規格的字幕語法產生技術。其次，由於藍光光碟在視頻壓縮技術上支援
VC-1、MPEG-2、MPEG-4 AVC 等幾種格式，而這些格式在編碼階段往往需要耗費
許多時間，因此本計畫提出一個可有效提升編碼效能的編碼參數選定方式。另外
擬由DVD 畫面內容中自動化進行字幕擷取，還原成文字，重新組成含time code 
之文字檔，以作為後續藍光光碟字幕製作時之字幕來源參考依據。在整合加值服
務方面，則提出一個海報印製自動比對校稿技術，利用影像比對的方式完成人工
校稿的工作，達到校稿準確度及品質控管的提升。透過前述計畫項目的發展，期
能對光碟壓製之數位影音前處理及整合服務加值技術提供有效的解決方案，達到
提昇產業競爭力的目的。
