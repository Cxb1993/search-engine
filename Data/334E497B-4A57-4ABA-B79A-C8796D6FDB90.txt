一、前言 
網際網路的盛行，傳統的消費習慣逐漸從實體行銷通路轉換成為網路購物型態。根
據資策會產業情報研究所統計，2011 年可達 3,256 億元，預估 2013 年達到 5,088 億元，
每年呈現 20%以上的高度成長[1]。電子商務平台業者(B2C)擁有媲美實體通路的銷售能
力，以國內最大的電子網購平台 Yahoo!奇摩購物為例，2009 年的營收已經突破百億，
超越任何實體通路百貨公司單店最好的業績，並以每年超過 30%以上的成長率，預估
2011 年後可高達三百億的營業額[2]。網路購物的優點，在於省略實體通路所需的店面、
人力等實體成本，改由電子商務行銷模式，提供更多元化以及便利的交易模式，因此創
造出更高的獲利。但是便利的優點也相對的帶來缺點，網路交易的糾紛因此層出不窮。 
根據警政署 165 反詐騙專線的統計，網路詐騙以每月 16%的成長率大幅增加[3]。
而網路交易所造成的糾紛不僅止於網路詐騙，根據消費者文教基金會的統計，商品瑕疵、
退貨遭拒、規格不符、帳號問題、詐欺等分居網路交易糾紛的前五名[4]。網路拍賣糾
紛造成的原因，可歸納成以下原因：(1)缺乏商譽的束縛。網路拍賣的參與者通常為消
費者個體以及小型企業體，規模通常較小，能夠輕易的轉移；(2)評價可信度的評估。
網路帳號創建容易，評價灌水的情形出現，導致評價可信度降低；(3)缺乏身分資訊。
網路拍賣中，除了網站上所公佈的資訊外，無法藉由其他資訊來評斷賣家是否值得信任。
雖然平台提供業者制定了許多相關的安全防護規範，但網路詐欺犯罪卻是層出不窮。就
消費者而言，為了避免網路交易糾紛，應對於網路賣家的信譽於購買前做深入的了解。
可以採取的手段如下：(1)選擇具規模信譽佳的購物網站。(3)檢視賣家過去拍賣評價。
(3)採用平台所提供的線上付費機制，避免私下交易。(4)採取面交或者貨到付款方式。
(5)要有良好的退貨機制。研究文獻當中提到，網路信任度的高低，會影響使用者對於
參加網路活動的意願，尤其是對於制度的信心[5]。在針對電子商務信任感的研究當中
指出，網站知名度與交易安全機制可能影響消費者網路購買意願，其中以網站知名度、
付款安全性、隱私權保護以及第三信任團體標章為最重要的影響因素[6]。綜合以上理
由得知，使用者對於網路購物對象的信任度，會影響網路購物的意願，因此網路口碑
(Online Word-of-Mouth)成為現今電子商務行銷上重要的課題。比起廠商砸下重金的廣告
行銷，消費者寧願相信其他消費者使用產品的心得，此種方式在網路上稱之為心得分享，
有時稱為開箱文，亦即所謂的口碑意見(WOM)。 
網路口碑行銷可以藉由多種方式來進行，例如電子郵件、網路論壇、電子佈告欄
BBS…等方式，其中又以個人部落格為最新興的口碑傳播模式[7]。部落格擁有多元化的
媒體呈現方法，例如文字、圖片、音樂、影片，因此對於口碑傳播效用與程度也相對的
來的強烈[9]。此外，部落格最重要的一個特性是，會經由好友關係串起線上社會網路
關係(Online Social Network)[10]。具有相同興趣的部落格，會經由線上社會網路關係的
傳遞，自然成為小世界群體(small world)[11]。舉例來講，台灣目前最大的部落格服務提
供者(Blog Service Provide - BSP)無名小站 Wretch(http://www.wretch.cc/)，針對不同的興
趣就擁有許多不同種類的社群。例如針對美食、消費、保養、旅遊…等相關於消費經驗
的社群，社群裡面的部落客會針對部落格文章所提及的內容做經驗分享以及討論，這也

三、文獻探討 
網路拍賣身份與部落格身份偵測需要針對部落客的身份進行辨識，在傳統的資訊整
合研究當中，將此類問題稱之為「名稱辨識問題(Name matching problem)」。Name 
matching problem 也可稱為 Record linkage、Name disambiguation、Merge-purge、Duplicate 
detection、Database hardening、Identity uncertainty 或 Coreference resolution[12, 13]。Name 
matching problem 可以應用在許多領域當中，例如論文 Bibliography 辨識作者身份，又
稱之為 authority control，目的是維護文章之間的參照關係與 bibliography 資料庫正確性
[14, 15, 16]。Bibliography 名稱對應問題可以分成三類[17]：第一類稱之為 Name sharing 
problem，代表一個名稱可以被多個作者所擁有，又稱之為 multiple-entity-one-name 
problem，大部分針對論文名稱問題對應的研究針對的是此類問題。第二類稱之為 Name 
variant problem，代表一個作者可以擁有多個人名。此類問題除了因為各個論文名稱縮
寫不同之外，隨著結婚更改名稱、譯名修改，都會造成此類問題，又稱之為
one-entity-multiple-name problem。第三類稱之為 Name mixing problem，此類問題結合上
面兩種問題，數個名稱會被多位作者所共有，又稱之為 multiple-name-multiple-entry 
problem。此類問題最為複雜，Shu et al 採用 LDA-dual (Latent Dirichlet Allocation)演算
法來解決此類問題[17]。 
Name matching problem 名稱對應考量的因素相當複雜，有的作者名稱格式不同、
有的縮寫不同、有的名稱缺乏、還有同名問題。其中最基本的技術是利用名稱字串對應
技術來比對，例如 Jaccard distance、TFIDF、Edit distance、Jaro Winkler distance…等[14]。
以下以 Edit distance 與 Jaro Winkler Distance 作為解說範例： 
 
 Edit distance 
Edit distance 是一種普遍應用在計算兩個字串間差異的動態規劃演算法[12]。最主
要的想法是將一個字串，經一次刪除一個字元、插入一個字元或取代一個字元等操作，
逐步演化為另一個字串，而每對一個字元作刪除、插入或取代時，必須產生一個代價成
本，應用各種演化的方法，最後累總代價成本最低者，即為該兩字串間的 Edit distance
值。Edit distance 的計算方法，是比較兩個字串必須要透過多少 Edit operations 可以把兩
個字串轉換成相同。通常Edit operations包含插入(Insert)、刪除(Delete)、取代(Substitute)、
拷貝(Copy)等運算。舉例來說假設字串 s=＂Willlaim＂，字串 t=＂William＂，那麼要把
s 字串轉換成 t 字串需要下列八個步驟： 
1. Willlaim  W (Copy ｀W＇) 
2. Willlaim  Wi (Copy ｀i＇) 
3. Willlaim  Wil (Copy ｀l＇) 
4. Willlaim  Will (Copy ｀l＇)

四、研究方法 
網路上的部落客與網路賣家、買家，到底哪些帳號屬於同一個人，必須進行網路身
份辨識，才能達到利用部落格社會網路指標來加強拍賣評價的參考性。本研究以 Yahoo
拍賣(http://tw.bid.yahoo.com/)與無名小站(http://www.wretch.cc/)做為實驗的對象。Yahoo
拍賣為國內營運最好的拍賣網站，無名網站為國內最大的部落格服務提供者(BSP-Blog 
Service Provider)，透過這兩大拍賣網站與部落格服務提供者，預期能找出相同使用者之
拍賣帳號及部落格帳號。

排名前五名的部落格，透過自行開發的 ILT_Crawler，根據其好友關係擴展抓取帳號的
範圍。網路拍賣網站帳號的抓取，是從 Yahoo 拍賣中的熱賣排行 23 個分類中，每日熱
賣排行前 20 名的網拍客(資料擷取從 2011 年 02 月 20 日至 2011 年 03 月 07 日)，使用自
行開發 ILT_Crawler 擷取帳號資料。在去除重複抓取的對象後，共取得無名部落格帳號
1,345,713 個、YAHOO 拍賣帳號 1,732 個。由於本研究是以網路拍賣評價作為比較標的，
因此網路拍賣帳號並不以全面抓取為目的。

六、結論與未來方向 
為了解決拍賣評價造假與濫用的情形，本研究利用部落格網路口碑來提升網路拍賣
評價之參考性。為了解決部落客與網路拍賣使用者之身份比對問題，本研究以 Yahoo
拍賣與無名部落格為例，採用 Edit Distance 與 Jaro Winkler Distance 兩種帳號比對演算
法，比對部落格與網路拍賣中相似的帳號，用來作為身分辨識之用。接下來透過分析部
落格好友關係形成之社會網路，擷取各種重要的社會網路指標並加入至網路拍賣評價當
中，進而提升網路拍賣評價之參考性。 
在考量比對可行性的情況下，本研究採用 Edit distance=1 與 Jaro Winker 
distance=0.95 作為帳號比對的門檻，找出相似帳號並進行比對，找出為同一個人的無名
部落格與 Yahoo 拍賣帳號，透過社會網路指標的計算，在拍賣網站使用者(買家與賣家)
加入相關的社會網路指標，提供給想要購買網路拍賣商品的使用者參考。 
在相似拍賣帳號與部落格帳號比對的過程當中，目前的技術仍需要輔助拍賣網站與
部落格提供的除了帳號以外的資訊，以及透過人力的介入，才能夠確認相似帳號是否為
同一人。因此除了較成熟的帳號比對技術之外，未來殷切需要建立跨異質性平台的身分
比對機制。此外，本研究解決了 Name sharing problem 類型的問題，對於較複雜的 Name 
variant problem 以及 Name mixing problem，仍需要花費更多的精力從事研究。
