中文摘要 
本研究主要為一RFID生技光碟檢測嵌入
式韌體控制系統之開發內容，其整體架構包括
ARM 平台 Linux-OS 之檢體影像處理系統、
影像擷取光機電控制系統、 ARM 平台 
WinCE-OS 碟片 RFID 辨別系統、生技碟片轉
速控制系統、及 PIC18F4431 機電致動微控制
器，在主文中詳細說明本檢測系統開發流程相
關測試與驗證結果。 
本報告針對RFID生技碟片檢測系統進行
系列實驗與結果討論，包括碟片 RFID Tag 識
別實驗、碟片轉速控制實驗、碟片對焦控制實
驗、變焦控制實驗、機構位移控制實驗、壓電
致動實驗，以及本研究自行開發嵌入式韌體系
統驗證，確認該韌體雛形架構可與微控制器建
立有效溝通機制，並有效對該生技碟片致動器
週邊電路進行驅動與控制動作。 
關鍵詞： 嵌入式系統、無線射頻辨識、生技
碟片。 
Abstract 
The object of this research is to develop an 
RFID mechanism to identify disks in an 
embedded BioCD inspection system.  The 
entire prototype consists of an Linux-OS based 
ARM platform for sample image processing, a 
WinCE-OS based ARM platform for BioCD tag 
identification, a BioCD rotational speed control 
scheme, and a PIC18F4431 microcontroller for 
driving the BioCD inspection mechanism. 
Keywords: Embedded system, RFID, BioCD.

一、前言 
二十一世紀今日人類文明針對生醫檢測
防治相關科技，均紛湧投入資源積極進行重點
研發，而生技碟片(BioCD)相關檢測技術，及
其微流道和檢測平台的設計與開發，成為未來
生醫科技研究重要主軸。透過生醫晶片系統的
微小化，可濃縮複數種生醫檢測到一微小晶片
內進行，藉以有效減少試劑的用量、縮短分析
時間、發揮易於攜帶和大量製造等之優點。 
二、研究目的 
近年來醫療保健網絡利用無線通訊技術
與無線射頻識別(RFID)技術，整合家用生技
檢測裝置將其內設生理訊號感測器，並積極透
過應用介面開發無線通訊模組，讓使用者在居
家環境中即可進行活動式生理監控，亦可透過
網際網路傳輸至遠端醫療單位做即時的病情
診斷，或可配置無線射頻識別技術晶片模組，
應用於檢測樣品的追蹤、緊急救護追蹤、與受
看護者之辨識與定位。因此，本研究鎖定自行
開發一整合生技碟片設備與無線射頻識別
(RFID)技術，是為一具有前瞻應用性之方向。

2. 生技碟片轉速控制實驗 
本實驗採用 PWM 開迴路驅動機制，控制
生技碟片主軸馬達轉速。開發過程要求轉速控
制準確，調變 Duty Cycle 比例並觀察轉速變
化如圖(十)所示，卻任期其線性度約趨近 1。 
其後，本研究整合 RFID Tag 識別與碟片
轉速控制機制並進行實驗。首先在 A、B、及
C 三個不同碟片上設置 Tag A、Tag B、及 Tag 
C，擷取資訊後以 ARM 平台內嵌應用程式與
所建立資料庫進行比對，並將比對後的資訊透
過 RS232 傳遞給 PIC18F4431 微控制器，再
送出 Duty Cycle 調變之 PWM 訊號進行生技
碟片轉速控制。該實驗設定 A 碟片與 B 碟片
上之 Tag 為有效標籤，且設定 A 碟片辨識確
認後應進行三段轉速流程分別為 500rpm、
3200rpm、及 2100rpm，B 碟片辨識確認後應
進行三段轉速分別為 1550rpm、750rpm、及
2350rpm，而 C 碟片上之 Tag 為無效標籤。實
測結果 A 碟片經 RFID 辨識後之轉速控制如
圖(十一)所示，B 碟片經 RFID 辨識後之轉速
控制如圖(十二)所示，而 C 碟片則因為其 Tag
設定為無效標籤，所以經辨識號微控制器並未
送出任何 PWM 訊號，主軸馬達因此也不進行
任何轉動操作。 
3. 鏡組機構縱向對焦控制實驗 
本實驗主要為驅動光學鏡組步進馬達，搜
尋機台上生技碟片之可能對焦平面，大致到達
該平面後鏡組往上調整400μm高度開始擷取
影像，如圖(十三)所示為每 100μm 擷取一幅
影像，共為步進馬達由上至下前進 800μm 連
續九幅影像。圖(十四)所示為將擷取的九幅影
像，經 ARM 平台 Linux 開發程式進行清晰度
函數計算後，所獲得每幅影像清晰度值之變化
曲線。 
經由所擷取連續影像與嵌入程式處理後
相對應清晰度值可知，檢體影像由模糊逐漸轉
為清晰的過程中，清晰度值亦隨之變大，其中
在此例可估計在第五張影像清晰度值最大，與
前後一幅的清晰度值皆有明顯差距，而當馬達
繼續往下行進時影像又逐漸轉為模糊，其清晰
度值亦隨之變小，因此可確認鏡組在第五張影
像擷取位置應可獲得最佳對焦影像。 
4. 鏡組機構變焦控制實驗 
本實驗機台鏡組放大倍率設計為 0.35×
1.5×20，其中放大倍率 0.35，為單眼變焦顯
微鏡與 CMOS 影像擷取設備結合的目鏡焦
距，單眼變焦顯微鏡所設定變焦位移為 1.5，
而本研究變焦倍率經鏡組機構位移可調變範
圍為 0.7~4.5，同時物鏡焦距設定為數值 20。 
該實驗主要調整單眼變焦顯微鏡，改變倍
率數值分別為 1.5、2、2.5、及 3，觀察其鏡
組機構變焦效果如圖(十五)所示。 
5. 鏡組機構橫向位移控制實驗 
本研究生技碟片檢測經組機構設計方
式，採用步進馬達配合齒輪與時規皮帶，驅動
控制步進馬達產生旋轉扭矩，傳動至鏡組位移
機構 X 軸方向的測微尺。位移過程主要透過
FS2410XP 之 ARM 平台送出位移控制訊號，
並配合執行 PIC18F4431 控制器韌體步進馬達
控制副程式激發步進馬達驅動器，橫向位移鏡
組機構擷取生技碟片所需部位影像，因此藉由
該位移控制機制達到觀察碟片上各部分檢體
影像之功能。 
該橫向位移實驗透過驅動步進馬達帶動
鏡組機構，如圖(十六)所示為每位移 0.5mm
擷取一張碟片影像，其機構由後至前移動總行
程 2.5mm 共擷取六張連續影像，因而獲得 X
軸線性位移變化之擷取影像。

z 計畫成果自評 
本文敘述研究過程開發一RFID生技光碟
檢測系統過程與實驗結果討論，歸納其研究成
果包括下列數項： 
(1) 有效開發 ARM 平台 WinCE-OS 系統
應用韌體實現生技碟片辨識功能。 
(2) 有效開發 ARM 平台 Linux-OS 系統應
用韌體實現生技碟片影像擷取檢測功
能。 
(3)  有效透過 RS232 通訊機制驅動鏡組機
構位移週邊電路。 
(4)  有效透過 PWM 訊號控制伺碟片主軸
達轉速。 
(5)  有效控制鏡組機構之複數個步進馬
達，進行系統對焦、變焦、與位移動作。 
(6)  有效控制壓電致動器，可獲得碟片上檢
體各深度最清晰影像。
