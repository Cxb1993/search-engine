與 H2O2 分解率，此乃因 OH‧較易有效率地礦化低分子量 DOC
所致。因此，鐵鹽混凝前處理去除高分子量 DOC 後，可提高
後續 H2O2/UV 程序去除 DOC 效果。相反地，後續 Fenton 程序
去除 DOC 則無顯著提高。綜合上述，鐵鹽混凝結合高級氧化
程序處理人工染整廢水之最適程序，建議(1)單獨 Fenton 程
序可應用於脫色，但不適用於去除 DOC，(2)脫色與去除 DOC
之最適程序為鐵鹽混凝結合 H2O2/UV 程序。

Photo-Fenton process, the major mechanism to remove DOC is ferric coagulation at the reaction time

predominated by DOC with low MW fractions (less than 10 KDa) more than 85%. Ferric 
coagulation can remove DOC with high MW fractions but low MW fractions. Pretreatment of 
ferric coagulation can promote the followed H2O2/UV process to remove DOC and degrade H2O2.  
This is due to that OH‧ can effectively oxidize residual DOC with low MW fractions. The 
pretreatment of ferric coagulation can enhance the followed H2O2/UV process to remove DOC but 
the followed Fenton process. It is concluded that (1) the Fenton process only is used for removing 
color but DOC, and (2) the optimal process combination for removing both color and DOC is 
ferric coagulation coupled with H2O2/UV process.

三、結果與討論 
3.1 鐵鹽混凝程序廢水脫色與去除 DOC 之機制 
1 鐵鹽混凝與 Fenton 程序脫色、去除 DOC 之比較 
鐵鹽（三價鐵）混凝與 Fenton 程序最適 pH 控制於 4.0 與 3.0-4.0(Kang et al., 2002; Hsueh et 
al., 2005)，人工染整廢水色度 1,200 單位、DOC 31 mg/L，鐵鹽混凝與 Fenton 程序脫色、去
除 DOC 之比較如表 1 所示。於鐵鹽低加藥量 5 mg/L 時，鐵鹽混凝與 Fenton 程序兩者皆對
DOC 去除率不佳（2.8%-12.7%）；相對地，Fenton 程序色度去除率高達 96%，單位鐵鹽色度
去除量為 1,053 unit/mg-Fe2+，為鐵鹽混凝 174 unit/mg-Fe3+鐵約 6 倍，顯示 OH‧氧化可有效
脫色，減少混凝劑加藥量與污泥發生量。同表 1 所示於鐵鹽高加藥量 50 mg/L 時，兩者程序
色度去除率皆達 95%以上，且單位鐵鹽 DOC 去除量分別為 20.5 mg-DOC/mg-Fe3+（鐵鹽混
凝）與 19.22 mg-DOC/mg-Fe2+（Fenton），無顯著差異。Fenton 程序兼具鐵鹽混凝與 OH‧氧
化雙重處理功能，OH‧氧化無法有效礦化 DOC，DOC 去除主要藉鐵鹽混凝作為用，故與三
價鐵混凝相同需高鐵鹽加藥量才能去除 DOC（同時去除色度）。 
2 Fenton 程序脫色、去除 DOC 之機制 
人工染整廢水色度與 DOC 濃度分別為 1200 單位與 30 mg/L，Fenton 程序於 pH 3 與 H2O2 
50 mg/L 時，亞鐵加藥量對色度殘留率之影響如圖 2 所示，Fe2+加藥量 5-50 mg/L 範圍之色度
殘留率皆低於 4%，顯示於低 Fe2+加藥量 5 mg/L，因 OH‧氧化破壞染料發色團，達到脫色
效果。同圖所示於 Fe2+加藥量 5 mg/L 之溶出後色度殘留率為 7.6%；Fe2+加藥量 20-50 mg/L
之溶出後色度殘留率範圍為 54.8%-66.7%。Fenton 程序於低 Fe2+加藥量即可有效脫色，且由
溶出後色度殘留率為 7.6%，僅增加 4%，得知脫色主要機制為 OH‧氧化，混凝作用不重要。
此外，隨 Fe2+加藥量增加溶出後色度殘留率亦增加，其主要原因為鐵鹽染料錯合物所造。 
亞鐵加藥量對 DOC 殘留率之影響如圖 3 所示，隨 Fe2+加藥量由 5 mg/L 增至 50 mg/L 時，DOC
殘留率由 87.3%降至 34.9%；但由溶出後 DOC 殘留率增至 73.7%-94.4%，以 Fe2+加藥量 20

2 鐵鹽混凝結合 H2O2/UV 程序礦化之比較 
H2O2/UV 程序於 H2O2 加藥量 400mg/L、照光強度 128W 及氧化時間 60 分鐘，殘留 DOC
為 21.1mg/L，氧化後殘留 DOC 分子量分布如表 2 所示，高分子量 DOC 佔 8.1%，中分子量
DOC 佔 3.2%，低分子量 DOC 佔 88.7%。高、中分子量 DOC 經 OH．氧化大多轉化為低分
子量 DOC，高、中分子量 DOC 佔之現象與 Fenton 程序相同。配製與鐵鹽前混凝處理後相同
DOC 但分子量分布相異之水樣，H2O2/UV 程序實驗條件為 H2O2 200 mg/L、UV 128 W 與氧
化時間 150 分鐘，混凝前處理有無對 H2O2/UV 程序礦化 DOC 之影響如圖 6 所示。以氧化時
間 60 分鐘為例，無鐵鹽混凝前處理之 DOC 及 H2O2 殘留率分別為 48%及 28%；相對地，有
鐵鹽混凝前處理之 DOC 及 H2O2 殘留率分別為 5%及 0%。顯示經鐵鹽混凝前處理再 H2O2/UV
程序氧化可顯著提高 DOC 礦化去除率與 H2O2 分解率，此乃因 OH‧較易有效率地礦化低分
子量 DOC 所致。H2O2/UV 程序乃主要依賴 OH●礦化去除 DOC，其處理設備與成本較傳統化

3.4 鐵鹽混凝結合高級氧化程序廢水脫色、礦化最適組合 
  鐵鹽混凝與高級氧化程序組合方式可為(1) 鐵鹽混凝結合 Fenton 程序、(2) 鐵鹽混凝結
合 H2O2/UV、及 (3) Photo-Fenton 程序，依脫色與去除 DOC 分述如下。 
1 鐵鹽混凝結合高級氧化程序脫色適組合 
  由表 1 與圖 2 所示鐵鹽混凝與 Fenton 程序脫色之結果，單位鐵鹽脫色量於鐵鹽混凝為 174 
unit/mg-Fe3+，於 Fenton 程序為 1,053 unit/mg-Fe2+，Fenton 程序於低鐵鹽加藥量衍生爭 OH‧
即可有效脫色，且由溶出實驗得知脫色主要為氧化。相對地，鐵鹽混凝須於高鐵鹽加藥量才
能有效脫色，亦同時去除 DOC，主要藉混凝作用，即混凝原理之電荷中和、吸附與架橋作
用，色度與 DOC 去除乃由液相吸附至鐵鹽膠羽。由圖 5 所示 H2O2/UV 程序亦可有效脫色，
因此，脫色處理可採用單獨鐵鹽混凝或單獨 Fenton 程序或單獨 H2O2/UV 程序即可，不必程

鐵鹽混凝普遍應用於染整廢水處理，許多研究亦探討 Fenton 與 H2O2/UV 程序處理染整廢
水處理或脫色，本研究探討鐵鹽混凝、Fenton、H2O2/UV 等程序組合對人工染整廢水脫色
與去除 DOC 之機制。研究結果首先闡明三價鐵混凝須於高鐵鹽加藥量才能同時去除色度與
DOC，Fenton 程序於低鐵鹽加藥量即可有效地氧化脫色達 95%以上，但 DOC 去除率低於 15%，
Fenton 程序單位鐵鹽去除色度量約為鐵鹽混凝 6 倍，且脫色以氧化機制為主。Fenton 程
序須於高鐵鹽加藥量才能去除 DOC 與脫色，DOC 去除機制主要為混凝而非氫氧自由基（OH‧）
氧化，且單位鐵鹽去除 DOC 與鐵鹽混凝相近。增加 H2O2 加藥量無法增加 DOC 去除率。此
成果指出鐵鹽混凝與 Fenton 程序應用分別為去除 DOC 與脫色，尤其 Fenton 程序不適應用
於去除 DOC。Photo-Fenton 程序於反應初期，主要以混凝機制去除 DOC，鐵鹽混凝結合
H2O2/UV 程序去除 DOC 優於 Photo-Fenton 程序。鐵鹽混凝前處理可先去除高分子量 DOC，
可顯著提高後續 H2O2/UV 程序礦化 DOC，但後續 Fenton 程序則否。故於應用面建議：(1)
單獨 Fenton 程序可應用於脫色，但不適用於去除 DOC，(2)脫色與去除 DOC 之最適程序為
鐵鹽混凝結合 H2O2/UV 程序。 
此外，於學術上本研究藉溶出實驗，說明 Fenton 程序與 Photo-Fenton 程序於反應初期，
主要以混凝機制去除 DOC，OH‧無法礦化去除 DOC。相對地，H2O2/UV 程序隨氧化時間持續
衍生 OH‧（存在 H2O2 ），可增加 DOC 去除。Fenton 與 H2O2/UV 程序雖皆衍生 OH‧礦化 DOC，
但因衍生 OH‧方式不同，故 DOC 去除效率相異。
