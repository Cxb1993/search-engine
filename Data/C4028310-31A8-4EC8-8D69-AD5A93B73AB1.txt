摘要 
在電纜發生故障後，故障點的偵測須快速及準確，以縮短搶修復
電時間。在電纜故障前，已經開始的劣化過程常藉由局部放電檢出電
纜的絕緣劣化，尤其是當距離長、電力系統架構複雜或工作環境較為
特殊，譬如佈置在海底的電纜，則有必要快速檢出局放點位置，以採
取適當的處理措施。  
 本研究針對局部放電點與故障點進行測距研究。首先，利用
EMTP/ATP Draw電磁暫態程式，建立25kV電纜之局放與故障分析模
型。所稱局放分析模型係模擬電纜絕緣層發生局部放電，觀察局放脈
衝在電纜中的折反射特性；所稱故障分析模型係模擬注入之高頻脈衝
在短路、開路的電纜中之傳導特性。在瞭解其特性後，二者皆運用時
域反射法找尋發生局放之電纜的局放點或找尋發生短路或開路之電
纜的故障點位置。 
 模擬結果顯示，時域反射法不但適用於電纜局放點定位，亦適
用於電纜的故障點定位。 
 
 
 
 
 
 
 
關鍵詞：時域反射法、電纜、局放定位、故障定位。

第一章 緒論 
1.1 研究背景 
 電纜發生短路事故時，大部分為絕緣劣化所導致。在絕緣劣化初
期，通常採用局放檢測方法以檢出絕緣劣化，若能正確找出劣化點位
置，特別是對於長距離、電力系統架構複雜或電纜工作環境較為特
殊，譬如位於海底的電纜，可以縮短絕緣劣化處理所需要的時間。 
 對於電纜之接地或開路故障，局放點定位通常須要分析局放脈衝
在電纜中的折反射特性，而電纜故障點定位亦通常注入高頻脈衝的方
式定位，由此二者皆在分析高頻脈衝的折反射特性，因此本研究探討
採用同一個分析脈衝的折反射方法作局放點定位與同時作故障定位
之可行性。如此，同一個定位方法可以兼具電纜局放點定位及故障點
定位之雙重功能。 
 
1.2 文獻回顧 
電纜局部放電檢測與定位的研究文獻頗多[1,2,8,9,26-28]，在局放
定位上，普遍採用時域反射法進行研究。其中，Villefrance等針對中
壓電纜，量測電纜終端的局放脈衝，根據脈衝振幅進行局放定位[2]。
Cavallini等運用頻域圖當中高、低頻振幅分布的特性，考量高頻衰減
與波傳導時間偵測局放點位置，稱為振幅-頻率圖(amplitude-frequency 
map)，量測132kV電纜局放脈衝，評估該方法局放定位準確度，並與
時域反射法相比較[9]。電纜故障定位採用時域反射法的研究文獻較少
[4,5,18]，其中，Navaneethan等運用時域反射法針對低壓地下電纜進

 此技術的原理係因當局部放電脈衝電流發生時，產生暫態電壓的變
化，而一部分的暫態電壓會穿過絕緣物散發於周圍空氣之中，此時可利用
電容性感測器與空氣中的雜散電容相耦合藉此偵測暫態電壓。其作法為取
一段靠近接頭部分的電纜，剝除其部份外部絕緣層來裝置感測器，以進行
偵測。但也因為剝除部分外層，限制了短路電流的承受能力。   
(2)超高頻法 
  現場量測時，超高頻下離感測器較遠的地方干擾衰減較快，所以超
高頻法適用於檢測電纜及其接頭附件。因超高頻下訊號衰減比低頻嚴重，
所以於監測時頇安裝多個，並裝置於靠近電纜接頭或端部附近。

3.3 故障定位方法 
 (1)電橋法[24] 
    測量方法藉著兩個內部電阻臂與兩個外部電阻臂之平衡。原理為
利用雙臂電橋測量出電纜導體上的直流電阻值，再量測電纜的實際長
度，最後使用電纜長度與電阻值正比的關係求出故障點。 
(2)脈波電流法 
 量測的基本原理係將地下電纜故障點採用脈衝高壓或直流高壓
訊號將其故障點擊穿，如：高阻抗接地故障時，此高壓會在發生故障
點導體與遮蔽層間產生電弧進而形成短路，波行進至故障點時會發生
反射現象，儀器收集並紀錄故障點於擊穿瞬間產生的脈波電流暫態訊
號，利用得到的反射訊號其行進時間與波在地下電纜內的傳播速度，
由公式推算出發生故障的位置。 
(3)時域反射法 
此法於測試時在電纜線注入一個脈衝，而這脈衝會沿著電纜線
行進，當遇到電纜線的阻抗不匹配點（短路點或接地點等）時，會有
反射波的產生並返回到達測試端，將可在測試端觀察到反射訊號的延
遲時間和振幅大小，藉此算出電纜線的不連續點位置和特性阻抗。特
性阻抗不匹配所引起的反射，依據傳輸線理論得知，若阻抗不匹配
(impedance mismatch)時，將有反射波發生，而反射波與線路上的阻抗
有關聯。而實際上當注入一脈波訊號至電纜線後所得到反射回來的脈
波，實際的延遲時間為量測延遲時間的一半，所以在有阻抗不匹配的
情形下，已知該電纜線上的傳播速度是 pv ，反射波與輸入波的時間差

4.2 電磁暫態程式基本原理[20,25] 
電磁暫態程式由H.W.Dommel博士在1960年發展出，起初用於電
力傳輸線的暫態模擬，目前廣泛的使用在開關切換和雷擊突波分析、
諧波與電力品質研究、保護電驛模擬。 
由電阻、電感或電容組成的 型電路或任意多相網路都可利用
EMTP求解，為了方便說明此處只考慮正序網路元件，電阻的電壓、
電流屬線性關係（指其離散波形與連續波形一致），所以不去考慮以
離散函數近似連續函數所造成的誤差；在電感、電容的電壓與電流屬
非線性關係，EMTP在處理求解這類型的數值分析，採用梯形積分法
(trapezoidal integration method)。梯形積分法主要功能用於求解定積分
運算法。圖4.1為一非線性圖形，通常用積分方式來求解t-∆t到t之間的

6.2 未來研究方向 
本研究尚有未完成之處，茲提出以下的未來研究方向： 
(1) 本研究在建立模型過程中，運用EMTP/ATP Draw軟體在輸入內外
半導電層參數遭遇困難，而未模擬該層，如何運用EMTP/ATP Draw
軟體模擬此層有待進步探討。 
(2) 在局放定位時，可以將運轉電壓列入模擬，進而建立較為精確的
模型，降低測距誤差。並將環境雜訊和電纜接頭列入考量進行定
位，使其與實務相符。 
(3) 本研究僅進行單相電纜故障點與局放點定位，尚未模擬三相電纜
故障點與局放點定位，建議可列為未來研究項目。 
(4) 未來可進行實測與評估時域反射法的定位誤差。

二、與會心得 
1. IEC61850 應用於配電自動化與饋線內分散式電源的控制 
目前 DNP3.0 仍屬集中式控制方式，未來採用 IEC61850 可應用該標準
內的 peer-to-peer Communication，亦即運用 GOOSE(Generic Object 
Oriented Substation Event)，達到分散式控制的功能。換言之，各個饋線
開關(LBS 或 DS)以及分散式電源(DG)的控制器內皆須符合 61850 的設
計，如此，DG 控制與 FDIR(Fault Detection Isolation Restoration)等饋線
自動化可以不但符合 Interoperability，亦可以符合 peer-to-peer 的要求，
後者可以符合 Scalability 的基礎。  
2. Plug-in Hybrid Electric Vehicle(PHEV)之影響評估 
分兩項評估：(1)對於饋線的影響；(2)對於環境的影響，前者是 EPRI
的報告，運用 EPRI 的 open-source Distribution System Simulator (Open 
DSS)分機平台執行，假設饋線架構、各個負載點的歷史負載、饋線參
數、PHEV 的充放電模式、各項負載中的市佔率、用車人充電習性、電
池狀態(在充電前的行車公里數)，其中，充放電模式係參考 SAE J1772，
