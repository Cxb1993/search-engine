三、 文獻探討 
在印刷業者眼中，產品的競爭力來自良好且穩定的印刷品質，為了讓印刷業者有客觀且準確地衡
量印刷品質的依據，印刷品質特性規格的建立就顯得非常重要。本章節將針對 UV 寬幅噴墨印刷和常
見之印刷品質特性等二部分做說明。 
（一）UV 寬幅噴墨印刷(UV Wide-format Inkjet) 
最常見使用於數位印刷中的無壓技術(Non - Impact Technology)就是噴墨印刷。這種電腦直接成像
(Computer to Print)技術是透過噴頭將墨水噴印於被印物表面，這表示噴墨印刷的印製過程並不需要透
過中間的影像載體。 
寬幅印刷產品的系統有很多，如海報與大幅面看板常採用寬幅噴墨印刷機。135cm 的寬度十分普
遍，也有印刷寬度為 5~8m 的系統。採用適合的油墨可以在各種材料、織物和塑膠膜上印刷。系統工
作時，噴墨頭掃描方向與紙張方向垂直。在大幅面寬度上必頇採用寬幅噴墨頭。用於噴墨頭的每個顏
色，可按行排列在印刷方向上、相鄰排列或兩個方向形成噴射矩陣(Kipphan, 2001, p. 726)。 
UV 印刷是非常具有發展潛力的印刷方式，最主要的原因除了環保因素之外，它適印的材料非常廣
泛。而這些適印的材料中有許多是傳統印刷所無法克服的(胡順華, 2004, pp. 1-2)。UV 乾燥的油墨能夠
廣泛地運用在各種不同的被印物料上，而且在玻璃、木頭、陶瓷與金屬這類堅硬的材料上有高品質的
印刷表現。這種油墨能使生產更加快速，因為它使用紫外光進行聚合而得以瞬間乾燥。但 UV 系統也
比其他的寬幅噴墨系統來得昂貴，而且無法完全負載在某些柔軟的材質上 (Robertson, 2008)。 
（二）常見之印刷品質特性 
傳統印刷影像是由一連串不同大小的網點所構成，在印刷製程中，必頇控制許多變異以確保完美
的印刷品呈現，這些重要的印刷品質特性包括濃度、明度、印刷對比、色相和色彩飽和度、印墨總量、
階調擴增、疊印能力和紙張表面等。業者必頇確認在如此多的變異中，如何透過適度地調整印墨、紙
張和印版才能有效的控制印刷製程，這些調整需將製程中的變異降到最低使得印刷機可印製出令客戶
滿意且與打樣稿吻合的印刷品(Southworth, M. & Southworth, D., 1989, Ch. 5-21）。

(1)跨產業發展 
UV 寬幅噴墨印刷機已經跨越傳統文化用紙的印刷範圍，多元的被印材應用，如紡織、木材、塑膠、皮革、
玻璃等，產業自行作加工處理或外包廠商作代工處理。使其跨足廣告、建材、家具、紡織、3C 產品等多項產
業，是屬於工業印刷範疇。因此 UV 寬幅噴墨印刷機以涵蓋在多樣產業的製造流程中，而非自行獨立的印刷產
業。在討論 UV 寬幅喷墨時，已不能用印刷產業的一語蓋之。因 UV 寬幅噴墨機的多元應用，使它像一個變型
金剛，靈活運用在不同的產業之間，且在不同的市場下形成新的商業模式。 
(2)主要產業成長趨勢 
目前台灣 UV 寬幅噴墨印刷機主要應用於三大產業：1.廣告業。2.建材業。3.3C 產品。在廣告業上的應用
成長幅度較小；除了市場較小外，與其他版式競爭較激烈，例如平面廣告上 UV 噴墨沒有絕對優勢，除非用在
特殊別印材上。而在建材業，UV 寬幅噴墨印刷具有彩色階調、高解析度等優勢逐漸取代網版，而快速成長，
但目前有漸緩的趨勢。而台灣為 3C 王國，配合近年來 3C 產品的發展，將 UV 寬幅噴墨印刷應用在 3C 產品上
是目前大家所看中的市場目標。  
(3)環保議題或口號 
UV 寬幅噴墨印刷具有 1.環保 2.快速 3.多樣的被印材等的優點，其中更以環保的優勢打敗水性油墨及溶劑
型油墨。不含揮發性的有機物 VOCs 是主要特色，在全球推展環保的口號下，廠商也被迫注重環保印刷的發展。
但在強調 UV 環保的議題下，真正實現 UV 為環保印刷卻是有待商榷的。UV 喷墨印刷雖無揮發性的有機物
VOCs，但過程印刷中所產生的其它惡氣、臭氣仍有害人體與環境。在生產製造業中，沒有絕對的零污染，如
果大家只是追求環保的風潮，配合消費者喜好的迷思，而不了解 UV 噴墨的市場狀況就輕易投資，很容易陷入
投資失敗、找不到客源的窘態。 
(4)潮流不等於錢潮 
UV 寬幅噴墨印刷在近年已成為風潮，越來越多投資者投資在此項設備中，但是否代表 UV 噴墨產業具有
豐厚盈利可圖?答案是：錯的。曾與一位在 UV 產業超過 10 年的廠商談到此問題，他回答說：「雖然 UV 噴墨
表面上具有環保且可多樣應用的優點，但實際上單靠一台 UV 噴墨機賺錢的並沒有。近年來越來越多人投資購
買機器，大家都看中這一塊的前景，但實際上的投資報酬率都不高。只能說不怕死的越來越多了。」目前大家
都在試探市場的水溫，也就是一個正在 try 的階段，追隨潮流前因先找到自己的目標市場，有市場後才投資，
才是明確的抉擇。

(1)廠商型態-輸出中心與工廠

A. 型態模式 
因 UV 寬幅噴墨印刷機具有客製化、多樣化等優點，小型的輸出中心採購此項設備可以用新的製程製作
少量多樣的產品，符合市場消費者的多變的需求。而傳統工廠面臨數位化的轉型，導入數位化製程取代傳統網
版短版印刷，因而購買 UV 寬幅噴墨印刷機。此外，UV 寬幅噴墨印刷機也會應用打樣上面。

B. 公司內部門檻 
○1 場地：UV 寬幅喷墨機，雖比傳統製程的生產線所需廠房相比，對廠房需求較小，但因幅度限制，添購公司

○2 價格：針對 UV 寬幅喷墨機的市場價格及耗材成本來說，添購公司仍需有一定的資本額，才能負擔琪設備維
修及耗材成本。 
○3 技術：UV 寬幅喷墨機的人員技術不高，對於人才雇用及員工教育較方便，且製程簡單，容易控管。 
○4 市場定位：購買設備前需要對市場需求作明確的定位，如是代工廠或自家產品的加工；木材（恩德）或紡織
（mimaki）等。還有對於市場的熟悉度也是很重要的。如作木材起家的恩德，對原先木材的熟悉度，進而發
展到材料設備上。再從設備材料推廣到通路行銷。 
○5 創新思想：因 UV 寬幅喷墨機的多元應用，購買 UV 寬幅喷墨機需具備有創新的思維，擺脫傳統的製程，嚐
試求新求變得新方法與新產品應用，才能開拓新的市場。 
小結，投資商應瞭解自己所需的市場定位為最重要因素，才能依所需要求購買設備耗材等。

目前 UV 寬幅噴墨所面臨到的最大困難為設備價格太高及墨水成本太高等兩大因素。但 UV 環保的趨勢
已逐漸成熟，歐美市場對環保觀念的普及，未來亞洲市場也會對環保意識有同樣的壓力，驅使社會環境改變產
業發展。待國內觀念的改變及成本的下降，UV 寬幅噴墨將會取代傳統製程或網版印刷。 
台灣是代工業的大工廠，待印刷技術門檻的降低，大公司對於生產一貫化作業將會廣泛應用在玻璃廠、
磁磚廠、3C 產品製造廠、光電廠、皮革廠等，都有自行採購 UV 寬幅噴墨設備的可能。 
油墨的成本與技術的改善將會控制 UV 喷墨的發展。在成本方面：當墨水成本下降或工資成本提高後，
使用 UV 寬幅噴墨印刷的廠家會大幅的提升。在墨水改良上：墨水的適印性，及其他特殊油墨的開發應用，如
導電油墨、生化墨水、螢光墨水、膠黏墨水等開發與技術的提升，將會開拓更大與更多元應用的 UV 喷墨市場。 
 
3. 研究建議 
(1)給產業界的建議 
經本研究實驗結果發現，UV 噴墨印刷應用於壓克力(Acrylic)、玻璃(Glass)和美耐板(Laminate)等被
印材之階調擴增值百分比曲線與一般紙張之階調擴增值百分比曲線有明顯差異，其階調擴增在亮部階
調最為顯著，建議業者可參考本研究結果進一步進行亮部階調網點擴增之印刷品質控制。 
本研究實驗由 6 家廠商依照日常流程進行印刷實驗，非針對特殊被印材調整印刷過程，研究發現
三種被印材壓克力(Acrylic)、玻璃(Glass)和美耐板(Laminate)之著墨能力仍有改善空間。建議業者可以
於依日常印刷流程強化前處理過程，以得到更好的色彩表現之印刷品質。 
本研究 UV 寬幅喷墨印刷機之印機油墨組合在壓克力(Acrylic)、玻璃(Glass)和美耐板(Laminate)之
色彩空間均不同，建議業者可依需生產之印刷產品，參考本研究結果選擇合適之印機油墨組合。 
因 UV 寬幅噴墨印刷產業跨足產品範圍廣泛，本研究建議業者在進行 UV 寬幅噴墨印刷產業投資
或採購之前，需先了解 UV 寬幅噴墨印刷產業之市場概況，針對特定市場需求進行採購投資，並發展
前後端整合之生產線。 
(3) 給未來研究方向的建議 
本研究因時間與人力限制，僅針對滿版濃度(Solid Ink Desity)、印刷對比(Print Contrast)、階調擴增
(Dot Gain/TVI)和色域圖(Color Gamut)作探討，建議後續研究可在針對數位印刷品質規格(ISO13660)進
行圖文細部品質探討，或增加被印材種類等後續研究。本研究發現 UV 墨與非紙張之被印材著墨能力
仍有改善空間，建議往印刷前處理作進一步之探討。

本研究藉由實驗探討台灣目前常見之 6種ＵＶ噴墨印刷機和油墨組合與三種特殊印材（壓
克力、玻璃翰美耐板）的印刷品質特性（滿版濃度、印刷對比、階調擴增和色域圖）。並
進一步藉由專家訪談瞭解ＵＶ噴墨印刷之市場狀況與未來發展趨勢。 
一、學術成就 
本研究廠商多樣且具代表意義，被印材料選擇具特色，非過往研究之紙張被印材，此創新
材料與印機油墨組合研究，深具學術研究之價值。本研究結果，不但可以提供後續研究之
參考依據，並為ＵＶ噴墨印刷與特殊印材（壓克力、玻璃翰美耐板）建立印刷品質特性規
格。本研究發現特殊印材（壓克力、玻璃翰美耐板）在階調擴增之曲線與一般傳統紙張階
調擴增之曲線不同，提供產業與學術界參考，並建議後續研究發展。本研究藉由深度專家
訪談，探訪北、中、南ＵＶ噴墨印刷之廠商、機器代理商和學界專家等，了解ＵＶ噴墨印
刷之市場狀況與未來發展趨勢，提供質化研究內容。本研究兼具質化與量化內容，具有全
面性研究參考之學術價值。
