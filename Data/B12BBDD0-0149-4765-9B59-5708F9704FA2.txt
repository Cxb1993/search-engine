壹、 概覽 
一. 動機與目的 
近年來，無線射頻辨識系統（RFID）的應用日益廣泛，國內、外有愈來愈來的業者開
始投入這項科技的應用，尤其是以物流產業的應用最為積極，由於這項科技的應用及推行，
未來勢必影響國內產業的生態。再者，處於知識經濟時代，創新不再是專屬於高科技產業
的議題，任何產業均需要創新甚至於教育。因此，本論文研究將針對教育的創新議題進行
探討，以提供教育於擬定創新策略之參考依據。本計畫希望藉由 RFID 系統的應用及相關效
益的估評說明，讓教育也對於 RFID 的應用有更深入一層的瞭解，並且及早發展系統的應用
及開發，以提昇教育競爭力。 
二. 重要性 
有鑒於國內教育導入 RFID 系統的應用尚不多見，且均屬研究理論尚未實踐與應用。因
此，本計畫研究將分別從學生、教師、教具管理以及學校的觀點，來討論 RFID 系統在教師
教學的應用情況、學生學習狀況及學校對於教具管理方面，並試圖將教育採用 RFID 系統的
原因及其應用的效益進行分析說明，以提供給教育導入 RFID 系統的參考。 
本計畫為了具體呈現 RFID 應用在教具物流的效益，將透過軟體的實作使用展現其真實
環境教具物流之運用，並且依據國立台北護理學院資管系教具借用的作業現況，建構出現
行作業與 RFID 作業模式的應用模型，藉以探討導入 RFID 系統前、後所造成的作業績效及
成本變化的情形。 
貳、 理論實踐 
一. 系統規畫與需求 
本次系統規畫主要平台採用微軟的 Windows Server 為基礎，手持式 Pocket PC 部份則
利用 Windows CE。並搭配 AMC RFID 硬體為主要架構。整體主要介紹硬體如下： 
RFID硬體部份 
標籤(Tags)：讀寫被動式ID標籤，可附加條碼於卡片上。採用國際標準13.56MHz。配合教
具物統系統中的教具管理，安裝於教具設備中，藉以管理相關設備。另與學生證做配合，
達到人員控管 
讀取器(Readers)： 
一、RFID 讀寫器採用 HF 標準，可同時讀取多個標籤。採用 UTP 連線方式可與多讀取器配
合達到多功多讀取處理。 
 
二、HF CF Card Reader 符合 ISO 15693 / 14443A 國際標準。安裝於手持式 PocketPC 中。
利用其移動性達成教具物流盤點管理功能。 
其他相關硬體：手持式 PocketPC。採用專供掌上型電腦使用的 Microsoft Windows Mobile 
5.0 超值版 及 Microsoft 行動軟體版本包含適用於掌上型電腦的 Outlook Mobile, Word 
Mobile, PowerPoint Mobile, Windows Media Player Mobile 和 IE Mobile 
 
RFID系統採用國際流通標準Object Naming Service and EPC IS流程。以其教具流程系統
與國際接軌。系統以ASP.NET為主做開發，後端資料庫為SQL Server為主要資料庫，查詢方
式以建立以WINDOWS 2000加上IIS為主要平台，而後台介面以Visual Basic6.0為主。開發
30 件不同教學器材數量組成管理情境。每管理區塊中央，配置有一 RFID Reader(讀取器)，
而且每件教學器材上皆貼有一片 RFID Tag(標籤)，該 RFID Tag 每秒鐘傳送封包給各管理
區塊中心的 RFID Reader，確定該教學器材目前位置處於管理區塊，若是讓 RFID Reader
有讀取到讓封包，則表示該教學器材未被借用，反之，則為借出。 
 
圖三 教學器材借用管理室模擬情境設計圖 
 
如圖四所示，右下角為教學器材管理室進單一出入門口，配置一台 RFID Reader。每
管理區塊之教學器材旁邊的綠色虛線代表以無線傳遞訊號，而藍色圈圈代表其無線電波傳
遞範圍。當每位管理區塊讀取不到某一件教學器材時，則代表該教學器材被借出，借出時
會經過右下角 RFID Reader，讀取借用之教學器材 RFID Tag 資料。 
由實驗結果得知，當教學器材未被借出時，五個管理區塊之 RFID Reader 所收到之封
包數量皆為各 RFID Tag 傳送的數量；而當教學器材被借用時，會經過右下角的 RFID 
Reader，其讀取到的封包數量，也為教學器材借用時所讀取到的 RFID Tag 封包數量。 
 
IV. 持學生卡與教具經由 A1 Reader，系統會以語音回報所借用的教具，防止系統遺漏。 
V. 教具管理者可以經由 A2 管理系統得知物品是否在物品櫃上。 
VI. 教具管理者亦可利用 A1 系統查尋各項目是由那個年級的負責人所借出，以便之後的追

肆、 成果與討論 
RFID 系統應用於教具物流中心，可在教具庫存水準、教具庫存週轉率、教具借用完成
率、缺貨率、以及人工工作比率改善上，與現有作業模式有顯著差異。 
從實際訪察並由單一教具的觀點來看，在庫存水準上：因有良好的控制使得在教具的
借用時程可以獲得控制，而顯少發生借用者借用不到教具情況。在教具庫存周轉率上：課
程在教學的教具組合使用上，可以透過本系統來得知教具的借用情況，並可即時做教具的
支援，使得整個周轉率提昇，透過本系統的教學品質上均可得更好的品質。以往傳統教具
借用是由人工做借用教具的主要方式。今天利用本系統將可使得人工工作比率獲得相當大
的改善，由早期的填寫方式進而改為 RFID 模式，而透過此系統來改變現有流程，使得教學
反應時間變快，進而提昇教育競爭力。 
在本計畫中，我們以教師及學生為出發點，並考量學校整個行政流程而提出「教具物
流的 RFID 展示環境及其模製系統之研究」計畫及架構，經由本系統的實行能夠於協助學校
能夠獲得更大的教育競爭力，並且使得教師能夠提高教學品質，學生可以獲得更好的學習
環境。 
經由實際操作後所得結果指出：學生以傳統填單借用教具由原本數十分鐘減少為數分
鐘。透過 RFID 系統可以使得效率增加百分之五十以上。也使得教具借用容易化。 
然而，對於 RFID 而言，並未將其百分之百的完全應用，我們將持續對 RFID 的技術再
做更深層的了解，以期發展出對於教育更有前瞻性的 RFID 技術。此外對於軟體開發我們也
希望再加入如 XML 的技術開發，並視為我們未來開發的標準之一。並且針對教具物流的 RFID
系統流程，於累積經驗後進行再檢討，期使系統更加精進更加完美。
