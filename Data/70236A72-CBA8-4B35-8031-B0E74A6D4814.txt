            摘要 
即時通訊協定早已發展多年，而相關
的研究包括網路品質控制、順暢度、延遲
改善及編碼演算法等，其發展可以分為線
上即時與非線上即時系統兩種，但大多數
研究都著重在非線上即時系統，例如線上
影片等。本研究期許可以設計出比現有免
費即時通訊軟體更方便的遠距即時影音教
學系統。在設計上本研究利用侵入 RTP 來
克服防火牆等問題、而對於一般 ADSL 上
傳頻寬不足的部分，則提出以基因演算法
配合分散式傳送機制等技術來加以解決，
除了解決網路與防火牆相關問題外，本論
文也設計了可搜尋的多媒體儲存格式，此
格式包含了影音、投影片與白板筆跡。 
最後本研究也依據理論實作出
Web-Based 即時遠距同步教學系統，在效
能評估實驗上採實地測試且證明了三個重
要項目，首先，本系統上傳頻寬不論多少
人均維持在一定的流量，不會令使用者端
的上傳頻寬無法負荷，更適合運用在一對
多交談。在 RTP 封包傳送方面可確保遺失
率不會過高，最後本系統在 CPU 的測試中
不論客戶端或伺服端也優於 P2P 軟體。 
關鍵字：防火牆、多媒體檔案格式、分散
式傳送、即時通訊協定、遠距教學 
一、 研究背景 
其實傳輸協定技術早已發展多
年，加上近年來寬頻的提升以及即時
通訊軟體的盛行，使得即時傳輸協定
漸漸受到重視。在目前的線上即時教
學中，一般的線上即時教學環境中都
是以完全硬體的方式來協助專家知
識的轉移，例如電視教學頻道，然而
在網路頻寬高度發展的今天，我們可
以將傳統的教學改進成線上即時與
非即時電腦輔助教學系統，運用此系
統我們可以直接利用個人電腦教室
當作傳統教學的環境，改變必須在同
一時間，同一地點上課的傳統教學方
式，也可以利用教學情境重現來做到
複習的效果。 
目前較常見線上即時影音系統
的產品，均屬於 Application 的型
式，必需由使用者自行安裝及更新。
只有 flash server 提供了 Web base 
的即時視訊產品。對 client 端而言也
沒有更新的需求，算是最方便使用者
的方案。不過 flash server 有嚴重的
頻寬限制及無法擴充性。 
二、 重要性及貢獻 
本論文之貢獻可歸納成五點。 
(一) 開發出可以模擬傳統教室的
遠距教學系統 
此系統所能提供的資訊有電子
白板、投影片、所有人的影像與語
音，因此除了肢體接觸外其它感覺大
致雷同。 
(二) 提出 RTP 可以穿過 NAT 之機
制 
傳送即時影音封包的 RTP 協定可
以不受 NAT 限制降低使用上的困難。 
(三) 提出 RTP 分散式傳送之機制 
利用此機制將網路流量分散給
其它的影音轉送伺服器來負擔，可以
減少單一伺服必需負荷過大的網路
流量，而使得封包遺失更嚴重。
