摘要 
 
無線感測網路可應用於健康看護及災害回報等技術，其資訊的重要性與準確性要求較為嚴謹，相對應的資訊安全運
用更應受到重視。無線感測網路入侵偵測系統相關研究顯示，系統偵測技術會影響到整個網路的生命週期。本研究結合
本體論(ontology)與入侵偵測系統，並提出輕量化本體論入侵偵測系統，以減少使用入侵偵測系統所消耗的能源。本研究
訓練攻擊型態於攻擊知識中，並且建置感測節點關係於本體論中，用以偵測偽裝攻擊(Sybil attack)。入侵偵測輕量化使用
巡守節點輪流裝載攻擊知識以偵測異常的感測節點，並且運用一部份的本體論以偵測 Sybil attack。最後利用隔離表持續
隔離異常節點，避免重複偵測所耗費的能源。初步實驗結果證明，我們所提出的方法可以減少無線感測網路入侵偵測的
能源消耗，並提升入侵偵測系統的可用性。 
關鍵詞：無線感測網路、入侵偵測系統、本體論、Sybil Attack。 
 
1. 前言 
 
無線感測網路資訊安全相關研究因應無線感測網路應用的發展，越來越受到重視。無線感測網路是一種透過數以千
計的低成本、微硬體及資源受限的感測節點進行環境感測的技術。其相關的應用包括戰場監控、災害回報及健康看顧等
[1]。當感測節點配置完成，網路會自我組織及自動建立路由路徑，並且將自身環境相關資訊傳回至基地台(Base Station, 
BS)。每個感測節點裝載受限制且不可更換的能源，因此能源消耗的控管以及延長整體網路的生命週期即為最重要的探討
議題之一。 
無線感測網路資訊安全可以區分為密碼學與入侵偵測系統(Intrusion Detection System, IDS)二大主題。密碼學使用加
密與解密方法保護資訊，認證每個節點以找出異常節點。密碼學在無線感測網路中，屬於第一線防護系統。而入侵偵測
系統透過異常偵測，比對每個節點的行為，以找出異常節點。如果感測節點行為異常，入侵偵測系統會提出警報讓管理
者進行手動處理。而在無線感測網路中，入侵偵測系統即為資訊安全的第二線防護系統。但在無線感測網路中，這二種
防護方式都會加速消耗整體網路的能源，因此如何讓網路能裝載資訊安全方法，並有效的使用就是目前研究重點之一。 
Sybil Attack 是一種攻擊者常使用來截取無線感測網路資訊的攻擊型態。入侵者會假扮自己成為感測節點、路由節點、
基地台等，以獲得網路中的資訊，並為下一個攻擊階段進行準備。由於攻擊者並無對資訊進行竄改，並且正常傳送資訊，
因此此種攻擊行為較難偵測。為了防禦此種攻擊，每個感測節點必需進行認證，並需擁有有效的節點認證資料。但認證
方法必需簡單，才能降低能源的消耗。然而，如果認證方法太容易解開，也會因此失去效用。因此，本研究提出在本體
論建置各個感測節點之間關聯，並結合至入侵偵測系統中，以提供新型偵測 Sybil attack 的方法[2]。 
將軟式計算技術應用於無線感測網路入侵偵測系統是近期相關研究探討的重點之一。由於軟式計算有高度萃取知識
的能力，因此已經被廣泛的使用在有線網路的資訊安全中。但是，真正使用軟式計算於無線感測網路入侵偵測系統的研
究並不多。軟式計算工具如支援向量機(Support Vector Machine, SVM)[16]、約略集合(Rough Set)[17]和類神經網路(Artificial 
Neural Network, ANN)[6]等在進行資料訓練及測試時，需要耗費較多的能源。但由於無線感測網路能源受到限制。因此為
了增加軟式計算的可用性，使用輕量化軟式計算技術於入侵偵測系統中變成重要的議題之一[3][7]。本研究提出的方法會
探討輕量化技術如何提升無線感測網路入侵偵測系統的效率。 
本體論是軟式計算的技術之一，它是用來實現語義網的技術。本體論的主要目標在於將獨立的知識分類到概念中，
並且定義他們之間的關係。而本體論分類過的知識會被用來產生新的知識。因此本研究運用本體論定義感測節點之間的
關係，用以區分出無線感測網路中的異常節點。 
在預處理階段，當無線感測網路配置完成，基地台會整合位置資訊，接著入侵偵測系統建置每個感測節點之間的關
係。管理者設置建立關係的門檻值。每個節點會依據計算出的關係值是否大於門檻值，而決定是否建立關聯。最後，入
侵偵測系統會根據關聯資料，比對該節點是否為異常節點。 
 
本研究主要貢獻是提出一個輕量化本體論入侵偵測系統，並且使用支援向量機和約略集合預先處理封包資料，以偵
測 Sybil attack。另外，本研究將方法實作於無線感測網路後，評測無線測網路入侵偵測系統的可用性與效能。 
本研究可分為五個步驟：(1)使用約略集合於封包前處理，並且利用支援向量機訓練攻擊知識；(2)比對感測節點與巡
守節點之間的關係以偵測 Sybil attack；(3)根據整體網路選擇攻擊知識進行偵測；(4)記錄異常資訊於隔離表中；最後(5)重
複這些步驟直至攻擊知識收斂。 
本研究文章組織如下：第二章整理出入侵偵測系統與軟式計算的相關文獻，第三章介紹研究方法及架構，第四章呈
現實驗結果與評估，第五章為結論與未來展望。

份的本體論以偵測 Sybil attack。 
(2) 叢集頭：它負責無線感測網路與基地台之間的傳送連結。叢集頭分派工作給巡守節點。而巡守節點會定期回傳資料給
叢集頭進行整合。叢集頭會將自己責任週期均分給巡守節點，接著依據基地台分派的資訊，將攻擊知識與本體論關係
傳送給巡守節點進行網路監控與資料整合。 
(3) 巡守節點：它們會分擔叢集頭的工作。由於無線感測路環境包含一般環境與攻擊環境，巡守節點會依環境狀況蒐集鄰

附近的本體論關係以偵測 Sybil attack。它會記錄異常的資訊到隔離表中，在一般的狀況下，巡守節點會周期性回傳資
料與隔離表給叢集頭。但若發生攻擊，這些資料會馬上回傳給叢集頭進行留存。 
(4) 感測節點：它們負責感測附近的環境資訊，如光、熱、濕度等，並會周期性傳送資料給巡守節點。

3.3 入侵偵測階段 
 
本體論建置完成後，分派系統會分析整體網路，並進行入侵偵測工作分派。且入侵偵測系統會預先選擇較常遇到的
攻擊知識，傳送至叢集頭進行分派工作。而本體論用以偵測網路中的 Sybil attack。 
正常狀況下，無線感測網路配置較不均勻，我們提出巡守式入侵偵測系統，使用攻擊知識以偵測異常[10]。巡守式入
侵偵測系統在巡守節點間轉送，並偵測鄰近節點是否異常，並依照不同環境選擇不同的攻擊知識。巡守式入侵偵測系統
只需裝載一部份的攻擊知識，以降低無線感測網路能源消耗。本研究提出的本體論入侵偵測系統結合本體論與巡守式入
侵偵測系統的優點。差別在於本體論入侵偵測系統會裝載一部份的本體關係，以偵測 Sybil attack。入侵偵測系統也會根
據回傳的資料與攻擊知識比對，如果無線感測網路中存在沒有選擇到的攻擊知識，則進行攻擊知識的替換與調整。 
本體論會被切割成幾個部分，根據巡守節點所負責的範圍，進行調整。巡守節點會輪流監控網路，以防止全部的網
路都耗費能源在進行網路監控，這不止可以延長整體網路的生命週期，也可以降低無線感測網路裝載入侵偵測系統所需
成本。當巡守節點責任週期全數結束後，叢集頭會重新收集無線感測網路資料。並且傳回至基地台進行分析，再依據網
路環境對入侵偵測系統及攻擊知識進行調整，以提高偵測準確率。系統會不停重複訓練攻擊知識與入侵偵測系統，直至
收斂。收斂後的攻擊知識會較具代表性，讓入侵偵測可以更輕量化。 
無線感測網路是一種自我組織網路，也就是路由表與叢集頭會變動，當無線感測網路的叢集頭變動後，整體無線感
測網路無法繼續隔離異常節點，因此我們使用隔離表將異常節點記錄於基地台中[11]。無線感測網路的叢集頭更換後，仍
然可以透過隔離表對異常節點進行隔離，並減少重複偵測的能源消耗。 
 
4. 實驗結果 
 
本研究使用網路模擬程式(ns-2)和 java 實作系統並評估效能。實驗硬體為 Intel® Core™2 Duo CPU T8100 2.10GHz 電
腦與 2GB RAM，作業系統為 Windows XP。我們模擬無線感測網路環境為 10,000 平方公尺，環境為靜態環境，並且配置
50 個感測節點，每個節點的廣播距離為 50 公尺。叢集頭被配置在無線感測網路的正中間，以便收集資料。 
實驗將我們的方法與巡守式入侵偵測進行比較，並使用 Sybil attack 測試系統效能。我們不考慮加解密技術，因為我
們重點在入侵偵測系統。總模擬時間為 3600 秒，於網路建置後 600 秒開始，每 20 秒隨機產生攻擊。本系統使用 protégée
建置本體論。 
實驗結果如圖 7 所示，存活節點是指整體無線感測網路中可以正常運用的節點數量。本實驗以無裝載 IDS、無裝載
IDS 於無攻擊環境、巡守式 IDS、巡守式 IDS 於無攻擊環境、本體論 IDS、本體論 IDS 於無攻擊環境等六種實驗模式進行。 
由於本體論 IDS 結合巡守式 IDS 與本體論，因此在能源消耗上，略高於巡守式 IDS，但存活節點數量相近。我們模
擬 50 個感測節點，結果顯示本體論入侵偵測系統的生命周期較長，且節點死亡速度較慢。由於巡守節點直接從感測節點
取得資料，叢集頭也透過巡守節點搜集資料，因此在整合資料與控管無線感測網路上，所需耗費的能源不高。實驗結果
證明本體論入侵偵測系統可以簡易的使用於無線感測網路上。

5. 結論與未來展望 
 
本研究提出將節點之間的關聯性建構本體論於無線感測網路入侵偵測系統上，以提供輕量化入侵偵測系統透過節點
之間的關係偵測 Sybil attack，並且不需使用額外的加解密機制。主要的貢獻為讓本體論可以建構整個無線感測網路的關
係，並且依據設置不同的相似性門檻值，用以偵測不同安全層級的 Sybil attack。實驗結果證明，本體論入侵偵測系統能
夠增加節點的存活率，並且對於系統造成的負擔不重。未來研究將對於該如何建置其它攻擊知識在本體論中，並且加以
應用，即為接下來需要探討的重點之一。
