三、研究方法及進行步驟： 
  計畫主持人原有的「智慧型遠距家庭照護科技創新與應用」之系統功能是由
裝置在被照護者胸前的「心跳感測器」提供了遠端醫療照護的偵測功能，胸腔範
圍內佩帶此「心跳感測器」後，可將心跳資料經由 Zigbee 與 Zigbee 之間的通訊
協定所形成的 WSN，再透過各個 RFD(Reduced Function Device)節點，將訊號發
送到 FFD(Full Function Device)調解器上，再利用 RS-232 與 PC 相連，使醫生透
過網際網路，達成完整的遠距醫療照護系統。  
 
本計畫根據上述「智慧型遠距家庭照護科技創新與應用」計畫，改進兩點，說明
如下： 
（1） 將使用裝置於被照護者胸前的「心跳感測器」，該胸前「心跳感測器」容
易造成被照護者使用上的不便性，尤其被照護者是女性時，往往被排斥不
願意配合使用，因此本計畫將改用兼具定位功能手錶型「心跳感測器」，
其形狀類似手錶，使被照護者在配帶方面沒有困擾，也維持被照護者在使
用「心跳感測器」的自尊，不容易被其他人發現，該手錶型「心跳感測器」
兼具定位功能，而且縮小了體積，便利性大為提高。

1.2 韌體設計 
在整個系統中，韌體的開發也佔了相當大的工作比例，主要的設計依性質區
分如下： 
1.2.1.ZigBee Protocol：在 ZigBee 協定中定義了三種角色，包括 Coordinator、Router
及 RFD。  
  Coordinator 主要工作是負責開啟一個無線網路，並管理這個網路的通訊，更
重要的是與網際網路做溝通(Bridge)。而 Router 則是負責傳遞該網路的封包。RFD
的工作則是負責管理並傳送感測器上的資料。在 Zigbee 協定上對於 NWK 層及
APS 這兩層又有複雜的協定定義，不容易獨力開發，本系統採用開放原始碼的微
處理機，而且提供軟體通訊的 ZigBee Stack，相信會使我們開發工作更加順利，
我們所使用的微處理機，該公司的 ZigBee Stack 是由 MicrochipC18 這個編譯器 C
語言撰寫，因此我們便在這個 Stack 上面再開發 APL 層，以整合我們的系統並符
合 ZigBee 協定的無線傳輸。 
 
1.2.2.Sensor Driver：由於 ZigBee Stack 並不是一個 OS，因為我們必須為「心跳
感測器」在 APL 層上撰寫合適的驅動程式。這次經過多方尋找比較選用的手腕
型「心跳感測器」，其搭配的外部介面是標準介面格式，擷取資料難度不高。  
 
1.3 軟體設計–網路技術 
  在本系統中生理資訊的傳遞除了在最前端是以無線作傳遞之外，一旦資訊被
Coordinator 收集之後就開始在有線上傳遞。由於本系統的目標為及時遠距醫療照
護及救護，因此我們必須使用許多的網際網路的技術才能夠實現這個系統。首先
我們以 Java 撰寫 Serial Forward，由程式把 RS-232 的資料轉換成能夠在 Internet
上傳遞的 TCP/IP 的封包格式，接著我們再利用 PHP 程式設計了一個 Socket 
Server，這個工作主要是接收從 Serial Forward 傳來的訊息，然後再以 TCP/IP 
Socket 介面和 Internet 相接。如此一來就可以把資訊送到全世界。最後使用者端
的操作介面則使用 Flash 撰寫一個即時動態播放器嵌入在網頁瀏覽器上，使用者
再使用網頁瀏覽器瀏覽到利用嵌入式的 Flash 播放器所發佈的即時訊息。

    參考比較了以上國內外類似本產學合作計畫之研究情況及重要參考文獻
後，本研究案將提供了比上述研究更多的功能，如提供遠距動態中之被照護者生
理資訊給健康照護體系，如發生不可接受之訊號時，除了監控端會發出警訊外，
被照護者的位置立刻被偵知，方便協尋。被照護者如在救護車上，生理資訊仍能
透過 GPS，GPRS 傳給將送往之醫師，醫院端也能透過終端機，隨時瞭解救護車
之行蹤。因此本研究案應是目前功能最齊全之即時動態目標無線遠距健康照護及
救護監控系統。 
 
五、結果與討論： 
   (1) 對於學術研究方面之貢獻: 無線感測器網路技術、通訊協定、資料管理系
統、測試及跨領域技術應用系統平台研發能量之建立、人員質量提升及系
統整合技術研發制度之建立。 
   (2) 對於國家發展及其他應用方面之貢獻: 前端使用任一種特殊功能的感測
器，我們仍然可以使用相同的後端傳送平台，擴大市場應用層面如在軍事
國防戰場環境資料收集、工業監控、農業自動化、城市智慧型交通管理、
汽車遠端健康檢測、生物醫療護理監控、環境監測、搶險救災、反恐、防
災、危險區域控制、危險物品管理及智慧型居家管理等領域之承載數據流
量較小及距離較小的業務任務監控應用。 
   (3) 本研究計畫已開發建置成功，將可為產業作研發及系統確立之先導工程，
不但可以培育學、研界之人才，更為學生提供完整的實際教學環境和職前
訓練，本校更可成為產業研發之重鎮，落實務實致用之重點特色。
