中文摘要 
本研究計畫為整合型計畫之總計畫，重點
在於原子層沈積系統之架設與測試，並整合各
子計畫之感測平台，進而而實現整合四種平台
之感測元件。於本年度之研究中，以架設好原
子層沈積(ALD)系統。並初步開始進行薄膜沈
積測試。子計畫之研究中，子計畫一以三氧化
二鉺與三氧化二釓應用於感測電晶體；子計畫
二，實現 Bottom gate 結構之離子感測器；子
計畫三則以氮化鎵為基底用於增測低密度脂
蛋白；子計畫四則是光定址電位感測元件之研
發，與鈉鉀離子選擇性之研究。而在整合之研
究中又分為兩部分，其一為利用 n與 p基底材
料，搭配差動放大之方式，實現無機參考電晶
體，以取代傳統參考電極。而另一部份，則是
開發聚二甲基矽氧烷(PDMS)為感測元件之封裝
材料，並進行基本電性與抗化性之評估。此封裝
方式將有助於整合各感測元件平台，並實現微流
道設計與整合多功能之感測陣列系統。 
 
關鍵字:原子層沈積系統、低密度脂蛋白無機
參考電晶體、光定址電位感測元件、離子選擇
性、參考電晶體 
Abstract 
This is a project to integrate four seed 
projects. The research was focused on the setup 
of the atomic layer deposition system. In project 
I, Gd2O3 and Er2O3 layer was used as the 
sensing membrane for ISFET; in project II, the 
bottom gate ion selective sensor was developed; 
in project III, GaN substrate was applied for the 
platform for oxidized LDL; and in project IV, 
LAPS system was developed, and the selectivity 
of the sensor for sodium and potassium ions was 
investigated. Besides, two integrated research 
was proposed, such as dual substrate for pH 
micro sensor and novel package technique 
based on PDMS. It will be very useful for 
integrate four seed project and realizing the 
hybrid muti-sensor array for blood detection. 
 
Keywords: atomic layer deposition, ISFET, 
GaN, LAPS, selectivity, PDMS

1. Introduction 
Ion sensitive field effect transistor (ISFET) is 
a kind of pH sensor device and has been 
investigated for more than thirty years [1]. It 
showed high potential to apply on biomedical 
detection and in-vivo monitoring because of the 
attractive advantages, such as small size, fast 
response, rigid structure, and high compatibility 
with CMOS technology. [2] However, to realize 
commercial ISFET sensors, some problems are 
still necessary to be solved. 
Just like other potentiometric sensors, a 
reference electrode is requisite to supply a 
stable reference potential or commonly ground 
for the sensor system. Until now, conventional 
reference electrodes, such as Ag/AgCl or 
calomel electrode, are still used for most of the 
ISFET systems. However, during the shirking of 
the sensor system to a single micro chip, the key 
limitation would be the minimized reference 
electrode. The volume of internal electrolyte for 
ion exchange is decreased with the smaller size 
of the reference electrode, which will influence 
the stability and lifetime of the reference 
electrode [3, 4]. 
To solve this problem, a concept of a 
differential system with ISFET/REFET 
(reference field effect transistor) pair was 
proposed to replace the conventional reference 
electrode in the sensor system [5]. The structure 
of REFET is very similar to ISFET, but 
performed with low ion sensitivity. Besides, 
based on the ISFET/REFET arrangement, a 
quasi reference electrode (qRE) made by noble 
metal, such as Pt or Au, is used to supply for 
commonly ground for the system. To fabricate a 
REFET, an ion-unblocking PVC membrane 
casted on the ISFET was performed with low 
pH sensitivity and high electrical compatibility 
with ISFET [6, 7]. However, there are still some 
drawbacks, including application range, short 
lifetime, and drift effect. 
In this study, a novel research of inorganic 
ISFET/REFET was proposed. Silicon nitride 
(Si3N4) layers were directly deposited on n and 
p type silicon substrate as the pH sensing 
membrane, respectively. To evaluate the pH 
sensing performance, pH sensitivity, long term 
stability, light effect, and hysteresis were 
measured.

on the package technology for EIS structure 
(研究成果已投稿於醫學工程年會, 2008) 
 
於整合計畫之重要性： 
    此研究開發出一新的封裝方式，以 PDMS
為封裝材料，除了可整合微流道與感測晶片
外，還可進一步的將子計畫一至四之感測元件
合併，開發出多功能整合感測元件陣列(hybrid 
multi-biosensor array)，再加上其優良的抗化性
與生物相容性，因此可進一步的發展至血液檢
測之應用。 
 
1. Introduction 
離子感測場效電晶體 ( ion sensitive field 
effecttransistor, ISFET ) 首先在 1970 年由 Pite 
Bervgeld 所提出[1]。它與金氧半場效電晶體 
( metal-oxide-semiconductor FET,MOSFET ) 結構
非常類似，差別在於ISFET使用絕緣層或感測膜取
代金屬閘極來感測溶液中離子濃度變化，並外接
銀 ( Ag ) /氯化銀 ( AgCl )參考電極，此外接參考
電極可提供一穩定電位，以防止電位漂移而產生
訊號干擾現象[2]。 
 
電解液-絕緣層-半導體 ( EIS ) 結構為離子
感測場效電晶體之核心部份，需要與溶液接觸形
成介面電位，為避免溶液與元件非感測區部分接
觸造成漏電流等感測上希望避免的非理想現象，
適當的封裝方式格外重要。 
 
    目前常用的封裝與定義感測面積方式有光阻
型、製具型...等方式，前者利用光學微影技術在
感測層上定義感測面積，常用光阻如SU8。後者
先造出固定面積的製具，再將其固定在感測膜
上，如O型環( O ring )。本研究首次提出以聚二甲
基矽氧烷PDMS用於EIS結構之封裝材料。為一個
新的封裝與定義感測區域並造出微流道方式。
PDMS由基底 ( base ) 、固化劑 ( curing agent ) 
兩種液體混合而成，待其固化即完成封裝[3]。此
種封裝方式的抗化性，製造出的EIS結構電性都是
本文討論的內容。

4. Conclusion 
本研究中，利用聚二甲基矽氧烷 ( PDMS ) 
定義感測面積、封裝及製造流道，可結合微流體
晶片的主流製程，將發展成熟的各式半導體感測
器導入微全分析系統 ( micro total analysis system, 
μ-TAS ) 。經過抗化性的測試，可證實此材料可
抵抗強酸、強鹼與有機溶液長達 3 天的時間。並
且經電解液-絕緣層-半導體結構之測試，由基本的
C-V 曲線與氫離子感測特性也可證實此材料用於
感測平台之封裝技術，確實可達到定義面積並保
護元件不因漏水或化學侵蝕的干擾而影響感測特
性。亦可進一步的用於多通道與生醫感測之領域。
