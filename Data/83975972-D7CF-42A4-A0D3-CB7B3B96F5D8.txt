是以平行並順著同一方向延伸出去，因此我們認為，T20-55
的硬相排列成平板狀(lamellar)的結構。經由影相分析可得硬
相寬度為 52.1 Å，而質長度均已超出 TEM image 的成相範圍
所以無法計算。T20-55 的理論平均分子量含有 7.52 個
MDI-BD 與 1.88 個 TDI-BD unit，折合長度約為 167.88 Å。 
圖 12 為 T30-55 的 TEM 圖。我們一樣能夠看到白色條
狀並列的情形，相較於 T20-55，此條狀的硬相並未延伸至相
當長的距離。圖中較膜糊的部分雖然可以看到白色的圓點，
但這並不像 HC=45%、HC=30%的樣品是由圓柱所產生，而
是因為影相重疊所造成。為了對此現象加以禪釋，必須涉及
TEM 取景的情況，在低硬質段含量時，TEM 圖象會隨焦距
調整而一起膜糊，而在高硬質段含量時，隨著調整焦距不同
可以看到不同方向的白色條狀硬相在同一個地方內出現。因
此我們認為，T30-65 也是 lamellar 結構。T30-55 的理論平均
硬質相長度含有 6.74 個 MDI-BD unit、2.89 個 TDI-BD unit，
平均 2~3 個 MDI-BD unit 就會出現 1 個 TDI-BD unit。 
圖 13 為 T0-65 的 TEM 圖。白色硬相部分已接近連續
相形態，反包埋住黑軟質相，在 T0-55 時硬質相是相當長的
圓柱狀，隨硬質段含量增加，兩硬質相會逐漸連接起來，將
軟質相包住，如圖 4-19(d)所示意。T0-65 理論平均硬質段分
子量含有 13.65 個 MDI-BD unit，折合長度約為 258.67 Å。
經影相分析測得硬相 domain 寬為 52.8 Å，因為硬質相已呈
現近乎連續，所以無法測得其長度。 
 圖 14 為 T10-65 的 TEM 圖。同 T0-65，白色硬相將黑
色軟相包埋其中，不一樣的是 T10-65 中的軟相 domain 略長
於前者，換句話說，硬相以稍長的圓柱相連結來呈現，如圖
4-19(e)所示意。經影相分析測得硬相 domain 寬為 52.4 Å，
長度則因硬相幾乎為完全連續而無法獲得。 
 圖 15 為 T20-65 的 TEM 圖。可以發現其軟相長度比
T10-65 來得更長，且硬相 domain 有扭曲線象。圖中黑白模
糊地帶應該是由於圓柱間相互連結程度上升，在即將轉成
lamellar 形態時的過渡狀態橫切面所造成。 
圖 16 為 T30-65 的 TEM 圖。可以發現在相同硬質段含量下，
硬相隨 TDI 含量的增加，由圖 44(d)的形態轉變成圖 44(g)的
lamellar 的形態。經由影相分析得知，質硬相 domain 寬度為
53.5 Å 略大於 T30-55 的 lamellar 形態。  
 
4-2 DSC 熱分析 
圖 17 至 20 為一系列不同硬質段含量與不同 TDI 含量
PU 的低溫 DSC 1st run 掃描圖譜，計算所得之 Tgs、玻△Bs、
△Cps 列於表 4-1 中。純 polyisoprene 的 Tgs = -59℃，而△
Bs = 5℃、△Cps = 0.56J/g℃。以 polyisoprene 為軟質段的
PU，1st run 測得其 Tgs 約在-56 至-53℃左右，△Bs 約為 11~5
℃，△Cps 則因硬質段含量改變而有所不同。比較不同硬質
段含量的軟質段 Tgs，不隨硬質段含量的上升而有很大的變
化，且△B 亦未改變很多，表示本系統之 PU 幾乎可以說是
完全相分離。所有一系列PU的Tgs與△B均比純polyisoprene 
prepolymer 來的高，這是因為在 PU 軟相中的 polyisoprene
鏈末端被鎖在堅固的硬質相表面，降低了軟質段的自由體積
和活動性所造成。 
由表 2 可以看到，隨著 TDI 含量由 0%逐次增加 10%直
至 30%時，系列樣品在 1st run 部分△Cps 是呈現上升、下
降再上升的趨勢，這與我們當初所想像的不同。就硬質相排
列而言，TDI 的增加會破壞 MDI-BD 硬質鏈段所形成的 order
程度，換言之軟質將會因此有較高的自由度，而△Cps 應呈
現單調遞減的趨勢，但結果卻均不符合此觀念，是非常值得
進一步思考的地方。至於回火後的低溫測試部分，我們可以
看到，除了 T10-30 之外，所有△Cps 均比未經 melt-quench
的 sample 來的大，這是因為 1st run 的樣品在乾燥的過程，
溶劑是緩緩揮發，提供了硬質段鏈有足夠的移動性以及時間
堆排形成 order，然而在 melt-quench 過程，高溫熔融的硬質
相為 spherular like 的形態，在聚冷後其硬相形態對軟相
entropy restriction 較小，所以 2nd run 可表現出較高的△Cp
值。 
另一個由 entropy restriction effect 所表現的情形出現
在 T20-55 樣品，在同為 HC=55%的 TEM image 中，以 T20-55
為最完整的 lamellar 結構，相對於所有 HC=55%的樣品而
言，以 T20-55 所受的 entropy restriction effect 也最大，所以
其△Cps 值也最低。 
觀察高溫 DSC 1st run 掃描圖譜(圖 25 至圖 28)可以發
現，硬質相 Tgh 約出現在 57~73℃的範圍之間，且隨著硬質
段含量的上升，有愈來愈明顯的趨勢，這是因為隨著硬質鏈
段的增長，硬鏈分子間氫鍵鍵結點增加，使之更易互相吸引
形成 order 結構，而表現出較為明顯的熱轉移行為。這個現
象我們亦可由 TEM image 觀察到，硬質段含量上升，domain
愈大且愈明顯。 
比較高溫 1st run HC=55%、65%(圖 27、圖 28)的掃描
圖譜，我們可以由純 MDI-BD 為硬質段的 PU 之中看到，
140~180℃溫度範圍內，非極性式 PU 常見的 T2 long range 
order 吸收峰，此吸收峰隨著 TDI 的加入之後不但往低溫偏
移(T10-65)，且變得愈來愈不明顯。這應該是由於不對稱 TDI
的加入，破壞了原本 MDI-BD 的 order 結構，造成 T2 peak
往低溫偏移、變得寬廣。隨著 TDI 加入至 30%之後，T2 peak
消失程度最大。 
比較 melt-quenched 高溫(圖 29 至 32)與 1st run 高溫(圖 25 至
28)DSC 掃描圖譜可以發現，在 melt-quenched 之後，原來高
溫 Tgh 區域的轉移現象逐漸變成一個像是 order 的吸熱峰。
這是由於 sample 在經 melt-quenched 之後，放置在室溫中，
相當於以室溫的條件進行回火程序，進而導致 PU 的硬相緩
緩排列出 order 的結構所造成 5，此現象亦即 6 所提出的物理
老化焓鬆弛引起。然而他們也証明了，只要 sample 在經
melt-quenched 之後，隨即進行 DSC 的分析掃描，此 peak 便
會消失不見。同時在這四張圖中我們還可以觀察到，
T10-45、T10-55、T20-55、T0-65 這四個樣品在過了 T1 之後
有放熱現象產生，這是因為在高於 Tgh 的溫度情況，硬質鏈
擁有足夠活動性能形成新的 order 所放出的熱量，隨著溫度
的持續上升，現另一個高溫吸熱峰瓦解了此 order 結構。 
 
4-3 動態機械性質測試 
圖 33、34 與 35 分別為 HC=30%、HC=45%與 HC=55%
以純 MDI-BD 以及不同 TDI 取代量系列 PU 的 DMA 圖譜。
圖中低溫區域可以看到在-50℃附近存在著一個變化率很大
的鬆弛現象，storage modulus 在很窄的溫度範圍內由
3000MPa 左右降至不到 1000MPa，而 tangent δ亦出現一尖
峰，稱之為α peak，對照 DSC 測試得知軟質段 Tgs=-55℃。
由此可知DMA圖譜 storage modulus曲線的第一個降伏與 tan
δ在-30℃左右所出現的 peak，亦即為軟質段玻璃轉移過程
所展現的鬆弛現象。觀察 tanδ曲線，第二個鬆弛現象出現
在高溫區域，稱之為βpeak，對照 DSC 高溫 1st run 掃描圖
譜得知，此鬆弛現象是由於硬質段玻璃轉移所貢獻，這是在
相分離的系統中常見的現象，証明 PU 中的確存在著兩相系
統。觀察 storage modulus 曲線在過了軟質段玻璃轉移溫度
後，隨著硬質段含量的增加，下降程度愈趨於平緩，同時使
得 tanδ曲線的αpeak/βpeak 的比值隨硬質段含量增加而變
大，βpeak 也愈往高溫方向偏移，此現象符合 Macknight 7
學者所提出的論述。硬質段因鏈長增加，不但分子間的內聚
力上升，也提高了硬相 domain 的體積分率，連帶提升具有
filler 功能的的假擬交聯密度，使得硬質相能夠持至較高的溫
度才逐漸發生鬆弛。觀察相同硬質段含量情況下，不同 TDI
含量的 tanδ曲線，當 TDI 含量上升時，βpeak 往低溫偏移，
造成此現象可能有兩個原因：1.以 MDI 為異氰酸鹽的 PU 比
TDI 有較高的軟化溫度 5，且由 Macknight8 所提出的非極性
式 HTPB 與 TDI-BD 為硬質段鏈之 PU，其高溫鬆弛範圍約
在 70℃左右來看，當我們的系統隨著 TDI 含量的上升，其
所表現出來的性質也將愈來愈接近以 TDI 為硬其段異氰酸
鹽之 PU 的性質。2.隨著 TDI 取代 MDI 程度愈多，硬質段鏈
間的排列則因不對稱性 TDI 的嵌入，破壞了硬質相內原本
MDI-BD 硬質鏈段較為緊實的規則結構，降低了分子間間作
用力，使得βpeak 往低溫偏移。 
另一方面，在 HC=45%下可以看到，tanδ在軟質段玻
璃轉移溫度區域有略往高溫飄移的趨勢，參考極性式 PU 的
文獻可知 9，當相互溶情形發生時，將大大提升軟段 Tgs，而
本研究的系統其 Tgs 僅微幅上揚，証明這並非相互溶的現
像。且由 Cooper 5 的研究指出，此現象是因為 TDI 對稱性較
MDI 為差，其所形成的硬質段 order 結構也較 MDI 者為鬆
散，增加軟硬相界面面積，因而略微提升軟質段 Tgs。 
系列 PU 的動態機械性質測試另一項特別的地方出現
在 T20-30、T30-30、T20-55 與 T30-55，這四條 tanδ的曲線
分別在 20℃、30℃、40℃與 47℃左右有 peak 出現，顯示這
四個樣品在 0~30℃之間應該是有 order 的結構瓦解，對照圖
25、27 卻發現無法由 DSC 升溫曲線中看到有吸熱的情形發
生。由於 DMA 測試過程升溫速率很慢，加上 DSC 亦未發現
吸熱峰產生。參照 Sadovsky et. al.10 所作的研究指出，PU 在
拉伸過程中不僅會破壞原有的形態，也會有新的 reorder 的
情況產生，且這種現象為不可逆程序，藉此研判，應該是試
片在緩慢的升溫、動態測試過程有新的 order 產生，隨即反
應在 tanδ曲線上所致。
