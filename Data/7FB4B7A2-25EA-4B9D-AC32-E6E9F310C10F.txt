摘要 
針對飛航事故的調查，本研究『水下飛機殘骸之搜尋，辨識與聲納影像定
位系統之整合』希望能藉由聲納的探測技術結合相關的設備，準確、快速且自動
的辨識出飛機殘骸及其周遭的海底地形、地貌，並將殘骸影像給予準確定位。本
計畫擬以三年時間，完成一具精確水下側掃聲納影像定位系統之測試載具，以供
理論驗證及後續研究之施行。本計畫的分年研究目標如下：第一年：調查與整合
國內現有相關水下偵蒐裝備與技術，規劃設計精確水下側掃聲納影像定位系統之
測試載具與系統模擬。第二年：發展水下航空器殘骸辨識系統，結合精確水下側
掃聲納影像定位系統即時提供疑似殘骸位置。第三年：建置及測試精確水下側掃
聲納影像定位系統測試載具，探討其作為海上空難的支援平台之可行性。本報告
為第一年之結案報告，分為二部分：「國內現有單位、機構水下偵蒐裝備與技術
之調查與整合」、以及「側掃聲納模擬系統之研究」。 
關鍵詞：飛機殘骸，聲納影像，定位系統，水下載具，水下救難

二、背景介紹：

2. 聲納掃瞄作業 
分配到責任區塊的單位、公司先以側掃聲納儀(Side Scan Sonar；拖曳
式的為佳；見圖 2)於此待測區內進行大射距(Range；如 600 M)、寬測線間
距(視設距而定)、覆蓋率(Coverage；見圖 8)為 100%的快速底床掃瞄作業，
並先由其中進行初步的判釋工作。其結果若是已經很確定並能由聲納資料
中獲得相關足以進行後續打撈的資訊(如：位置、深度、尺寸、於底床上的
姿態…等)，則交給打撈的團隊進行後續工作；倘若確定為殘骸但由此資料
還無法得到可供打撈作業的訊息，則需再進行水下確認的工作，此部分將
詳述於後；倘若僅只是懷疑而無法確認是否為殘骸，則需針對此疑似水域
或區塊再進行一次較小射距(Range；如 300、150 或更小)、較窄的測線間距
(視設距而定)、覆蓋率(Coverage；見圖 8)200%的詳細底床掃瞄作業，以進
行疑點確認的工作，若確認為殘骸，則依其所得之資訊是否充足來決定是
移交打撈團隊或再進行水下確認工作。 
一般來說，由於進行全面搜尋的時間與發生事故的時間相距極短(最多
3 至 5 天；搜索時程最多 2 至 3 個月)，殘骸應多半散佈、裸露於水(海)底床
面上，被沈積物掩埋的機會不大，全程使用側掃聲納應可完成此項任務；
除非有特殊狀況發生(如：當地沈積速速率極快或底質鬆軟有沈陷的現象發
生)或是要搜尋已成歷史的失事遺骸，在以側掃聲納完成初掃瞄後若發現有
疑似的情況稍露於底床上，則此時需再以高解析的地層剖析儀(Sub-Bottom 
Profiler；見圖 2)或磁力儀(Magnetometer；適用於金屬質的殘骸)以適當間距
的”之字形”測線來對此疑似水域進行一次詳細的掃瞄作業；若為被掩埋的
殘骸，則其於地層剖面圖上會有一雙曲線(Hyperbolic；見圖 9)現象發生，
於曲線的頂端到底床間的距離為其埋藏深度；若殘骸為金屬質，則以磁力
儀去掃瞄後所得的磁力異常圖上會產生一個磁耦合(Couple；見圖 10)，而於
兩者中間即為殘骸所在的位置。經過以上的再確認後，若所得的訊息足以
進行後續的打撈作業，則移交相關資訊給相關團隊；倘若仍未能確認，則
需再以進行水下確認作業。 
由於大範圍初掃瞄所得的資料量極為龐大，倘若全以人工解讀的方式
來判釋，勢必將花費相當的人力及物力，且於時程上未必能符合現場的需
求；因此，初步的目標物自動判釋技術(Automatic Target Recognition；ATR)
的發展有其必要及重要性；此部分將另述於後。

A. Hausdorff Distance Transform： 
關係式如下： 
H (Aa , Bb ) = max { h ( Aa , Bb ) , h ( Bb , Aa ) } 
h ( Aa , Bb ) = max { min { d ( a , b ) } } 
h ( Bb , Aa ) = max { min { d ( b , a ) } } 
其中，Aa、Bb所指的是分屬兩不同集合的點。 
經過反覆不斷的計算後，我們可以找到兩集合間 HD 的關係，當
HD 值越小時，代表兩集合間的配對指數越高，越近似。而此法的缺點
是計算龐大且複雜，會耗費相當多的硬體資源。

期末報告(2/2): 側掃聲納模擬系統之研究 
 
摘要 
本報告探討側掃聲納訊號之模擬技術，藉由聲波在海水中之傳播將聲能傳
達到目標物，接著經由目標物的反射與散射，再將部分聲能，經由反向路經傳回
聲能接收器，並考慮拖魚本身的運動特性，最後在經由信號處理方式，轉換成聲
納圖像，達到側掃聲納訊號模擬的目的。模擬過程中，利用碎形原理產生海床，
其地形為一種隨機碎形地形，而利用雙站散射模型獲得目標物之反射訊號。本文
主要貢獻為結合聲音在水中傳播特性、反射模型與拖魚運動特性，產生側掃聲納
之模擬訊號。 
關鍵詞：側掃聲納，雙站反射模型，聲納圖像，碎形原理，模擬平台 
 
Abstract 
This work presents the development of a side-scan sonar images simulator by 
considering the image formation processes of a side-scan sonar acoustic model and 
properties of terrain. The simulator involved acoustic propagation through the water 
medium, seabed scattering and towfish motion characteristics. Synthetic images are 
obtained by applying the bistatic scattering model. The seabed is generated using the 
fractal theory and the terrain is represented as a random fractal terrain. The outcome 
of the simulator has two aspects: the synthetic side-scan sonar signal and synthetic 
side-scan images. The simulator can be used as a tool to generate simulated side-scan 
signals for underwater exploration tasks. 
Keywords:  side-scan sonar, bistatic scattering model, sonar image, fractal method, 
simulator.

音(echo)的信號，轉換成圖像，由於目標物之反射係數各異，其聲學反射強度不
同，聲納接收器接收到之聲學能量強度便不同，形成濃淡不一之聲學影像圖(如
圖 2.6)。聲波反射訊號越強，聲納影像灰階之濃淡程度越濃。聲波反射訊號的
強弱則與聲能在水中傳遞之損失、目標物之反射係數與目標物之粗糙度等有關。
當目標物反射訊號與海床背景反射訊號強度有明顯之差異時，即可根據聲納影像
上之訊號特徵，進行目標物判別。

5.3 拖魚運動之特性 
側掃聲納蒐集資料時之拖魚穩定性，影響聲學影像之扭曲變形。如同光學相
片，若於按下照像機快門之同時移動照像機，則拍攝之相片，會模糊不清。在資
料蒐集之同時，拖體穩定的移動，聲脈波將以穩定之間隔抵達海床面。配合脈波
寬，聲脈波可完整涵蓋海床，顯現海床地貌。拖魚之移動可分為水平移動的衝動 
(surge)、 側動(sway)、浮動(heave)及翻轉(pitch)、 扭轉(yaw)。其移動之動
力主要受載台之拖動及波浪影響。

6. 結語 
本報告探討側掃聲納訊號模擬方法，模擬程序考慮了成像過程之物理模
式，包含：基本聲學模式，海水之吸收模式、海床與物體之反射模式以及載具運
動的影響。所形成之影像與實際側掃聲納影像相似，因此本方法可以有效預估側
掃聲納影像，以做為水下實驗前置規劃之用。本文成果亦可應用於未來改善無人
水下載具之聲納影像自動判讀，以及提昇根據海床物體位置之水下載具自動導航
之能力。
