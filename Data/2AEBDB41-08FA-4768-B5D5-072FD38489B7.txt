2麴，再加入約1.2-1.5倍23 %鹽水混合成醬油醪(使食鹽含量為16-18%)，經過大約六
個月至一年的發酵後經由壓榨而得生醬油(raw soy sauce)以及醬油粕(shoyu mash
residue)。其中，生醬油經由調味、殺菌、包裝而製成醬油商品；目前醬油粕則主要
廉售做為飼料，然而隨著畜產業之精緻化，醬油粕當作飼料用途之比例減少。目前
日本已有直接丟棄或焚燒處理之趨勢(Yamamoto et al., 2006)，如此一來則會對環境
造成污染。
許多研究指出醬油粕中含有胺基酸及異黃酮(isoflavone)等有用之機能性成
分。本研究室於過去研究中(Yeh et al., 2005)亦發現脫鹽醬油粕含有大量之纖維(52.4
%)，其保油力(5.35 mL/g)、保水力(10.85 mL/g)、膨脹性(11.0 mL/g)以及陽離子交換
率(519 mequiv/Kg)皆較控制組(cellulose)佳。此結果顯示，脫鹽醬油粕具有供作纖維
素補充劑之良好物化性質。脫鹽醬油粕對葡萄糖之結合能力以及抑制α-澱粉酶活性
之效果亦證實其具有開發為減肥食品之潛力。脫鹽醬油粕同時含有高量之異黃酮
(7000μg/g)，其甲醇萃出物亦證實具有與抗氧化劑α-tocopherol 相當之抗氧化力。
綜合上述，發現醬油粕應具有回收利用的價值。然而，在製備脫鹽醬油粕的過程中，
所產生之水洗鹽液含有豐富胺基酸等營養性成分，若直接排放，將會造成環境污染。
因此，本研究目的為探討將此水洗鹽液回收再利用之可行性，使醬油粕之回收利用
更加完善。本研究擬利用此含胺基酸之水洗鹽液當作醬油釀造之鹽水來源，期望能
將此水洗鹽液回收利用，藉此提高醬油之產量及降低生產成本。
二、文獻整理
醬油是以脫脂大豆及小麥為原料，前者經灑水蒸煮後，與炒焙並磨碎之小麥混
合，接種製麴，經2-3 天後出麴，然後與1.2-1.5 倍(w/v)之23 %之食鹽水混合下缸，
經6-12 個月微生物發酵後壓榨而得生醬油，最後經調味、殺菌而成醬油產品。
製做醬油之主要成分大豆，其一般成分為：水分11.57-13.70 %、灰分3.89-4.44
%、粗脂肪13.38-19.05 %、碳水化合物17.08-23.54 %、粗纖維3.43-6.64 %及粗蛋白
39.46-42.20 % (續，1991)。現在製作醬油時多使用脫脂大豆片取代整粒大豆，脫脂
大豆片比整粒大豆蛋白質含量高，而且表面積大，故蛋白質易被水解，製麴時間縮
短，蛋白質利用率亦提高。大豆蛋白中主要是由11S 及7S 等蛋白質緊密地結合在
一起，因此酵素若要將這些蛋白質充分分解，大豆蛋白在製麴前就必須經過適當的
處理，使摺疊之蛋白結構打開，才能與酵素充分接觸並分解(Fukushima, 1968)，而
熱處理是最常用的一種方法。1955 年(Noda Shoyu Co., 1955)發現蒸煮後馬上冷卻至
低於40℃，可將蛋白質利用率提高至70-73 %；此即現在所稱的NK法(New Kikkoman
method)。Taira等(1965)則指出大豆蛋白之過度加熱對其中之離胺酸(lysine)、精胺酸
(arginine)及胱胺酸(cystine)會造成破壞，影響鹼性蛋白酶(alkaline protease)之分解效
果。
食鹽為醬油之重要原料，可防止腐敗，並為鹹味之來源。台灣醬油廠使用之食
鹽大部分是粗製食鹽。食鹽為防腐之重要成分，水量不準確，如濃度過低，醬油醪
會酸敗，製品即受影響產生異臭味；濃度若過高，防腐力加強，相關微生物如酵母
菌、乳酸菌之生長及各種酵素作用將受到抑制。食鹽之溶解法有冷水溶解法及加熱
溶解法，由於食鹽之溶解性受溫度影響甚小，幾乎全部工廠均採用冷水溶解法，如

（五）脫鹽醬油粕之物性分析
(a) 容積密度 ( bulk density )
參考Ralet et al. (1993)方法測定樣品之容積密度，精秤約1 g樣品，小心均勻放
入已校正過刻度之量筒中，加壓使樣品高度不再降低為止，記錄樣品體積，樣品之

(b) 吸水力 ( water-holding capacities )
參考並修飾Chau and Huang (2004)方法測定樣品之吸水力，精秤約1 g樣
品(W1)加入10 mL蒸餾水，在室溫(25 ℃)下充分攪拌30 分鐘，以2200 g離心30 分
鐘，分離上層未被樣品吸收之水加以秤重(W2)。樣品之吸水力以每克樣品可保持的
水量表示(g/g)。
計算：
吸水力 ( g/g ) = ( 10–W2 ) / W1
(c) 吸油力 ( oil-holding capacities )
參考並修飾Chau and Huang (2004)方法測定樣品之吸油力，精秤約1 g樣
品(W1)加入10 mL大豆沙拉油，在室溫(25℃)下充分攪拌30 分鐘，以2200 g離心30
分鐘，分離上層未被樣品吸收之油加以秤重(W2)。樣品之吸油力以每克樣品可保持
的油量表示(g/g)。
計算：
吸油力 ( g/g ) = ( 10–W2 ) / W1

參考Ralet et al. (1993)方法測定樣品之膨脹性，精秤約1 g樣品(W)，放置

品重可吸收的水量表示(mL/g)。
計算：
膨脹性 ( mL/g ) =V / W

四、結果與討論
（一） 醬油粕之一般成分及鹽分測定
醬油粕之一般成分及鹽分含量分析結果，醬油粕約含水分33.3 %、灰分11.2 %、
粗纖維30.5 %、粗脂肪5.1 %、粗蛋白11.6 %以及碳水化合物8.3 %，而鹽分含量約為
7.5 %。若以乾重計算，其粗纖維可佔約45.8 %(dry base)，顯示醬油粕可供作纖維之
提供來源。而醬油粕之鹽分高達7.5 %，若能完全回收，則每10 公斤之醬油粕可回
收約750 g之食鹽。
（二）脫鹽醬油粕之一般成分及鹽分測定
經過脫鹽過後之乾燥醬油粕之一般成分及鹽含量分析結果(表一)，脫鹽醬油粕
約含水分5.2 %、灰分2.1 %、粗纖維53.4 %、粗脂肪9.0 %、粗蛋白23.9 %以及碳水
化合物6.4 %，而脫鹽過後之醬油粕鹽分含量約為0.5 %。與Yeh et al. (2005)比較，其
脫鹽醬油粕約含水分6.2 %、灰分2.2 %、粗纖維52.4%、粗脂肪9.2 %、粗蛋白23.8 %
以及碳水化合物6.2 %，本實驗結果與文獻兩者間差異不大。結果顯示，經過脫鹽過

（四）利用醬油粕水洗後回收鹽水釀造醬油之經時變化
利用醬油粕水洗脫鹽後回收鹽水以及利用普通鹽水(對照)下缸釀造醬油成分之
經時變化其詳細結果如表三及表四所示。結果顯示，經過六個月後，利用醬油粕水
洗脫鹽後回收鹽水下缸釀造醬油之總氮可達1.94 %，高出Yokotsuka (1981)及
Fukushima (1989)指出良好品質濃口醬油之總氮1.5-1.8 %，且較利用普通鹽水下缸釀
造醬油之總氮(1.85 %)高。而利用醬油粕水洗脫鹽後回收鹽水組醬油之甲醛態氮可達
1.15 %，亦較利用普通自來水釀造醬油之甲醛態氮(1.05 %)高。氨態氮之結果顯示，
實驗組以及對照組之氨態氮於各時期均低於0.23 %，顯示兩者品質均屬優良；兩者
之各時期鹽分含量趨勢亦相近，後期之鹽分兩者皆約18.1-18.3 %。利用醬油粕水洗
脫鹽後回收鹽水下缸釀造醬油之胺基態氮於釀造六個月後可達0.94 %，較利用普通
自來水釀造醬油之胺基態氮(0.86 %)高。結果亦顯示，經過六個月後，利用醬油粕水
洗後回收鹽水組醬油之水解率可達59.3 %，較對照組醬油之水解率(56.7 %)高出2.6
%。而利用醬油粕水洗後回收鹽水組醬油之總氮利用率可達83.6 %，較對照組醬油
之總氮利用率(79.7 %)高出約4 %，此增加之4 %總氮利用率推測係由回收之鹽水所
得。本研究亦針對市售醬油及利用醬油粕水洗後回收鹽水組或對照組醬油之香氣成
分進行分析，結果顯示(圖一、圖二 及圖三)，本研究所釀得之醬油其香氣成分大致
相同。綜合以上結果證實以醬油粕水洗脫鹽後回收鹽水下缸醬油之可行性，回收鹽
水除了降低所需食鹽成本外，亦能增加醬油生產之產率及品質。
