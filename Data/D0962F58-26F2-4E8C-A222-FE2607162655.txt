研究摘要(500 字以內)： 
本產學合作目標為設計一「現地搜救機器人之教育及原型開發套件」，此
套件可用於搜救機器人教學之用，學習者所設計之機器人除可在災害發生時協
助搜救人員外，也可在承平時期協助工地作業人員施工。設計此機器人所需之
開發及模擬環境將採用 MSRDS (MicroSoft Robotics Developer Studio)，
MSRDS 為免費、通用的機器人開發環境，採用 MSRDS 將易於向世界各地的程式
設計師分享開發結果；而在機器人硬體零件方面，本研究將採用利基應用科技
之機器人硬體模組，利基應用科技為國內個人單板電腦(PSBC)廠商，其產品具
有模組化、易整合等優點。本計畫所研發之搜救機器人原型及模擬環境，可提
供往後相關研究參考，並有助於本國機器人產業在國際上之推廣，增加本國機
器人知名度。 
本教育套件主要分為「機器人硬體套件」與「模擬環境套件」兩大頪，「機
器人硬體套件」採用利基應用科技所生產之模組。「機器人硬體套件」部分，
本研究將 13 種不同模組合成「控制器、感測器套件」，並採用「輪型機器人模
組」及「六足機器人模組」組成「載具機器人套件」。「模擬環境套件」部分，
共設計三種不同的模擬環境模組，依學習順序由淺入深分別為「無家俱公寓環

服務及模擬環境模組，當模組開發完成後可直接提供 VPL 所需之服務；VPL 為
MSRDS 中控制機器人的程式語言，為三種程式語言中最接近學習者的語言，學
習者可直接使用較容易的 VPL，撰寫搜救機器人的控制程式，如此可避免接觸
到較為複雜的 C#服務。 
本研究團隊在與利基應用科技討論後，決定以 serial com port 透過藍芽
做為溝通介面，並由利基應用科技以 innoBasic 開發實體機器人控制程式，本
團隊以 C#開發 VPL 中所需之機器人控制程式。由於 VPL 較適用於初階的機器
人學習者，因此學習者可透過 VPL 撰寫簡易的機器人控制程式，本研究團隊所
撰寫的 C#機器人控制程式會將 VPL 程式轉為 serial com port 訊號，並透過
藍芽傳給實體機器人，實體機器人接收到訊號後，會根據利基應用科技的
innoBasic 機器人控制程式將控制訊號實作為機器人動作，例如：轉動伺服馬
達、接收偵測模組訊號等。 
本研究團隊依「輪型機器人模組」建置其在模擬環境中的虛擬機器人。本
輪型機器人模組包含動作輪組、循跡感測器、可動式測距模組，機器人可藉由
上述感測器在模擬環境中擷取資料及移動。 
「模擬環境套件」共設計三種不同的模擬環境，依學習順序由淺入深分別
為「無家俱公寓環境」、「一般公寓環境」與「地震後公寓環境」，以下將分別
介紹這三種模擬環境。 
「無家俱公寓環境」為最簡單的模擬環境，為了降低初學者的學習門檻，
因此移除了公寓中所有家俱，初學者可在此模擬環境中練習撰寫基本的機器人
控制程式，公寓環境及機器人所得感測資訊。模擬環境中主要視窗為概視圖，
當機器人在模擬環境中移動時，學習者可藉由此畫面明瞭機器人現況；右方視
窗上方為機器人在模擬環境中所得影像，下方為測距模組所得資料。由於所得
影像為機器人的視角，因此可視範圍較小；而測距模組量測範圍是機器人前方
180 度，所以量測角度比影像視角寬廣，測距資料圖中顏色愈深代表物品離機
器人愈近。 
「一般公寓環境」中包含日常家俱。當學習者已熟悉基本的機器人控制，
便可進一步練習如何在有簡單障礙物的環境中移動。本模擬環境包含之家俱
有：電視機、沙發、餐桌椅、床、書櫃等。 
「地震後公寓環境」模擬地震後家俱傾倒的情況。由於傾倒的家俱會阻礙
機器人行進路線，因此學習者可藉由本模擬環境研發自己的搜救機器人演算

技術說明
(中文) 本計畫擬設計一套PC-based教學用之「現地搜救機器人之教育及原型開發套件」，
其兼顧電腦運算能力與機器人機構之變化彈性。在搜救機器人硬體機構設計部分，
將採用利基應用科技之模組及套件，利基應用科技的產品在硬體設計方面相當具
有彈性，可讓學習者自由組裝各類型搜救機器人之構件。在搜救機器人軟體設計
部分，將採用Microsoft Robotics Developer Studio (MSRDS)，MSRDS之視覺化
程式設計語言(Visual Programing Language)及視覺化模擬環境(Visual 
Simulation Environment)，可讓學習者易於學習程式語言，並在模擬環境中測
試自己設計的搜救機器人。
(英文) This project would develop an "in-situ rescue robot module for education and prototype 
developing". The project would use innoVati as hardware and Microsoft Robotics 
Developer Studio (MSRDS) as software. The visual programing language and visual 
simulation environment can help student developing their own rescue robot and testing 
their robot in the simulation environment.
