研究摘要 
針對微細切削加工與傳統大尺寸切削之特性差異、所面臨的挑戰、以及理論模型建立與模
擬方法都不盡相同，在傳統大尺寸中刀具監測常用的方式在微細切削中，因微細刀具磨耗造成
之訊號變異較傳統刀具切削相對微弱，所取得訊號抗雜訊能力更差，造成切削力與功率等變化
之量測更為困難。因此針對微細切削刀具細小特性，探討微細刀具狀態的發展方向。 
本計畫設備使用 CIZITEN 的 Cincom R04 走心式車床加工機，訊號擷取裝置為 Kistler Type 
8152B1 的 AE SENSOR，擷取介面採研華公司訊號擷取卡 PCI-1714-UL，並使用 LabVIEW 開發
人機介面。本計畫對此加工機所使用微細鑽孔刀具，針對其微細切削刀具細小特性，開發刀具
狀態監測系統達到提高產品良率的目標。在刀具狀態監測系統中，首先找出刀具斷裂時產生 AE
訊號的振幅，以此為參考，設定加工中 AE 訊號的上限值，當 AE 訊號達上限值時系統發出警報
通知操作人員處理。以本計畫中所使用的刀具狀態監測系統在工廠中使用可以減少因刀具損壞
造成的產品不良率，並且減少人力資源在檢驗產品區分良與不良品。 
關鍵字：線上刀具狀態監測、微細鑽削、工具機、聲射  
Abstract 
For micro-machining and cutting characteristics of the traditional large size difference, the 
challenges faced, and theoretical modeling and simulation methods are different, the large size of the 
traditional tools used to monitor the way in micro cutting, the tool wear caused by fine The signal 
variation is relatively weak compared with conventional cutting tool, made even worse signal noise 
immunity, resulting in changes of cutting force and power and so the measurement more difficult. 
Therefore, micro-cutting tools for small features, explore the direction of development of micro-tool 
Monitoring. This project device CIZITEN Cincom R04 the Swiss type lathe, signal acquisition device 
for Kistler Type 8152B1 of AE SENSOR, capture interface mining Advantech signal acquisition card 
PCI-1714-UL, and use the LabVIEW development interface. 
This project used this machine for micro drilling tool, small cutting tools for its fine features, the 
development tool monitoring system to improve product yield targets. In tool monitoring system, first 
identify tool fracture generated AE signal amplitude, as a reference, set AE signal processing in the 
upper limit, when the AE signal when the upper level system alerts the operator to inform treatment. 
To this project, used in tool monitoring system used in the factory can reduce the damage caused by 
tool product defect rate, and reduce the human resources in testing products distinguish good and bad 
products. 
Keywords: Online Tool Condition Monitoring, Micro Drilling, Machine Tools, Acoustic Emission 
人才培育成果說明： 
本計畫除一位兼任助理參與實驗規劃外，另有 2位研究生臨時助理參與實驗及資料處理工作。 
兼任助理工作項目：實驗規劃、實驗設備校驗與架設、夾置具設計與製作、工廠聯絡、實驗資
且加上自動停機功能，使操作者適時更換刀具使生產線運作正常，可以確保加工品質與系統的
可靠度。發展一套有效且可靠的刀具狀態監控系統以取代人力又能提高產品良率，為本研究技
術特點。 
微細刀具跟一般傳統刀具的監測上有許多的不同，因為微細加工的技術上有許多的挑戰，
包含刀具強度低與選擇性少、主軸高轉速需求、設備高精度穩定度需求、刀具狀態與加工品質
監測困難以及工件夾持困難等問題。因此在一般尺寸上使用的監測方式，未必能在微細加工的
監測上使用。微細切削由於刀具與切削深度等尺寸相當微小，刀具在切削過程容易有大的變形，
實驗的設計執行與切削狀態及品質的觀察量測也屬不易。 
以刀具與量測技術之觀點來看，ㄧ般微細刀具壽命短損壞不易被察覺，狀態控制相較傳統
刀具更為不易。因此發展線上即時刀具狀態偵測系統在微細切削加工中較傳統切削系統難度更
高但也更為重要。 
刀具狀態偵測系統包含刀具系統、感應器、機電整合與訊號處理等單元，偵測訊號與機台
刀具結構、製程參數、周遭環境與感應器安裝息息相關，例如以振動訊號變異為監控訊號，因
刀具系統結構、材料與製程參數之差異，造成振動量變異的水準與形式也隨著改變，加上環境
雜訊，感應器規格與訊號處理方式的選擇對偵測品質均有相當大的影響；另外，刀具型式的不
同，也常造成感應器型式與安裝方式不少的差異；在偵測系統實際應用上，系統運作成功與否
與製程系統特性息息相關，此類系統一般並非工具機之標準模組，必須對特定製程設計出最佳
之即時偵測系統，因此要從過去使用在傳統刀具的監測方法中選出可以適用在微細切削的監控
上。 
本研究另一方面的目的除了在微米等級的微細刀具的監測並且加上改變加工參數田口法實
驗設計縮短目前的加工時間。選定在加工現場中能夠修改並且最能影響加工時間的參數為轉速
與進給，並且考量到現場修改參數的可能性，因此選用轉速與進給來最為參數設計的因子，在
每個因子給予三水準，最後找出在生產週期與刀具損耗中最佳平衡。進行加工參數田口法設計，
規劃新的加工參數將目前的生產速度提高1.2~2.8倍，進行實驗找出生產速度與刀具損耗的最佳
平衡點。
