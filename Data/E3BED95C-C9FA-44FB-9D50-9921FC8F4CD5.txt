中文摘要 
鉻鉬合金鋼具有優良的製造特性，廣泛使用在航太工業、核能電廠及國防科
技。SAE 4130 製作大型精密有縫高壓容器時，須經由兩種銲接製程，分別為直
縫銲接( AWST ; Annealing+Welding+Solution +Tempering )與圓周銲接( ASTWR ; 
Annealing+Solution+Tempering +Welding +Stress Relief ).。然而，此大型精密有縫
高壓容器大多使用於高溫環境，服役時易受高溫及高壓影響。為此，本研究為模
擬 SAE 4130 兩種銲接製程於高溫環境的顯微組織、機械性質及高溫腐蝕影響，
藉由得到結果做為結構設計的參考。 
研究結果顯示：常溫衝擊時，衝擊韌性依序為 ASTWR >AWST >AST，與
其缺口處的顯微組織有關；於高溫衝擊時，易受析出物影響造成機械性質下降。
室溫至 600℃之試驗溫度範圍，抗拉強度皆為 AST >AWST > ASTWR，此與試驗
前銲接製程有關。然而，高溫拉伸性質易受氧化影響，Si 的析出顆粒大小對不同
試驗有不同影響。比較AWST與ASTWR兩銲接製程於實驗溫度範圍內，ASTWR
的衝擊值均為最高，顯示可承受較大的單位時間-應力作用(即瞬間應力)；拉伸試
驗結果顯示，AWST 銲後調質處理製程，可承受較較高的穩定應力作用。 
模擬在熱段區的結構組件遭受到高溫環境侵蝕時，SAE 4130 的兩種銲接製
程所承受的抗熱腐蝕能力評估。試驗前，先將銲件披覆約 2 mg/cm2 NaCl，再置
於管狀熱處理爐進行高溫熱腐蝕行為的測試。除探討兩種銲件的熱腐蝕行為外，
同時針對母材、熱影響區及銲道分別進行測試與顯微組織觀察。實驗結果顯示披
覆 NaCl 的試件明顯比未披覆者腐蝕嚴重。而且，因金屬氯化物與氯氣的催化作
用，使腐蝕皮膜呈現厚和多層的結構，並有向母材內侵蝕的現象。AWST 及
ASTWR 兩種銲件均以銲道腐蝕最為嚴重，研判是因為銲接的高溫熱循環作用，
導致在晶界形成貧鉻敏化的偏析反應，使銲道的晶粒間易產生腐蝕作用。由於，
ASTWR 的銲後應消溫度僅 550 ℃，導致鉻元素在銲道的分布較 AWST 者不均
勻，因此更易產生貧鉻敏化反應，致使 ASTWR 銲件比 AWST 銲件熱腐蝕的更
為嚴重。

2.1 構造用鋼[6] 
 構造用鋼之含碳量較低 (約 0.2 % C
以下) 屬低碳鋼，可分為普通構造用鋼與
機械構造用鋼。普通用鋼多用於建築、橋
梁、船舶、車輛和其它一般構件上。由於
機械構造用鋼用之機械性質皆高於普通用
鋼，使用於製造機械零件或重要結構物件
等。 
 以往為增加鋼材銲接性、成形性、抗
腐蝕性及機械性質及降低成本等，藉由添
加少量合金元素以改善奇等性質。然而，
高強度低合金鋼[2]為低碳鋼之一種，含碳
量偏低( 0.05 % ~ 0.25 % C )，且有較高的
含錳量 (＜2.0 %)，藉由不同元素的添加
(如 Cr、Ni、Mo、N、V、Nb、Zr 等元素)，
以增加其抗蝕性、銲接性、成形性與機械
性質( Y.S.＜275 MPa)。相較一般低碳鋼，
高強度低合金鋼有較優異的機械性質。依
使用目的與性質區分，可分為熱處理型及
非熱處理型兩大類。 
1. 非熱處理型 
 此種鋼材用於橋梁、道路、船舶等大
型工件，由於不易施以熱處理，為此希望
製造完後即具備一定強度，一般則在出廠
前施正常化熱處理，且製造物件時，板件
與板件接合需施以銲接。然而，為避免銲
接後的熱脆裂影響，選擇含碳量低於
0.17% 鋼，強度的增加則藉由添加特殊元
素於肥粒鐵基地內，形成固溶強化效果。 
2. 熱處理型 
 利用熱處理方式產生相變化，一般需
經兩次熱處理，分別為固溶熱處理 
( Solution ； S ) 與 回 火 熱 處 理 
( Tempering；T ) 合稱調質熱處理。行固
溶熱處理時，須將溫度提升至 A3溫度以上
的沃斯田鐵單一相後，後藉由快速冷卻通
過 TTT 曲線鼻部，形成麻田散鐵組織，雖
麻田散鐵硬度較高，但韌性及延展性極
低，且易產生脆性。為此需進行回火熱處
理，施以低於 A1溫度之熱處理，將麻田散
鐵相轉變成回火麻田散鐵相，以增加材料
韌性及延展性。 
 然而，熱處理型合金鋼須具備質量效
應低、硬化能高與對回火軟化抗性佳等特
性。質量效應低可減少材料因受不同厚度
大小所產生之表裡硬度不同程度；硬化能
高即鋼淬火時較易得到麻田散鐵；回火軟
化抗性佳可減少回火處理後硬度的降低。
然此三者皆可經添加特殊合金元素，改善
其性質。

2.5.1 全退火 
完全退火 ( Full Annealing ) 將溫度
升至 Ac3或 Ac1溫度以上，持溫一段時間，
使顯為組織變成沃斯田鐵單一相後，依造
在於爐中冷卻 ( 緩慢冷卻 ) 。在這過程
中，亞共析鋼將會從 γ（沃斯田鐵）→ P 
（波來鐵）+ α（肥粒鐵），可使材料強度
下降，亦可得到較微細晶粒，以便改善切
削性或塑性加工性。 
2.5.2 調質熱處理 
調質熱處理可分為： ｢固溶+淬火+回
火｣三種步驟。固溶熱處理  ( Solution 
Treatment ) 為升溫至Ac3溫度以上，持溫一
段時間後，將材料加熱至單一相；淬火處理
則為急速冷卻方式，如油淬或水淬等，形成
麻田散鐵相。由於材料內碳原子來不及擴
散，導致碳插入晶格內部，使原來形成 γ 
( FCC ) → m ( BCT )，雖前後原子相對位置
不變，但體積變大，造成內部殘留應力的形
成，此時麻田散鐵組織硬且脆。 
為增加材料韌性與延性，則施以回火
熱處理( Tempering ) ，低於 A1溫度之熱處
理溫度。由於回火熱處理微藉由提高溫
度，使差排移動，亦降低材料內部殘留應
力，並未產生相變化。由圖 2.3 所示[22]。
經回火過後，原麻田散鐵相轉變成回火麻
田散鐵相，此相雖強度與硬度些微下降，
但其韌性與延性大幅提升。

3. 施銲接速度： 
當銲接速度越快，雖可減少熱循環對
銲道 ( Fusion Zone; FZ ) 兩側的母材形成
熱影響區的影響，亦可減小熱影響區的範
圍，但亦使銲道塌陷，降低銲後機械性質
與銲道中心裂紋等現象；反之，若銲接速
度慢時，導致熱影響區範圍擴大。 
4. 熱輸入量： 
當銲接速度為等速，給予熱入量越
大，則熔透量越高。依使用需求不同，藉
由關係式為 Q=VI/S，可獲得相同熱輸入
量。 
Q：熱輸入量 ( J/mm ) 
V：電壓 ( V ) 
I：電流 ( A ) 
S：銲接速度 ( mm/min )。 
 
依銲前與銲後熱處理區分為：預熱處
理、續熱處理及銲後熱處理三種： 
1. 預熱處理 ( Pre-heating )  
減少銲接時材料表面所吸附的水
氣，藉以降低氫脆的可能性。此外，預熱
處理可使銲接後，銲道不易因冷卻速率過
快，導致熱影響區形成麻田散鐵相，造成
材料冷裂[28]。 
2. 續熱處理 ( Inter-pass heating ) 
銲接時，同時並持續加熱工件，藉以
紓緩殘留應力與軟化工件。 
3. 銲後熱處理 ( Post-weld Heat Treatment；
PWHT )  
即藉由回火熱處理，減少銲道與熱影
響區殘留應力，以增加銲後工件機械性
質。此為銲後必要處理程序。 
 
2.7 合金鋼的銲接 
2.7.1 銲道 
該區域為工件與工件熔融接合處，因
熱輸入量集中於此處，以至快速冷卻，易
形成麻田散鐵相，造成殘留應力於此處。
一般在銲後，須盡快施以回火熱處理，以
免銲道因殘留應力過大而脆斷。 
2.7.2 熱影響區 
由於銲接過程中，電弧所給予的熱輸
入，導致靠近銲道兩側遭受僅次於熔點溫
度熱循環影響，亦呈現不同的金相組織，
而此處稱之為熱影響區。一般銲件熱影響
區機械性質較低，主要為其易產生硬且脆
的顯微組織，如麻田散鐵 （Martensite）、
費德曼肥粒鐵 （Widmanstatten Ferrite）及
粗波來鐵（Coarse Pearlite）等。因此，當
被要求銲後工件品質時，熱影響區的寬度
越小越好。 
由 Ohmori, Y.等[29]研究，回火麻田散
鐵較回火變韌鐵更易產生 350℃回火脆
性，其結果顯示回火變韌與回火麻田散鐵
所組成之雙相組織有較佳的機械性質。因
此，若在熱影響區內形成回火變韌與回火
麻田散雙相組織，可增加其機械性質，亦
可降低其回火脆性。

3.3 小結 
1. AW 銲件之銲道硬度值均較 HAZ 及
BM 為高；HAZ 與 BM 間則無明顯起
伏。其係因 BM 組織為硬度不高的波
來鐵與肥粒鐵，無法與 HAZ 有明顯區
隔。銲道內之各銲道界面間有硬度遽降
現象，主要為受到多層次銲接之熱輸入
影響，使晶粒迅速成長粗大。 
2. AW 銲件之 LW 與 R3W 側 GGHAZ，
因鄰近銲接熔池受到略低於熔點之高
溫，使其晶粒成長粗大，造成硬度值劣
化之現象。而在 LW 側的 RZHAZ 因受
到高溫度而為沃斯田鐵相，隨之受到較

第六章 結論 
 
1. AW 銲件之最後一道與倒數第三道側
晶粒成長熱影響區，因鄰近銲接熔池受
到略低於熔點之高溫，使其晶粒成長粗
大，造成硬度值劣化之現象。而在最後
銲道側的再結晶熱影響區因受到高於
A3 溫度而為沃斯田鐵相，隨之受到較
快的冷卻速率影響，而形成脆硬之麻田
散鐵組織，此結構會使該區域機械性質
下降，不利於工程上之運用。此現象可
藉由銲後熱處理改善其機械性質，便於
工程上之應用。 
2. 歷經同樣的銲接與熱處理製程，
ASTWR 與 ASTW 比較，銲後應力消
除對於拉伸機械性質不具有明顯效
應。但是，對於破壞韌性而言，在
ASTWR 熱影響區有非常正面的提昇
效果。 
3. 比較 AST 與 ASTWR 之拉伸與衝擊試
驗結果，可推論：常溫環境下，穩定的
應力作用下，熱影響區先發生破壞的機
率較母材為高。然而，發生瞬間應力作
用時，母材發生先破壞的機率反而較熱
影響區為高。 
4. 比較三種試件，ASTWR 於實驗溫度範
圍內均有較高的衝擊韌性質。故
ASTWR 在高溫下能承受較高的單位
時間應力作用。 
5. AWST 與 ASTWR 400℃之拉伸破壞組
織，有明顯的片狀類熔融組織摻雜。而
且，當片狀類熔融組織數量越多，抗拉
強度有下降的趨勢。 
6. 若精密有縫高壓容器在 400℃穩定應
力作用下，依研究結果推論可得：工件
在圓周銲道(ASTWR)附近優先發生破
壞的機率較高。 
7. AWST 銲後經超過 A1 溫度的固溶處
理，使鉻元素重溶回基地相且分佈較為
均勻。由於，減少碳化鉻在晶界析出，
減少了晶粒間腐蝕作用，因此耐蝕性
為：AWST＞ASTWR。 
8. 由微硬度分佈曲線可看出，AWST 銲件
產生腐蝕破斷的位置應為熱影響區靠
近銲道處；ASTWR 則為銲道，其原因
係原均勻分佈在鋼中的鉻元素，因為銲
接的高熱影響使在晶界產生碳化鉻的
偏析反應，此時晶粒間因貧鉻效應較強
而易產生腐蝕作用。 
9. 由 EDX 及 XRD 分析顯示：兩種銲接
製程的腐蝕皮膜從最外側往最內側依
序為 Fe2O3、Fe3O4、FeCr2O4及 FeO。
然而，ASTWR 因貧鉻效應導致鉻貧乏
而無法產生具保護性之 FeCr2O4，使之
ASTWR 腐蝕速率大於 AWST。
