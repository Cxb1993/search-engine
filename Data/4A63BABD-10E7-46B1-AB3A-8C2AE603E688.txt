end-users can efficiently set the related parameters

技術說明
(中文) 我國資訊工業技術蓬勃發展之際，各型產業機械搭配PC-Based控制器建構出智慧
化的功能已成為精密機械的發展潮流。近年來國內外各控制器大廠，紛紛投入開
發高附加價值產品於PC-Based控制器上，其中以輔助加工系統為主要功能，由此
系統可藉以提升控制器的附加價值進而提高工具機的市場競爭力與產品差異性。
因此本計畫將以智研科技有限公司的PC-Based控制器M450S為基礎搭配三軸CNC工
具機建構一套教導式PC-Based三軸銑削加工系統，主要目的在於讓從未接觸NC程
式撰寫的使用者，能透過「教導式」的方式有效率的設定刀具參數(刀把類型、
刀具幾何、加工刀具庫等)與加工參數(素材尺寸、加工形狀的選擇等)以及提供
刀具對不同銑削材料的銑削性能參數。最後利用圖控元件與步驟式的設定方式，
讓使用者能輕易且明確的了解銑削加工參數的功用，並且快速產生一組具備最佳
化的NC加工程式，如此將可減少程式編寫的時間，且「教導式」的設定具備查核
功能可降低人為編寫NC程式上的疏失。
(英文) This plan is based on the INTEK Technology Company Limited’s PC-Based controller, a 
model M450S. The controller is combined with the 3-axis CNC machine tool to build a 
set of work system of the instructional 3-axis milling. The destination is for beginners 
whom are capable of using the way of “ Instructional ” to set the tool parameter (cutter’s 
type, cutter’s geometry, cutter room) and the work parameter (workpiece’s size and

system so that the operator compiling the NC program can avoid the human errors.
