 II 
一、計畫摘要 
在第一年計畫中，針對非線性時變系統提出一創新型的以卡曼濾波器為基礎(Kalman 
filter-based)的適應性觀測器。利用卡曼濾波器的高增益(high gain)特性，可以應用於具大變化
的參數的最佳化估測，然後再提出一致動器錯誤偵測(fault detection)的方法，利用所估測出的
致動器錯誤(actuator faults)，提出一補償致動器的法則以修復致動器的錯誤。此外，吾人更引
入線性最佳化法(optimal linearization)，針對非線性系統，求得在每一取樣點的線性化模型，
因此上述所提出的錯誤偵測與致動器補償及修復性能可成功地應用於非線性時變取樣系統。
並且在此計畫中，更提出數位重新設計的方法，以設計等效類比控制器之數位控制器。 
在第二年計畫中，提出一個針對非線性隨機混合系統的嶄新狀態空間自調式控制方法。
在這裡我們試圖在狀態空間自調式控制中利用一個可以調整的 NARMAX (nonlinear 
autoregressive moving average with exogenous inputs)模型來建構出非線性隨機系統模型，並透
過最佳線性化法則來對該非線性隨機系統模型提出一個相符合的適應性數位控制理論，而這
個非線性隨機系統的系統參數未知、狀態不可得知、不可量測的隨機雜訊及非隨機雜訊等。
在這個計畫中所提出的方法使得對於非線性隨機混合系統的進階數位控制演算法更具發展
性。 
在第三年計畫中，提出當系統或輸入有局部錯誤產生時，嘗試加入一種新的容錯控制，
它是修改傳統的狀態空間自調式控制法則；藉著比較在卡曼濾波器估測演算法中的誤差值，
一種量化的準則被發展出來：權重矩陣重新設定(reset)技術，它是藉著調整和重新設定在卡曼
濾波器估測演算法中用以估測參數的協方差矩陣（covariance matrices），這方法可用於改善系
統回復的參數估測，並且有效地處理局部突發式或逐步式錯誤的系統錯誤、以及突發式或逐
步式的輸入錯誤。因此，在此次的計畫中，嘗試著使用這個修改過的自調式容錯控制方法，
加入前次計畫中基於 NARMAX 模型的狀態空間自調式控制方法，並使用於未知的多變數非
線性隨機系統，以提高控制系统的可靠度。 
 
關鍵詞：錯誤偵測、卡曼濾波器、適應性控制器、數位重新設計、狀態空間自調式控制、
NARMAX模型、非線性控制系統、最佳線性化法則、主動容錯控制

二、緣由與目的 
 
第一年研究背景及目的 
線性系統的適應性觀察器自 1970年之後以開始被研究，但多數限制於單變數系統且系統
的參數是常數，使得這樣的觀察器在應用到多變數時變系統時會遭遇到數值分析以及在穩定
度分析的困難[1,2]。在本計畫中提出一嶄新的以卡曼濾波器為基礎的適應性數位觀察器，主
要是參考在[3]當中所提出的類比式的自適應觀察器設計法，以其類比設計法為基礎，吾人利
用數位重新設計的方法，數位化此類比式的自適應觀察器，並導出一創新並適合運用於資料
取樣系統以卡曼濾波器為基礎的適應性數位觀察器。 
對於一控制系統，基本上，所設計的控制器會使得系統的輸出可以追蹤所給定的參考輸
入軌跡，但假如錯誤一但發生在致動器或控制器或狀態也產生錯誤，則會影響輸出的響應並
使得性能也會變差[4,5]，因此在本計畫中，以上述的卡曼濾波器為基底的數位適應性觀察器
來偵測致動器的誤差，提出利用主動式的錯誤診斷與偵錯方法，應用在被動式的容錯控制使
其能應用在非線性時變的取樣系統，並設定控制輸入會失去一些百分比的動力作用在受控系
統上。 
此外，由於現今數位技術與微電腦的快速發展，使得設計成本可以降低並且獲得更好的
可靠度以及在設計上更有彈性等優點，在現今的控制系統設計上越來越傾向設計數位的控制
器以取代傳統的類比控制器[6]，因此在本計畫中也提出數位重新設計的方法來建構有關容錯
系統的數位控制法。 
在本計畫所提出的致動器偵測與性能修復的方法，不僅能用於線性時變系統，更能推廣至
非線性時變系統中。針對非線性系統的容錯控制，利用線性最佳化的方法獲得的局部線性最
佳化模型在工作點狀態，因此在此線性的模型下，便可依上述提出的容錯控制法應用在非線
性的系統中。  
 
第二年研究背景及目的 
非線性自回歸滑動平均模型(Nonlinear Autoregressive Moving Average with eXogenouos 
inputs model)是在1981年的時候由[7]所提出，並提供了一個一致的表示式來表示許多種類的
非線性隨機模型，然後在[8]之中的研究使得NARMAX模型不再受限於多項式的表示法，而且
可以做成一個合理的模型，合理的模型可以利用為數不多的參數來將嚴密的非線性系統特性
給表示出來，其相關的結果由[9]提出；當延伸到隨機的例子時，這些模型提供了一種可以用
來在參數估測演算法中當基底的合理模型。而狀態空間自調式控制理論是由[10]以及[11]所提
出的，然後[12]證明這個理論可用於設計線性多變數隨機模型的進階合適控制器，在[10]和[11]
的方法之中，標準的卡曼狀態估測演算法[13]可被用於線上的參數估測演算法。 
系統識別(system identification)是利用所獲得的輸入與輸出數據以來發展一套計算程序，
並找出一個最能符合系統特性的數學模式，而這數學模式可以有效地用來分析、預測以及控
制此物理系統。而非線性輸入自回歸滑動平均NARMAX(nonlinear autoregressive moving 
average with exogenous inputs)模型在系統辨識的應用之中將是一個重要的工具，因為它不僅
結構簡單、收斂速度快，而且重要的是還具有辨識準確度高的優點。 
在有關 NARMAX 模型的選擇方面，本計畫選擇合適的模型來使用，至於系統的參數以
及狀態的估測，則是經由系統的輸入和輸出資料傳遞至上述的 NARMAX 模型來進行估測而
得，當估測的模型是正確的或是與系統本身非常相近時，再經過所設計的控制器輸入，可以
讓所估測到的系統參數收斂到一個範圍內，並且可以得到一個與真實系統的輸入與輸出非常
相似的結果。

It is clear that the system could contain large uncertainties when the failure dynamics are large. 
Under this situation, the controller has to take an appropriate control action for the uncertainties 
occurring at any time instant vt . This is an adaptive control problem, in which controller 
parameters are adjusted according to the system parameter estimates. In this paper, the method 
based on the modified STC scheme is proposed to accomplish the fault-tolerant control. 
Three assumptions required for the proposed fault-tolerant control are addressed as follows: 
Assumption 1: The system is controllable and observable even if faults occur. 
Assumption 2: The system is persistently excited by control inputs. 
Assumption 3: Before the fault occurs, the system is healthy or has fully recovered from the 
previous fault. 
 
3.2 Modified self-tuning control with active fault tolerance 
The STC scheme should be modified to cope with parameter variations due to system faults. 
When a partial fault occurs, the system parameters vary accordingly. The time-varying parameter 
estimates obtained via the RELS algorithm in the conventional STC scheme would give large 
parameter errors and lead to poor system performances. However, based on the Kalman filter 
interpretation algorithm of RELS method [17], a modified scheme is proposed to estimate 
parameter variations. The modified state-space STC scheme can be applied to multivariable 
stochastic faulty systems without requiring prior knowledge of system parameters and noise 
properties. 
In short, in the beginning, a healthy but unknown system is well tuned by the conventional 
STC scheme, and then the self-tuning structure with the reset covariance matrix of the parameter 
estimate is modified to enhance the parameter estimation and output response when the system 
states and/or inputs are partially faulty [14]. It postulates that the parameter vector is time-varying, 
i.e., it is modeled as a random walk or a drift:

本計畫在計畫提案內容之相符程度、達成預期目標情況等方面，皆達成預期之目標。其
中，第一年計畫以卡曼濾波器為基於的數位適應性觀察器來偵測致動器的誤差，提出利用主
動式的錯誤診斷與偵錯方法應用在被動式的容錯控制，使其能應用在非線性時變的取樣系
統，並設定控制輸入會失去一些百分比的動力作用在受控系統上。在當時計畫研究中，尚無
相關論文提出可用於線性時變系統更能推廣至非線性時變系統中的致動器偵測與性能修復的
方法，可見其學術創新性與實用性。再者，根據目前已知的文獻，有關 NARMAX 模型的研
究，大部分都是使用在非線性單輸入單輸出的系統，而第二年計畫所提出的是可以使用在多
輸入多輸出的非線性隨機系統的 NARMAX 模型，並且可以將此 NARMAX 模型套用在創新
型態的狀態空間自調式控制器上面。如此一來，只要能夠知道系統的輸入數與輸出個數，就
可以立即對該多輸入多輸出的未知非線性隨機系統來建立 NARMAX 模型，並且進行估測的
動作，使得我們能夠快速且有效地知道該系統特性參數，並追蹤到我們對該於未知系統所設
定的參考輸入。最後，根據之前所推導出來的創新 NARMAX 模型的狀態空間自調式控制，
本計畫再加入了修改後的自調式容錯控制。這樣一來，對於未知的多輸入多輸出非線性隨機
系統，除了能夠有效率地辨識出它的系統特性，並且因為修改後的演算法中使用了重設的方
法以進行容錯控制，所以可以快速地克服不可預知的系統故障或是輸入故障，使得可以更準
確地追蹤到對於未知的多輸入多輸出非線性隨機系統所設定的參考輸入，並且不需要去擔心
因為系統或輸入端所產生的錯誤所造成的系統不穩定或錯誤，使得系統在局部故障產生時仍
然能夠有效地正常運作，進而提升控制系統的可靠性。經過以上這三年的計畫研究，主持人

II. WHAT MAKES A GOOD REGUATOR 
－ Jie Huang, Chinese University of Hong Kong 
 
能解決輸出調整問題的控制器被定義為調整器，一個調整器能夠容許不確定
之參數則被定義為好的調整器。就線性系統而言，強健輸出調整問題在 1970年
徹底地被探討，此研究導至著名的內部模型準則。一個好的調整器必須含有完善
定義的動態系統──內部模型，而內部模型是經由外部的干擾與參考輸入所決
定。而 Wonham以每一個好的調整器必須含有外部模型來總結內部模型原理。
然而，以非線性系統而言，調整器含有外部模型是不足夠的。 
本演講介紹穩態發電機的概念與目前發電機內部模型的特性描述，則內部模
型能看成穩態發電機的漸近線估測。若在非線性系統，如果控制器要能夠解決強
健輸出調整問題，則控制器必須含有一個穩態發電機。因此，以線性系統而言，
穩態發電機是經由外部系統決定，而以非線性統而言，則是經由外部系統與特定
設備兩者所決定因素。故通常每一個好的調整器必須含有外部與內部的穩態模型
來概括標準的內部模型準則。

 A5 
新式的改良型學習控制機制，以設計非線性系統的最佳學習追蹤器。即使沒有非
線性系統詳細的系統方程式資訊，本文提出的機制仍然能夠設計出最佳學習追蹤
器。為解決這非線性系統的最佳重複學習追蹤控制問題，首先，利用最佳線性化
的方法來找出非線性系統的精確線性模型。其次，建構線性二次學習機制以設計
最佳學習追蹤控制器。然後，運用極點重置的技巧來改善學習機制其學習過程的
效能。最後，使用設計出的改良式學習追蹤器以控制非線性系統。利用最佳線性
化方法在操作狀態找出精確線性模型後，即可將預先使用在線性系統的學習機制
擴展到非線性系統。而線性化模型的特徵值如何影響最佳重複學習機制的性能，
也在本文中被探討，並提出用極點重置的方法以改善學習機械裝置的一些缺點。
最後舉例來證明本文所提出方法的有效性。

標。最後舉一個實際的例子來描述所提出方法之效能。

適用於取樣線性時變系統以預測型卡爾曼濾波器為基礎的錯誤估測與控制 
Tsai, J. S. H., Wei, C. L., Guo, S. M., Shieh, L. S., July 4-7, 2010, “Predictive 
Kalman filter-based fault estimator and control for sampled-data linear time-varying 
systems,” IEEE International Symposium on Industrial Electronics (ISIE-2010), 
Bari, Italy, pp. 292-297. 
 
    此篇論文，乃是針對之取樣線性緩慢時變系統，此系統對於致動器與狀態
中，含有無法預測衰減因子的錯誤估測問題，提出一具有全狀態空間適應性為基
礎之錯誤診斷的觀測器與高增益的控制器；而所提出之改善型卡爾曼濾波器的適
應性觀測器，其錯誤估測效果與性能恢復都比起傳統的觀測器獲得較好的估測；
同時也提出留數產生架構與自動切換機制的方法，達成高性能錯誤估測與診斷。
實體應用上，利用具有高增益的數位再設計的方法論，此方法的優點，不具消耗
的損失下，可利用低增益數位的觀測器與追蹤器等效於高增益類比的觀測器與追
蹤器，得到較高的錯誤觀測與性能恢復的追蹤效果。
