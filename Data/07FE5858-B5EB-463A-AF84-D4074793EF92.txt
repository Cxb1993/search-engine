人才培育成果說明： 
本計畫 Web 技術建構的聯合採購資訊系統，能使系統的整體處理能力得到明顯提升，
也能讓企業營運成本下降，既能滿足企業日常採購業務的正常要求，又能滿足企業在供應
鏈中斷時的快速採購需求。本計劃提出的系統，對滿足電子採購中的即時性與動態性採購
需求。 
1. 電子商務與電子採購理論層面，加強本案研究人員對電子商務應用技術與電子採購
的關連性、電子商務的發展現狀與趨勢，以及電子採購的概念、特點及電子採購的
模式與相關成功案例的研討。 
2. WEB 服務與應用技術方面，整體性地分析的相關應用的技術與知識，包括定義、特
徵、架構與模型、應用領域及如何應用 WEB 技術解決電子採購問題等。 
3. 在系統開發與實作方面：建立本案研究人員對系統發展的架構與各階段應完成的事
項有更清楚的認識，以及更成熟的運用專案管理之理論和方法；開發中更累積各項
WEB 應用技術的實作經驗，並學習雛型系統之建立與驗證之方法。 
4. 學習如何快速建立企業圖形和技術繪圖的工具。 
5. 了解汽車製造業管理之採購流程與相關汽車製造業基本知識。 
6. 瞭解類神經網路的基本概念；並對倒傳遞神經網路的運用方法深入了解與應用。 
7. 了解聯合採購之應用領域，與實際進行聯合採購時所發生問題之解決方法。 
 
技術研發成果說明： 
1. 能有效降低生產成本，提升企業營運利潤。 
2. 有效提升服務品質。 
3. 降低人工處理造成的失誤率且不易管理之問題。 
4. 透過資訊系統，可以對未來需求進行預測之外，也協助了汽車製造業人員迅速了解
該地工廠之財務狀況與庫存量之問題。 
5. 使企業建原本的競爭與對立關係轉化而成產業共同體。 
 
 
 4
採購系統可以對企業的各種採購需求進行支援，尤其是在由於生產中斷或供應鏈中斷時導
致的快速採購需求方面，以 Web 為基礎的採購系統以其具有之即時性、動態性對快速採購
達成有效的支援。本研究在相關研究的理論基礎下，確定聯合採購之方法能有效降低進貨
之成本，並透過資訊系統的導入，整合 Web 技術在聯合資訊系統的實際應用，提供更迅速
的電子採購平台服務，讓企業營運更加順利與成長。
