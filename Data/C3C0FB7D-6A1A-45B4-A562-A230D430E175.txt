壹、 前言
現行營建專案資源規劃與排程問題的文獻，於資源需求型態上大都為針對各種不同資源型態
作個別探討，尤其於專案物料需求規劃與採購問題上則少有探討，工程物料的成本，一直在
營建工程專案總成本中，佔了相當大的比例[1]，對於大型工程而言，所佔的比例更高，常常
超過 50%，因此物料管理在營建管理中，一直是不可忽視的討論重點之一。在物料管理的流
程中，物料採購計劃的擬定是相當重要的一環，它決定了採購時間及採購數量，訂料的時間
太早造成存貨成本的提昇、資金週轉及利息增加的壓力，太晚亦可能造成生產力的損失進而
工期的延誤，訂料的數量太多，亦可能造成呆廢料的增加，太少亦不敷工程進行的需要，因
此有必要將此專案物料規劃與排程問題獨立出來研究之。
此外，在營建工程專案規劃與排程之研究中，大都針對演算法的問題進行探討，常停留於理
論的研究，因而實際應用於營建專案規劃較少，其原因之一在於現行專案管理的軟體在進行
資源分配與拉平的功能時均有其既定的演算法處理之，其演算法經一些研究證明[2]，求解效
率並不甚佳，故而應有改善的空間，而此一系統必須要有能確實估計專案資源系統的功能，
而市面上的套裝軟體，或多或少僅能滿足部分需求，故在進行專案資源規劃時，功能應可再
加強，方可使資源更有系統且精確的估計出來，方才符合規劃的功能，故參考本計劃將根據
其他產業的相關應用，建立符合營建專案特性的專案資源規劃系統是必須的，將本計劃所提
出之演算法並結合此一系統以期具有實用性。本研究目的主要是延續上年度的計劃，其主要
目的包含：
1. 對專案物料需求規劃(project material requirements planning: PMRP)問題進行探討。並對
現行用以求解專案物料需求規劃問題的二階批量法(two-step lot-sizing rule)的缺失進行
分析，以作為設計有效的法則之重要參考依據。
2. 針對專案物料需求規劃問題的特性提出整合專案排程與物料訂購決策之最佳解(optimal
solution)模式與有效的探索解法(heuristic rule)，以提供專案經理作為最佳的專案作業排
程與物料訂購決策之規劃參考。
貳、 研究流程
本計畫先分析製造業之物料需求計畫與營建專案物料需求計畫之異同外，提出本研究整合專案
排程與物料訂購批量決策之整合模式(整數線性規劃法)外，也將針對現行求解此類問題的二階批
量法的缺失進行說明。並分別針對二階批量法與最佳解之績效差異進行評估，同時本年度計畫
亦將延續上年度之計畫，提出以禁忌搜尋法為基礎之混合型探索解法，與本計畫所提整合專案
排程與物料訂購批量決策之最佳化模式測試比較後，選擇求解效率較優的演算法作為資源限制
下的的多專案排程演算法，根據不同專案特性因素對探討本計劃所提出混合型演算法績效進行
輸入參數的敏感度分析，提出在不同專案特性下，對於專案完成績效較佳的演算法的輸入參數
設定結論。在幾個重要的專案特性因素下進行本研究之排程法則績效評估，以對各法則之有效
性進行比較分析。

3. 營建專案物料規劃子系統
本研究以Microsoft visual basic 6.0 作為程式開發工具，結合 MS.Project 2000，將本研究所建
構之分析演算模式程式化，同時，本研究提出一營建專案物料規劃子系統如圖三所示：營建
專案MRP 架構中，主要有預算施工進度計劃，專案物料清單資料庫，庫存紀錄檔案等三個資
料檔，及將這三個資料檔聯接的部分，這三個資料檔之關聯及資訊之傳輸流程如下：首先由
工程專案的進度計劃中，找出各作業之物料供應點，然後進入專案之物料清單資料庫中，找
出各作業所需要的資料，而後由存貨記錄檔中，找出每一物料之安全存量、現存量、採購時
間、規格等資料，而後列出施工期間，不同時期的物料需求，到此便完成了物料需求計劃。
而後由物料需求計劃及物料需求之前置時間，透過適當既定之批量模式，即可完成訂單計劃。

中文：
本計畫主要在針對專案物料需求規劃於排程問題進行探
討，本計畫先分析製造業之物料需求計畫與營建專案物料需求計畫
之異同外，提出本研究整合專案排程與物料訂購批量決策之整合模
式(整數線性規劃法)外，並提出以禁忌搜尋法為基礎之混合型探索
解法，與本計畫所提整合專案排程與物料訂購批量決策之最佳化模
式測試比較後，選擇求解效率較優的演算法作為資源限制下的的多
專案排程演算法，再利用套裝程式Microsoft Project, Visual Basic 和
Microsoft Access 建立工程專案資源規劃系統雛形，完成之後，將
尋數個案作系統模擬驗證，以增加系統的實用性，將本計劃所提出
的演算法，撰寫成電腦程式，結合本研究所提出之工程專案資源規
劃系統，此一系統將可在考慮資源供應的狀況下，執行對專案完成
績效有助益的較佳或最佳排程計劃。同時對於本計畫所提之有效的
探索解法(heuristic rule)，將可提供專案經理作為較佳的專案作業排
程與資源供需決策之規劃參考。技術說明
英文：The research will focus on investigating the problem of
project materials procurement problems in multi-project resource
constrained scheduling. The research will analyze the difference
between the project materials requirement planning (MRP) and MRP
of manufacturing industry to provide an optimization model of
RCMPS considering the impact of project material. Also, the research
will provide hybrid heuristic algorithm using TABU search algorithm
as tool. Also, the project will implement Microsoft Project, Visual
Basic as a development tool to establish a construction project
resource planning system prototype. Finally, the research will provide
a better algorithm to the RCMPS based on the performance of the
optimization model and heuristic algorithm as the conclusion.
