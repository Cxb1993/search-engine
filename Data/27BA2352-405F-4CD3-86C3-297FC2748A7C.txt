二、前言與研究目的 
 
   資料挖掘(Data Mining)系一極為熱門
且可以實際幫助企業進行決策支援之重要
研究議題。資料挖掘的方法主要可以分為
分 群 (Data Clustering) 、 分 類 (Data 
Classification)、循序特徵(Sequence Pattern)
及關聯法則(Association Rules)等[1]-[4]，
而其中資料分群的方法扮演相當重要的角
色。資料分群是人們使用資訊的主要活
動，其應用範圍極為廣泛。資料分群之目
的是讓我們可以洞悉資料之內部結構或讓
我們發展一企業策略以客製化個別顧客之
分群以增強企業效率。由於資料分群可應
用於趨勢分析、顧客剖析、相似性搜尋、
分類、影像處理、圖型辨識及多重物流配
送等應用，所以此一研究領域也隨之成為
熱門且實際可應用於企業決策與工業技術
上之重點科技。   
 
三、文獻探討 
目前已有許多頗具效率且實用性高之
智慧型啟發式方法被發展出來。其中較著
名的 K-means、自我組織圖(Self Organizing 
Map; SOM)[5]及基因演算法 (Genetic 
Algorithm; GA) 已被成功應用於資料分群
問題上。SOM是最廣為人知的一個方法，
但不幸的是 SOM 計算所需花費的時間非
常的長，因此有許多學者們遂對於如何減
少 SOM 的計算時間極感興趣，其中的研
究包含將 SOM 結合 Fuzzy、K-means、
Genetic Algorithms等或是在 SOM的計算
過程中加入一些技巧。此外，近來有兩個
極為有效率之方法稱為 FSOM+K-means[5]
及 GKA[6]被提出。此二方法在資料分群
問題上之效率均較前面三者為佳。 
最近筆者亦提出數個已在 IEEE 國際
會議或國際期刊發表之新的資料分群方
法 ， 例 如 ICA (Intuitional Clustering 
Algorithm) 、 ACODF (Ant Colony 
Optimization with Different Favor) 及
MSGKA (Multiple Searching Genetic 
K-Means Algorithm)等。經由理論分析與實
際的應用結果，這些方法在許多資料分群
問題的錯誤率上均較前述方法為低。其中
尤以 ICA在資料分群問題的錯誤率上及時
間成本上成效最佳。 
在本研究計畫中，筆者所提出三個之
分群方法分別稱為蛛網分群演算法、精英
分群演算法及二次 k-means 分群演算法。
經由筆者的實際模擬結果，此三方法分群
的效能均遠勝於上述國際著名分群方法。

 摘  要 
Edge detection is an extremely significant pre-processing step 
of image processing. An excellent edge detection approach can 
improve the accuracy and speed of image processing system. In 
general, edge detection is an operator to find edge pixels from an 
image. Formerly, numerous edge detection approaches based on 
mathematical ideas such as Laplacians, Marr-Hildreth and Sobel 
were proposed. Lately, smart and advanced methods based on 
intelligent computations such as fuzzy, neural network, genetic 
algorithm were developed. As the result of the popularization of 
digital image processing devices, more and more real-time 
applications are designed to solve special real-life problems. This 
paper presents a new edge detection method named M-Sobel 
(Modified Sobel) to satisfy the requirement of real-time image 
processing system for colorful true pictures. As compared with 
many other edge detection methods, our new edge detection 
technique is fast and suitable for color pictures.

ABSTRACT 
This paper presents a new edge detection method named M-Sobel (Modified Sobel) 
to satisfy the requirement of real-time image processing system for colorful true pictures. 
As compared with many other edge detection methods, our new edge detection technique 
is fast and suitable for color pictures.  
Keywords: edge detection, image processing, real-time system 
 
1.  INTRODUCTION 
Edge detection is an extremely significant pre-processing step of image processing. 
An excellent edge detection approach can improve the accuracy and speed of image 
processing system. In general, edge detection is an operator to find edge pixels from an 
image. Formerly, numerous edge detection approaches based on mathematical ideas such 
as Laplacians, Marr-Hildreth and Sobel were proposed [1], [10], [12], [13]. Lately, smart 
and advanced methods based on intelligent computations such as fuzzy, neural network, 
genetic algorithm were developed [2]-[6], [9]. As the result of the popularization of digital 
image processing devices, more and more real-time applications are designed to solve 
special real-life problems. There are several kinds of real-time image processing system 
such as automatic vehicle license-plate recognition, handwritten character recognition and 
surveillance System for intruder [7]-[8], [11]. To implement these systems, edge detection 
plays a key role to extract the information of edge pixels. 
To separate edge pixels from the whole image, the gradient is the most common used 
attribute. Usually, to combine masks is a simple technique for computing the gradient.

3. SIMULATION RESULTS 
To verify the performance of our proposed edge detection approach, we conducted 
some experiments on a PC with Pentium Ⅳ CPU. We compare the proposed method with 
Sobel, Marr-Hildreth and Laplacians approaches for image edge detection. Fig.1

real-time images continuously to get the average value of images processed by each edge 
detection method (per second). We recorded memory usage and CPU time for the final 
experiment. This experiment is a strong evidence to demonstrate the contrast between our 
method and others.  
 
4. CONCLUSIONS 
A new edge detection method named M-Sobel to satisfy the requirement of real-time 
image processing system for colorful true pictures is presented in this paper. As compared 
with many other edge detection methods, our new edge detection technique is fast and 
suitable for color pictures.  In sum, the M-Sobel approach has some obvious advantages 
as follows: a) the requirement of resources is less than other methods; hence, this feature 
lets our method be able to work on mobile devices with poor capabilities, b) average time 
for processing an image is shorter than other methods, c) it is easy to implement 
applications based on our method.
