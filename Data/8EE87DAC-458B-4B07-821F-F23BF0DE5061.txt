研究摘要(500 字以內)： 
    近年來，先進雷射二極體常應用於長距離高數位化之光通訊系統。如：光纖到家及高密度分波多
工系統。為了提升傳輸性能，雷射二極體輸出雷射波長品質是一個很重要的因素。一般而言，優質
雷射輸出波長應具有窄線寬、單頻模式、調整範圍寬廣、高強度功率及高旁模抑制比。然而當調變
信號注入雷射後，雷射線寬將會變寬。尤其是傳送10 Gb/s以上的訊號時，則會產生嚴重的擾頻現象。
傳輸過程色散效應也因此就會相當的明顯，導致系統會引起相當大的非線性失真，再加上雷射光源
本身仍然會有殘存擾頻存在，造成輸出波長不穩定，種種因素將會導致於系統傳輸信號品質劣化進
而傳送距離受到限制。曾經有很多種技術被提出用來克服前述的問題，但是這些技術都會增加系統
的價格及複雜度。對實際建構高數位化光通訊系統而言，發射端雷射光源之成本因素乃是首要考量
之務。 
     此次計畫報告裡，針對雷射二極體單頻模式及旁模抑制比做大幅度之改善。我們提出超窄頻帶
光帶通濾波器，超窄頻帶光帶通濾波器由摻鉺光纖、光循環器及全反射鏡所組成。摻鉺光纖可以降低
模態跳動及燒孔效應影響，穩定雷射駐波波長及產生高強度雷射輸出。可以預期的是，一只雷射二極
體中加入超窄頻帶光帶通濾波器，雷射輸出波長會具有窄線寬、單頻模式、調整範圍寬廣、高強度功
率及高旁模抑制比等優越特性。超窄頻帶光帶通濾波器之研製，整個過程不需要使用昂貴的元件，而
且元件均為光纖化、體積可以很小，未來便於封裝。應用於產業界是相當受歡迎的。

圖一、超窄頻帶光帶通濾波器實驗架構圖 
    本報告所提出之架構，乃由一個摻鉺光纖放大器（Erbium Doped Fiber Amplifier；EDFA）、半導
體光放器（Semiconductor Optical Amplifier；SOA）、光循環器（Optical Circular ；OC）、一段摻
鉺光纖（Erbium Doped Fiber；EDF）及一個全反射鏡（mirror）所組成。其工作原理是利用輸入
光與 mirror 反射的光於摻鉺光纖中產生駐波（Standing wave）進而產生干涉，利用干涉產生光源
頻寬縮小的目的。本計畫將利用現有商品化的摻鉺光纖，摻鉺光纖的長度將是達成此技術的重要
關鍵因素，如何調整摻鉺光纖的長度來達到極窄頻寬的光源輸出將是本計畫需要解決的問題。 
另一個預期影響此特性因素是 Return Loss（RL），為了降低元件間的 Return Loss，本報告將於各
元件連接時，將使用 APF（Angled Fiber Connection）光纖接頭，並使用光纖熔接機熔接摻鉺光纖，
藉著此兩種方法將 Return Loss 降至最低。然而光纖熔接機的熔接參數，例如放電功率（Arc 
Power）、預熔時間（Pre-fuse time）、熔接時間（Arc Duration）、Z 軸距離（Z-push distance）、清潔
時間（Cleaning time）……等，皆會影響 Return Loss 的反射量，因此，熔接機\的熔接條件亦是此
計畫成敗的關鍵因素之ㄧ。

可利用之產業及可開發之產品： 
1. 高密度分波多工系統在現今光通訊產業是多方被採用的通訊方式，高密度分波多工系統的串音問
題也是常困擾設計工程師。串音的成因是由於多模態雷射波長因擾頻效應使線寬增加而在發射端
相互干擾，或傳輸過程中受到非線性雜訊而使雷射波長線變寬，在接收端受到干擾。因此自製低
成本的超窄頻帶光帶通濾波器可使多模態雷射波型平坦化及線寬窄小化，這對於高密度分波多工
系統是有很大的貢獻。 
2. 在生活中，傳統的「光纖有線電視系統」業者方面在主幹線幾乎都以改鋪設成光纖，為因應光纖
到家數位家庭整體服務，高畫質的數位訊號需求必然大增。頭端發射源的品質需要更進一步要求。
因應此一趨勢，可以讓參與相關計畫研討之研發人員對「可調式窄線寬雷射二極體」及「超窄頻
帶光帶通濾波器」有一深入之了解與實習。光源線寬若因此大幅改善，則高品質的數位寬頻整體
服務必然實現。 
3. 光感測系統：由於超窄頻帶光帶通濾波器使雷射二極體產生穩定雷射駐波波長及精準單頻頻帶，
在工廠自動化輸送帶上做為產品質判讀的感測裝置其精確度能大幅提升。 
4. 顯示影像科技：超窄頻帶光帶通濾波器能使光源的光譜更明確化，光譜明確化對於顯示器畫質 有
正面的提升。 
5. 醫用雷射：醫療用途的雷射光束、雷射刀廣泛運用於各種治療面。雷射光束及雷射刀需要高功率、
波長穩定及高穿透率。這需要高品質的雷射光源，超窄頻帶光帶通濾波器可以提升雷射光束的性
能。 
6. 其他尚有很多應用到超窄頻帶光帶通濾波器，如：奈米通訊、電漿顯示器產業、核熔合系統診斷、
大氣離子物理方面、雷射雷達、光學定向測距及太空光通訊等高層次產業方面。以上所列舉均為
光電產業。從小型系統到大系統都會應用到超窄頻帶帶通濾波器，範圍相當廣泛。
