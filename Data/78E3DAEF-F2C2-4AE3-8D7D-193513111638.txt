一、 摘要 
新型靜態式全頻譜標準色彩影像系統可得到待測物在光學上所有的資訊，其包含光譜
資訊、空間資訊、強度資訊、色彩資訊、影像資訊，其應用在可見光、不可見光頻譜量測、
物體(動物/植物/礦物)影像光譜、近紅外光 NIR 光譜影像、多通道光纖光譜攝像系統、遠端
感應、太空遙測、生醫光譜影像、顯示器色彩調校、智慧搜尋飛彈…等，由於應用領域廣
泛，因此產業價值非常可觀。 
目前超頻譜影像裝置在靜態影像方面如欲取得二維影像，必頇將整個裝置或其物體移動，
若應用在生醫或是顯示器方面時，將會不利於量測的方便性，再者更換其前端光學影像系
統(量測鏡頭)時，後段光學系統也必頇重新設計，依此目的本計畫欲設計一機構，其主要架
構為光學影像系統、可微控之延遲透鏡組、超光譜系統，其中可微控之延遲透鏡組可輔助
超光譜儀達到 2 維影像與光譜的掃描，而光學影像系統可以是顯微鏡，望遠鏡等成像鏡頭，
或者單純是影像輸入，因此可在不移動影像、光學成像系統、超光譜儀系統三者的情況下，
量測影像的頻譜，此將適用各式光學影像系統，而不需再重新設計後端之光學系統。如此
機構之研發不僅是科學領域的前瞻性，也是未來商品化的影像量測標準裝置，也是作為高
解析度超頻譜動態影像的開端。 
 
Abstract 
   
The hyper-spectrum standard image system can obtain all of the optical information of test 
object including spectral information、spatial information、intensity information、color 
information and image information. The system can apply to visible spectrum measurement、
invisible spectrum measurement、objective(animals/plants/minerals) image information、near 
infra- red spectral information、multi-channel fiber spectral camera system、remote induction、
space and remote sensing and biophotonic spectral image…etc. 
Until now, it has some problems to obtain 2D image through hyper-spectrum setup. One 
problem is not convenient to move the hyper-spectrum setup or object in order to obtain 2D 
image. The other problem is, when we change the measuring lens the behind part of optical 
system must redesign.  
The purpose of this project is to design a new hyper-spectrum setup which includes optical 
image system、relay lenses module and hyper-spectrum system. The new hyper-spectrum setup 
can obtain 2D image through only move relay lenses module. The new hyper-spectrum setup is 
more convenient to apply to measurement and the beginning of commercialization.

四、 技術特點說明 
 
主要架構為可微控之延遲透鏡組(Relay Lens)，可輔助超光譜儀(Hyper-Spectrometer)達
到 2 維影像與光譜的掃描。所謂延遲透鏡可將影像從一處轉播放到另一處，而影像及物體
本身的大小關係固定，常應用在來福槍瞄準器，潛望鏡系統，顯微鏡，以及軍用紅外線影
像系統，屬於非成像(Afocal System)系統。此機構另一端亦可與光學元件連結，因此可在不
移動影像、光學元件、超光譜儀三者的情況下，量測影像的頻譜。在此光學元件可以是顯
微鏡，望遠鏡等，或者單純是影像輸入。而由於此一架構在搜索現有光譜影像相關的專利
與研究論文中，並無依此方式來達到光譜影像的掃描，因此相關的專利正在撰寫申請中。
而其相關專利將可由其光學系統架構、成像品質校正、掃描系統整合、微動系統控制、光
譜影像處理、與產品應用等方向進行專利布局。 
本計畫可取代光譜儀在不同的水帄量測環境需要訂製不同掃描系統、以及取代光譜儀
應用設備 Mirror Scanner 在不同角度下，光學路徑不同所造成的像差等缺點。機構內部之微
動裝置可由經由電腦控制，用步進馬達達到精準的掃描速度、方向以及位移之控制，以便
將影像及其光譜精確的記錄下來。如 Figure 1 所示，當我們藉由前端的光學影像系統(Optical 
Image System)擷取物件 x 軸之影像資訊，藉由延遲透鏡將影像傳遞到超光譜系統
(Hyper-Spectrometer System)中物鏡的前焦距上，進而透過超光譜系統將影像及光譜資訊傳
到到 2 維的 CCD 成像。 
超光譜儀成像系統如 Figure 2 所示，包括一個物鏡(Objective Lens)，一個狹縫(Slit)，如
Figure 8 所示，稜鏡組，以及分光光柵組成。超光譜儀成像系統，可將物面(Target)延 x 方
向一行的影像資訊藉由物鏡擷取進來，透過狹縫進入到稜鏡組以及分光柵組成的分光系統
進行分光，在成像面上產生影像以及光譜的資訊，如 Figure 2 所示，其成像面上，延 x 方

膠囊內視鏡的使用並非目前檢查的常態，它最重要的功能在於提供了傳統內視鏡無法

為最不具侵襲性的診察利器，醫師可藉由所獲得的影像，從血跡、血塊等蛛絲馬跡中尋找

閱覽程式中病理影像辨識功能，輔助醫生作病理的判斷，及病歷資料的整理，更進一步提
供消化系統各樣的病理資訊，以達到最完整的診斷。

    第二類型是待測物不動，而達到掃描的目的，這類型的又可分為鏡反射式、點矩陣式、
移動狹縫式及遙測式。鏡反射式(如 Figure 8 所示)是在超光譜儀前面加裝一個可自動旋轉的
反射鏡，利用反射鏡的旋轉將不同區塊的影像，反射到超光譜儀內成像及分為光譜的資訊，
此種掃描方式可用於不能移動的物體掃描，例如遠方景物影像的掃描。鏡反射式的缺點在
於，不同區塊的影像光程不同，也就是離光軸越近之影像光程越短，越離軸區塊之影像其
光程越長，如此會造成離軸區塊影像的像差變差，造成影像的失真。另外，由於入鏡窗口
的有限大小會造成大角度的光線無法射入，因為對於尺度大的物體，需鏡反射式超光越遠，
而造成能量的損失，也無法在有限距離內做大尺度物體的整體掃描。
