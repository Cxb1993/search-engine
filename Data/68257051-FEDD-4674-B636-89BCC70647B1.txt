中 文 摘 要 
本報告旨在探討結構加裝單一非線性調諧質塊阻尼器 (TMD) 之最佳化理論
及參數研究，進而推導出非線性 TMD 在白噪音風力擾動 (White noise) 下之最佳
阻尼係數建議公式。首先提出結構加裝非線性調諧質塊阻尼器之動力分析方法，
求解結構動態反應。在剪力型結構加裝單一調諧質塊系統下，以結構歷時反應之
平方合最小化為訴求，推導出一非線性 TMD 最佳化理論，並利用 MATLAB 現有
之最佳化工具：直接搜索法 (Direct Search Method)，可作為驗證最佳化理論可行
性及快速搜尋 TMD 最佳參數之工具。參數分析中，採用單自由度化之台北 101 大
樓視作結構相關參數，探討非線性 TMD 最佳化參數間之相互關係與對結構減振影
響，並針對不同結構阻尼比與風力倍率對最佳化 TMD 參數之影響說明之，最後利
用五組白噪音設計風力，推導出 TMD 最佳阻尼係數之建議公式。引入兩組結構物
裝設單一 TMD 之案例分析，第一採用台北 101 大樓結構及其調諧質塊阻尼器之相
關參數，並另外考量三種非線性阻尼型式，由分析結果顯示，不同阻尼型式之 TMD
均能達成所需要求；第二分析案例採用五層樓剪力鋼構架，驗證非線性調諧質塊
系統應用在多自由度結構物上亦有不錯的減振效果。 
 
關鍵詞：非線性、調諧質塊阻尼器 (TMD)、最佳化理論、台北 101

第一章 緒論 
1.1 研究動機及目的 
地震起因於大地板塊間之相互碰撞擠壓而傳送應力波於地表產生振動，而台
灣島位處於歐亞板塊及菲律賓板塊交界面，每年均會造成多起地震，位於台灣本
島上之建設因之無可避免必須承受地震之侵襲；相對於地震，造成建築物產生振
動之外力亦有外在承受風力擾動之影響，但在大多數情況下，風力所造成之災害
遠小於地震，惟當建築物往高處發展時，風力所產生之影響則反過來凌駕於地震
之上，故在超高樓層建築物設計觀念，多數採用風力控制。隨著台灣建設發展，
建地狹小以致於諸多建築物均往高樓發展，而往往高樓林立區均是人口密集區
域，以致於建築物在抵抗地震及風力下之安全性考量乃當今土木建設重要課題之
一，位處於強風強震區之建設，諸如橋樑、高樓等建築，早期多採用硬性抵抗模
式以對抗外在地震及風力擾動，硬性抵抗亦即加大結構設計斷面，使得結構物強
度提升，但如此勢必造成結構自重增加，導致結構設計下之設計地震力增加而不
符合經濟效益。而最近數年間，多數土木結構物設計乃採用「韌性設計」的觀念，
利用梁柱本身材料降伏特性，伴隨著結構變形進入塑性階段時可進而消散外力導
入之能量，但此法勢必造成一些問題，如在大地震來襲時，結構是否真能如預期
般產生塑鉸是值得商議的，另一則是即使真如設想般利用變形消能，但大災害過
去後伴隨而來的往往是維修上的不易。隨著現今土木行業技術提升，發展出結構
控制技術，結構振動控制之設計揚棄傳統抗風耐震設計之概念，在傳統結構系統
外，裝設控制元件，經由適當之配置方式，使控制元件能巧妙地消散或隔絕外力
輸入土木結構之能量，將土木結構在風力與地震力作用下之振動量大幅降至可以
容忍之範圍內，以確保人員與結構之安全，並可提供舒適之居家與辦公環境，更
可進一步節省土木結構之建造費用。結構控制在土木工程上之應用乃世界潮流之
所趨，土木結構加裝控制機制愈趨普遍，目前尤以被動控制最具實用價值，因為
被動控制無需提供額外能源，且易於養護，故較易為土木工程界所接納。但是，
隨著科技之進步，半主動控制、主動控制及主被動混合控制之可靠度將不斷提昇，
其應用及推廣指日可待。

1.3 本文內容 
目前調諧質塊阻尼系統之研究概括為三大類，其一主要針對單 TMD 系統之研
究，大多集中在於結構控制效果以及最佳參數之求得，為使控制效果達到最佳化，
主要使 TMD 之振動頻率調諧至主結構振動頻率附近，並選擇適當阻尼，如文獻探
討所述，已有許多學者專家致力於此，但單一 TMD 存在著一項缺點，由於其對於
結構自然頻率之波動相當敏感，若設計出頻率偏離主結構頻率，則有效性將會降
低許多，因此外力擾動若為窄頻，則單一 TMD 之減振效果較良好，反之為寬頻時，
則效果較不佳；為了改善單 TMD 所產生之問題，產生了多重調諧質塊阻尼器 
(MTMD) 的研究，根據多個 TMD 分別裝設於不同樓層上，可控制結構多模態行
為，則對於位移及加速度控制有較良好的效果，亦可改善單 TMD 系統所產生之離
頻效應 (detuning effect)；第三類研究則是將 TMD 之概念應用於工程問題中，如
基底隔震系統結合 TMD 系統，可更有效地降低基底之相對位移，或利用結構內部
既有建築物視為質塊系統，可有效節省建造經費及增加構材實用性 [27]。 
本研究主要針對第一類研究：單一或多自由度結構加裝單一非線性調諧質塊
阻尼系統作為分析對象，由前段文獻探討可知當前研究大多針對單至多自由度結
構物裝設線性 TMD 之最佳化理論作一探討，而對於非線性調諧質塊系統則少有著
墨之處，非線性調諧質塊可區分為勁度非線性 (質塊回復力) 及阻尼非線性 (施加
阻尼力)，有鑑於此，採用非線性黏滯阻尼形式之調諧質塊系統，探討結構加裝非

3.3 直接搜索法 (Direct Search Method) 
直接搜尋法(Direct Search Method) [30-32]為求解最佳化問題之方法，與傳統最
佳化方法最大差異在於直接搜尋法不需求得目標函數 (objective function) 之梯度
値，直接搜尋法搜尋目前點 (current point) 周圍的數點，以尋找目標函數值低於目
  25
模式 (Pattern) 為一組向量集合，視所要搜索之最佳化問題之獨立變數個數而
定，當最佳化問題參數值越多時，模式向量之維度即越大，將使得起始點週遭搜
索點數量增多，因此運算時間增長，一般模式向量由 2N 個向量組成，N 為最佳化
問題中之獨立變數，此模式向量為： 
1[1 0 0 0] N×L 、 1[0 1 0 0] N×L 、 … 1[0 0 0 1] N×L 、
1[ 1 0 0 0] N×− L 、 1[0 1 0 0] N×− L 、… 1[0 0 0 1] N×−L  
網格 (Mesh) 為每一步搜索過程中，所欲搜索起始點週遭之一系列點稱之。網
格點之求得取決於當步之模式向量，計算方式則為當步之起始點加上當步之模式
向量乘上一網格尺寸 (Mesh size)，其中網格尺寸乃視每步搜索過程成功與否決定
之，若搜索成功則網格尺寸可調大，即網格尺寸變動因子大於 1.0，一般設定為 2.0，
反之若搜索失敗則將網格尺寸縮小，即網格尺寸變動因子小於 1.0，一般設定為
0.5。可於網格選項中 (圖 3. 5) 進行設定。 
表決 (Poll) 為直接搜索算法於每步搜索成功與否之依據。當每步之模式向量
給定之後，繼而可計算起始點週遭之網格點位置，此時定義此最佳化問題所欲探
討之目標函數指標，並計算網格中各點之目標函數值，一旦找到目標函數值小於
起始點所對應之目標函數值則停止對網格點之計算，此時稱之為表決成功，並以
此點作為下一次迭代時之起始點。反之若未能找到使目標函數值得到改善的點，
則稱之為表決不成功，則下一步迭代之起始點保持不變。 
直接搜索工具即經由上述三步驟流程，一步步更新起始點，使得每步所得之
目標函數值均小於上一步目標函數值，如此迭代次數越多則所求得之參數值則會
越接近最佳化參數，惟大多數情況下，必須設定一迭代之停止條件，以供搜索工
具用以判斷是否跳出搜索迴圈之依據，而當搜索迭代停止時，則最後一步之起始
點以及所對應之目標函數值，即為所求。可由圖3. 6設定其迭代停止條件，當滿足

4.4 最佳化建議公式及非線性 TMD 之設計流程 
在第三章中，說明了本文所推導之非線性 TMD 最佳化理論以及引入之最佳化
搜尋工具，以往所作之研究中，亦有許多針對線性 TMD 最佳化設計公式之研究，
然而對於非線性 TMD 仍未有較完善可行之建議公式，因此本文擬嘗試利用多組非
線性阻尼力次冪 TMD 與風力擾動進行時域分析，以直接搜索法獲得各相對應之非
線性 TMD 最佳化參數，並利用 Warburton [8]所提之最佳化 TMD 設計公式為基礎，
分別針對無阻尼、阻尼比 2%與阻尼比 5%之單自由度結構，在白噪音風力作用下，
予以曲線擬合出適當之非線性 TMD 最佳化設計公式。由上節靈敏度分析結果可
知，頻率比對於 TMD 系統減振效果影響相當大，反之阻尼係數對於 TMD 系統減
振效果影響較微小，因此在設計 TMD 最佳阻尼係數時，僅需給予最佳值合理變化
範圍之阻尼係數值，即可有良好之減振效益；惟頻率比變化靈敏度高，在時域分
析下，頻率比往往沒有較固定之變化趨勢，在擬合適當之頻率比最佳設計公式有
其難度，故本文僅以獲得最佳化阻尼係數設計公式為目標，在可接受範圍內設計
一套非線性最佳阻尼係數，而最佳化頻率比即利用直接搜索法或最佳化理論求
値，如此一來僅需搜尋頻率比最佳值即可，可大幅縮減數值迭代及搜索過程。

第六章 結論與建議 
本文係針對非線性調諧質塊系統之參數行為作一整體說明，成功驗證非線性
調諧質塊阻尼器之實用性，無論是線性或非線性調諧質塊阻尼器，經最佳化之設
計，均能發揮其減振效益；並發展出非線性 TMD 之最佳化理論，主要依據結構物
在加裝單一非線性調諧質塊阻尼器下反應歷時最小化為最佳訴求，並引入直接搜
尋法作為最佳化理論驗證工具及快速搜索最佳 TMD 參數值之用；以台北 101 大樓
之結構參數作為主結構系統，對調諧質塊阻尼器之設計參數，進行參數分析及靈
敏度分析，瞭解非線性 TMD 參數在變動 TMD 質量比、結構阻尼比、阻尼力次冪
與風力倍率下，對於結構位移與加速度之減振反應；利用曲線擬合之概念，給予
五組白噪音設計風力，參考 Warburton [8]所提之線性 TMD 設計公式，進一步延伸
推導至非線性 TMD 之阻尼係數建議公式，並於實例分析中驗證阻尼係數建議公式
之可行性，茲可歸納結論與建議如下： 
1. 本文所提之最佳化理論可用於多自由度系統，且因於時域下探討最佳結構
減振，故對於風力擾動與結構阻尼比並無限制，應較具有實用性，經驗證
下非線性最佳化理論確可迭代出最佳 TMD 參數值，惟在加速度最佳化控
制理論中，為簡便計算，因此將狀態向量代回計算結構加速度反應時，乃
假設 TMD 阻尼力及勁度力不作用，其並非合理假設，後續仍須考量加速
度控制理論之合理性；而在進行最佳化參數迭代時，在起始點之設定與迭
代至局部最小值問題上，尚有待進一步探討。 
2. 在結構位移或加速度控制下，最佳非線性 TMD 參數相差無幾，因此若僅
利用位移控制進行最佳化分析，對結構加速度之減振應有良好效果。 
3. 不論 TMD 阻尼力次冪為何，進行最佳化參數分析後，TMD 對於結構之減
振效果均相近，即線性 TMD 與非線性 TMD 均能有效降低結構反應。 
4. 不論 TMD 阻尼力次冪為何，最佳頻率比大約落於 0.9~1.1 之間，並隨著
