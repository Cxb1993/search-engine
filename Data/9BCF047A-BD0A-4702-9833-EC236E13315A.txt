三、研究方法 
 
本研究以 chamber ( 圖二 )實驗探討稻作生質燃燒懸浮微粒 ( PM2.5、PM10 ) 
排放特徵，並解析粒子三種脫水醣類 ( Levoglucosan、Mannosan、Galactosan )、碳
粒子 ( OC、EC )、水溶性陰陽離子成份分佈，及其氣狀污染物 ( CO、CO2 ) 分佈
特徵與排放係數等。實驗藉由 PEM 及 MEM 兩種採樣器採集稻作煙塵兩種粒徑 
( PM2.5、PM10 ) 粒子；並即時監測 CO、CO2 變化，實驗流程架構如圖一。 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
1. chamber 燃燒實驗 
  目前相關生質燃燒 chamber 研究多數以烈焰型 ( Flaming ) 燃燒方式探討其
排放特徵為主，但台灣高溫高濕環境使得稻草有較高的涵水量，使稻作露天燃燒
往往產生大量濃煙，故本研究將模擬台灣稻稈真實燃燒情況加以討論。研究將乾
稻稈置於燃燒腔點火燃燒，明顯火焰外露視為烈焰型 ( Flaming ) 燃燒，完全沒有
火焰外露僅是濃煙排放視為悶燒型 ( Smoldering )；如混合前兩者 ( 有時火焰外露
有時只排放濃煙 ) 視為混合燃燒型 ( Mixing ) ( 圖三 )。 
( 1 ) 乾稻稈涵水量調理 
  本研究中稻草涵水量依據環檢署事業廢棄物水份測定法—間接測定法 ( NIEA 
R203.02C ) 進行涵水量檢測，本研究使用稻草燃燒材料皆以曝曬後之乾稻稈 ( 仍
會有少量水分 ) 為探討對象 ( 見表一 )。

四、結果與討論 
 
( 1 ) 稻作燃燒微粒質量濃度分佈 
    首先討論 chamber 三種燃燒方式 ( Flaming、Smoldering、Mixing ) 之微
粒粒徑 ( PM2.5、PM2.5-10、PM>10 ) 質量濃度分佈特徵 ( 圖四、表四 )。結果顯示
Smoldering 三種粒徑粒子排放濃度皆明顯高於另兩種燃燒方式，Smoldering > 
Mixing > Flaming；其中 Smoldering PM2.5 粒子濃度為 Flaming 5 倍，PM2.5-10 與 PM>10
粒子亦有 Flaming 3 倍濃度；Mixing 三種粒徑 ( PM2.5、PM2.5-10、PM>10 ) 排放亦大
於 Flaming 分別為 3 倍、2 倍、2 倍。顯示稻作燃燒方式對細粒子 ( PM2.5 ) 影響
遠大於粗粒子 ( PM2.5-10 ) 與粗大粒子 ( PM>10 )。 
 
 
 
 
 
 
 
 
 
 
 
 
 
  煙塵粒子粒徑分佈百分比方面 ( 表三 )，結果顯示三種燃燒方式主要排放皆
以 PM2.5 細粒子為主，約佔 66.5 ~ 77.6 %，粗大粒子 ( PM>10 ) 次之，約佔兩成左
右 ( 15.8 ~ 22.7 % )；粗粒子 ( PM2.5-10 ) 佔 TSP 一成左右 ( 6.6 ~ 10.8% )。本項結
果與研究室前人研究 ( 陳怡杏，2009 ) 顯示雲林地區大氣生質燃燒粒子濃度 
PM2.5 佔總懸浮微粒 TSP 60%，粗大粒子佔 TSP 20%左右之結果相似。此外，本研
究尚討論高濕度環境( 80% RH vs 60% RH ) 對細粒子 ( PM2.5 ) 與粗大粒子 
( PM>10 ) 粒子排放濃度的影響，結果呈現高濕度下 ( 80% RH )，細與粗大粒子濃
度皆有增加的效果，如 Flaming 在 60% RH 大氣環境濕度下，PM>10 粒徑粒子排
放僅有 2.4 mg / m3 ( 佔 TSP 中 5.0 % )，但在 80% RH 高濕度環境，粗大粒子明
顯增長至 27.51 mg/ m3 ( 佔 TSP 中 28.6 % )，可能因 Flaming 火焰燃燒旺盛，粒子
排放後接觸到高濕度環境，容易產生粒子聚合，使粗大粒子 ( PM>10 ) 明顯增長，
或稻草吸有水份其燃燒過程可能釋放粗大粒徑粒子。但 Smoldering 方面，隨大氣
溼度增大 ( 60% RH 至 80% RH ) 細粒子 ( PM2.5 ) 濃度有明顯增高現象，但粗粒
子 ( PM2.5-10 ) 濃度反而下降 ( 37.54 mg / m3 降至 11.73 mg / m3 )，可能因悶燒排放

4.0，顯示生質種類、燃燒條件 ( 涵水量、烈焰、悶燒 ) 皆可能影響生質纖維素或
半纖維素熱解比例，進而影響脫水醣特徵比值 ( Levo / Mann )。 
  因此，本研究藉由燃燒條件推估樣本脫水醣回歸比值，結果顯示如圖六，若
以X軸為基準 ( Mannosan )，X＝1 µg / m3時，三種燃燒條件 Levoglucosan ( Y 軸 ) 
濃度分別為 31、36、57 µg / m3，表示 Levo / Mann 比值分別為 31、36、57 ( 若
X＝0.8，則 Levo / Mann 比值分別為 34、43、67 )。顯示燃燒方式 ( 即是熱裂解
條件 ) 可能影響 Levo / Mann 比值。故本研究推估稻作燃燒 Levo / Mann 比值應
在 40 ± 20 範圍，甚至隨燃燒條件 ( 如涵水的悶燒 ) 其比值有更升高的可能性 
( ＞60 )。相關生質燃燒文獻 ( Sheesley et al., 2003 ) 顯示稻稈燃燒產生 PM2.5 粒
子 Levo / Mann 濃度比值為 41.6；其他榖類 ( Cereal straw ) chamber 燃燒文獻 
( Zhang et al., 2007 ) 顯現 Levo / Mann 比值為 55.7，兩種 chamber 文獻比值皆在
本實驗迴歸比值範圍之內。 
 
 
 
 
 
 
 
 
 
 
 
 
 
  此外，相關大氣田野實驗呈現台灣冬季稻草 Near-source 田野燃燒粒子 Levo / 
Mann 比值為 25.5 ( Engling et al. ,  2009 )；本研究室相關研究 ( 謝濠駿，2008 ) 
曾針對台灣中南部冬、夏兩季進行大氣採樣分析，Levo / Mann 比值為 21.7、50；
田野 Levo / Mann 比值較低，可能因大氣尚有其他生質燃燒排放源粒子 ( 如軟木 
Levo / Mann 比值僅有 3.6 ) 之老化氣膠 ( aged aerosol ) 存在，使大氣脫水醣比值
大幅下降。總而言之，稻作主要由纖維素組成，與相關文獻木材類含大量半纖維
素及木質素不同，隨生質燃燒條件 ( 裂解溫度、涵水量等 ) 皆可能影響 Levo / 
Mann 比值，本研究初步結果推估稻作燃燒 Levo / Mann 比值應在 40 ± 20 範圍；
而稻作燃燒若為不完全燃燒 (如悶燒或混合燃燒 ) 方式容易使其比值上升 ( ＞
60 )，因不同燃燒方式與條件皆可能造成纖維素與半纖維素裂解程度差異，使 Levo 
/ Mann 比值變異性增加，以致相關文獻報導 Levo / Mann 比值不盡完全相同。

( 4 ) 水溶性陰陽離子濃度 
  本研究尚討論稻作生質燃燒陰陽離子 F-、Cl-、NO3-、SO42-、NO2-、PO42-、
K+、Ca2+、Mg2+、NH4+、Na+等離子濃度 ( 其中 NO2-、PO42-濃度均低於偵測極限 )，
表七顯示十一種離子在三種燃燒方式 ( Flaming、Mixing、Smoldering ) 排放濃度。
稻作燃燒粒子以 Cl- 排放濃度最高 ( 佔陰離子 30 ~ 50% )，SO42- 及 F- 次之，NO3- 
最少。本研究高 Cl- 濃度與相關文獻報導結果相似，相關文獻 ( Ryu et al., 2007；
Cheng et al., 2009 ) 顯示當有生質燃燒發生時，Cl-、NO3-、SO42-離子濃度皆較背景
濃度明顯提升，氯 ( Cl ) 是農作物生長發育重要的微量元素，主要來源為肥料 
( 氯化銨、氯化鉀 )。此外，硫元素 ( S ) 是蛋白質主要構成分子，也是 SO42- 離
子主要來源；而氟離子 ( F- ) 則可能因植物吸收空氣或水污染物，使得氟蓄積於植
物體。NO3-、Na+、Ca2+、Mg2+等離子主要來自肥料 ( 如硝酸態氮肥等，詳見表
A-3 ) 所得。相關生質燃燒文獻 ( Cheng et al., 2009 ) 討論台灣大氣中 PM2.5 陰陽
離子濃度，NO3- 濃度 ( 1.8 µg / m3 ) 高於本研究結果 ( 0.9 ~ 1.3µg / m3)，可能是大
氣 NO3- 主要來自交通及其他燃燒來源，而生質燃燒 NO3- 貢獻無異雪上加霜，影
響區域空氣品質。
