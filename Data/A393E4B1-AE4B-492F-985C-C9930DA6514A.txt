Abstract 
This project aims to use the FPGA to implement the GSM IF down converter and 
channelizer. We use the numerically controlled oscillator (NCO) to realize the IF down 
converter and use the polyphase algorithm to design the channelizer. In addition, we show that 
our channelizer can separate the channels exactly. In the hardware structure, the channelizer is 
composed of the FIR filters and the IFFT. We first use the 2-bits parallel distributed arithmetic 
algorithm to design the FIR filter. One feature of this algorithm is that it uses the look up table 
(LUT) to replace the complicated multipliers that may consume much area and time. Also, the 
parallel-processing feature of this algorithm can reduce the processing time and save the area. 
Then, we use the Raxix-2 DIF FFT algorithm to design the IFFT so that the number of the 
multipliers can be reduced, and we employ the parallel processing on each signal to reduce the 
processing frequency. Finally, we adopt the hardware description language ‘Verilog’ for RTL 
design and use Precision RTL and Modelsim development software to execute the logic 
synthesis and simulation. Also, we compare the simulation results with the results that come 
from the MATLAB program to finish the design and synthesis for the channelizer.  
Key words: software defined radio, down converter, channelizer
