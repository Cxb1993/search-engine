追跡法之光學鏡組(lens module)來模擬厚透鏡組之初階及像差特性[8]。初階參數用以決定近
軸入出射光線的高度與角度之相依性。Code V ®的lens module鏡組初階參數包含厚度、焦距
(MFL)、前焦距 (MFF)及後焦距(MBF)。設定MFF及MBF等同於設定前後主平面位置。像差
方面先設定成像縮小率(MRD)、入瞳位置(MEN)、入瞳直徑(MED)及最大視場(MFD)四個參數
值。單色像差再給予對應之六個單色Seidel像差係數值。依據以上設定參數，Code V 可計算
出任意入射近軸光條件下，鏡組對單色像差之影響力。此功能非常益於光學鏡頭設計之初始
佈局，Park and Shannon [9]展現了應用四個lens modules於變焦鏡頭設計之方法。然而，Code V
的方法仍存在若干不便與缺陷：(1) 物體不能在無限遠或前焦點處，即MRD不能為零或無限
大，必需用一極小或極大之值模擬(2)無法直接用法物方遠心鏡組，因為其入瞳位置及孔徑均
為無限大。同樣的必需用較大數值取代(3)無焦鏡組之焦距及前後焦距均為無限大，此時Code 
V的 MFL、MFF及MBF為無效參數，不能正確計算光線出射高度及角度(4) Code V雖能計算
鏡組對系統像差之影響，例如點列圖(Spot diagram)或垂軸像差曲線(Ray aberration curves)，但
無法直接給出鏡組在特定組態下之像差值，此資訊卻是光學設計時關注的焦點。在前報告指
出，我們的方法對上述Code V模型的種種缺陷均完全解決。 
在色差部份，Code V可設定於參考組態之三種波長軸向及垂軸向波色差係數，據此計算
色差及二次色差特性，以及當組態改變時，新的色差特性。然而，我們發現Code V的預測不
盡正確。以下以實際鏡頭與Code V lens module 模擬結果做比對。圖5上部顯示一個實際雙膠
合透鏡結構，使用於二個光學組態條件，及色球差曲線圖。第一組態顯示透鏡之消色差特性，
即紅光與藍光焦點重合。然而在第二組態，物距改變，無法再完美消色差，可見紅光與藍光
焦點分開的情況。圖下部應用Code V lens module功能模擬雙膠合透鏡，右方模擬之色球差曲
線圖中，第一組態與實際相同，但第二組態之紅光與藍光焦點仍然重合，與實際不符。值得
注意的是，Code V只考量軸向及垂軸向色差，並未加入軸向瞳色差。

五、鏡頭像差演算法 
在建立各種鏡頭之像差演算法方面，完成以Code V編寫之基本光線截點及實際(非近軸)
像差函數，包含：弧矢面截點座標、成像面x, y截點座標、軸向球差、寬光束子午彗差、寬光
束弧矢彗差、寬光束子午場曲、寬光束弧矢場曲、畸變。由於無法直接由Excel 透過Code V
呼叫以Code V 巨集語言所編寫之函數，亦編寫相應Visual Basic 版本。例如

Public Function CV_LSA(ByVal z As Integer, ByVal w As 
Integer, ByVal yp As Double) As Double 
  Dim InArray(1 To 4) As Double 
  num_w = CVSession.GetWavelengthCount() 
  If w < 1 Or w > num_w Then 
    w = Val(CVSession.EvaluateExpression("(ref)")) 
  End If 
  If yp = 0 Then 
    yp = 0.00001 
  End If 
  f = 1 
  InArray(1) = 0 
  InArray(2) = yp 
  InArray(3) = 0 
  InArray(4) = 0 
  ok = CVSession.RAYRSI(z, w, f, 0, InArray) 
  If ok = 0 Then 
    dT = Val(CVSession.EvaluateExpression("(y si)"))   
    M_si = Val(CVSession.EvaluateExpression("(m si)"))
    N_si = Val(CVSession.EvaluateExpression("(n si)"))
    TTan = M_si / N_si 
    If TTan <> 0 Then 
      TVal = -dT / TTan 
    Else 
      TVal = 10000000000# 
    End If 
  Else 
    TVal = 10000000000# 
  End If 
  CV_LSA = TVal 
End Function
