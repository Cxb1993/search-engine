子計畫一：叢集式隨意點對點多媒體會議
服務系統及資源最佳化管理之研究與開發 
多媒體會議是一個實現多方即時互動
並且以多媒體應用為主的通訊服務，例如
影像、語音、文字等傳輸。傳統以電信傳
輸模式所提供的遠距視訊會議服務是使用
事先所部署的昂貴設備和專線的方式來達
成，雖然會議傳輸品質較穩定、可靠，但
是也因為成本高以及應用不易開發等原
因，使用的情況並不普及。由於網路技術
快速地發展及普遍地使用於一般的家庭使
用者，並且受惠於網路資源共享及無所不
在的特性，期望能夠藉由建構在以 IP網路
為傳輸媒介的方式來提供廉價、易取得的
多媒體會議服務。 
本子計畫提出一個以 SIP 為基礎的叢
集式隨意點對點系統架構來提供多媒體會
議服務。這個系統架構整合了緊密耦合模
型和完全分散式模型的概念，透過分享使
用者資源來降低系統的通訊成本。此外也
因為其優點互補，能提供一個穩定、可靠
且適合大型服務規模的多媒體會議服務系
統 
 
子計畫三：多層次網路跨曾設計效能優化 
一般的結構化疊層網路如 CAN、
Chord、Tapestry、Pastry 雖具有良好的結構
及特性，如能快速傳送訊息、具延展性和
錯誤容忍等，但實際上它們的效能與其上
的應用及其下的實體網路都有直接的關
係。本子計畫之研究訴求不僅在保留既有
Overlay網路的特性，更是需要充分利用軟
體的彈性，隨著應用、政策和環境參數而
作調整其拓樸結構，不僅對上層應用 
(conferencing and multimedia streaming 
applications) 提供最大的彈性，並兼顧下層
實體網路的穩定性、安全性 (含 VPN) 與
效能。為達成上述目標，本子計畫以資源
管理及品質確保的觀點出發，以最佳化
Structured Transport Overlay網路與其下層
之實體網路間的跨層設計為主要目標 (在
本群體計畫中為以 IP/MPLS 為核心，
WiMax, WiFi, WLAN為周邊的異質網路)。 
 
子計畫四：虛擬路由與轉送之設計及其在
虛擬私有網路之應用 
由於網路的普及，使得人們逐漸希望
可以隨時隨地取得網路資訊及通訊服務，
對於網路服務的依賴日益增加，因而帶動
了無線通訊技術以及相關應用的蓬勃發
展。不論是有線網路技術或無線網路技
術，如何提升網路環境的安全性，一直是
一個不退流行的研究議題，因此我們希望
可以建構一個擁有特定安全層級的網路環
境。為了建構一個無所不在的網路資源，
人們開始亦步亦趨提出相關的研究議題，
而我們也努力的朝向建構一個利於提供行
動性 (mobility) 網路服務的方向邁進，並
且找出面臨的問題及可行的解決方法。 
除了以 VPLS 提供客戶端虛擬私有網
路之外，我們考量客戶端對於資料流進行
分隔的需求，運用 VLAN (Virtual LAN) 的
概念提升網路環境的安全性。例如：同一
間公司會計部門和資訊部門某些資訊不該
互通有無，就可以使用 VLAN的技術將之
分隔，使得公司網路環境安全性更加完善。 
我們將以 Linux Bridge結合 VLAN tag
實作出 Tag-Based VPLS與Mobile VPLS的
網路環境架構，提供客戶端附有安全性及
便利性的通訊環境，並且實現Mobile User
使用Mobile VPLS架構所提供的通訊服務
時，還能保有其原有 IP 位址進行通訊，大
幅縮短換手 (handoff) 的時間。

子計畫四： 
本計畫成功整合 Virtual Router 與
Mobile VPLS 系統架構，並且將原有的
Port-based Mobile VPLS 擴充為 Tag-based 
Mobile VPLS之外，我們考慮到 VPLS架構
中並未限制客戶端只能使用第二層的網路
設備，因此，我們也在 Tag-based Mobile 
VPLS架構中加上第三層的網路裝置，並且
透過 CE 裝置協助路由器執行路由表的更
新，使得移動性使用者可以在移動至另一
個網段下仍使用原有的 IP 位址進行通訊。 
我們也針對所建構的環境進行系統效
能的測量，藉此呈現網路封包處理的延遲
時間以及Mobile User 移動至不同網段時，
系統的反應速度。量測結果顯示當 Mobile 
User 在取得網路連線或通過 802.1X 認證
後，最快只需花 7.8575毫秒即可將路由更
新命令傳遞至路由器，由於在我們的架構
中，移動至不同網路的Mobile User等同於
從總公司移動至分公司，而非著重於快速
換手之議題。因此我們認為在 CE裝置上所
開發的程式足以應付Mobile User在連上網
路後，欲立即進行通訊的狀況。 
 
子計畫六： 
本子計畫提出的系統環境乃基於一般
網路傳輸業者 (ISP)，在其上部署我們的網
安設備 Security Router (SR)、Security Policy 
Decision Function與Mobile Communication 
Controller，建置一架構於其上的軟性安全
防禦網路，提供共同聯防的安全網路環
境。而爲了提供差別式服務，因此我們將
使用者分為三大等級  (Service Level 
Agreement)，提供的網路安全服務內容分成
七項種類 (Protection Type Profile)，使用者
可以依據自己的需求，向我們購買不同的
使用者等級及所需要的安全服務。 
本系統 IDS 使用兩種入侵檢測方式，
利用攻擊型態的特性 (如攻擊封包種類、
大小或針對的 Port)，去做統計後比對根據
歷史資料所計算出來的異常流量門檻值，
以判斷是否被入侵或遭受攻擊，並可察覺
還未知的攻擊。另一種檢測方法為傳統的
特徵比對，先收集已知攻擊的特徵建立一
特徵資料庫，即時比對即時察覺，但只能
偵測已知攻擊。故我們混合這兩種檢測方
式，以防範所有的攻擊入侵。雖然大部分
攻擊是針對某些重要的伺服器或主機，但
也有些攻擊是希望癱瘓整個網路  (如
DoS、DDoS 或蠕蟲等)，因此當 SR 或 MN
偵測到入侵行為時會依攻擊型態判斷出是
否會危害到整體網路的正常運作，若為此
類型攻擊即依照攻擊來源位置搭配
Routing Table與 Notification關聯性，找出
最靠近攻擊來源的 SR 設定防火牆規則將
其隔離，以免繼續危害網路。 
 
四、成果自評 
 
計畫一已完成原訂計畫中第一年研究
內 容 ， 共 三 大 研 究 成 果 ， (1) 在 
Cluster-Based P2P Ad-Hoc Multimedia 
Conferencing 需求分析中，我們發現會議
建立時間之重要性，以及 P2P 結構與 IP 
Address 結構不匹配問題，造成服務搜尋速
度影響；(2)因此針對服務搜尋速度問題，
完成叢集式點對點多媒體會議系統設計；
(3) 最後透過 OverSim 模擬工具，完成本
系統之效能驗證。 
因此，本研究成果與原計畫完全相
符，本研究成果將作為未來延續性研究之
重要基礎，探討叢集式隨意點對點多媒體
會議服務系統資源分配與管理最佳化等研
究議題。
