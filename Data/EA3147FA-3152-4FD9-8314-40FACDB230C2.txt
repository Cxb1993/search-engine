Abstract-- This project is to develop the related technologies to support various AMI functions based on 
the previous results obtained by all professors within the integrated project which has been approved for 
only one year. With the good research results achieved, it will be very important for this research team to 
carry on the further research work to investigate the smart grid by using the ICT technology with 
integration of control modules, application software to support various application functions of the smart 
grid. 
This project is to develop the integration of advanced metering infrastructure and dispersed 
generation to enhance the cost effectiveness of smart grid by considering the existing distribution 
automation system and outage management system in Taipower. Based on the previous project 
“Enhancement of distribution system planning, analysis and operation applying the OMS system”, this 
project will perform develop the application software for various distribution system functions by using 
the OMS to generate the network topology and data for the application. To extend the function of DAS 
to support more advanced customer automation (CAS) application, the power line carrier and Zigbee 
module will be developed to support the communication of last mile application by integrating the 
existing optical fiber network of DAS system in Taipower. In this way, the value added application of 
DAS and OMS can therefore be enhanced by automatic meter reading and demand response. Therefore, 
the integration of PLC, Zigbee and optical fiber is very critical for smart grid application in this project 
to provide effective emergency backup power by DG for essential loadings. The multi-agent system will 
be developed to perform the fault detection, isolation and restoration (FDIR) more efficiently with 
distributed processing capability as compared to the existing centralized DAS system in Taipower. The 
embedded system is developed to support the work order management and switching operation for 
serious fault contingency. The PLC module is developed to support two way communications of 
customers and utility for remote control of customer appliances. The OMS is applied as the data center 
for the distribution system planning and operation. DG units will be included in the computer simulation 
to support the islanding operation of microgrid by determining the timing for tie line tripping and load 
shedding. To increase the DG capacity connected to distribution system, the PV generation system with 
energy storage is developed to study the DG penetration. To achieve better distribution system planning, 
the probabilistic generation models of wind generators and PV generation are developed so that more 
effective design of main transformer expansion planning can be obtained.

場與離場工作區的模糊推論，考慮工作區的模糊工作時間及工作隊的模糊進場與離場時間。對於工
作停電規劃進行最佳化求解。本文以實際配電系統模擬測試所提出方案的可行性。 
 (Ref: 以模糊時間基因演算法求解之配電系統工作停電規劃--中華民國第三十屆電力工程研討會) 
 
2.本研究提出一種混合型進化演算法來求的故障檢修的人員與維修區的最佳的排程策略。本方法考
慮工作區的調度與人員的排班等實際限制條件並結合現場的排程經驗規則，加速求得最佳的特性。
本研究中考慮多組工作隊、多類型工作類別、多工作區的工作時間及工作隊的進場與離場時間。 
(Ref: Intelligent crew and outage scheduling in electrical distribution system by hybrid generic algorithm -
- 2009 International conference of ICIEA) 
 
3.本計劃提出一種改良型的基因進化演算法並且結合 Google -Map 執行配電系統的計畫性停電故
障維修的排程。該改良型 GA 利用特性型動態分佈之交配率與突變率，大幅改善解題的收斂特性與
穩定性，此外，該方法也提出利用智慧型手機讓維修現場的工作隊能夠與維修資源調度中心作即時
性的最佳路線與排程規劃。 (Ref: An Improved GA approach for distribution system outage and crew 
scheduling with Google map integration– 2011 International conference of ICMLC) 
 
4. 本研究提出模擬免疫系統中的抗原、抗體、B 細胞和 T 細胞的專一性、記憶性和適應性等特性，

工作時間差異以及工作隊的於工作區間轉移的行車距離等因素，提出抗體的相似度與集中度的計算

5. 風機故障之臨界清除時間之分析。若系統電壓低於感應機之臨界故障電壓，將使得感應機的輸出
最大感應轉矩低於風機之輸入轉矩，則風機轉速會持續上升，若系統電壓未能在感應機之臨界清除
時間恢復，則風機將會迅速加速而消耗大量之無效功率，必須將風機切離系統。此外，亦完成
STATCOM 對於系統電壓與風機臨界清除時間之影響分析，STATCOM 之容量越大則風機之故障臨
界清除時間越長。 
6. 不同卸載量對孤島微電網系統的影響分析。當饋線微電網外部台電系統發生重大故障，且無法及
時排除而必須解聯形成孤島運轉時，於必要時必須執行卸載策略與維持風力發電機組之持續運轉，
以提高供電可靠度及降低用戶損失。在不同負載與風速情況下，經不同的卸載大小後，檢視系統的
響應，以獲得不同運轉狀況之最適合的卸載量。 
7. 含類神經網路卸載之孤島系統運轉分析。建立在不同風速與負載量下之最適卸載量表格，以此表
作為類神經網路之輸入與輸出訓練資料，經 L-M 倒傳遞法訓練此類神經網路。經由饋線 24 小時之
實際運轉測試，其最大誤差率不會超過 2%。 
8. 電池儲能系統併入後之暫態穩定度分析。模擬風機輸出因風速變化而造成功率輸出變化時，藉由
此電池儲能系統的控制，達到穩定電力系統之目的。 
2-5 子計畫六-蓄電池儲能系統之太陽能光伏電模組分析與研製於智慧型配電系統規劃與運轉之研
究 
本計畫第一年利用並聯負載共振式轉換器於太陽能蓄電池充電器之研製，以較少數的電路元
件，即可達到高效率的特性，將電路操作在高頻下，電路具有體積小、重量輕及成本價格低廉等優
點。藉由挑選適當的共振槽電路參數，使充電器操作在零電壓切換導通情況，可減少開關元件的切
換損失，以達高效率的要求。 
因光伏電模組會受日照強度、環境溫度所影響，故本計畫在光伏電模組中加入閉迴路昇壓式轉
換器，使光伏電模組有穩定的輸出。傳統的切換式轉換器，因開關的散熱與電磁雜訊問題，使轉換
器無法有效在高頻操作，因此本計畫第二年採用串並聯負載共振式轉換器，因開關在零電壓與零電
流切換，切換頻率為高頻，因此不需加入複雜的保護與緩衝電路，使電路體積減小，其效率與傳統
的切換式轉換器做比較，有明顯的改善。

2-8 子計畫九-運用粒子群集及其衍伸演算法於配電系統重構問題之研究 
本計畫提出一合宜有效率的饋線重構演算法，以尋求最適切的開關操作策略，演算法的主體為
粒子群演算法(Particle Swarm Optimization, PSO)，於求解多目標饋線重構問題的範疇則應用了粒子
群演算法結合柏拉圖最佳化(Pareto Optimality)的技術。 
就粒子群演算法應用於單目標饋線重構問題方面，本計畫提出增強型整數編碼粒子群演算法
(Enhanced Integer Coded PSO, EICPSO)，EICPSO 使用整數變數的編碼方式，考量每一個整數變數
代表配電系統中的一個常開開關，並整合了本計畫所提之區域最佳解列表(Local Optimal List, LOL)
的概念，有效利用區域最佳解，以強化粒子群演算法的收斂速度與效能，經模擬驗證的結果，可說
明 EICPSO 已能夠有效應用於求解饋線重構問題，且已發表相關論文於著名學術期刊。 
於多目標饋線重構方面，本計畫參照多目標粒子群演算法 (Multi-Objective Particle Swarm 
Optimization, MOPSO)成功運用於多目標連續函數最佳化問題的文獻，分析 MOPSO 演算法是如何
引入柏拉圖最佳化的概念，擴張粒子群演算法的應用，然後構思 MOPSO 要如何用於多目標饋線重
構問題，並嵌入外部儲存庫(External Archive)與品質函數(Quality Function)的概念於增強型整數編
碼粒子群演算法，從而建構出多目標增強型整數編碼粒子群演算法(Multi-Objective Enhanced 
Integer Coded Particle Swarm Optimization, MOEICPSO)以求解多目標饋線重構問題，目前所發展的
演算法已可成功的求得一組可行最佳開關操作策略解集合，而非傳統權重方法之單一最佳解。 
為驗證本計畫所提演算法效能，本計畫除參考 IEEE 33-Bus 配電系統的與台電公司(Taipower 
Company, TPC)的實際配電系統。為測試更複雜且擁有更多電力開關的系統，本研究擴充了 TPC 的
測試系統(稱之 Modified TPC)，使之擁有比原來多一倍開關之配電系統。另做為對照組之相關演算
法則有基因演算法(Genetic Algorithm, GA)、饋線重構之專家，考量二位元粒子群演算法(Binary 
PSO, BPSO)的缺失，經修正過後的二位元粒子群演算法(modified BPSO, mBPSO)與離散粒子群演算
法(Discrete PSO, DPSO)。各演算法於單目標線路損失的結果中，EICPSO 於每個測試系統皆可將系
統的損失降到最低，且平均獲取最佳解的時間也是最低的，其他做為對照的演算法於較大的測試系
統中則落入區域最佳解，且平均搜尋到區域解的時間也相對的較久。本計畫另驗證區域最佳解列表
機制對於 EICPSO 演算法的影響，由模擬結果可知嵌入區域最佳解列表機制的 EICPSO，其收斂速
度相對較快，且於較大型的配電系統時，其不合法解的比例也比較低。
