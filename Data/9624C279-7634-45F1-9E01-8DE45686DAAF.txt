研究摘要(500 字以內)： 
本計畫開發一項「以語音辨識為基礎的互動式行動廣告服務」，提供廣告主一
種吸引消費者參與行銷活動的媒介。有別於傳統廣告所使用之單向的「推播式」
行銷，本計畫所發展的廣告策略是一種雙向互動的「吸拉式」行銷，透過讓民
眾主動參與廣告宣傳活動的機會，增加其消費欲望。我們利用語音辨識技術，
結合行動通訊，讓民眾「開口」索取贈品或優惠。其過程為撥打手機進入本系
統後，說出某產品促銷活動的廣告詞，因此系統自動辨識來電者所說的內容是
否與廣告詞相符。判斷結果再以簡訊方式回傳給來電者，告知是否獲得贈品或
優惠。初步調查，參與的民眾將基於好奇、有趣、與挑戰贈品的動機進行嘗試，
而在嘗試的過程中，周遭的人也將因為聽到或看到這種「開口」索取贈品的活
動，進而加入來電「開口」的行列。 
 
人才培育成果說明： 
本計畫涉及多方理論、技術、知識、與人文素養，因此參與本計畫的人員除能
習得「數位訊號處理」、「語音辨識」、「資料庫理論」、與程式設計等工程相關
知能外，更能學習關注社會動態與趨勢，並培養創新與創造的能力。在與方寸
經緯公司合作的過程中，參與的學生可獲得許多寶貴的實務經驗，對於學生未
來進入就業市場將有很大的助益。本計畫參與人員已有四位研究生於計畫結束
後取得台北科技大學碩士學位。 
 
技術研發成果與技術特點說明： 
本計畫所發展之「以語音辨識為基礎之互動式行動廣告服務」是國內外極少數
使用自動辨識技術做為廣告服務的核心，也是國內唯一針對廣告應用而發展的
語音處理技術，因此計畫之完成不僅能提供學術研究一項新的思考方向，更能

技術，利用語音辨識與行動通訊管道，提供廣告主一種吸引消費者參與行銷活動

廣告詞才算合格，因此系統除了自動辨識來電者所說的內容是否與廣告詞相符外，
更進一步判斷所說的語氣或口吻，以及模仿之相似與否。判斷結果再以簡訊方式
回傳給來電者，告知是否獲得贈品或優惠。參與的民眾將基於好奇、有趣、與挑

(英文) This invention provides a voice recognition-based interactive advertising system 
including a receiving module, a verification module, and an advertising message 
generating module. The receiving module is used for receiving a voice signal. The 
verification module is connected with the receiving module and used for generating a 
verification result according to the voice signal and a voice criterion. The advertising 
message generating module is connected with the verification module and used for 
generating an advertising message according to the verification result.
