Due to the very nature of wireless sensor networks, applying wireless sensor networks for 
agricultural fields is deemed an important application. Note that sensors have several 
resource constraints such as power consumption, limited computing capability, smaller 
storage space and low bandwidth in communications. These resource constraints on 
sensors call for the design of data management in wireless sensor networks. 
 
Our primary goals include (1) developing data accuracy guaranteed mechanism, (2) devising 
energy efficient data collection methods, and (3) query processing and in-network mining 
mechanisms. More specifically, we will first develop trustworthy relations among sensor nodes 
and explore such relations for identifying faulty sensor nodes so as to ensure the correctness of 
data. Next, by exploiting the spatiotemporal correlations in readings, we can exploit this feature 
to develop in-network query processing by aggregating the similar reading results, and a data 
collection method which selects only few representative nodes for reporting their readings such 
that the goal of energy conservation can be achieved. Finally, we focus on KNN query processing 
mechanisms and develop a group-based object tracking mechanism. 
 
Keywords: approximate data collection, spatial and data correlation, object tracking, group 
mobility, clustering, query processing, KNN, in-network aggregate query, faulty detection
