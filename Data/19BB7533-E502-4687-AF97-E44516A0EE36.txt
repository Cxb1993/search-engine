三、 網路資料庫伺服器建置 
本計畫以 Linux 網路作業系統建置資
料庫伺服器，及作業平台，目前已利用致遠
管理學院教學研究設備，建置一 Linux 網路
作業系統之伺服器，同時藉由網路共享平台
與分散式資料庫架構，目前已將海洋大學黃
文政教授所建置「台灣地區河川流量資料庫
水文分析」之建構介面及資料庫映射至致遠
管理學院，以作為資料庫端對端協定傳輸技
術(peer-to-peer protocol)測試與研發之用，致
遠管理學院所建置網路伺服器網址為
http://cee.dwu.edu.tw/~river，如圖 3. 所示。 
 
圖 3 台灣地區河川流量資料庫水文分析 
為連結即時水情資訊，本計畫除建立
水位資料庫外，另以資料庫聯結更新方式，
試驗階段將連結經濟部水利署即時水情資
料庫，於平時此資料庫透過各河川局「多工
多埠自動化水文資訊傳輸系統」，每日定時
上傳即時水位資料更新資料庫一次，因此本
計畫建置網伺服器即可透過資料庫端對端
協定傳輸技術(peer-to-peer protocol)，進行即
時水位資料更新之架構，而有關即時水位流
量分析模組，將以 Practical Extraction Report 
Language(Perl)程式語言為基礎，目前 Perl
程式語言不僅在 Linux 作業平台發展迅速，
同時目前也支援 Windows 所建立之 Web 
Server 工作環境，因此 Perl 除程式包含了
C、sed、awk 和 sh 這幾個工具最好的特色，
而且主要的語法很接近 C 語言，對原本熟悉
C 語言的人來說非常容易上手外，程式語言
之開發已非常簡便。最重要的 Perl 語言極具
擴充性，亦類似 Linux 建構一全球化綜合程
式軟體模組網路(comprehensive Perl archive 
network; CPAN)，在各大 FTP Archive 網站，
可搜尋上千萬個眾人發展之模組或程式，可
加入欲開發之程式中，例如本研究日流量資
料庫查詢統計分析 Perl 程式介面，即利用數
個模組於程式介面，包括繪圖介面模組
GD.pm、資料庫介面模組 DBI.pm(內含連結
驅動程式模組 DBD::CSV.pm、資料檔存取
模組 Text::CSV_XS.pm 及結構化查詢語法
介面 SQL::Statement.pm)、統計分析模組
Statistics::Descriptive.pm 及數個全球資訊網
常利用之模組 CGI.pm 等，則利用 Perl 編寫
資料庫之全球資訊網程式將輕而一舉可達
成。尤其透過 Perl 獨特處理字串能力正規算
式(pattern matching)，或相關內建函數，因
此本研究採用 Perl 程式語言建構即時水位
流量率定分析模組及資料庫查詢統計，擬利
用 Perl 程式提供強大分析運算功能，而能展
示出完善即時水情資訊與資料庫查詢分析
