附件二 
前言 
   市場研究公司 In-Stat 發表的最新報告，2006 年 Wi-Fi RFID TAG 出貨量為 135,000 張，
預計該市場到 2010 年間，每年都將加倍成長。該機構預測，該市場到 2010 年的規模將
超過 210 萬張 TAG。本研究團隊有鑑於 RFID 已成為全球實務導入與學術研究策略方向，
其中又以 Wi-Fi Active RFID 為即時定位系統為主要發展。本研究團隊將與合作企業
SYRIS 團隊雙方共同發展，目標為發展出一套國內自主技術之 RFID 室內及時定位系統
（Indoor Real Time Location System），該企業具備設計與生產主動式 RFID 硬體的團隊。
國際標準組織 ISO 於 2007 年正式批准並公佈了 2.4GHz 主動式  RTLS 技術標
準—ISO/IEC 24730。ISO/IEC 24730 為企業提供 2.4 GHz 有源 RTLS 系統應用的空中介面
協定和應用程式介面。 
 
研究目的 
    LBS 定位服務為基礎且最為人所熟知的全球定位系統 GPS ，主要是藉由用戶端 
GPS 接收器接收 3 個以上之定位衛星訊號並利用三角定位法進行定位運算，其精確度
可達到 5m 至 40m；但其缺點為 GPS 接收器必須與衛星保持視線(Line Of Sight, LOS)
才能精確地收到訊號，如此就限制了在室內環境之行動端位置感知的相關應用範疇。本
研究團隊以長榮大學 RFID 研究中心為發展基礎，透過與「璽瑞企業」友善的產學互動，
並取得該主動式 2.45Ghz 頻段 Reader 之 API，進行深入研究，將以開放資源型平台軟體
來整合其硬體物件事件庫，快速且直接的回應到企業應用系統本身所需之「室內及時定
位系統」。本研究將以建構出一主動式 RFID 自由軟體定位系統共用平台，追求技術本土
化的發展目標。本研究將發展主動式 RFID PDA READER，實際接收到資產定位的 TAG
編碼，並呈現出 RFID 資產盤點功能。本研究採固定式 TAG 在平面位置上，並透過主動
式 Reader 不斷的接收訊號，進一步得到移動位置的參考點，達到定位的功能。 
 
文獻探討 
    WhereNet 公司認為 ISO 24730 標準對於即時定位系統應用，有助於進一步解決客戶
應用需求，擴展供應鏈應用，增強 RFID 標籤的實用性和有效性，提高硬體設施的便捷
度和降低標籤價格。ISO/IEC 24730 標準共包含兩個部分： 1. 應用程式介面：包括可延
伸標記語言 XML 和簡單物件導向協定 SOAP，保證多 RTLS 系統/產品分享定位資料資
訊。2. 空中介面協定，分為三部分：(1)定位 2.4GHz 標籤/讀寫器空中介面和定義網路定
位系統的 x-y 座標；(2)低頻(120 kHz 左右)設備的通訊協定；(3)開關通訊協定，允許
2.4 GHz 主動標籤回應簡單、低成本的掌上型讀寫設備。Ni (2003) 提出了以 LANDMARC 
分析室內定位，主要的優點是它可以使定位的正確率提高、不需要用到很多的 RFID 
Readers、可以使用便宜的 RFID tags，他們使用了二種不同的 RFID Tags，分別是當作是
Reference Tags (RT) 及 Tracking Tags(TT)，在環境的佈置中 RT 是固定位置的，且 RT 位
置資訊是事先系統己知，RFID Readers 接收 RT 訊號強度資料 LANDMARC 系統判斷
TT 其可能的位置資訊是位於那三個 RT 間，這個方法可以逹到精密度 2 公尺以下，但
是事實上根據我們實際運作經驗，仍很難達成。Saxena et.al, (2007)提出虛擬空間的概念。
由事先佈署於實驗環境中的 RT 所建立的虛擬空間。需要找尋欲追蹤的 Object Tags 的位
置時，利用資料庫記載的虛擬空間中鄰近的 RT，可經由相對位置而找到約略的放置點。
當要被定位的 RFID Tags 的訊號被三台以上的 RFID Readers 接收到時，便可利用接收
到的訊號強弱計算出 RFID Tags 的相對應位置。然而，三角定位法較適合使用在空曠的
空間裡，沒有障礙物的阻隔，可得到較為準確的結果。但在一般的室內環境中，有許多
研究方法 
本研究將結合 RSSI 收訊強度 RSSI 與到達時間差 TDOA 兩種之間的各自優點，透過
兩者同步進行之實驗設計與參數累積，進行室內及時地位系統演算法的開發，當中也必
須考量 NLOS，當 TAG 與 Reader 之間的直射路徑被障礙物擋住，無線電波只能在經過
反射和衍射後到達接接收端，此時測量到的數據，如到達時間、時間差、入射角度等，
將不能正確反映發送端與接受端的真實距離，這種現象被稱為非視距傳播（NLOS）。非
視距傳播造成定位演算法性能的顯著下降，事實證明即使在無多徑效應和採用高精度定
位技術的情況下，NLOS 傳播也會引起測量誤差。 
 
（1）RFID RSSI 強度實驗 
測試人員攜帶一張名片型 tag，從 Reader D 走至 Reader C，再返回 Reader D，測試時間約
為三分鐘，並以每分鐘為一基本單位，分析每一台 READER 的訊號強度值。 
設定參數： 
1. 當 TAG 距離 READER 距離 N 公尺內的訊號強度區間值（RSSImax），以此訊號強度值
判斷 TAG 是否於 READER 附近。 
2. 取得 READER 最遠的偵測範圍（Rangemax），若該範圍無法確定，需要進一步的測試與
實驗，並且假設最遠範圍的訊號強度值（RSSImin）。 
附註：假設得到 RSSImax、RSSImin 與 Rangemax 三個數值，可藉 RSSImax 與 RSSImin 求取最高
訊號強度平均值（RSSImaxAvg）與最低訊號強度平均值（RSSImin），藉由（RSSImaxAvg - RSSImin）
/ Rangemax 得到遠離 READER 一段距離（Distance），會造成訊號衰減的數值

A.第一分鐘分析資料 
根據收到的實驗數據發現，14 分鐘內大多為 R1 的訊號，因為此時移動人員位置較靠
近 R1，而系統僅收到兩筆 R3 訊號資料，而且沒有收到 R2 的訊號資料，因此以下僅
對於 R1 進行分析。由下圖 1 可發現，假設除去誤差較大的數值，可得到一個下降的曲
線，代表 Tag 逐漸遠離 R1 的範圍。 
B.第二分鐘分析資料 
根據收到的實驗數據發現，15 分鐘內夾雜 R1、R2、R3 的訊號，因為此段時間移動人
員（TAG）正由 R1 點往 R3 點移動，。由下圖 2 可發現，假設除去誤差較大的數值，
可得到一個上升的曲線，代表 Tag 逐漸進入 R3 的範圍。由下圖 3 可發現，假設除去誤
差較大的數值，可得到一個下降的曲線，代表 Tag 逐漸離開 R3 的範圍。由下圖 4 可發
現，假設除去誤差較大的數值，由於 tag 剛開始靠近 R1 的範圍，所以 R2 收到 tag 之
信號較不明顯，訊號強度曲線呈現較為平緩，當 tag 進入 R3 範圍，根據畢氏定理可得
到 tag 距離 R2 越來越近，也就是說，tag 已經進入 R2 的偵測範圍，所以信號強度明顯
增強許多。由下圖 5 可發現，假設除去誤差較大的數值，可得到一個上升的曲線，代
表 Tag 逐漸進入 R1 的範圍，可能已經逐漸遠離 R3 偵測範圍。
