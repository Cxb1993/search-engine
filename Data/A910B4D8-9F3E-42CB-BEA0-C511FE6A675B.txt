極紫外光直寫微影技術之真空系統設計與製造(3/3) 
 
中文摘要 
 
    本子計畫為三年期整合型計畫，是「以波帶片陣列為基礎之極紫外光直寫微影系統之研發」
的第四個子計畫。本子計畫負責極紫外光直寫微影系統的真空系統的設計與製造。本子計畫已依
據前兩年之計畫工作項目發展了一套參數化繪圖程式設計 EUV系統，並且完成了 EUV系統之真
空腔體配置、分析了真空腔體結構安全性與抽氣性能。第三年針對抽氣性能調校訂定了抽氣操作
步驟，對 EUV 系統的初型製作進行了系統性能測試，包括了真空幫浦抽氣性能測試，烘烤過程
對壓力時間曲線之影響。EUV系統在第三年與其他子計畫進行了實驗整合，將平台與光學鏡組放
入了 EUV系統中。抽氣實驗結果證實了，以此子計畫發展的參數化繪圖程式所設計的 EUV系統，
不僅製作成本低，系統操作方便，也的確能夠有效提供足夠的實驗環境進行 EUV光學實驗。 
 
關鍵字：真空腔體、真空系統、極紫外光微影技術、參數化設計

1. 文獻探討 
此子計畫所負責的主要內容為極紫外光直
寫微影系統技術之真空系統設計與製造。從早
期體積龐大、成本昂貴的 EUV 真空系統[1]，
依照其他子計畫實驗硬體設備的空間要求，改
良成體積適中，成本較低的 EUV系統。本實驗
室共製作過兩代真空腔體系統，第一代及第二
代真空度皆要求介於 1.0E-8 Torr等級範圍，
實驗溫度設定於 373K以上。實驗環境要求皆高
於極紫外光微影技術之實驗環境需求。極紫外
光微影技術實驗真空度為 1.0E-5 Torr 至
1.0E-7 Torr，實驗溫度設定為 290K。EUV系統
主要參考了美國專利[2]。此篇專利內所設計的
真空腔體系統是根據 EUV光學鏡組架設路徑所
設計，與此子計畫所設計的真空腔體有相關性。
EUV系統中的真空元件包含真空幫浦、壓力計、
控制閥、連接管以及烘烤加熱器等[3-5]。本子
計畫參考 EUV系統之真空腔體及真空元件的設
計規範，發展了一個參數化模型程式，可在電
腦輔助設計軟體中迅速自動產生腔體結構之三
維實體模型[5-7]。真空腔體之實體模型可傳入
有限元素分析軟體進行腔體之結構分析，確認
腔體結構的安全性。最後將設計與製造真空腔
體及真空系統之初型進行整合，進行相關抽氣
實驗及抽氣性能改良[8-9]，比較烘烤過程對系
統抽氣性能的影響，根據抽氣性能結果，調整
操作系統的步驟，以達到最佳效率的抽氣性
能。 
 
2. EUV系統之整合設計與性能量測 
    本子計畫在第一年發展了一套參數化繪圖
程式，針對極紫外光直寫微影系統技術設計了
一套 EUV 真空雙腔體系統。在第二年利用
ANSYS 軟體分析確認腔體的結構強度安全無
虞。在第三年則針對 EUV系統的初型製作進行
整合與抽氣性能測試。

2.3 EUV系統真空系統之性能測試 
   此子計畫對EUV系統做了初步的性能測試。
不對兩個真空腔體進行腔體加熱烘烤，而僅僅
進行抽氣實驗。當真空壓力值為 2.0E-2 Torr
時開啟渦輪分子幫浦，實驗時間設定為 48小時，
溫度設定為 298K。由圖 2-3-1的實驗數據圖可
以得知 EUV系統在抽氣 48小時後，腔體內真空
壓力為 7.9E-8 Torr。 
 
圖 2-3-1 EUV系統性能測試數據圖 
2.4 EUV系統烘烤過程壓力時間曲線之
性能量測 
    經由 2.3的性能測試實驗結果，可知 EUV系
統在未有烘烤過程的情況下，連續抽氣 48小時
後，所達到真空壓力為 7.9E-8 Torr。之後再繼
續測試，進行抽氣實驗，也依然是 7.9E-8 Torr，
因此可得知 EUV系統的真空壓力極值為 7.9E-8 
Torr。為了要縮短實驗時間，提高抽氣速率，本
子計畫購買了加熱元件，將加熱帶包覆在腔體外
表面上，如圖 2-4-1。
 
圖 2-4-1 包覆加熱帶的 EUV系統 
    
   依照 2.3相同的抽氣步驟，在真空壓力值為
2.0E-2 Torr 時開啟渦輪分子幫浦，實驗時間
為訂定為 48小時，加熱帶加熱溫度設定為 358K 
，預估加熱烘烤腔體時間 8小時之後就關閉加
熱器，但繼續讓 EUV系統進行抽氣實驗。在有
加熱烘烤腔體 8 小時的情況下，EUV 系統到達
真空壓力極值7.9E-8 Torr時，僅需18.7小時。
在 2.3得知，若沒有烘烤真空腔體而直接進行
抽氣性能實驗，則需要 48小時才能達到 7.9E-8 
Torr。比較圖如圖 2-4-2所示。為了確認有烘
烤與沒有烘烤真空腔體對抽氣壓力時間的影響
性，再次進行一次實驗，以抵達 8.5E-8 Torr
為目標。以沒烘烤作為對照組，而有烘烤作為
實驗組。實驗結果如圖 2-4-3，對照組需要
27.33小時，真空壓力抵達 8.5E-8 Torr，而實
驗組只需 14.4小時，就可到 8.5E-8 Torr，實
驗時間將近為對照組的一半。由此證明了，烘
烤過程是對腔體內部真空壓力是具有一定的影
響程度。

The vacuum components library including parametric 
information of the vacuum components is a library in the CAD 
system that wraps portions of the Windows API in C++ classes. 
The functionality enables vacuum components library to use a 
default application framework to create the vacuum system. The 
flowchart of the presented CAD system for vacuum system is 
shown in figure 3. 
The flowchart is about the designing process of the vacuum 
system. The first step is to input the specifications of vacuum 
components into the CAD system, and then locate the 
characteristic points of each vacuum components by the 
dialogue interface which inlayed in the presented CAD system. 
Thus, we obtain the parametric models of vacuum system 
effectively and conveniently. The locations of vacuum 
components in the vacuum system are also accurate. The 
location parameters for vacuum components are inputted into 
customized dialogue box and saved in a parametric text file. 
The AutoLisp program can read this parametric text file and 
then construct the CAD models automatically.

CONCLUSION 
This paper develops a parametric CAD tool for generating 
solid models of vacuum systems. Parameters for vacuum 
components are specified in a user dialogue interface, and solid 
models of the vacuum system are built automatically. The main 
design procedure has been introduced for interactive generation 
of Con-flat flanges of vacuum chambers. The design procedure 
simplifies the dimension input and covers the wide range of 
parametric parameters. The functions of developed CAD tool 
include building parametric models of the vacuum system and 
also analyzing the pumping efficiency of the designed vacuum 
system.
