摘要 
本研究之目的在於探討平面八連桿雙蝴
蝶機構之無缺陷尺度合成，所提出之方法可
適用於函數機構、路徑演生機構以及剛體導
引機構之尺度合成。 
針對各型平面八連桿雙蝴蝶機構的構
造，本文首先根據已知條件──輸入桿及輸
出桿之需求角位置，提出一套獨立迴路篩選
法則，選擇出最適當的三個獨立迴路。再以
此三個獨立迴路的位移方程式，來求得演生
機構輸出桿角位置與耦桿點位置之顯函數表
示式。由於演生機構之輸出桿角位置或耦桿
點位置，會與原輸出桿之需求角位置或耦桿
點位置存在誤差值，求得此誤差值之方均根
並無因次化後，即可建立最佳化尺度合成之
目標函數。另外，並藉由探討雙蝴蝶機構之
相鄰分支與死點之 Jacobian 值，歸納出避開
順序、迴路與分支缺陷之限制式。然後，將
目標函數配合所歸納出之避開缺陷限制式，
以 SQP 最佳化方法進行尺度合成，得到符合
設計需求之無缺陷機構。 
本文針對各種不同設計問題，利用上述
方法推導出各型雙蝴蝶機構最佳化尺度合成
所需之目標函數，並以範例驗證其正確性。
藉由範例之結果可得知，以本文提出之方法
所合成出之機構，不僅能夠達成所設定之設
計需求，且在運動過程中亦能夠確實地避開
各類缺陷問題。 
 
關鍵字：平面八連桿組、雙蝴蝶機構、尺度
合成、順序缺陷、迴路缺陷、分支缺陷 
 
Abstract 
The main purpose of this project is to 
propose a unified method for the defect-free 
dimensional synthesis of planar double butterfly 
linkages for function generation, motion 
generation and path generation. 
 For each double butterfly linkage, a 
selective methodology is proposed for choosing 
three suitable independent loops according to 
the specified input and output variables. The 
explicit equations relating the generated output 
variables and coupler point positions are then 
derived by eliminating and transforming some 
angular variables in the three independent loops 
and by separating the output link into two links. 
The objective function of the optimal synthesis 
is defined as the dimensionless root mean 
square of deviation of the angular positions of 
the output link and/or the coupler point 
positions between the generated output and 
required positions. The Jacobian limit of each 
double butterfly linkage is used as the constraint 
equation for avoiding order, circuit, and branch 
defects in the optimal synthesis. Thus, defect- 
free mechanisms can be synthesized by using 
the Sequential Quadratic Programming method. 
For different design problems, this project 
provides several examples to demonstrate the 
feasibility of the proposed method. The results 
of the examples show that the synthesized 
mechanisms not only satisfy the design 
requirements but also avoid defect problems. 
 
Keywords: planar eight-bar linkage, bouble 
butterfly mechanism, dimensional synthesis, 
order defect, circuit defect, branch defect

研究方法 
在雙蝴蝶機構中，根據所選擇之桿件不
同，共有七種不同的子迴路形式。本文提出
一方法，用以選擇最適當之三個獨立迴路，
進而建立最佳化尺度合成之目標函數。三個
獨立迴路的選擇步驟如下： 
1. 第一個獨立迴路需包含固定桿、輸入桿及
輸出桿，且其所有角位置變數扣除已知變
數（即輸入桿與輸出桿之角位置）後，所
剩下之未知變數總數需等於 2。利用第一
個獨立迴路的兩個純量方程式，可推導得
兩個未知變數的顯函數表示式。 
2. 第一個獨立迴路以外，在其餘的六個迴路
中，其所有角位置變數扣除已知變數及步
驟 1所求得之兩個變數，剩兩個未知變數
之迴路可作為第二個獨立迴路。這兩個未
知變數即可用第二個獨立迴路之方程
式，解得其顯函數表示式。 
3. 第一個及第二個獨立迴路以外，其餘的五
個迴路中，包含輸出桿且其所有角位置變
數扣除已知變數及步驟 1、2 所求得之四
個變數後，僅剩一未知變數的迴路，可選
取為第三個獨立迴路。 
若利用以上步驟完整地排列出三個獨立
迴路，以第一型雙蝴蝶函數機構為例，共可
得到 33種獨立迴路組合。事實上，所選擇之
獨立迴路組合不同，推導未知變數之顯函數
表示式的工作量亦有所差別，考慮到減少計
算上的複雜度，未知變數求解次數較少的獨
立迴路組合為較佳之選擇。 
另外，為了清楚地表示其演生機構之構
形，在演生機構之構圖過程中，勢必將第三
個獨立迴路之輸出桿件拆解成兩個分離桿
件，包括原機構之輸出桿及演生機構之輸出
桿。同時，宜保持前兩個獨立迴路構造的完
整性，以便將該組合適當地建構為演生機構
之構形；若是無法保持前兩個獨立迴路之構
造完整，則表示所選擇之組合無法繪出適當
的演生機構構形。

(2) 避開迴路缺陷之限制式 
若雙蝴蝶機構在其運動範圍內，並不存
在死點位置，則可在八組演生機構之輸出桿
角位置顯函數表示式中，選定一組來進行尺
度合成，即可避開迴路缺陷。若雙蝴蝶機構
在其運動範圍內具有死點位置，則迴路缺陷
無法僅靠選一組顯函數表示式來避開，因在
同一組顯函數表示式中進行尺度合成，仍有
跨越不同迴路的可能性[7-9]。但可確定的
是，若能夠避開分支缺陷，就能避開迴路缺
陷，因此本文打算用避開分支缺陷之限制式
來同時避開迴路缺陷，故針對避開迴路缺陷
之限制式不需另外訂定。 
(3) 避開分支缺陷之限制式 
Chase 和 Mirth[5]明確定義分支之辨識，
當 Jacobian 值變換正負號時，即代表機構跨
越死點至其他分支；而當機構處於死點構形
時，其 Jacobian 值則為零。

結論與建議 
茲整理本研究之主要成果與未來研究方
向於下：

1. 針對平面八連桿雙蝴蝶機構，本文提出一
套獨立迴路篩選法則，藉由所給定之己知
條件（輸入桿與輸出桿的需求角位置），
來判斷並選擇在推導演生機構之輸出桿
角位置的顯函數表示式所需之三個獨立
迴路。 
2. 針對平面八連桿雙蝴蝶機構，本文透過已
知條件──輸入桿及輸出桿之需求角位
置，再以所選擇之三個獨立迴路，透過變
數消去與轉換，以及輸出桿拆解之方法，
來求得演生機構輸出桿角位置與耦桿點
位置之顯函數表示式。其中，演生機構之
輸出桿角位置或耦桿點位置，會與原輸出
桿之需求角位置或耦桿點位置存在誤差
值，求得此誤差值之方均根並無因次化
後，即可建立最佳化尺度合成之目標函
數。 
3. 針對無閉合解之平面八連桿雙蝴蝶機構
之無缺陷尺度合成，本文針對雙蝴蝶機
構，歸納出避開順序、迴路與分支缺陷之
限制式。其中，在避開分支缺陷之部份，
本文利用 Jacobian 值在各個設計點中不
得變號之需求，藉以建立避開分支缺陷所
需之限制式。然後，將目標函數配合所歸
納出之避開缺陷限制式，進行無缺陷最佳
化尺度合成，以合成所需之無缺陷機構。
並列舉範例以驗證其正確性。
