研究摘要(500 字以內)： 
 
關鍵詞：超精密車床，Z 軸床台，導軌 
本研究計畫在於配合銘家機械股份有限公司，設計並製造一種超精密車床用的Z 軸花崗
岩床台基座，上面舖設有不鏽鋼導軌，使導軌在300 mm 的行程內有上1µm 的直度精度，當
裝上線性馬達配合光學尺後，欲使Z 軸上線性移動的刀具軸具有10-7 m (次微米)的定位精度
及10 nm的解析度技術。若刀具軸在此床台上將利用線性馬達帶動配合精密光學尺的回饋控
制，配合反扣裝置可用來加工在超精密車床夾頭上之工件以達成奈米級的精度水準，即鏡面
加工的效果。由於Z軸床台基座是採用花崗岩材質，具有不變形的優點，但須經過加工且要
研磨 (Lapping)及人工鏟花。至於鎖在Z 軸花崗岩床台上的不鏽鋼導軌採用 420 J2 材質，
經熱處理後也須研磨加工，下方之固定導軌座與上方之V 行滑動導軌配合，藉由鑽石膏當介
質互相推磨，以達成鏡面之表面光度，且在300 mm 的行程內要有1µm 的真直度，若配合精
密光學尺的回授配合，利用氣靜壓技術，使活動滑軌與固定導軌成非接觸式之組合，將可使
刀具座在滑軌上有最佳的定位精度效果。同時刀具座將採用碟型彈簧作反扣以確保其在導軌
上作精確的直線運動，以便作鏡面加工或是鋁料、銅料等大曲面反射鏡之加工。由於國外製
造的超精密車床甚為昂貴(約900~1000 萬元)，本研究已設計出經濟實用的超精密車床Z 軸
床台，導軌及進給系統技術，無論是從零件圗設計開始、裝配圗、尺度標準或是如何加工製
造精密組件，並配合不同的量具及量測方法，皆是本研究計畫的重點。本計畫完成後，組裝
出一高剛性、高精密度的Z 軸床台組合，將來裝上德製線性馬達(Auto Servo) 通電後，利用
Heidheim 光學尺做回饋，在恆溫約20 °C 的控制下，可以使用於高速而精密之車床鏡面零
件加工，並作為業界及學術界之參考。至於定位系統也將於Z 軸床台及導軌完成後於明年申
請計畫繼續完成。 
  
Abstract(英文)  
Keywords: Ultra-Precision Turning Machine, Z Bed, Guide way, 
 
The main purpose of this planning is to support a cooperate company, Ming-Chia to design a 
Z –Bed, guider and the feeding system in the Ultra-Precision turning machine. The stainless 
guiderare locked on the Z-Bed, which straightness is 1µm during 300 mm stroke. If the tool saddle 
assembles on this Z-bed, its feedbacks using an optical scale and the tool saddle drives by a linear 
motor, then it can get a 10-7 mm position’s accuracy and the resolution is 1nm. If we set a spindle 
linear moving on this Ultra-Precision Turning Machine’s Z bed (includes feeding system) which 
can process a workpiece to get a Nano accuracy and no Albe error during in 300 mm stroke. The 
bed makes from good quality stone, which has highly stiffness and undeformation properties, it 
needs to manufacture to become a V form, then lapping, finally scratching. The guiders make from 
420 J2 stainless steel, after heat treatment, then lapping it to straightness becomes 1µm. Using an 
optical scale for feedback system, let the tool saddle slides on this guider, it can get an excellent 
position accuracy. Because we input the Ultra-Precision turning machine from foreign country is 
expensive and difficulty (costs nearly 9000,000~10000,000 NT$) , if we can design and build it by 
our self, which are not only save the cost of graphics, machine parts design, processing but also to 
improve the skill of the inspection, assembly and testing skill of our machine tool manufacturing. 
They are both key points and significant to us. So I hope I can get the financial support from NSC, 
and I wish I will be finished it as best as I can.
