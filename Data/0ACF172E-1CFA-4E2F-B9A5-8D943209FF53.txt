2中文摘要
因應台灣高齡化社會之趨勢，本研究擬研發出一套適合高齡者的新型態銀髮
族居家衛浴輔具系統，透過人因工程、工業設計、電腦通訊工程以及科技應用等
的觀點，並與本校電腦與通訊工程系共同合作，提升現有輔具產品層次，藉由科
技可能的應用及人性化的設計，解決現有輔具的不足，不僅針對事故發生防範，
更進一步的思考事故發生後如何及時的給予協助、告知他人意外已發生等，以增
加產品價值，提升市場競爭力。
本研究將建構一實驗系統，來探討新衛浴輔具導入居家在設計開發流程所帶
來的影響變數，研究方法與步驟如下：以新衛浴輔具設計為例，並實際探討新衛
浴輔具設計案的開發流程，在設計討論的過程中適時的加入此系統，試圖找出居
家新衛浴輔具在設計過程中所產生的影響及效益，再輔以問卷訪談的方式了解廠
商、使用者，對於此設計開發流程中的價值與益處，並作綜合分析與討論在設計
討論過程中所產生的互動關係。
期望本研究所完成的高齡者居家衛浴輔具設計系統產品設計能令每個家庭
憂心的高齡者起居問題獲得解決，並還給高齡者安心、自在與尊嚴的居家生活空
間。不僅能給高齡者一個安全無虞的衛浴空間，對其他家庭成員的使用亦是種保
障。
關鍵字：衛浴輔具、銀髮族、無障礙空間
4人才培育成果說明：
因為本計畫的屬性為研究設計與開發，所以參與計畫的研究團隊藉由計畫的
執行，不論是研究初期的文獻探討與資料蒐集，或是研究調查資料的統計分析，
甚至經由發現的問題，轉換為設計規範與要素進行設計開發，研究與設計流程完
整的執行。對於銀髮族意外發生的情形、銀髮族在衛浴空間使用上的困擾，以及
現有的衛浴輔具概況，均有深入了解。不僅在研究的養成乃至於設計的訓練，都
歷經實戰的洗禮，對於研究團隊的專業素養有著極為顯著的成長。
本研究的研究團隊在計畫執行過程中，對於台灣的居家衛浴空間現況及現存
的衛浴輔具，以及銀髮族在使用衛浴空間所遭遇的問題有深入的了解，並將發現
的問題轉化成設計規範與要素，經構想發展→提案檢討→模型試做→Prototype
製作，共計產出新型態衛浴輔具1個系列(含6件)，研究成果合乎預期計畫，相關
研究也已發表於2010華梵大學設計與文化學術研討會，產品設計成果也積極申請
專利中。
技術研發成果說明：
本研究實務的設計方針主要是以通用設計觀點，將科技帶入衛浴空間，以創造更
友善的衛浴環境，所得結果如下：
1. 操作的方便性：透過科技簡化操作流程與介面的設計。
2. 環境搭配：材質與色彩的選擇以衛浴空間為主要考量。
3. 造形：以簡單及溫潤作為造型的依據，達成「友善空間」的訴求。
4. 產品的安全性：針對使用者在浴廁空間所發生的意外做事前的預防以及事後
的援助。

本計畫所研發之高智能整合型衛浴設備，主要是利用各種感測器之整合，並搭
配超音波感測技術來提供智能化的貼心服務。我們可以利用對超音波訊號的感
測來紀錄使用者之進出狀況，並自動開始或關閉照明、UV光殺菌等功能。由此
所設計的一款高智能整合型衛浴設備，可與其他非接觸式量測元件作整合，提
供自動感應與即時的燈光照明，以及能夠確實偵測到使用者在該衛浴設備中的
狀態。當非接觸量測元件判定使用者在浴室中摔倒或昏倒時，可以立即給予警
示與自動完成應有之緊急通報處理動作，此需求是特別為了年長或有特殊疾病
之使用者而設計的。另外，還加強了自動化殺菌以及溼度控制之設備，並可藉
由資訊平台主機與其他功能結合實現系統整合。
利用上述之方式，能夠完善地提供使用者高規格的生活要求。
The objective of this project is to design an intelligent and
automatic system for bathrooms. The automatic system is designed to
integrate a number of sensors including infrared sensors, temperature
sensors, humidity sensors, UV light, video camera, dryer, etc. As an
elder user is in the bathroom, this system can detect the state of
whether he/she is in a falling state. If it is so, this system will
notify outside. At the same time, the system will issue an alarm to
the other family members. In addition, this system can detect the
humidity of the bathroom. If the bathroom is too wet, the dryer fan
will be activated accordingly. If the bathroom is too dark as a
person is in it, the system will detect the situation automatically
