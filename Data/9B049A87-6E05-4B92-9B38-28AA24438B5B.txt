improve its security and flexibility. In this study, 
we focus on reaching the specifications of CSI 
monitoring systems by deploying the hardware and 
implementing the system to do the systems stress 
testing in real environment. This research includes 
the following seven key developments: 
(1) Originate an active and passive mix-mold RFID 
Inventory system. The system will adopt 
active and passive mix-model RFID readers to replace 
the passive RFID inventory model. 
We can use different RFID readers in different 
context. The system not only can inventory 
automatically but also can increase the accuracy for 
several times. 
(2) Originate the SOA architecture of cloud computing 
to develop the web service systems. All computing 
logics will be stored in the service layer and all 
the services will be deployed in distributed ways. 
The systems will be flexible, high efficiency and 
suitable for different kinds of information 
environments. 
(3) Originate the Smart Seal System. This study 
develops the Smart Seal System for 
None-Line-of-Sight container intrusive detection, 
automatic information pushing, GPS, 
Google Maps GIS. The system is also developed with 
the goal of patent invention. 
(4) This study develops a decision support system 
with automatic information pushing service. The 
integration ability will be the main target of this 
study. 
(5) The data produced by the systems will be 
encrypted. 
(6) This study develops a container security 
monitoring system to achieve the specification of 
CSI. The CSI(Container Security Initiative, CSI) was 
proposed after 911(2002). 
(7) The study will be cooperated with RFID 
manufacturing enterprises.
