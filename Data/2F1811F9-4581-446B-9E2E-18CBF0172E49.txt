1. 前言  
近年來，隨著時代的演進、科學的進步與市場的需求，產業界已從目前的半導體
(Semiconductor)工業逐漸發展成微機電(Micro Electro Mechanical System, MEMS)工業。MEMS
是以原本用於微電子產業的半導體製程技術來製作微米（百萬分之一公尺）尺度的機械結構，
並整合為一種微型系統，所應用到的層面包含機械、電子、物理、光電、電聲、通訊、材料、
控制、生化醫學、化學等[1]。 
在經歷長時間的技術研發與經驗累積，MEMS的技術發展已獲得相當大的進步，其中以表
面微加工製造技術所製作的智慧型感測器最早商業化，如汽車用的壓力感測器就是最先商業化
的MEMS產品之一。現在新車上都具備MEMS產品，如進氣岐管絕對壓力(MAP)引擎感測器、
主動式懸吊系統所使用的加速度計、自動門鎖、防鎖死煞車系統、安全氣囊、電子式穩定控制
系統(Electronic Stability Control)以及胎壓檢測器[2]。 
目前MEMS的應用已經從工業領域擴展到其他如通訊、資訊以及消費型的市場，特別是在
遊戲方面的MEMS裝置，早在1998年ADI已開發出第一款針對遊戲產業所設計的低重力(Low G)
雙軸加速度計，到近年日本任天堂公司所推出的Wii遊戲機所具備3 軸加速度計的遙控器，甚
至是在2008年由台灣工業技術研究院研發，結合動作偵測與空間定位功能，具備3軸加速度計
與2軸陀螺儀晶片的CyWee Z控制器[3]。 
現在市場趨勢及科技產業演進，各種產品都傾向輕薄短小，而在產品內部的微小MEMS
裝置中所應用的材料更是縮小至微、奈米尺寸。由於在元件被微小化的同時，其材料特性會展
現出許多異於宏觀世界中同樣塊材的物理原理與化學特性，並且會隨著製程的環境與參數的不
同而改變，所以在進行微小薄膜結構分析時，塊材的機械特性即不再適用。因此假使能得知薄
膜材料正確的機械行為，就能提高MEMS元件的性能、精確性、穩定性、可靠度與製程品質良
率，並且降低成本。所以如何得知微小材料的機械行為就變得極為重要。 
以往的參考文獻，都只著重於求取薄膜材料的楊氏模數與一些文獻外加求取的殘留應力，
未量測薄膜的波松比，如此是無法完整的提供作為MEMS製程的參考依據。有鑒於此，本文嘗
試提供一種能測量薄膜材料楊氏模數(Young’s Modulus)、波松比(Poisson’s ratio)的方法。 
2. 研究方法 
本研究主要是以透過MEMS製程製做出一鎳微懸臂樑，採用微懸臂樑試片測試法中的非週
期性變化之靜電(Electrostatic)驅動方式，搭配相移術電子光斑影像干涉術 (Electic Speckle 
Phenomenon Interferency, ESPI)（如圖1所示）並依據微懸臂樑受靜電負載之變形理論公式獲得
鎳微懸臂樑的楊氏模數，再結合奈米壓痕試驗以截取鎳微懸臂樑的波松比。同時使用ANSYS
套裝軟體來進行有限元素數值模擬，驗證靜電驅動相位移光學干涉術試驗結果。

4. 實驗結果與討論 
4.1 試片製作 
試片製作方式主要是利用微機電系統製程技術中的微電鍍製程技術所製作而成，電鍍製程
參數如表1所示。 
4.2 懸臂端20000μm x 2000μm x 26.33μm之鎳微懸臂樑 
在懸臂端 20000μm x 2000μm x 26.33μm 之鎳微懸臂樑的靜電力驅動實驗中，施加直流電
壓範圍約為 0～5 伏特，試片測得最大平均之面外位移量約為 4.29μm（條紋數約 14 條）。將試
片自由端位移量根據由 ANSYS 模擬分析，雜散電場相對於自由端位移量的影響（如圖 2 所

的值再求平均，可得到楊氏模數為 172.368±1.58GPa。 
由於懸臂端長度 24mm 的微懸臂樑試片是和懸臂端長度 16mm 的微懸臂樑試片在同一匹
晶圓上以相同電鍍參數製程出來的，所以其奈米壓痕測試的數據可直接採用懸臂端長度 16mm

在 ANSYS 模擬中，將實驗所計算得到的懸臂端 24000μm x 2000μm x 31.97μm 鎳微懸臂樑
的材料參數代入後，施以電壓 3.2 伏特，所得之位移量如圖 16 所示，模型最大位移量為
2.1835μm，此分析數值與實驗量測之最大面外位移量 2.1410μm（如圖 17 所示）相差 1.94% 
(0.0425μm)，差異甚小，故實驗量測之機械性質具準確性。 
接著個別將同樣施以直流電壓8.9伏特之圖16與圖17中長度方向的垂直位移量作成距離－
位移量圖來比較，如圖18所示圖中實驗值的擬合曲線稍微比模擬值擬合曲線彎曲，可能是由於
試片寬度兩側面以及其上表面的雜散電場影響，使試片寬度方向也有微微地變形並導致試片長
度方向中端有微拱，拱形區域位移量最大相差約17.4%(0.21μm)，但對自由端的變形量沒太大
影響。而大致上雙方鎳微懸臂樑長度方向的位移量趨勢都相當近似，更可驗證實驗量測的可靠
性。 
4.5 結果討論 
前述實驗量測結果綜合整理如表4所示。 
1. 鎳微懸臂量尺寸16000μm x 2000μm x 31.97μm與24000μm x 2000μm x 31.97μm雖然為同一
匹晶圓上電鍍製程出來的，其楊氏模數應該相同。但從表4.2發現，懸臂樑長度24000μm的
楊氏模數比長度16000μm稍小，可能為(1)微懸臂樑兩側面以及兩側上方表面的雜散電場影
響沒有扣除而產生的誤差。但產生的誤差只有0.31%（0.529GPa），影響不算太大，可能是
因為厚度比長寬度來說相當薄，以致兩側面以及兩側上方表面的雜散電場影響面積比較小。 
2. 由於鎳微懸臂樑的波松比是藉由其楊氏模數、複合模數以及鑽石奈米壓頭材料性質反算得
到的，因此試片相同寬度厚度下，長度變化影響波松比不同的差異原因與結果討論第一點
相同。 
3. 由圖19可知，施加相同電壓，在相同厚度寬度的條件下，長度越長的變形量越大。並且電
壓與自由端位移量成非線性關係。 
4.由表1可知，電鍍鎳微懸臂量尺寸20000μm x 2000μm x 26.33μm以及尺寸16000μm x 2000μm x 
31.97μm與24000μm x 2000μm x 31.97μm的電流密度(Current density)分別為16.95(mA/cm2)
與16.48(mA/cm2)，再由表4可發現尺寸20000μm x 2000μm x 26.33μm的楊氏模數比尺寸
16000μm x 2000μm x 31.97μm與24000μm x 2000μm x 31.97μm來得小。由此可知電鍍製程中
溫度固定時，電流密度越大，材料的楊氏模數越小。此點與圖20[10]所表現的特性不謀而合。 
5. 由圖7、圖13、圖18可知，將所得材料性質代入有限元素模型中模擬分析，分析結果與實驗
結果相仿，可說明所得數據具可靠性與準確性。 
6. 從圖20[10]可看出，電流密度為定值時，電鍍溫度從60°C降至40°C時，鎳的楊氏模數上升大
約20GPa。再從圖21[10]可得知電鍍溫度60°C時，電流密度16～17mA/cm2鎳的楊氏模數約為
150GPa。以此數據為基準，並加上圖20[36]的趨勢，可推算在電鍍溫度40°C，電流密度16
～17mA/cm2的楊氏模數約為170GPa，與本文實驗所得數據非常相近，也可證實本實驗的精
確度。 
5. 結論 
1. 本文提出利用靜電驅動相位移光學干涉術(PS-ESPI)，配合有限元素軟體模擬雜散電場影響
作為理論修正，並輔以靜態奈米壓痕法量測鎳薄膜之機械性質，其楊氏模數量測結果與文
獻值比較極為相近，證明為可行且精準的方法。 
2.  現已發表之文獻中所量測金屬薄膜機械性質僅能得出楊氏模數，本文增加使用靜態奈米壓
痕技術，則可量測出金屬薄膜極重要的波松比。

二、與會心得 
  本次會議主題為201X的濺鍍技術(Topical Theme: Sputtering Technologies in 201X)，下分Plasma 
Processes,Plasma Induced Process Technologies,Micro & Nano Technologies,Thin Films,Fundamentals 
of Sputtering and Plasma, Sputtering Processes, Applications 等次主題，其中薄膜的論文最多，計有
62 篇。三天下來，本人對濺鍍及薄膜研究現況與未來進程的廣度與深度增進，自不待言。 
  由本次發表的研究成果言，日本與德國在上述的主題中之製程研發成果仍居領先地位。本次
會議中未有一篇論文來自中國大陸，這也符合中國大陸現有政策是在自家舉辦，所以網路上的邀
稿國際會議終年不斷，各類大小會議均有，略可窺見中共「作東不作客」的心態。 
  本次會議運作期間適逢日本本島東北部遭遇 311 大地震、海嘯、及核放射汙染，確實對產學
界參與的意願有極大影響，有不少論文即撤稿(withdraw)，國內有些論文則由口頭報告改為論文張
貼，這也是口頭報告(含邀請論文)論文只有 39 篇，張貼論文有 126 篇的不符合慣有國際會議比例
的現象，參加人數亦明顯受影響。本人的論文原先是登錄口頭報告，由於三月十一日的日本災難
後，大會網路資訊一度停滯，環境負面消息不斷出現，後再向大會申請改為論文張貼。 
三、建議 
  國際學術研討會主辦單位於會議開幕前一年即開始籌畫準備，工作備極艱辛繁瑣，若個人無
安全威脅，為提升及展現台灣在濺鍍、電漿、薄膜領域研發成果，應不輕言缺席，此亦為善盡學
術界一份子之義務。
