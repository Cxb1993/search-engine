具取樣保持輸入不確定時間延遲系統之強健性控制之研究 
計畫編號    ：   NSC 99-2221-E-022-003 
計畫主持人  ：  連 長 華  教授 
計畫參與人員： 張 皓 欽、周 佩 穎 
執行單位    ： 國立高雄海洋科技大學輪機工程系 
 
中文摘要：本計畫之主要目的是探討具取樣保持輸入不確定 T-S 模糊及切換時間延遲系統之強健性控
制器(robust control)之設計問題。具取樣保持輸入設計易於實現在 T-S 模糊及切換系統之各別平行分
散補償器(控制器)及切換控制器設計問題上。取樣保持輸入技術是基於電腦之數位控制理論且可用於
取代類比元件之方法。故設計出一個符合實際需求之迴授控制策略在完成實現上是相當重要的工作。
此計畫內容主要是考慮具取樣保持輸入及時間延遲之 T-S 模糊及切換系統之強健保證代價值及 H 控
制器設計問題之研究。將以時變時間延遲之方法來處理取樣保持輸入設計之問題。狀態取樣時間之允
許上限值將也可由我們所提之 LMI 最佳化法求出。此計畫能提供一個設計強健控制器以滿足對系統
性能要求之進階研究。此計畫已依照當初規畫完成執行，結果依序摘錄於本文報告中。 
Abstract: The robust controls for uncertain T-S systems and switched system with time delay and 
sampled-data input are investigated in this project.  Sampled-data input is easy to implement in parallel 
distributed compensators and switching controls for T-S fuzzy systems and switched systems.  The design 
of sampled-data input technique is based on a computer digital control theory and used instead of analog 
elements.  Hence designing a practical and useful feedback control strategy is a very important work in 
implementation.  In this project, the robust guaranteed cost controls and robust H  controls for uncertain 
T-S fuzzy and switched systems with sampled data input and time delay are considered.  Time-varying 
delay approach is used to deal the sampled data input problem.  The upper bound for the sampled time of 
state is estimated from the proposed LMI optimization approach.  This project provides an extensive 
research for designing the robust controls of systems to achieve the requirement of system performance.  
All the proposed schemes in this project have been done and shown in the following sections.
