Abstract 
This project is planed to be a two-year 
project and expressed as following 
respectively. For the first year, a three 
dimensional atomistic finite element model is 
established. For the second year, the IC test by 
using cantilever type probe is simulated. But 
approve one year only, so this project carry out 
part for first year only. 
   In this year project, a three dimensional 
atomistic finite element model is established. 
For a micro system, the trajectory of particle 
motion is determined by the interaction 
between the two adjacent particles. Different 
potential functions may influence the 
simulation results. The Mohr’s potential 
function attempts to be used in the interaction 
between the adjacent particles; the element 
involving nonlinear spring, damping and 
lamped mass is used to establish the atomic 
model in this project. The lamped mass is 
located in a node to simulate a mass of atom 
and nonlinear spring and damping are used in 
connection with the adjacent nodes to simulate 
the bonding force between atoms and it 
produces an attractive force. The fracture 
elongation of spring is used to simulate the 
interrupted length of atomic bond. Finally, a 
tension and compression of three dimension 
atomistic finite element model for copper 
micro structure is using as an example, the 
stress and strain relationship, Young’s modulus 
and Poisson’s ratio are compared with the 
results of Material Studio software to verify 
the feasibility of this developed model.
