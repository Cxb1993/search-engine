研究摘要(500 字以內)： 
近年來政府相關單位對於洪災的防治，已不僅考慮以工程手段來禦洪，而進一步思考，
採用非工程方式的洪氾區劃設來管制洪氾土地的利用。本研究以AutoCad Map 3D結合二維
漫地流淹水模擬劃定「防災地圖」的技術及圖資需求，在整個洪氾區非工程管理課題上相
當重要：由於洪氾邊界範圍具有不確定性，洪氾邊界範圍的界定方式、圖籍精度是否能夠
符合劃設區域的需求、劃設圖資項目是否足以說明洪氾區劃設成果。因此，本研究建立結
合二維漫地流淹水模式分析與 AutoCAD Map 3D的地形模型，讓二維漫地流淹水模式分析
分析所得結果，以淹水展示於 AutoCAD Map 3D地形模型上的洪氾潛勢與分布狀況，研究
中以苗栗縣為案例，展示出洪氾區域邊界之概念。 
 
人才培育成果說明： 
透過本計畫之合作，聯合大學與昌漢科技股份有限公司，共力開發出以AutoCad Map 3D
結合二維漫地流淹水模擬劃定「防災地圖」的技術及圖資需求模式。其結合兩單位之專業，
包含聯合大學於防災、水利、空間資訊上之專長，及昌漢科技之系統開發及工程軟體整合
能力。 
 
技術研發成果說明： 
本研究以防災應用為主體，透過AutoCAD Map 3D作為應用平台，借以教導使用者如何
利用AutoCAD Map 3D的地圖建立與操作，進行防災地圖的製作。畢竟，區域型防災的首要
工作，即是建立一張防救災地圖作為防救災戰略施行時的指揮圖。透過防災地圖的建立，
可以讓指揮團隊清楚的了解地方上的災害類型、避難路線、救災支援等相關資訊。 
在本研究中，化繁為簡去蕪存菁的教導使用者如何熟悉工程上最為通用的AutoCAD軟
體進行防災避難空間的地圖製作，讓一般工程人員可以很快的進入具備工程化地理資訊系
統(Engineering GIS)軟體AutoCAD Map 3D的世界中。其中，使用者可以在各章節中學習
AutoCAD Map 3D的軟體操作，進而應用到防災地圖的製作。 
性製圖工能，能輕鬆的幫助使用者快速建立地圖資料，如鎖點功能，也讓使用者在繪圖的
精準度及正確性大幅提升。在AutoCAD Map 3D 2008以後所提供的FDO資料連結功能，更
讓AutoCAD Map 3D進入GIS應用的另一個境界，由傳統刻版的製圖工具，搖身一變成為GIS
空間分析應用的整合性軟體。其中，FDO(Feature Data Object)更打破了商業應用的限制，
由OSGEO提供的開放式程式碼加值，讓許多空間資料在應用與管理上，突破格式上的疆
界，能夠整合包含Oracle Spatial、MySQL、SQL Server、SQL Server Spatial、PostgreSQL、
SQLite等空間資料庫應用，提高資料的管理與安全層級。 
而這麼多年的技術演進，AutoCAD Map 3D依然沿用Autodesk公司所創建的DWG檔案
格式，讓使用者在資料交換上，降低格式上不統一的疑慮與困擾。因此，在AutoCAD Map 3D
中，使用者可以利用DWG檔案格式在單一圖面上的圖檔能力，亦可運用DWG檔案於空間
資料管理、分析、展示上的專案整合能力，協助企業內部降低管理成本。有鑑於廣泛的DWG
檔案格式應用與操作，本章節將針對DWG檔案格式於CAD與GIS應用中的應用差異進行說
明，幫助使用者更快、更有利掌握如此強大的CAD與GIS平台。 
此外，在防災圖資應用中，使用者常常苦惱於空間資料的收集與資料處理。AutoCAD 
Map 3D 提供各種CAD資料與GIS圖資的整合能力，使用者(尤以各鄉鎮市公所防災人員)可
以輕鬆的由政府所提供的基本圖資料(詳見二萬五千分之一（經建版）地形圖數值檔
http://www.land.moi.gov.tw/land/map/Html%26Img/25k_topo.htm) 網站中下載到基本地形圖
資料，協助作為區域防災的基本圖資。進而將圖資轉換為GIS的空間資料，輔助區域防災的
基本圖資建置。
