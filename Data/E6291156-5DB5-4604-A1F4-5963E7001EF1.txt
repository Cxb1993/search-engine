一、概論 
研究均勻流通過圓柱之流場，是個傳統
且複雜的流體力學問題，在工程應用上也具
有相當的重要性。當雷諾數（ 0 0Re 2V r υ= ，
0V 為上游流速， 0r 為圓柱半徑，υ為流體運
動粘性係數）大於 60~100 時，圓柱後方即形
成交替射出之渦流，導致作用在圓柱之力量
產生週期性的變化，而使圓柱發生週期性之
流向及橫向振動。當作用力之週期與圓柱振
動週期同步時，即發生共振。在共振時結構
物所產生之振幅，最大可達二倍結構物的直
徑，這對結構物之安全造成極大的威脅。因
此在工程、載具及器具之設計皆必須考慮之
共振的影響。 
根據前人研究（Panton【6】），流體通過
固定圓柱，圓柱軸與流向垂直，在雷諾數大
約 60~100 時，在圓柱下游的尾痕（wake）區，
會產生交替射出的渦流。不對稱交替射出之
渦流會影響圓柱表面之壓應力與流速場，從
而引起昇力與阻力週期性之變化。昇力之頻
率與渦流射出頻率（ 0f ）相同，而阻力之頻
率則為渦流之二倍。昇力與阻力都是工程設
計上的基本考量。再進一步，就須要考量週
期性昇阻力引起彈性圓柱之振動，因為振動
就可能造成破壞。 
根據前人研究（Hurlbut、Spaulding、and 
White【3】，Cheng、Hong、and Aung【1】、
Tanida【5】），流體通過橫向強制振動圓柱，
當圓柱振動頻率（ cf ）接近前述固定圓柱渦
流射出頻率（ 0f ）時，振動圓柱之渦流射出
頻率（ 0f ′），不會等於 0f ，而會變為與 cf 同
步，這就是所謂的鎖定現象，渦流射出受到
圓柱振動鎖定。而能發生鎖定現象的頻率區
間 則 稱 為 鎖 定 區 間 。 鎖 定 區 間 約 為
0 40%f ± 。在鎖定區間，昇阻力迅速增加。
前人研究（Hurlbut 及 Tanida），也由昇力之
虛數部份（ ( )Im CL ，正比於每個週期流體對
圓柱所做之功），分析圓柱之振動穩定性。 
前人（Gu、Chyu、and Rockwell【2】, Lu 
and Dalton【4】）曾觀察到渦流轉換的現象。
這現象是說，在固定的圓柱振動振幅及雷諾
數之下，當 cf 逐漸增加，附著渦流（附著在
圓柱上，尚未射出之渦流）會逐漸縮短，直
到接近一個最終位置時，就會讓渦流突然轉
換到圓柱的另一側。而轉換時的 cf 會隨雷諾
數及振幅的增加而減少。 
 本文將建立數值模式，以研究橫向強制
振動圓柱對流場之影響。影響參數有三，為
雷諾數，圓柱振動頻率，以及圓柱振幅。由
於前人數值研究之案例不多，且對於緩衝區
間（介於前述鎖定與不影響區間）尚未深入
研究，因此本文將以大量的案例更精準的描
述振動圓柱振幅之影響。本文先利用流線
圖、等壓圖，以及渦度圖等三圖，探討渦流
的觀察方法與渦流特性。接著以等壓圖與壁
壓圖分析圓柱振動對昇力之影響。之最討論
圓柱振動頻率對於昇力之影響。最後則討論
偏向混沌之流場。

3.4.2 渦流與壁壓之關係 
圖 25 為在 Re 150= ， max 0.28X = ，
0 0.998cf f = 案例下，渦流生成過程圖。在
T=155 時，圓柱左側下游 B 渦流初生成，接
著以 1 t∆ = 的間隔時間作紀錄，直到 T=164。
而圖之下方 Amax、Bmax、及 Cmax 值代表
各渦流的低壓中心值，Ap、Bp、及 Cp 值代
表各渦流最大的等壓力環（及圖上之封閉曲
線）之壓應力值。T=155 為圓柱左側 B 渦流
射出之前一時刻，T=161 為圓柱右側 C 渦流
初生成，T=164 為圓柱右後方渦流射出。由
圖中之渦流的壓力封閉曲線可知在渦流的射
出歷程中，渦流面積大小的變化。 
由圖 25 可知，隨著時間的增加發現，當
B 渦流慢慢增大時，圓柱左側表面壓應力分
佈將隨著渦流的增大而逐漸加大，並且隨著
渦流位置往下游移動。圓柱左側表面壓應力
分佈也會隨之抬高，這顯示當渦流出現於圓
柱後方時，渦流之能量與所在位置可能對圓
柱表面壓應力分佈造成一定程度的影響。接
著在 T=158 時，昇力為最大，渦流中心低壓
恰好最強，也恰好要射出。另可以發現到圓
柱左側負 x 軸向最大的壓應力分佈。而到
T=159 時，圓柱左側負 x 軸向壓應力分佈開
始逐漸縮小。B 渦流持續遠離圓柱，直到
T=161 時圓柱右側後方也生成 C 渦流，此刻
因為 C 渦流的影響，使得壁壓分佈開始往右
邊扭動，而左側渦流因為已經遠離圓柱有一
段距離，所以對於壁壓之影響相對減弱。隨
著 C 渦流慢慢的增長，壁壓也跟著往右偏，
而在 T=164 時達到最大的扭曲。因為此時刻
也是昇力最大的時候。

最遠離圓柱。這是因為高頻鎖定區間會強化
渦流，低頻鎖定區間會弱化渦流，而緩衝區
間將更弱化渦流。 
圖 27 為在Re 150= ， max 0.28X = 支按下
中，鎖定區間內之渦流能量比高、中、低三
個頻率（ 0 1.02cf f = 、0.998、0.928）之昇力
最大時的壁壓圖分佈。由圖中可發現，在
0 1.02cf f = 時，正 x 軸上的表面壓應力（ BP + ）
最大，而 0 0.829cf f = 為最小，這似乎意謂著
圓柱的振動產生了不同能量的渦流，而渦流
能量愈大時， BP + 愈大。由此可知最強的渦流
能量應該能夠造成最大的 BP + ，而最大的
BP +，將產生最大昇力。此處因 BP − 都差不多，
故只要以 BP + 即可判定昇力相對大小，不須如
前使用（ BP + － BP − ）。最後，根據觀察得知，
當 BP + 最大時，圓柱此刻即受到流場的最大昇
力所作用。 
圓柱在流場中以一特定頻率振動，進而
在圓柱後方產生渦流，渦流因為圓柱的振動
影響，其能量將受到振動頻率的控制，接著
渦流因為不同的能量與位置對圓柱表面作用
不同壓應力，而不同的壓應力分佈情況將轉
換成不同的昇力情形。
