一、中文摘要 
在此篇報告中，使用高介電常數材料
的奈米晶粒，並以傳統SONOS型式的快閃
記憶體元件結構，取代現今傳統氮化矽
(Si3N4)材料。用不同的寫入/抹除的操作方
式，在低電壓下來操作快閃記憶體。以達
成電荷捕捉效率佳、有快速的寫入/抹除速
度、大的記憶窗口、儲存資料持久性以及
寫入、抹除操作造成的性能退化少的非揮
發性快閃記憶體。 
我們使用氧化鉿(HfO2)奈米微晶粒作
為捕陷電荷層來製作新穎的SONOS型式非
揮發性快閃記憶體。此氧化鉿奈米微晶粒
快閃記憶體在一萬次的寫入/抹除下，仍然
擁有好的儲存資料持久性以及寫入、清除
操作造成的性能退化少。 
 
關鍵詞 : 矽 /氧化矽 /氮化矽 /氧化矽 /矽
(SONOS)；非揮發性快閃記憶體；氧化鉿
(HfO2) 奈米微晶粒；儲存資料持久性
(Retention)；讀取 /寫入抹除操作忍耐力
(endurance)。 
 
Abstract: 
In this report, we design nonvolatile 
memory with a high-k dielectric nanocrystal. 
That nonvolatile memory device structure is 
SONOS type. This high-k layer replace the 
silicon nitride layer in the SONOS structure. 
Different program/erase methods and also 
proposed for low power applications. This 
nonvolatile memory structure will have 
superior characteristics in terms of 
considerably large memory windows, high 
speed program/erase, long retention time, 
and excellent endurance.  
We present a nonvolatile SONOS-type 
flash memory that was fabricated using 
hafnium oxide (HfO2) nanocrystal as the 
trapping storage layer. These HfO2 
nanocrystal memories exhibit excellent data 
retention, endurance, and good reliability, 
even for the cells subjected to 10k P/E 
cycles. 
 
Keywords: 
Si/SiO2/Si3N4/SiO2/Si (SONOS), nonvolatile 
flash memory, HfO2 nanocrystal, Retention, 
endurance.  
 
二、緣由與目的 
近年來，由於記憶體元件尺寸微縮，
使得傳統SONOS型式的非揮發性記憶體元
件，氮化矽層被高介電係數材料取代，而
作為電荷捕陷層，在使用不同的高介電係
數材料，由於能階上的差異，因而影響記
憶體元件本身儲存資料的持久性以及寫
入、抹除的速度…等，且高介電係數材料
會隨著製程溫度升高，而使得元件的儲存
資料的持久性受到考驗。
