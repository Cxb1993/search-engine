一､中文摘要  
此計劃的主要目的在於研究並實現一個以位
置資訊為基礎(location-based)，具有服務導向
(Service-Oriented)性質的智慧型居家生活平台並
開發所需要的軟體元件，以滿足現今社會中智慧型
生活空間與機能。由於 SOA 架構可以利用 Web 
Services 等標準化元件來提供前端程式與後端資
訊系統互相溝通的通訊介面，所以可以藉 SOA 架構
將使用者所需的服務客製化，讓使用者取得所需的
相關訊息，更可藉由 web services 技術架構將多
資訊系統整合。 
因此，在此一計畫中，我們將整合多媒體網
路與閘道技術、人臉辨識技術，Location-Based
平台技術、ZigBee 傳輸技術與嵌入式系統技術與
行動裝置來發展一個可以提供服務隨插即用
(service plug and play, SPNP)的智慧型居家環
境。並以一個實驗例來說明整個操作概念；在此一
實驗例中，使用者可以享受 (1) 隨意服務 
(service-on-demand)，利用手持裝置隨意取用多
媒體與資訊服務 (2) 在地個人化服務 (location-
based personalized service)：居家系統可以藉
由感測網路所得位置資訊，為使用者選擇合適的服
務內容。 (3) 保全服務 (security service)：
利用優化的人臉辨識技術提供安全安心的認證服
務。此外由於感測器將普遍部署於居家環境中，為
解決感測網路與感測器介接問題，提高部署的便利
性，此一計畫亦將依 IEEE1451 標準開發智慧型
ZigBee 裝置，此一標準可以提供 XML 軟體控制功
能，因此可以藉此開發具有 SOA 介面的 ZigBee 感
測元件。綜言之，我們希望透過此一計劃的成果執
行，能夠滿足現代人對於居家環境，包含安全，娛
樂、照護以及健康需求，利用資訊與通訊技術的整
合應用，來增進未來生活的安全、便利與舒適！ 
關鍵詞：服務導向式架構、適地性服務、健康照
護、多媒體服務、感測網路、人臉識別 
 
二､緣由與目的 
數位家庭技術已經是二十一世紀初對大眾生活
影響非常重要技術之一。數位生活技術所涵蓋的範
圍廣泛，包含資訊處理技術、資訊傳輸技術以及資
訊硬體技術。為有效整合現有資通訊技術並且透過
軟體開發增進其應用價值。本計畫結合跨校、跨團
隊的教授與專家，來建立一個符合 SOA 架構的數位
家庭服務平台。在此一平台上提供滿足現代家庭對
於安全、娛樂、照護與健康的要求。本研究的主要
目的在於研究並實現一個以位置資訊為基礎
(location-based)，具有服務導向(Service-Oriented)性
質的智慧型居家生活平台並開發所需要的軟體元
件，以滿足現今社會中智慧型生活空間與機能。在
Sensor Network 上 具  Location Based 
Service 的 Service Oriented Architecture 系
統，我們將利用 SOA 架構概念來整合多媒體網路與
閘道技術、人臉辨識技術，Location-Based 平台
技術、ZigBee 傳輸技術與嵌入式系統技術與行動
裝置來發展一個可以提供服務隨插即用(service 
plug and play, SPNP)的智慧型居家環境。

子計畫四: 應用於 LBS 之智慧型 ZigBee 感測裝置
(Smart ZigBee Sensor Design for LBS 
Application, ZLA) 
 
LBS 智慧型 ZigBee 感測裝置系統架構，如圖七所
示。此系統架構包含有行動生理感測裝置 (Mobile 
Sensor)、路由節點(Router)、及後端資料採集器(Dongle)
等，使用可在居家進行生理監測與照護服務。首先使用
者進行生理訊號量測(血壓、血糖、脈搏…等)，經由數
位轉換並透過微控制器處理，最後進行 ZigBee 無線模
組傳送生理資訊，並將此資訊以 multi-hop 的方式傳至
後端資料收集器，最後經由人機介面可經由 TCP/IP 透
過網路以供遠端下載以及監控等應用。 
圖五 ZBS 無線感測硬體設備 
 
監控介面通訊為 RS232 通訊和 Internet 通訊，以
Server 和 Client 區分。當感測模組定時回報或命令
控制透過 RS232 送回至監控端電腦上，亦可透過
Internet 通訊傳送至遠端監控電腦上，其監控介面
如圖 3 所示。為了避免感測節點傳送時變更了資訊，
以原始封包和繞送封包的方式傳送，在感測節點取得感
測資訊後，以原始封包傳送至上一層節點，當節點收到
感測節點傳送的封包後，改用繞送封包做為傳送，傳送
至監控電腦。 
由於本裝置之傳輸範圍有限，且使用者在室內可
能會到處移動，若超出其傳輸範圍，則生理訊號將無法
傳遞至後端 Gateway 來進行資料的儲存。為能延長 LBS
智慧型 ZigBee 感測裝置與後端 Gateway 的通訊範圍，
故擬於系統中引入無線感測器網路技術(Wireless Sensor 
Network, WSN)，以克服室內通訊死角的問題。到目前
為止，無線感測器網路技術所使用之通訊協定與實體模
組有許多種形式，如 RF、Wi-Fi、Bluetooth、ZigBee、
Z-Wave 等，本計畫因考量到市場需求性與通訊協定標
準化等，故擬採用 ZigBee 模組進行無線感測器網路設
計。
