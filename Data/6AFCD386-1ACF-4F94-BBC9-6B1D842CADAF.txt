英文摘要(Abstract) 
For protecting secret data delivery in public computer network, Cryptography technique and Information 
Hiding technique are two most important research issues in recent years. For data hiding technique, secret 
data is concealed into a cover media (e.g., Text, Image, Video, and Audio) and delivered to a receiver. 
Because the properties of cover media, unexpected user will not pay too much attention on it. On the other 
hands, mobile devices became very popular communication device in recent years. Using mobile device to 
deliver secret data also becomes an important research issue. The power consuming is most conserved factor 
for designing a data hiding technique. In this project, low computation cost data hiding methods were 
presented to achieve the goal of secret data delivery.  
Side-match vector quantization is presented for solving the problem (i.e., Block effect) of VQ. Because 
the boundary pixels between two neighbor blocks have similar pixel distribution, SMVQ using state 
codebook concept to select a suitable coding codeword. In order to prevent the ambiguous for decompress 
image, the extra indicator for every remaining block is required. Thus, we have presented an indicator 
elimination scheme for SMVQ to further improve the performance in terms of compression rate. The main 
idea of the proposed method is using de-clustering concept to conceal indicator into the compression code. 
The experimental results shows that the proposed method can significant improve the compression rate with 
the similar visual quality of decompressed image.  
In order to increase the usability of data hiding technique works on mobile devices, we focus on 
developing low computation cost data hiding technique. The proposed reversible data hiding method utilizes 
the property of Sudoku to construct a guidance matrix and help to conceal secret data into the cover image. 
First, secret data is converted into base-9 numeral system. Then, secret data is concealed into cover image by 
looking up the guidance matrix. The proposed method generated two stego images. The secret data can be 
fully extract when the user hold both two stego images. Thus, the security of secret data delivery can be 
satisfied. Furthermore, the original cover image can be fully restored after secret data have been extracted.  
 
Keywords: Indicator elimination, data compression, information hiding technique, reversibility, Sudoku

影像區塊截斷編碼技術(Block truncation coding，BTC) [7]及 JPEG [10, 15, 29]等均是常被使用的技術。
其中，向量量化法的編碼/解碼流程簡單且能快速地還原影像，且其壓縮效率非常好，再者向量量化概
念亦被廣泛應用於多種領域，如監視系統的入侵偵測。 
邊緣吻合向量量化編碼法(Side-match vector quantization, SMVQ)[18]其目的在改進傳統 VQ的壓縮
效果。其中觀察自然影像的特性之後發現，影像中的區域裡相素分佈相似性很高，這也代表這個區域
中的區塊可用已編碼過的區域預測出來，而且經過實驗預測出來的效果很接近原始 VQ 的視覺品質。
可惜的是預測不是每次都見效，當預測出來的結果與影像的實際結果差異太大時便無法使用 SMVQ必
須改用 VQ，為了記住該區塊使用的是 SMVQ 或 VQ「標示碼」位元便被採用來標示編碼字來源。而
這多出來的標示碼反倒是讓壓縮效果提昇的幅度打了折扣，於是本計畫成功發展出將標示碼隱藏起來
的方法，除了讓 SMVQ 順利運作同時讓壓縮效果獲得更進一步的提昇。 
 
2. 低計算量之可還原原始影像資訊隱藏技術 
無線網路的廣泛應用使得許多新興的資訊科技產品均具有無線傳輸功能，而無線環境下常使用行
動裝置進行連線與資訊傳遞，行動裝置最重要的兩個限制是電力及傳輸頻寬，因此，許多現有的機密
資訊傳遞技術恐無法在行動裝置上勝任。此外，數位影像的產生是多樣化的，可以是數位相機拍攝而
得，可以是電腦繪製，也可以是經由掃瞄而得，這些數位影像均是由像素組合而成的，此類影像稱之
為空間域影像(Spatial domain image)。數位影像的另外一種表示方式是經各種轉換方式得到許多係數，
經轉換後的影像稱之為頻率域影像(Frequency domain images)，其中轉換技術有離散小波轉換(Discrete 
wavelet transform, DWT)、離散餘弦轉換(Discrete cosine transform, DCT)及離散傅立葉轉換(Discrete 
Fourier transform, DFT)等。頻率域的影像有利於一般的影像處理操作或進行資料壓縮，但其轉換過程
具有較高的計算成本，因此本研究將僅著重於空間域影像的資訊隱藏。 
使用數位影像的像素值進行資訊隱藏最大的優點是其所需的計算成本較低，在實作上較為可行。
Mielikainen在 2006年提出空間域的資訊隱藏技術[24]，其主要精神是將影像中的像素予以配對，假設
負載影像 I = {pi | i = 1, 2, …, H × W}，其中 H與 W分別是影像的高與寬，機密資訊為 S = {bi | i = 1, 2, …, 
K}，K的最大值為 H × W。作者定義一個二元函數 f(pi, pi + 1) = LSB(pi / 2 + pi+1)，(pi, pi+1)是配對後的
像素對其可用來藏入兩個機密位元(bi, bi+1)，透過下列四種隱藏規則進行像素值調校以達到藏入資訊。 
規則 1：若 bi = LSB(pi)且 bi + 1 = f(pi, pi + 1)，則 stego 像素(pi’, pi + 1’)由計算 pi’ = pi 及 pi + 1’ = pi + 1求得。 
規則 2：若 bi = LSB(pi)且 bi + 1 ≠ f(pi, pi + 1)，則 stego 像素(pi’, pi + 1’)由計算 pi’ = pi 及 pi + 1’ = pi + 1+1求得。 
規則 3：若 bi ≠ LSB(pi)且 bi + 1 = f(pi-1, pi + 1)，則 stego 像素(pi’, pi + 1’)由計算 pi’ = pi - 1及 pi + 1’ = pi + 1求
得。 
規則 4：若 bi ≠ LSB(pi)且 bi + 1 ≠ f(pi-1, pi + 1)，則 stego 像素(pi’, pi + 1’)由計算 pi’ = pi + 1及 pi + 1’ = pi + 1
求得。 
這樣的作法得到不錯的影像品質，但還有改進空間，且其無法達到還原原始影像。

邊緣吻合向量量化編碼(Side-match vector quantization, SMVQ)技術是為了改善向量量化編碼壓縮
所造成的區塊效應(Block effect)，區塊效應是因為影像在進行編碼字尋找時是區塊各自獨立的，但影像
的色彩分佈是具有區域性的，因此，解壓縮後的影像以肉眼看起來會呈現區塊相鄰處有不協調的情形。
此外，SMVQ 也達到了進一步提昇向量量化編碼壓縮技術的壓縮效能，其最主要的概念是利用自然影
像中附近區域的像素分佈會很雷同的特性，再搭配建造臨時的小編碼簿來達到縮減編碼長度的目的。 
SMVQ 首先將欲壓縮的影像區分兩大類，分別是種子區塊(Seed blocks)及剩餘區塊(Remainder 
blocks)(如圖二所示)，其中種子區塊採 VQ壓縮法壓縮，用意在保持這些區塊的真實性以用來預測剩餘
區塊。剩餘區塊在壓縮時則有兩種可能，一是用 SMVQ壓縮，另一個是 VQ壓縮進行編碼。針對一個
剩餘區塊 X，其先用相鄰上方區塊(以 U 表示)及相鄰左方(以 L 表示)的部份像素值進行預測並建立一個
臨時編碼簿(State codebook)，這裡所提到的臨時編碼簿通常遠小於 VQ壓縮之主編碼簿(Main codebook)。
假設主編碼簿包含有M個編碼字(Codeword)，臨時編碼簿具有 S個編碼字，且M > S。建立臨時編碼
簿的作法是利用算出 X的部份臨時像素值並用其從主編碼簿中找出 S個最相似的編碼字組成。如圖三
所示，X = {x0 = (u12 + l3)/2, x1 = u13, x2 = u14, x3 = u15, x4 = l7, ∆, ∆, ∆, x8 = l11, ∆, ∆, ∆, x12 = l15, ∆, ∆, ∆}，其
中∆表不參與計算。經到主編碼簿中查找到相似的編碼字所組成的臨時編碼簿將提供壓縮區塊 X之用，
如果 X 可以在臨時編碼簿中找到合適的編碼字，則索引表便以該編碼字在臨時編碼簿中的位置進行編
碼。這裡所提到「合適的編碼字」指的是編碼字與區塊的歐基理德距離(Euclidean distance)(以 SMD 表
示之，如式(1))小於預先設定的門檻值(Threshold) TH的情況。

透過分析數位影像的特徵及邊緣吻合向量量化編碼法的特性以達到標示碼消除的目的。由於 SMVQ在
壓縮剩餘區塊時必須使用額外的一個位元做為標示，這樣的做法使得壓縮效果打了折扣，因此，本計
畫利用資訊隱藏技術直接將標示碼隱藏於壓縮碼之中，以提昇 SMVQ的壓縮效果。Lin與Chang於 2006
年提出將機密資訊藏於 VQ 壓縮碼中[20]，其使用的概念是將編碼簿裡的編碼字進行配對，配對方式
是將最不像的兩個編碼字配成一對，搭配藏入規則來達到藏匿機密資訊的目的。我們從 Lin 與 Chang
的方法中得到靈感，採用相似概念設計改良後的 SMVQ方法。 
自然的數位影像存在區域裡的像素分佈很類似的特性，這一點與 SMVQ的中心思想很接近，因此，
我們便利用這樣概念進行標示碼的藏入。為利於說明我們定義主編碼簿為 MCB ={mcwi | i = 0, 1, …,

1, …, S-1}，scwi = {scwi(j) | j = 0, 1, …, k × k - 1}，scwi(j)∈{0, 1, …, 255}。首先，將影像 I切割成以 k × k

六、結果與討論 
資訊科技的發展與網路的普及使得人們的生活變得更加便利，面對這樣的時代需要新的技術以應
付傳遞機密資訊的要求，除了加密技術之外資訊隱藏技術更適用於在公開網路上傳遞機密資訊。由於
多媒體檔案的多樣性讓機密資訊的藏匿有了更多的選擇性。本計畫除考量在公開網路傳遞機密資訊的
需求，同時在網路終端使用的設備除了傳統的桌上型電腦之外，目前有許多的行動設備也可連接網際
網路，例如智慧型手機與個人數位助理(Personal digital assistant, PDA)等，這些行動設備最大的限制就
是電力的供應，因此本計畫朝向低運算量之機密隱藏技術以提昇在行動設備上的可使用性。 
機密資訊隱藏技術除了可用在公開網際網路上傳遞機密資訊，本計畫更進一步擴展其可應用範圍。
在瞭解邊緣吻合向量量化編碼壓縮的特性後，本計畫研究過程將其所產生的標示碼當作機密資訊，透
過修改壓縮程序讓這些標示碼可以被藏入壓縮碼中，我們所提出的方法並不會增加太多的計算。在計
畫執行過程中我們曾遭遇許多困難及瓶頸，但經過參與人員不斷討論與嚐試各種可能的解決方法，最
終還是能順利完成新技術的開發。

邊緣吻合向量量化編碼之標示碼消除技術 
影像壓縮技術除了可以維持應有的視覺品質之外，最重要的是其能大幅度減少其所需
的儲存空間，不但有利於儲存更有利於節省網路頻寬的使用，我們所發展出來的技術
不但可擴大資訊隱藏技術的應用層面，更可使壓縮效果獲得更進一步的提昇。 
 
低計算量之可還原原始影像資訊隱藏技術

算量非常低，這樣的特性將有助於應用行動裝置上。

A High Payload Data Hiding Scheme for Color Image Based on BTC Compression
Technique

Abstract—In the daily life, secret data delivery is an im-
portant application for private communication among parties
over computer networks. Data embedding technique is a good
way to achieve the secret data delivery. A color image is a
popular cover medium used to send the secret. Block truncation
coding (BTC) is a lossy image compression method, which
can significantly reduce the image size. In this paper, a novel
data embedding method is proposed to conceal secret data
into color BTC compression code. To compare with Chang et
al.’s method, the proposed method can effectively generate a
common bitmap for saving the size of the compressed image.
Furthermore, the proposed method can conceal more than four
secret bits into a block in average. From experimental results,
the proposed method not only successful achieves the goal of
secret data delivery but also significantly reduces the cost of
common bitmap generation.
Keywords-Block truncation coding (BTC); color image; data
hiding; information hiding; Steganographic;
I. INTRODUCTION
Data hiding (also called information hiding or data em-
bedding) is a way of secret delivery that is different from
cryptography. Data hiding is not easy discerned by humans’
sense organs. After embedding, the digital secret can be
extracted by using the specified data extracting procedure.
Moreover, information hiding technique is easier than cryp-
tosystem to cheat unexpected users for achieving the goal of
secret data delivery. Because multimedia (e.g., text, image,
video, audio, etc.) exist a lot of redundant spaces that
can be used to conceal secret data, data hiding technique
uses multimedia as a cover medium to achieve secret data
delivery.
From cover image reconstruction in secret data extracting,
data hiding techniques can be briefly divided into “Irre-
versible” and “Reversible”. Irreversible data hiding means
that the original cover cannot be fully reconstructed when
secret data has been extracted [5]. This kind of technique
focuses on the secret data delivery not the visual quality of
the reconstructed cover image. On contrary, reversible data
hiding focuses on the visual quality of the reconstructed
cover image rather than the performance of data hiding
in terms of embedding capacity. For instance, the distance
medical treatment cares more about patient’s X-ray image
than patient’s private data. In “Reversible Data Hiding” (also
called “Lossless Data Hiding”), the cover image can be fully
reconstructed as original one after secret extraction [1], [3],
[6], [10].
For saving the bandwidth of image transmission and
the cost of image storing, image compression technique
development has become a popular research topic. Generally,
the image compression technique can be briefly divided into
two categories, namely lossy and lossless compression. The
lossy image compression (e.g., JPEG, VQ, BTC, etc.) tries to
reduce the size of image as many as possible with tiny image
distortion. In contrast, the visual quality of a compressed
image is a main criterion of lossless compression technique.
Chang et al. proposed a data hiding method to conceal
secret data into a color image by using color BTC technique
[2]. Chang et al.’s method utilizes genetic algorithm to figure
out a common bitmap to reduce the bitmap requirement and
adopts arrangement rules to imply the secret data.
A novel data hiding technique is proposed in this paper in
order to improve the weakness of Chang et al.’s method. The
proposed method uses a simple common bitmap generation
technique to generate a common bitmap effectively with
better visual quality of stego images. Furthermore, the
proposed method can carry more secret data than Chang
et al.’s method.
II. RELATED WORKS
A. Color Block Truncation Coding (Color BTC)

posed by Delp and Mitchell in 1979 [4]. BTC is a simple
way to significantly reduce the size of grayscale images.
Guo and Wu proposed an improvement the performance of
BTC using void-and-cluster dithering approach in 2009 [7].
Because a color image is composed by red, green, and blue
planes, the color BTC method applies the traditional BTC
method for every plane to achieve the goal of saving the cost
of storing a color image. The compression code of a color
image consists of three bitmaps and three pairs of high mean
and low mean for three planes. The key steps of the color
