parameter learning is defined based on structural 
risk instead of empirical risk minimization in order 
to 
achieve generalization ability. Tuning all of the 
free parameters in the FS-RGLSVR using linear support
vector regression (SVR) is proposed for minimizing 
the cost function. Each of the consequent and  
antecedent part parameters is expressed as a linear 
combination coefficient in a transformed input space 
so that the linear SVR is applicable. This project 
introduces iterative linear SVR to tune antecedent 
and consequent parameters. This project demonstrates 
the capabilities of FS-RGLSVR by conducting 
simulations in clean and noisy regression problems. 
Comparisons with fuzzy systems with different types 
of learning algorithms verify the performance of the 
FS-RGLSVR.
