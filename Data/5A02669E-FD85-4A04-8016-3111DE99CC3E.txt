四、數值模擬 
針對上述之實驗迴路，其物理模型為一圓管如圖 2
所示，圓管分成加熱段與絕熱段二部分。上游部分為絕
熱之邊界，中間部分為等熱通量加熱之邊界，下游部分
為絕熱之邊界，管內流體為 PCM 微粒/奈米流體混合
液，為了方便分析在此之前先做幾項基本假設，其假設
如下所列： 
 
( 1 ) 管內流體流動為層流、二維穩態不可壓縮流、且完
全發展。 
( 2 ) 混合液的物理性質不隨溫度變化且黏滯散溢
(viscous dissipation)效應忽略不計。 
( 3 ) PCM 微粒在固-液相變化過程中之密度變化忽略不
計。 
( 4 ) 管內流體為牛頓(Newtonian)流體。 
( 5 ) 管內流體濃度均勻分佈且為均質性。 
( 6 ) 除了 PCM 微粒於固-液相變化過程中皆滿足局部熱
力平衡。 
( 7 ) PCM 微粒/奈米微粒完全懸浮於流體中。 
( 8 ) 管壁內傳導熱傳遞為軸對稱。
