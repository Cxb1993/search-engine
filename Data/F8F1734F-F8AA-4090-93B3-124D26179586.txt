二、緣由與目的 
 
隨著無線網路技術的發達，日常生活中的行動
裝置也愈來愈多。生活中有很多的服務需要位置資
訊的提供，例如：老人護理、生態保護、快遞追蹤、
自動倉儲、急難求助(如手機直撥 112)、行動導覽、
自動車輛導航、資源利用最佳化及自動計費等應用
都需要位置資訊的提供。本計畫原規劃內容為一三
年之計畫，第一年的計畫著重在室內定位技術之研
究。主要研究標的說明如下: 
A. Localization techniques performance evaluation 
主要在說明各種定位技術的摘要說明。例：IrDA
是用在室內短距離的傳輸。藍芽是用在電腦與電
腦之間的檔案傳送，三合一的行動電話，電腦上
的免持聽筒應用…各種方面。WLAN 是用在隨拆
隨架或架設在有線主幹線連接無線存取點的網路
環境中。RFID 之應用相當廣泛，最常見的應用
為：門禁管制、貨物管理、醫療應用、交通運輸、
防盜應用。GPS 是衛星可偵測到的範圍。 
 
B. Indoor 802.11e WLAN network deployment for 
hot-spot request. 
由於不能破壞古蹟的內部設施及建設，所以無法
以有線的方式來提供所需的網路資料傳輸，因此
必須靠感測器來以無線感測網路方式來提供資
料，所以主要分別在做室內訊號衰減研究、
Room-based Network Deployment、以 Multi-hop 的
方式傳送資料、依定位精確度要求佈點、實作室
內佈點…等重要議題方面作研究。 
 
C. Indoor localization based on wireless sensor 
network 
主要是在作室內定位的研究，其研究議題如：
Single-point position estimation：在 Single-point 的
環境之下，無線裝置所能收到的訊號只有在通訊
範圍內的 Anchor，雖然提高 Anchor 的數量有助
於定位的精確度，相對的會提高系統所需的成
本，如何減少 Anchor 的數量，並維持定位精確度
是一個值得探討的研究議題。Multi-point position 
estimation：在 Multi-point 的環境之下，裝置與裝
置之間可以透過無線傳輸彼此交換相關資訊，並
且可以和鄰居進行合作運算，但無線裝置具有行
動性（mobility），行動性可能導致無線裝置必須
時常重新定位，何時必須重新定位以及如何避免
將錯誤的資訊提供給鄰居將是一個必須解決的問
題。 
 
D. Y1-4: Indoor fast/efficient handoff of 802.11e 
WLAN 
Handoff 即是指Client由一台AP的涵蓋面遊走到
另一台 AP 的涵蓋面而不會中斷連線，AP 必須在
Client 不察覺的情形下將 Client 交給另一台 AP，
並確保連線不會中斷。當多台 AP 的涵蓋面重疊
時，client 能選擇信號最強的 AP 作結合。並且不
斷的找尋更好的 AP。實作 Handoff：在已架設完
成之網路環境中，利用具有無線上網功能的
Notebook、PDA 等工具… 測試如果在環境中不斷
的移動，服務是否因此中斷，資料能否持續正確
的送達。

四、結果與討論 
 
有關本計畫執行的成果，說明如下。 
 
A. Localization techniques performance evaluation 
常用的通訊技術用於定位之可行性評估如下: 
IrDA 因有方向性之限制且通訊距離較短，因此不
適合用來定位。藍芽通訊技術用於定位的主要好
處是越來越多的裝置支援藍芽，使得藍芽定位之
可行性大為增加，但缺點為定位所需時間較久。
使用 WLAN 進行室內定位的可行性很高，且已有
相當多的研究被實驗證明其可行性。本計畫後需
RFID 用於定位的主要缺點為低成本之 RFID Tag
其通訊距離較短。GPS 用於室外定位依目前的技
術而言有很高的精確度，但其缺點為無法使用於
室內定位。 
 
B. Indoor 802.11e WLAN network deployment for 
hot-spot request. 
在具保存價值之古蹟建築物中，由於不能破
壞古蹟的內部設施及建設，所以無法以有線的方
式來提供所需的網路資料傳輸，因此必須靠感測
器來以無線感測網路方式來提供資料，所以主要
分 別 在 做 室 內 訊 號 衰 減 研 究 、 Room-based 
Network Deployment、以 Multi-hop 的方式傳送資
料、依定位精確度要求佈點、實作室內佈點…等
重要議題方面作研究。 
(1) 訊號衰減研究：使用 WLAN 的 AP 進行定位之
前，使用 Range-based 的方法或 Range-free 的
方法，都需要了解訊號衰減的情形，以進行位
置之推估。有關訊號強度用於定位的方法，大
部分使用衰減模型的方式來測距，在室內的環
境中會有必較大的衰減係數，亦即訊號衰減較
快。 
(2) Room-based Network Deployment：由於室內無
線電訊號對牆壁的穿透率不佳，如果有很多房
間收不到訊號將無法利用 WLAN 進行定位。
因此，利用無線網路訊號強度來提供較佳之無
線上網服務及利用無線網路訊號來定位是同
一個需要解決的問題。我們提出的方法是將房
間依相近位置進行分群，每一群使用一個 AP
服務該群房間。若使用於定位，AP 間之聯繫
則不需列入考慮。若用於提供上網服務，則
AP 之間可以使用有線網路連接或以 multihop
的方式連接。 
(3) Multi-hop 的方式傳送資料：若以 Multihop 的
方式連接 AP，則 AP 間之資料代傳可以使用
Wireless Distributed System(WDS)的方式進行
資料代傳。提供服務的頻帶及用來代傳的頻帶
需不同以提升系統效能。例如：提供服務的頻
帶使用 802.11n 2.4G ISM 頻帶，用來代傳的頻
帶使用 802.11a 5G ISM 頻帶。 
(4) 依定位精確度要求佈點：不同的房間有不同的
定位需求，所需用於定位之 AP 數量及位置亦
不同。本計畫提出以階層式 Grid 的架構畫分區
域。每個區域由一個 AP 負責提供上網服務或
用於定位。需要較高定位精確度之區域或稱為
hot spot area 可以使用較細之 Grid 區域劃分，
其中之每個 AP 使用較小之功率進行傳輸，精
確度要求不高的區域可使用較大之 Grid，Grid
中之 AP 使用較大之功率進行傳輸。 
 
C. Indoor localization based on wireless sensor 
network 
在室內的通訊環境中，無線電波容易遭受遮
蔽或多路徑衰減（multi-path fading）的影響，導
致訊號減弱，無法利用訊號強度（Received Signal 
Strength, RSS）直接進行裝置之間的距離估測，由
於在相同方向上所收到的數個訊號，訊號較強者
代表距離較近，而訊號較弱者代表距離較遠，但
至少可以得知一個相對的比較關係，雖然無法直
接將訊號強度轉換為距離資訊，可以利用這個比
較關係所提供的資訊，以及 Anchor 之間的合作運
算找出無線裝置的相對位置。在定位中，我們可
以將位置資訊分為兩種：精確的座標資訊和位置
資訊。假使系統所要求的位置資訊是精確的座標
資訊，系統中就必須包含較多的 Anchor，相對的，
假使系統所要求的位置資訊是位置資訊，我們就
可以減少 Anchor 的數量。Anchor 將其位置資訊
flooding 至網路上，增加其位置資訊的可利用度。
利用 Partial-range 資訊來降低直接使用 Hop 數所
造成的誤差。Sequence-based 的定位演算法可以避
免 Shadowing 和 Multi-path fading 所造成的影響，
並可以有效利用訊號強度進行定位。改良 DV-Hop
定位演算法，每一個裝置保留所有 Anchor 的位置
資訊，當移動後需要再次定位時，發送定位要求
給周圍鄰居，收到這個訊息的裝置將本身所擁有
到所有 Anchor 最小 Hop 數傳送給發訊者，而發
送者比較所有資訊後選取最小的 Hop 數做累加的
動作，並將累加過個最小 Hop 數作為本身到達各
Anchor 的距離，並進行位置估測。利用訊號強度
估測與鄰居之間的距離，並利用 Mass-spring 演算
法進行最佳化。
