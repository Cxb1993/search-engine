一、 摘要 
 
新型靜態式全頻譜標準色彩影像系統可得到待測物在光學上所有的資訊，其包含光譜
資訊、空間資訊、強度資訊、色彩資訊、影像資訊，其應用在可見光、不可見光頻譜量測、
物體(動物/植物/礦物)影像光譜、近紅外光 NIR 光譜影像、多通道光纖光譜攝像系統、遠端
感應、太空遙測、生醫光譜影像、顯示器色彩調校、智慧搜尋飛彈…等，由於應用領域廣
泛，因此產業價值非常可觀。 
目前超頻譜影像裝置在靜態影像方面如欲取得二維影像，必須將整個裝置或其物體移動，
若應用在生醫或是顯示器方面時，將會不利於量測的方便性，再者更換其前端光學影像系
統(量測鏡頭)時，後段光學系統也必須重新設計，依此目的本計畫欲設計一機構，其主要架
構為光學影像系統、可微控之延遲透鏡組、超光譜系統，其中可微控之延遲透鏡組可輔助
超光譜儀達到 2 維影像與光譜的掃描，而光學影像系統可以是顯微鏡，望遠鏡等成像鏡頭，
或者單純是影像輸入，因此可在不移動影像、光學成像系統、超光譜儀系統三者的情況下，
量測影像的頻譜，此將適用各式光學影像系統，而不需再重新設計後端之光學系統。如此
機構之研發不僅是科學領域的前瞻性，也是未來商品化的影像量測標準裝置，也是作為高
解析度超頻譜動態影像的開端。 
 
Abstract 
   
The hyper-spectrum standard image system can obtain all of the optical information of test 
object including spectral information、spatial information、intensity information、color 
information and image information. The system can apply to visible spectrum measurement、
invisible spectrum measurement、objective(animals/plants/minerals) image information、near 
infra- red spectral information、multi-channel fiber spectral camera system、remote induction、
space and remote sensing and biophotonic spectral image…etc. 
Until now, it has some problems to obtain 2D image through hyper-spectrum setup. One 
problem is not convenient to move the hyper-spectrum setup or object in order to obtain 2D 
image. The other problem is, when we change the measuring lens the behind part of optical 
system must redesign.  
The purpose of this project is to design a new hyper-spectrum setup which includes optical 
image system、relay lenses module and hyper-spectrum system. The new hyper-spectrum setup 
can obtain 2D image through only move relay lenses module. The new hyper-spectrum setup is 
more convenient to apply to measurement and the beginning of commercialization.

六、 推廣及運用的價值 
 
光譜影像之所以廣泛為人所研究，其中一大特點就是在彌補人眼視覺觀測上的不足，
以癌症診斷為例，癌症連續多年蟬聯國人十大死因榜首，也是現代人健康的最大威脅，衛
生署統計國內每年死於癌症人數，將近三萬八千人，佔總死亡人數的 28.1%，而國人十大
好發的癌症，前五名依序是肺癌，肝癌，結腸直腸癌，乳癌，以及胃癌。其中肺癌是女性
第一大殺手，而肝癌則是男性罹患癌症的第一名，抗癌也成為國內醫學研究單位最積極投
入的項目之一。而在一般病徵診測癌細胞或腫瘤組織時，需要依賴專業醫師的豐富經驗與
知識來判定其可能性以及疾病的嚴重程度，也因為有誤判的事件發生，因此提供專業醫生
更精密的生醫診斷影像與技術，而獲得客觀的判斷標準，以期精準地做正確的治療與判斷，
將是左右生醫影像診斷最重要的方向。而由於癌細胞與腫瘤組織相較於正常的細胞組織，
具有不同的吸收與反射光譜，由其特點使得光譜影像就順理成章而成為癌症與腫瘤病徵診
測的一大利器，因此發展精密的光譜影像量測系統就成了刻不容緩的要項。 
光譜影像量測亦可應用腸道內視鏡及膠囊內視鏡等內視鏡系統，忙碌緊張的作息和飲食習
慣的改變，是現代人的生活中無法避免的，這也加劇了消化系統的負擔，人類消化系統病
變長久以來始終是個困擾。人的消化道包括食道，胃，小腸 (含十二指腸、空腸、迴腸)和
大腸，由於內視鏡儀器和技術的進步，內視鏡檢查已經成為診斷和治療食道、胃、十二指
腸和大腸疾病的重要工具。 
內視鏡，可改善內外科手術具破壞性檢查的缺點，但目前的檢查仍需由嘴巴進入，經
喉嚨、胃、十二指腸，最多只能深入人體幽門內約 1 公尺深；或由肛門，經直腸、結腸、
甚至到小腸末端。然而，就腸胃科醫師來說，小腸疾病仍是一個最棘手的問題，因為它的
長度約 7 公尺且位於腸胃道的深處，一般的內視鏡並無法到達，故診斷和治療小腸疾病仍
然非常困難。 
傳統推進式的小腸鏡(Push-Type Endoscopy)是最常用來診斷小腸疾病的內視鏡，但最遠
只能到達空腸的近端。因此，非內視鏡方法，例如影像檢查，仍是必要的，但這只能用於
診斷較大的病兆而且無法用於治療。但最近幾年有兩個重要的發明-膠囊內視鏡 (Capsule 
Endoscopy)和雙氣囊小腸鏡(Double-Balloon Endoscopy)，使我們有機會可以檢查整個小腸。
膠囊內視鏡的是一顆像膠囊大小且具有照相功能的儀器，吞下之後，在腸道內移動時它會
記錄影像並傳到背在體外的記錄器中，它可記錄 8 小時，一般可記錄整個小腸，結束後再
由專家判讀影像並找出異常之處。它的好處在於病患只需吞下一顆膠囊並不會痛，故接受
度高，但仍有其限制。膠囊內視鏡提供一種可以觀察整個小腸的工具，但無法調整角度也
無法對一個病兆做更詳細的觀察。此外，它無法做切片也無法做內視鏡的治療，疑似小腸
阻塞時也不適合施行膠囊內視鏡。 
膠囊內視鏡本身是一個相當精密的電子儀器，在長寬 33 × 11 毫米，如同魚肝油大小
的體積中，含有鏡頭、無線電發射機、影像感測器、天線、精密電池等多種裝置。內視鏡
前端鏡頭設計成圓拱形，以減輕分泌物對它產生的阻礙問題。為了提供如同相機閃光燈般
功能的照明光源，採用省電的白光發光二極體。在拍攝效果方面，膠囊內視鏡最小拍攝的
物距小於 0.1 毫米，以每秒兩張彩色照片的速率進行拍攝，所得到的影像品質尚稱清晰，
部分照片甚至可以拍攝到小腸的細緻絨毛。  
基文影像公司(Given Imaging)的第一代膠囊內視鏡因為無法控制方向，且受到一秒兩張
的拍攝速率等條件的限制，使得膠囊內視鏡成效最卓越的部分是在小腸病症的診斷方面。

就學校研究單位而言，一味的投入心力於製程技術的研究上，並不是一個有效提升國
內產業技術的研究方向，理由是，製程設備造價高昂，對於經費有限的學校實屬一種沉重
的負擔，再者，製程或材料的研發成果，很少有機會於國內各大顯示器產業生產線上實際
付諸實行，因為生產機台及材料皆源自國外，廠商並不願意冒險更改既有的配套程序。而
國外的顯示器研究上，已漸漸有由追求大尺寸製程技術改成提升顯像品質的趨勢，而這些
研究，於設備上並遠不如製程設備般昂貴，但對於顯示器效能的提升，人類視知覺的滿足，
卻能有最顯著的改善。 
為了帶給人們視覺上的享受以及產品品質的完備，透過自動化的光譜影像量測可以精
確地得知液晶顯示器的背光與液晶上的濾光片所能夠成的色域，進而快速地透過調校軟體
進行色彩調校。而液晶顯示器產品品質的缺陷偵測一直為液晶顯示器產業極力希望能整合
的部份，然而至今仍然沒有最佳的解決方法，若透過掃描與分析液晶面板的光譜影像，將
可輕易解決缺陷偵測的問題。這些僅只是光譜影像應用的區區一角。 
在量測系統架構上，現今的光譜儀量測系統中，絕大部分為點量測方式，但此架構若用於
大尺寸面積光譜量測、配合顯微鏡的微小物光譜與影像量測、或者與其他透鏡組等光學元
件連結時，會因為不同需求而需要訂製不同的量測環境與架構，在時間與成本上極為可觀，
也因為這些需要客製化的設備無法大量生產而使得產業的市場需求量小，因此需要新的光
譜量測方法來取代那些需要客製化的設備。 
由於目前之膠囊內視鏡往全頻光譜式膠囊內視鏡發展，其將在生醫方面帶來更多貢
獻。其中要膠囊內視鏡於生物體內獲取光譜影像，掃描的方式不能再考慮“移動物體”、“移
動裝置”，如此要在物體與裝置間沒有相對的移動以達到掃描的目的，即是此計畫所提出的
泛用型掃描方式的光譜影像系統可達成。 
針對以上眾多需求，本計畫欲發展一套光譜儀的輔助裝置，相對於傳統需要針對量測
環境、待測的影像大小而客製化的系統所耗費的資源，本計畫將取代大多數的客製化系統，
以更簡便、更快速的輔助架構來發揮光譜儀在影像掃描與光譜分析的各種優越特性。 
目前市面上有生產之超光譜掃描系統，其掃描的方式大致可分為兩種類型：第一種類
型是欲量測之物體移動來進行影像及光譜掃描，如一些農產品、肉類的檢測(如 Figure 6 所
示)，以及由顯微鏡檢測影像之光譜(如 Figure 7 所示)，此類掃描方式缺點為由於待測物必
須穩定移動，需要設計穩定移動之平台，這對於顯微鏡系統而言就需要改裝原本的待測平
台，造成使用上的不便。

移動狹縫式超光譜儀(如 Figure 9)是利用外部程式去控制超光譜儀內狹縫上下，來達到

造成離軸像差，影響成像品質。而相對的物鏡也可能必須特別再另外設計，以解決像差問
題。

點矩陣式的產品如 Figure 10 所示，將多通道的光纖接連於顯示器上，藉由比較顯示器
上不同位置的光譜，而判斷出顯示器的背光光源分佈、各位置的色域表現、Mura 偵測等問
題；遙測式的應用如 Figure 11 所示，利用高空飛行器，如偵察機、熱氣球、航空衛星、或
遙控飛機等載具將光譜影像掃描系統於高空中掃描土地，由於高山、雪地、森林、沙漠、
住宅、道路、及工廠皆具有不同的光譜分佈，透過利用光譜的差異便可以分辨不同的物質，
解決單純使用照片對同樣顏色之不同種類物體造成誤判的情形，因此透過頻譜分析便可以
輕鬆地將其不同的種類區隔開來。而隨著感測器技術的改進，遙測影像的光譜解析度由先
前多光譜影像的數個或數十個波段，進步到目前高光譜影像的數百個波段。由於光譜解析
