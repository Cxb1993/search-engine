三、研究報告內容 
1. 系統架構設計 
本研究主要為探討機器人的搜索與路
徑規劃問題，主要分為兩個主題，主題一
為機器人執行單一目標點之搜索與路徑規
劃 ( 簡稱機器人單點搜索路徑規劃 )，主題
二為多機器人執行全域搜索與多目標點路
徑規劃  ( 簡稱多機器人搜索多點路徑規
劃 )；單一機器人搜索部分，動作完全由主
控電腦端下達命令 ( Command ) 而後動作
之，待動作執行完成，隨即向主控端回報
動作狀態並由主控端決定下一動作；多機
器人搜索部分，於主控核心上已寫入相關
演算法則，動作完全由多機器人自主決定
下一動作決策，並具有搜索時行進較短路
線與遇到死巷離開之能力，以節省搜索時
間，電腦端監控介面所扮演的角色為執行
搜索任務中資料記錄者，並具有搜索完畢
引導多機器人執行多任務點路徑規劃能
力。系統架構如圖 1 所示，建構於主控電
腦端 ( Master ) 與受控之機器人端 ( Slave )，
兩大系統，其電腦端的監控介面與機器人
端之間的通訊皆是建立在無線射頻 ( Radio 
Frequency，RF ) 通訊平台上，通過所配置的
nRF905 以無線射頻方式進行資料的傳輸。

4. 機器人搜索與路徑規劃 
本研究中執行搜索以及多任務目標點
路徑規劃，由於搜索演算法眾多，配合不
同的需求使用不同的方法，才能在所需環
境中達到最佳效果，本研究中，在方法使
用上，採用演算法與法則互相搭配，於單
點搜索路徑規劃上採用“Flood-fill 演算法 
+ 向心法則＂、多機器人搜索多點路徑規
劃上採用“深度優先搜索法 + Flood-fill演
算法 + 右手法則＂，增進人工智慧的思考
模式，目的皆為使機器人擁有好的搜索效
果及時間效率。 
在執行搜索上，可利用單隻或多隻機
器人進行，機器人或監控端必須隨時沿途
紀錄座標位置、障礙點與可行點，避免搜
索重複節點，否則可能進入搜索迴圈，並
擁有行走最佳路徑能力，以減少最低成本
與時間；又依照不同設計，可定義為找到
特定點即停止搜索或規劃範圍內全域搜索
直至探索完畢，依照不同目的，於同一平
台上設計了兩種不同的類別，來驗證提出
之方法為可行。 
 
單點搜索路徑規劃 
執行單點搜索路徑規劃任務時使用到
向心搜尋演算法與 Flood-Fill 演算法
(Flood Fill Algorithm)，任務目標為:「使
用單隻機器人執行搜索任務，當搜索至任
務目標點即停止搜索，並執行回程較短路
徑規劃與行走」，首先利用 Flood-fill 演算
法建立虛擬權重表，並依據向心搜尋演算
法則，遇到岔路節點時優先往權重較小的
格點前進，採用此法可預先推算出如何行
進為較短，加入方向性參考提升搜索速
度，而非盲目搜索。 
一開始除了起點與終點外無其他資
訊，地形皆未知，在此使用向心法每走一
格即利用Flood-fill演算法計算出預測最短
距離，每次都有一條較短建議路徑給予機
器人，如果建議路徑有兩條以上，則會隨
機擇一，讓機器人不斷朝目標點前進，相
關步驟如下所示。

5. 結論與討論 
本研究由以上兩大主題所構成，主題
一為機器人單點搜索路徑規劃，主題二為
多機器人搜索多點路徑規劃，皆為探討機
器人搜索與路徑規劃問題，所使用之機器
人個體小，並以模組化概念構成，已具備
移動、方向感知、語音提示、避障與環境
偵查等功能，並皆已融合搜索理論，可再
搭配瓦斯偵測、火焰感測、有害氣體偵測、
無線攝影機等模組，更可應用於救災場
所，相關搜索理論搭配機器人實驗已得到
實際驗證，除了可有效即時監控機器人探
測結果，並可避開孤島區域之搜索，不需
全域盲目搜索，讓機器人加入人工智慧元
素，可自身決策搜索動作，監控端再採以
輪詢方式收集相關資料，且無論於機器人
或監控端上皆有記錄相關探測資料，若有
一方資料不慎遺失或損毀，可做相關補救
動作。

技術說明
(中文) 本計畫主要在開發教育用移動機器人，並探討移動機器人的路徑規劃，首先設計
新型架構之教育用機器人，使機器人具備行走路徑長度計算與方向感測的能力，
再利用利用A*搜尋演算法(A* Search Algorithm)實現單一機器人的最短路徑問
題(Shortest Path Problem)，此方法將為發展多機器人路徑規劃的基礎。第二
個主題則是利用A*搜尋演算法結合貪婪演算法(Greedy Algorithm)建構單一機器
人的途程路徑，並以2-opt交換法改善其路徑。第三個主題則為利用上述的單一
機器人路徑規劃方法結合連續單項拍賣(Sequential Single-item Auction)解決
多機器人的任務分配(Task Allocation)問題，並藉此求得接近最佳解的多機器
人路徑。使得機器人之間可以以避免避撞的方式，由不同的出發點到達各自的目
標點，以解決多部機器人的任務分配(Task Allocation)問題，最後以本計畫所
開發出來之群組移動機器人系統作實際驗證。
(英文) The project design mobile robots, and develop path planning method of the education 
mobile robot. We design the mobile robot that can move autonomously. The mobile robot 
can programs the motion trajectory, and computes the orientation in the free space. The 
research in this project includes three topics. The first manner is the application of A* 
search algorithm to solve shortest path problem of single mobile robot, this object can 
developed path planning foundation of multiple mobile robots. The second manner is the 
combination of A* search algorithm and greedy algorithm to solve the routing problem of 
single mobile robot, and apply 2-opt exchange heuristic to improve the path. The third 
manner is the combination of the second manner and sequential single-item auction 
method to solve the problem of multi-robot task allocation.
