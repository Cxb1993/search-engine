2新型五螺箍SRC矩形柱之強度與韌性研究
翁正強
國立交通大學土木工程系暨研究所教授
摘 要
本研究進行一系列採用新型五螺箍 (5-Spirals) 之大尺寸預鑄矩形斷面鋼骨鋼筋混
凝土（Steel Reinforced Concrete，SRC）柱之軸壓試驗，目的在探討五螺箍在矩形 SRC
柱之使用效益。由於本研究採用的五螺箍是以自動化機械加工製造，有助於大幅降低箍
筋綁紮之人力及縮短工期，因此極具經濟效益，很適合應用於預鑄工法。再者，由於
SRC 柱中之鋼骨亦能夠對 SRC 柱核心之混凝土提供有效的圍束，因此有助於降低圍束
箍筋之用量，達到更為經濟的設計結果。試驗結果顯示，配置五螺箍之矩形 SRC 柱可
以發展出良好的強度與韌性。五螺箍除了圍束混凝土的能力比傳統橫箍筋好之外，亦沒
有橫箍筋在角隅處可能發生彎鉤脫落之問題，因此能夠發揮較佳的韌性。本研究發現，
在相同的斷面尺寸與縱向總用鋼量之情況下，五螺箍 SRC 柱之韌性明顯的比配置傳統
橫箍筋的 RC 柱好很多，亦優於同樣配置五螺箍之 RC 柱。整體而言，採用五螺箍之試
體其韌性皆優於採用傳統橫箍筋者。最後，本研究並建議一套適合應用於這種新型 SRC
柱之圍束箍筋量設計法，以提供設計者參考。

3.2 試體設計
在箍筋用量方面，本研究採用三種不同的方法來配置 SRC 柱之箍筋，分別為（1）
ACI-318 Code、（2）Taiwan SRC Code 及（3）Weng’s Formula[8] 。其中 ACI-318 Code
與 Taiwan SRC Code 設計法已於前面敘述，以下針對Weng’s Formula之設計法進行說明。
3.2.1 Weng’s Formula設計法
Weng’s Formula設計法[8]主要考慮到SRC柱內之「高度圍束區混凝土（Highly Confined
Concrete），指受鋼骨所圍束之混凝土」可以由SRC柱內之鋼骨斷面之翼板來提供圍束，而SRC
柱之箍筋則主要用來圍束「普通圍束區混凝土（Ordinarily Confined Concrete），指箍筋內側之混
凝土但不包含鋼骨翼板所圍束之區域」，因此SRC柱圍束箍筋之用量得以進一步放寬。以下為

4.1 SRC短柱受軸壓之破壞行為
圖4顯示2支配置不同型式箍筋之RC柱的軸力-位移曲線，由圖中可看出，配置五螺箍的RC
柱（RC2）比傳統橫箍RC柱（RC1）的韌性明顯優越。在張國鎮、尹衍樑等[4]之研究中亦可發
現類似之曲線。本研究發現，橫箍柱角隅之135度彎鉤，因為混凝土側向膨脹與主筋挫屈造成彎
鉤脫開之現象，如照片1所示。另一方面，在極限載重過後，五螺箍柱因為外圍小螺箍筋先行斷
裂，此時試體之軸力稍微下降，但仍可支撐大部分的載重，展現良好韌性，如照片2所示。直到
試驗後期，大螺箍發生斷裂之後，軸力才急速下降。本研亦究發現，在相同縱向總用鋼量之下，
所有五螺箍SRC柱之韌性皆優於五螺箍RC柱。如圖5所示，SRC3、4及5之韌性皆比五螺箍RC柱
及橫箍RC柱佳。另一方面，如圖6所示，鋼骨用量較高之五螺箍SRC柱（SRC10、11），雖其箍
筋間距較大，但韌性仍較五螺箍RC柱及橫箍RC柱佳，顯示五螺箍應用於SRC柱時，能有效減少
箍筋用量，亦能增進柱的耐震能力。
4.2 試體韌性之探討
本研究根據軸壓試驗之軸力-位移曲線，以 SRC 柱軸力下降至其極限軸壓強度七成時所測
得之應變量，ε0.7Pu，與其在達到極限軸壓強度時之應變量，εPu，之比值，即ε0.7Pu /εPu，作
為 SRC 柱之韌性指標（Ductility Index），μ。當 SRC 柱之韌性指標愈大時，表示 SRC 柱的延
展性愈好，愈不容易發生脆性破壞。

五、結 論
本研究進行一系列 SRC 及 RC 柱之軸壓試驗，試體型式包括四類：（1）五螺箍 SRC 柱，（2）
傳統橫箍 SRC 柱，（3）五螺箍 RC 柱，（4）傳統橫箍 RC 柱。根據試驗結果，本研究分析以上
四類試體的強度、韌性及箍筋用量之經濟性等。分析結果顯示，在相同的縱向總用鋼量之下，
採用五螺箍之 SRC 柱具有最佳的強度、韌性與經濟效益。本研究初步歸納出以下重點：
（1） SRC 柱之箍筋用量與經濟效益：
在相同的縱向總用鋼量之下(包括主筋與鋼骨)，五螺箍 SRC 柱所需之箍筋用量明顯低於
傳統橫箍 SRC 柱。例如，五螺箍 SRC 柱(試體 SRC5)之「每單位長度箍筋用量」為橫箍
SRC 柱(試體 SRC1)之 79％，經濟效益明顯。對於同樣採用五螺箍的 SRC 柱與 RC 柱而
言，SRC 柱之箍筋用量比 RC 柱少，但前者之韌性表現皆優於後者。例如五螺箍 SRC 柱
(試體 SRC5)之箍筋用量僅為五螺箍 RC 柱(試體 RC2)之 65％。
（2） SRC 柱之韌性：

5預鑄五螺箍矩形 SRC柱之軸壓
與反覆載重抗震試驗
翁正強 1 尹衍樑 2 王瑞禎 2 梁景裕 1
   ( 1.台灣交通大學土木工程系, 台灣新竹 30050; 2.潤泰集團, 台灣臺北 10492 ) 
摘 要：本研究成功的進行一系列大尺寸鋼骨鋼筋混凝土( Steel Reinforced Concrete，SRC )矩形柱之
軸壓試驗與反覆載重抗震試驗，旨在探討新型的複合螺旋箍筋（本文稱之為「五螺箍
( 5-Spirals )」或「尹氏螺箍 ( Yin’s Spirals )」）在矩形 SRC 柱之使用效益。本研究的五螺箍
採用自動化加工製造，具備高度經濟效益，很適合用於預鑄工法。此外，由於 SRC 柱內部
之鋼骨亦能夠圍束混凝土，有助於降低 SRC 柱箍筋之需求量，達到更經濟的設計結果；且
螺旋箍筋對混凝土之圍束效果比傳統水平閉合箍筋更好，亦可提升柱之強度與抗震能力。試
驗結果顯示，SRC 柱之反覆載重遲滯迴圈（Hysteretic Loop）均相當飽滿，試件的層間變位
角（Drift Angle）高達 6.0% 弧度，且 SRC 柱之強度可以持續的維持在高檔。此一現象顯示
在 6.0% 弧度的大變形之下，五螺箍 SRC 柱仍然具備十分穩定的強度，可以有效的抵抗強
震引致之水平剪力，而不致於發生快速崩塌之現象。因此，五螺箍 SRC 柱在抗震方面具有
卓越的特色，對於保障大地震來襲時之生命與財產安全有重要意義。 
關鍵字：預鑄工法；鋼骨鋼筋混凝土（SRC）；矩形 SRC 柱；五螺箍；圍束效應；軸壓試驗；反覆
載重試驗；抗震能力；經濟效益 
 
1、前言
SRC 建築構造結合了鋼骨（S）與鋼筋混凝土（RC），它可以兼具「S」構造與「RC」構造之特
色。由於 SRC 具有許多結構力學的優點，因此台灣近年來採用 SRC 構造的建築物有增加之趨勢 [1]。
以 SRC 柱為例，包覆鋼骨的混凝土可以發揮以下功能：（1）提供鋼柱防火被覆；（2）增進鋼柱防銹
功能；（3）提昇鋼柱之抗壓能力；（4）增加鋼柱之勁度。另一方面，由於 SRC 柱中之鋼骨亦能圍束
混凝土，因此有助於降低 SRC 柱箍筋之需求 [2]。如圖 1 所示 [3]，目前矩形 SRC 柱之箍筋大多配置
水平閉合箍筋（Hoop，簡稱橫箍筋）。由於橫箍筋之綁紮很耗費人力，因此若改採自動化機械加工
製造的螺旋箍筋，將可大幅降低人力需求。再者，螺旋箍筋具有連續性，不像每支橫箍筋均為獨立
個體且需在角隅以彎鈎固定，因此螺旋箍筋不僅對混凝土的圍束效果更好，亦能節省箍筋用量。
五螺箍的配置方式是以一個中心的大圓螺箍，再搭配四個角落的小圓螺箍交織而成。這種創新
的配置方式突破了傳統上螺旋箍筋僅適用於圓形截面柱的限制，巧妙的克服了螺旋箍筋應用於矩形
截面柱的缺點，使它對矩形柱四個角落的混凝土發揮良好的圍束效果。再者，由於五螺箍可以採用
自動化加工製造，可大幅降低傳統箍筋綁紮之人力及縮短工期，因此極具經濟效益，很適合用於預
鑄工法。關於柱箍筋採用新型五螺箍的配置方式，已經由張國鎮、尹衍樑等 [4, 5] 證實可以成功的應
用於矩形 RC 柱上。

2、Taiwan SRC Code 設計法

[7]，（2）Taiwan SRC Code [8] 及（3）Weng’s Formula設計法 [9] 。以下針對 Taiwan SRC Code

3.Weng’s Formula設計法
翁正強等 [9] 經由SRC柱軸壓試驗與力學分析，提出一稱為 Weng’s Formula 之新設計法，該
法主要是考慮到SRC柱內的鋼骨翼板也能夠提供混凝土有效的圍束，因此可以使SRC柱之箍筋用量
更節省。如圖3所示，Weng’s Formula 認為SRC柱內之「高度圍束區混凝土」可以由柱內鋼骨之翼
板來提供圍束，而SRC柱之箍筋則只需用來圍束「普通圍束區混凝土」。由於鋼骨翼板已經圍束了一
大部份的混凝土，因此SRC柱所需要的箍筋用量將可以進一步以予減少。以下為 Weng’s Formula

7、結 論
本研究成功的進行一系列配置新型五螺箍(5-Spirals)的矩形 SRC 柱之軸壓試驗與反覆載重抗震
試驗。根據試驗與分析結果，在本文探討的參數範圍內，可初步歸納出以下重點：
（一） SRC 柱之強度：
在相同的軸向總用鋼量下，配置五螺箍之 SRC 柱其箍筋間距可較傳統橫箍 SRC 柱及 RC 柱
寬，但試驗之軸壓強度並不會較差。試驗結果顯示配置五螺箍之 SRC 柱可以在較少之箍筋
用量下，達到優於傳統橫箍 SRC 柱及 RC 柱之軸壓強度。
（二） SRC 柱之韌性：
五螺箍 SRC 柱之韌性表現比傳統橫箍 SRC 柱更好，亦優於同樣配置五螺箍之 RC 柱，並且
明顯的比傳統橫箍 RC 柱好很多。五螺箍除了圍束混凝土的能力比較好之外，亦沒有橫箍筋
可能發生彎鈎脫落之問題，因此具備更優越的抗震潛能。 
（三） SRC 柱之抗震能力：
五螺箍 SRC 柱之反覆載重遲滯迴圈均相當飽滿，層間變位角高達 6.0% 弧度，具備卓越的消
能與抗震能力。本研究發現，在 6.0% 弧度的大變形狀況下，SRC 柱之強度仍可維持在高檔，
顯示 SRC 柱具備穩定的強度，可有效抵抗強震引致之水平力，而不致發生快速崩塌，此一
特點對保障地震來襲時之生命安全有重要的意義。
