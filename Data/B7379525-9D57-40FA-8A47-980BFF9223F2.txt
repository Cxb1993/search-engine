摘要
近年來人因工程強調以「使用者為中心」的設計原則，逐漸受到重視。產品設計的最終目的，在
於滿足使用者的需求，唯有把「使用者」放在設計的過程中，根據人因工程的原理，符合「人」
的行為特性，才能設計真正滿足使用者需求的產品，進一步營造人性化的環境。人因工程如何應
用在產品設計，牽涉到心理與生理的層面，其中生理方面的人體計測資料的應用，在設計實
務的應用上是一個值得探討的實務問題。
現今社會科技資訊發達，視訊技術不斷的推陳出新，從傳統的類比式電視，發展至衛星電視，有
線電視與新的數位電視。在硬體方面有液晶電視與電漿電視的開發；現今的電視節目接收不再限
於傳統定點的欣賞，更增加了行動性，甚至走向個人化設定，讓電視節目不再受限於定點定時的
欣賞，更成為現代人數位娛樂生活所不可或缺的一部分。本研究主要是針對目前市面上新款的數
位視訊轉換器產品為例，藉由操作使用性評估與人因工程評估比較，針對目前市面所流通之數位
視訊轉換器，深入瞭解﹝1﹞使用者選購數位視訊轉換器的因素；﹝2﹞數位視訊轉換器經常性使
用的功能；﹝3﹞操作數位視訊轉換器時所感受到的不方便與舒適度；希望能藉由現況分析、設
計分析、使用者為中心的研究方法來找出相關可能原因，作為未來改良與設計之依據。
關鍵字：數位視訊轉換器、操作使用性評估、人因工程、設計。
Abstract
The assertion in this project has been concentrated mostly on the human set top box interaction.
The procedures adopted for the assessment included the current set top box survey, a design
analysis and a subjective preference evaluation. First, identify the merits and demits including
interface factors of set top box used through task analysis. Second, to compare different set top
box working conditions. Finally, to establish the design approach bases on the user center system
and human interface interaction finding.

STB; Usability; Design
