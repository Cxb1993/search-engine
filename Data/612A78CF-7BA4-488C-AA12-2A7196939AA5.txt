研究摘要(500 字以內)： 
今日隨時隨地都存在無所不在，如電視、網路攝影機，和手機等的互連網多
媒體物件(Xmedia Objects)。但是將這些隨處網路多媒體物件 NUXO (Network 
Ubiquitous Xmedia Objects)有效的連結起來是一有趣又具挑戰性的應用。 將兩個
數位網路電視連起來可以讓兩個使用者一同欣賞或分享共同喜歡的節目。而根據
觀賞內容或觀賞時間點與習慣的相似性，可形成一社會網路。 
社會網路是一具某相似特質的社群結構。 社群的個體利用這些相似特質特徵連結
起來。這些相關性可連結兩個或多個簡易相關個體或兩個或多個親密相關。本計
畫將建立在既有的數位電視網路的基礎上發展一平台以開發不同的應用。這平台
將使用 P2P 技術以結合既有的 Skype 網路。 Skype 是目前網際網路最普遍使用的 
voice over IP。 本計畫將在 Skype 上加入多媒體物件連接以達到 NUXO 服務- 
SkypeX(a Skype Network with Xmedia connection)。基於 P2P 架構和 Xmedia 的內
容，一社會網路可因此形成。這些關鍵技術不僅可增加既有產品（如數位電視和
網路攝影機）的銷售，而且可創造新的終端產品與應用。

技術研發成果說明： 
本計畫所完成之研發項目及成果主要包含以下七點：Skype Agent、Database、
Online engine、Xmedia 整合、Webcam、Cell phone、Crossover media。以下將
逐項說明： 
z Skype Agent：主要負責處理來自使用者的 Skype 的動作。為系統與使用者的
互動。透過 Skype Agent 實現註冊確認通知、電視節目預約通知等事宜。Skype 
Agent 實現了透過即時訊息或簡訊來對 NuTV 社群網路的成員發送註冊通知
以及節目通知等功能。想要加入 NuTV 社群網路的使用者，只要在 NuTV 社
群網頁上註冊自己的資料，Skype Agent 就會透過即時訊息發送確認註冊資料
description /metadata，系統可讀取電子節目表的資訊，或是節目內容
描述資訊與頻道性質描述資訊。 
 Filtering Agent 簡稱 FA。主要負責語意分析與推演的工作。 
 Schedule Agent 簡稱 SeA。主要的功能進行管理系統事件排程。 
 Skype Agent 簡稱 SA。主要負責處理來自使用者的 Skype 的動作。為系
統與使用者的互動介面，系統可透過 SA 與使用者直接互動，例如：Social 
Search 等。 
 Query Agent 簡稱 QA。主要負責資料查詢的工作。 
z Xmedia 整合：今日隨時隨地都存在無所不在如電視、網路攝影機和手機等的
互連網多媒體物件(Xmedia Objects)。本計畫研究主要項目便是將如此的隨
處網路多媒體物件作有效的連結整合，本計畫主要將分做四大方向做研究整
合：單一 webcam、多 webcam、Cell phone 介面開發、交互媒體介面。 
 
技術特點說明： 
Vpeye 中介系統相較於一般 virtual camera 系統所佔用的系統資源有明顯的優
異之處，Vpeye 提供類比電視訊號、數位電視訊號、多媒體影音資料等不同類型
來源端的整合。Vpeye 中介系統利用 MFC 軟體開發讓使用者可以更輕易的操作，
並且可以輕易地知道現在視訊所顯示的影像。除了顯示多媒體檔案外，系統同時
支援 MMS 網路串流的播放以及桌面影像的顯示，並且可以選擇已安裝在系統中
的視訊設備。 
 
可利用之產業及可開發之產品： 
z Nu 手機 
隨著寬頻無線網路(WiMAX)的發展，手機行動通訊越趨重視隨身攜帶的
手機就像一部迷你的筆記型電腦形成手機網路讓加入 Nu 手機的同好可
以共享彼此的創意鈴聲或是創意照片，影片而不必透過電腦的傳輸這些
繁瑣的步驟並利用手機行動的優勢將自己現在的生活及時的在個人 Blog
更新讓社群中的好友可以更即時的知道使用者現在的狀況解決現在 Blog
無法提供的即時性應用。 
 
z NuTV 
NuTV 不同一般網路電視(IPTV)的部分在於 NuXO 平台的整合和分散式
影音內容的即時性，NuXO 平台能支援 TV 社群服務，實作系統具有
context-aware 能力的電視服務和調整服務方式的特性，能隨時隨地根據
使用者狀態調整。影音分享與傳送建置在 Skype network 基礎上，Skype
所扮演的角色類似影音分享與傳遞的連接器，而 NuXO 系統頻台則是這
些連接器的資訊交換中心。 
NuMusic 
類似 NuTV 的概念，將大家所擁有的音樂聯結起來，到社群朋友的音樂
資料庫中試聽音樂並可以利用 NuXO 的網站，討論流行音樂或是古典音
樂欣賞的論點以及看法，尋找的自己所喜歡的音樂或是喜歡創作的朋
友，也可以將創作的歌曲利用這個平台快速的分享給其他喜愛音樂的人。
