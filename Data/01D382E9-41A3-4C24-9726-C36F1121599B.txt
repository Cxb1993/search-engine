三. 預期完成之工作項目 
 
本計畫擬於三年內，設計、研製、並分析數種降頻器微波電路，其分年之研
究子題如下： 
第一年 進行降頻器單晶微波積體電路之設計與製作， 
第二年 進行量測第一年所製作之電路以及設計多功能整合之晶片，並進行下
列項目： 
1. 高頻電路單晶片之設計及製作， 
2. 元件模型之驗證及改進。 
第三年 進行第二年所設計多功能整合晶片之量測，並完成系統整合。 
 
 
四. 第一年 ( 98.8.1-99.7.31) 研究項目執行進度 
 
1. 設計降頻器之晶片 ( 已完成 ) 
1) 低雜訊放大器之設計 ( 已完成 ) 
2) 混頻器之設計 ( 已完成 ) 
2. 元件等效電路之驗証及改進 ( 進行中 ) 
3. 實驗系統建立與量測 ( 近完成階段 ) 
4. 撰寫報告 ( 準備階段 )
