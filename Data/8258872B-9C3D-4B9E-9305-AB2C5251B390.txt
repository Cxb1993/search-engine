1摘要
本研究計畫之主旨在於研發以奈米碳管（Carbon nanotube, CNT）批覆及成長於載體上，並應用於
環境或製程水樣微量分析之新型固相萃取管(Solid-phase extraction tube, SPE)。研發目的是充分利用
CNT對苯環分子(鄰苯二甲酸酯類)之特殊吸附性，研發具高附加價值之萃取管。此固相萃取管除了具
備CNT本身之優異吸附性質之外，也需同時具有高度之熱穩定性，以便高溫脫附之操作；及高孔隙度，
以使水樣快速通過。惟CNT本身因受限於其微小的尺寸，若直接填充於空管中，將不利於通水性的要
求，因此需將其批覆於載體之表面。因此，本計畫的構想即欲充分利用奈米碳管之吸附能力，將其批
覆、成長於適當的粒狀載體表面後，製備一適用於水樣分析之固相萃取管，以達到符合具自動化熱脫
附之層析質譜儀的儀器分析要求功能。此研究計畫之先期研究曾嘗試利用化學鍵結法（即自我組裝程
序）進行奈米碳管之批覆於粒體玻璃載體的表面，雖能達到披覆之效果，但是披覆均勻度欠佳，且存
在高溫脫附時鍵結穩定性的疑慮。因此，本計畫以一年的時間，分別進行三項工作：（一）改良化學
鍵結法，達成披覆率及披覆均勻度提昇的程序；（二）以跨學門合作方式，與本校電機系實驗室合作
以常壓熱化學氣相沉積系統在陶瓷顆粒載體表面成長奈米碳管（三）測試成品之熱穩定度、固相萃取
效率、及再用性。其中化學鍵結法主要包括玻璃粒體表面之胺氮化、奈米碳管表面之官能基化、及奈
米碳管間之相互鍵結等程序最佳化。熱化學氣相沉積系統的部分，則是先以含浸方式在陶瓷顆粒載體
表面鍍上金屬催化劑，再將載體置入熱化學氣相沉積系統中成長奈米碳管。研究結果顯示；在化學鍵
結法部份，含CNT之附合材料對於低碳數之鄰苯二甲酸酯類化合物有較好之吸附能力，但隨著吸附質
含碳數的增加；吸附效果漸漸下降，可是對於鄰苯二甲酸酯類化合物吸附；整體效果仍然比Tenax TA
與Carbotrap C 還要來得好也較平均。在熱化學氣相沉積法部份雖未盡全功；得到如當初預期的群落狀
生長之CNT，但亦製造出了具有相當程度石墨化結晶之產物，其中包含了奈米碳球、立體網狀結構等，
此部份產物可能亦具有一定程度之吸附能力，因此對於這部份產物未來可進行進一步的吸附效率測試。
關鍵詞：奈米碳管、固相萃取、鄰苯二甲酸酯類、化學鍵結法、化學氣相沉積法、奈米碳球、立體網
狀結構
3一、前言
自從二次世界大戰之後；塑膠材料被引進至商業使用中，於是有越來越多的商業產品中具有這種
耐熱、堅固、絕緣且化學性質穩定的材料，直至今日 PE (Polyethylene)、PVC (Polyvinyl chloride)、PP
(Polypropylene)與 Nylon 等塑膠材料在人類日常生活中，幾乎已經是無處不見，塑膠材料為工業的發展
帶來了幫助、人類的生活帶來了便利，成為了人類生活中不可或缺的一部份，但卻也為人類的生活環
境帶來了污染。具良好延展性的塑膠材料其原因在於製造過程中添加塑化劑(Plasticizer)，而在塑化劑
當中；鄰苯二甲酸酯類(Phthalate esters, PAEs)是一種常見的塑化劑，鄰苯二甲酸酯類為鄰苯二甲酸酐與
醇類反應所生成的酯類的統稱，當作為塑化劑時；由於鄰苯二甲酸酯類並非以共價鍵結的型式與聚合
物結合，因此在塑膠產品的生產、使用過程中；由於時間、酸鹼、溫度等因素的影響下，會釋放至環
境中。
鄰苯二甲酸酯類除了在工業上扮演塑化劑的角色外，當此類含苯之物質進入環境後則成為了「外
因性內分泌干擾物質 (Endocrine disrupting chemicals, EDCs)」亦可稱為「環境荷爾蒙」，此類物質若於
環境中進入生物體內後，將會模擬生物體內的天然荷爾蒙，進而影響生物體內的生理調節機能，造成
生殖功能、神經與免疫系統、性徵發展與胎兒發育等方面的不良作用。
為了去除自然環境水體中的鄰苯二甲酸酯類，一般商業吸附材料是可作為考量選項之一，但商用
吸附劑具有高背景值、低選擇性及成本較高等問題，因此本研究群嘗試；利用商用多壁奈米碳管以化
學鍵結的方式披覆於玻璃珠載體上，製造一種奈米碳管附合材料用於水相中的鄰苯二甲酸酯類吸附、
萃取分析，此外本研群亦嘗試以更直接、省時的方式將碳材料成長於載體上，預期將此複合材料應用
於水中鄰苯二甲酸酯類之吸附。
二、研究目的
奈米碳管（Carbon nanotube, CNT）因具有高度石墨結構，而被預期可能為是對苯環結構分子之有
機污染物的一種良好吸附材料。近幾年亦有多項研究開始著力進行將CNT 當作吸附材來吸附水相中之
污染物。本研究群之先期實驗曾利用奈米碳管當吸附劑來吸附水相中的鄰苯二甲酸酯類之有機污染
物，結果發現奈米碳管的確較一般商用吸附材有較高的飽和吸附量。但是，有別於一般吸附材質，奈
米碳管之造價甚高、且有必要之純化程序，更由於奈米級之尺度使人們對於其大量使用於環境中造成
之健康風險存有疑慮，使其不適於大量應用於處理環境污染物。本實驗室基於此考量，認為以CNT 為
吸附材質來取代商用吸附劑，做為環境水樣分析的前濃縮吸附劑，即固相萃取（Solid phase extraction,
SPE）材料。
本計畫主要規劃以兩種方式將奈米碳管披覆在載體上，一種為化學鍵結法，另一種為熱化學氣相
沉積法。前者主要的優點為可在常溫下操作且可多層的披覆，後者的優點為反應時間較短且不需使用
具腐蝕性之化學藥劑。
鄰苯二甲酸酯類（Phthalate esters）屬已知之賀爾蒙干擾化合物，長期以來用於塑膠材質中作為塑
化劑、可塑劑等用途，卻其與塑膠之間並不是以化學型態方式鍵結，因此可經由製造、使用、丟棄而
釋出於環境當中造成污染。同時，也由於塑膠之高穩定性，工業製程等級的水管也以表面處理之塑膠
管為主，雖經處理但仍會釋出少許污染物，形成微量但足以影響製程之污染問題。本計畫亦擬以該種
污染物為對象，測試CNT萃取管之分析敏感度是否優於商用萃取管。

由 SEM 圖 5-2 觀察可得知經過化學鍵結可將奈米碳管多層披覆在玻璃載體上。在實驗吸附劑對

品注入超純水並定量至 1 L，將吸附管與蠕動採樣幫浦連接後，以流量 25 mL/min 將水樣全部導入吸附

分析。而吸附容量的定義是，將通水後吸附管熱脫附出來 GC 分析後的濃度，除以在吸附管內填充的
商用吸附劑含量。

該吸附劑對於 PAEs 的吸附容量如圖 5-3 所示，Carbotrap C 對於 PAEs 的吸附容量對於低碳數的
PAEs 化合物如 DMP 及 DEP 較其他 PAEs 化合物還要高，隨著含碳數的增加吸附容量也逐漸降低，推
測原因是因為 Carbotrap C 對於含碳數較高的 PAEs 化合物通水後的損失率較高，因此 Carbotrap C 吸附
PAEs 化合物後，會因為水樣的持續通過而損失其吸附的 PAEs 化合物量。
Tenax TA 對於 PAEs 化合物的吸附容量，雖在低碳數的 PAEs 化合物如 DMP 及 DEP 的吸附容量
沒有 Carbotrap C 來得好，但對於較高碳數的 PAEs 化合物的吸附容量都較 Carbotrap C 還要高，所以若
以 Tenax TA 與 Carbotrap C 兩種商用吸附劑對於 PAEs 的吸附容量相比，Tenax TA 是較 Carbotrap C 還
要適合用來吸附 PAEs 化合物。
而以奈米碳管填充之吸附管對於 PAEs 吸附量來說，奈米碳管對於含碳數較低的 PAEs 化合物如
DMP 吸附效果最好，隨著含碳數的增加吸附效果漸漸下降，但對於 PAEs 化合物吸附容量整體效果仍
然比Tenax TA與Carbotrap C 還要來得好也較平均，因此MWCNTs比商用吸附劑Tenax TA與Carbotrap
C 還要適合用來吸附 PAEs 化合物。
