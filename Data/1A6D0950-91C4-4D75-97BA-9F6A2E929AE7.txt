社交網路分析(Social network analysis) 
社交網路分析是將社交關係以網路理
論建構，包含端點(nodes)與聯繫線(ties)。其
中端點是網路之中的個別單體，而聯繫線則
表示個別單體之間的關係。以圖形基礎所表
現的結構通常是很複雜的，例如可能有許多
不同種類的聯繫線介於不同單體之間。在許
多學術領域中的研究顯示社交網路在多層
面中運行，可以從家庭層面提升到國家的層
面，並且社交網路在解決問題中可扮演決定
性的關鍵角色，於組織中執行而使得個體最
終達到其目標。本分項計畫可藉由社交網路
的建立，提供使用者針對特定主題找尋潛在
能夠解決其疑問之專家，經過推薦的專家較
有機會順利解決使用者的疑問，進而幫助使
用者進行數位學習。

子項計畫 B1：動力馬達之分析與設計 
在無刷永磁馬達的設計與分析上，熱傳
問題是一項重要的課題，因為馬達的熱傳特
性直接影響到馬達的尺寸與壽命。文獻回顧
之結果顯示，馬達之熱傳分析大都以傳統馬
達為主[8][9]，即使涉及無刷永磁馬達，也
都以2D熱傳分析模型為主[10][11]，極少使
用3D熱傳分析模型。在過去，馬達的熱傳
場問題不是用數學分析法就是利用熱網路
法[12]，但在計算機快速發展之後，利用數
值解析就成為最熱門的工具，如有限元素法
[13]和有限差分法[14]，如圖2所示。其中有
限元素法更適合處理複雜的邊界條件以及
材料特性不連續的問題。

而通常在馬達系統中會有三個損耗情
形發生：（1）定子與轉子的鐵損（Iron loss），
（2）線圈的焦耳損（Joule loss），（3）磁
鐵的渦流損（Eddy Current loss），本研究
子計畫在熱分析過程中先進行電磁模擬以
確定馬達的磁通密度分佈，主要是運用
JMAG軟體的“Iron Loss Tool”來確認鋼鐵材
料中的鐵損。如此再加上線圈的焦耳損以及
磁鐵中的渦流損所計算得的總損耗，便可與
量測的損耗數據相互比較。如果二者的數值
近似時，則模擬的損耗數據便可用來計算馬
達的熱傳響應。  
此外，為了解決過去電動載具所面臨的
過重問題，將永磁無刷馬達與車輪結合為一
體式之直驅式輪轂馬達 (in-wheel direct 
drive motor)，已成為電動載具動力源發展的
趨勢。本計畫第一年所發展之輪轂馬達亦是
同上述的架構。此種輪轂馬達設計無須傳動
機構，可避免變速器與減速傳動機構所造成
的能量傳遞損失，同時可有效地降低動力系
統的體積與重量。然而受到體積與電氣規格
的限制，單獨靠輪轂馬達本身之輸出動力，
仍難以同時滿足高啟動扭力與高極速之行
車動力需求，因而發展出許多解決方式以增
加馬達動力輸出範圍之策略，如弱磁控制
[15][16]、調變氣隙大小[17-19]、調變永久
磁鐵等效磁通[20][21]、複激式馬達[22][23]
等。但上述方法一般會增加系統之複雜度、
體積與成本，為克服此一問題，本子項計畫
參考電子式變結構繞組專利[24]，在不需增
加額外機構、體積與成本下，透過電子式的
切換，變換馬達的繞組結線以拓展馬達動力
輸出範圍。故在第三年度計畫中，針對計畫

轉子錯誤，換流器與功率電路錯誤、感測器
錯誤、及機械(如轉軸、軸承)錯誤。馬達定

當定子繞線錯誤發生時，電壓、電流或
熱應力會作用於繞線上，發生初期時馬達仍
可運轉，但在短路繞線內的大循環電流會產
生過熱因而造成嚴重傷害。偵測定子繞線錯
誤 的 相 關 文 獻 有 小 波 轉 換 (wavelet 
transform)[30]、負序向電流觀測[31]、線電
流之頻率分析法[32]、電流偵測法[33][34]、
及類神經網路等智慧型控制[35-37]。  
而實施高頻PWM技術會給切換開關更
多應力，造成開關比上述其他元件更易發生
問題。而切換開關短路可能由於過電流造
成，但一般驅動器皆有此種保護，因而不會
有其他問題或傷害發生。切換開關開路錯誤
因為比短路問題反應較慢，因而初期易被忽
略，但它會造成電流失真，而使驅動系統產
生雜訊與震盪。一般依據電壓感測器擺放位
置，偵測技術可分成 (1)端電壓量測法
(terminal voltage measurement method)，(2)
線電壓量測法，(3)相電壓量測法，(4)中性
點電壓量測法。但這些方法會增加硬體成
本，且因PWM高頻工作之故，需使用高解
析度之元件。因此，上述問題為本子項計畫
第三年度研究方向之一 。 
於輪轂馬達與驅動器之應用方面，由於
近年來節能減碳與綠色環保意識之抬頭，使
得各種以馬達為動力源的電動載具蔚為發
展之風潮，特別是以輪轂馬達為主之左右直
驅型電動載具最被廣為重視。其原因主要在
於以電子(虛擬)差速同動技術來取代機械
式差速機構，以提升較佳之操作性能。但在
戶外操作時，常因環境阻抗複雜而不易維持
路徑，此乃因目前多數之左右輪驅動控制仍
採用開迴路電流或電壓控制，而當左右側之
驅動輪受到環境抗力不均或馬達規格參數
不一致時，會因左右兩側受力不同而偏離原
移動路徑，造成使用者需不斷地修正方向，
增加操作的困難度。因此，本子項計畫針對
差速同動之誤差補償進行設計與研究。  
子項計畫 C2: 智慧型能量管理系統 
電池模組殘餘電量方式之評估，常藉由
量測電池開路電壓值以反推電池可持續使
用時間，但電池模組開路電壓之非線性變
化，卻常造成預估殘餘電量之誤差。故本子
項計畫將建立新型電池殘餘能量評估準
則，並依電池內部儲存電荷量不同，予以計
算電池殘電量，以提高預測精確度。同時，
本子項計畫亦製作圖形化人機介面，期以透
過圖形方式呈現電池模組運轉狀態，諸如輸
出電壓、輸出電流與殘餘能量，以供使用者
快速掌握電池模組即時資訊[38-40]。  
為更精確分析及評估電池運轉特性及
評估電池可正常操作之狀態，本子項計畫建
立鋰電池模組之等效電路模型，此等效電路
將涵括電池電壓參數及電池內阻參數，其中
電池電壓參數將可協助預估電池內部可利
用之殘電量，而電池內阻參數則用以分析建
立電池壽命評估準則，俾於確認電池健康變
化[41-44]。  
電池模組於使用時，因運轉環境之不
同，可能造成電池操作於異常情況，諸如過
大電流放電或電池模組本體溫度過高等，若
長期處於此操作情形，將可能降低電池模組
使用壽命。因此，本子項計畫建構電池運轉
資訊記錄平台，透過平台之協助，將有益於
使用者掌握用電歷史資料，且因其具備異常
警示功能，透過即時警示訊號之協助，可有
助於減少電池模組重覆操作於異常狀態
中，裨於延長電池壽命[45-47]。  
針對實車測試試驗，除與其它分項子計
畫針對功能性整合外，同時本子項計畫亦分
析各整合程序中所產生之問題，並進行分析
提出可行性改善方案，裨使本計畫中之技術
更可符合實際運轉操作。

技術說明
(中文) 本專利針對電動載具上具Wye 接線和Delta 接線結構繞組的馬達，提出Delta 
180°分段切換驅動法。相較於先前Delta 型接線以120°驅動法驅動馬達無需輸出
0°換相訊號線，另外也改善傳統式Delta 
180°驅動會產生大振幅的突波電流。其控制方式是以原變結構切換繞組架構為基
礎，令Delta 180°驅動法時未導通相的繞組開路，防止因delta 180°驅動特性造
成馬達某一相繞組為發電機或煞車模 
式，因而令馬達出力不平均，進而讓馬達產生速度不穩的情況。
(英文) This patent proposes a 180-degree excitation of delta connection for a motor with both 
wye and delta configuration. Compared with the previous 120-degree excitation 
approach, this patent provides merits of eliminating the need of the second set of hall-
effect signals starting at 0 degree and suppressing the current surge caused by 
conventional excitation. Based on the previous variable-winding method, this patent 
opens the switch of the floating phase to prevent the generating mode of the floating 
phase. As a result, the motor will run smoothly.
產業別 電機及電子機械器材業；運輸工具製造業
技術/產品應用範圍 電動汽機車，代步機，跑步機等需要提升無刷直流馬達轉速之應用面。
技術移轉可行性及
預期效益
前技術雖已具馬達變繞組驅動功能，但因Delta 接線是以120°驅動法驅動馬達，所以馬
達需再輸出0°換相訊號線，而本技術於Delta接線改以180°驅動法驅動馬達，其換相角度
與Wye 接線120° 驅動 
法相同，馬達可減少出線數，降底製程困難度與成本，並提高系統可靠度。
