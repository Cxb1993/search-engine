一、 摘要 
 
新型靜態式全頻譜標準色彩影像系統可得到待測物在光學上所有的資訊，其包含光譜
資訊、空間資訊、強度資訊、色彩資訊、影像資訊，其應用在可見光、不可見光頻譜量測、
物體(動物/植物/礦物)影像光譜、近紅外光 NIR 光譜影像、多通道光纖光譜攝像系統、遠端
感應、太空遙測、生醫光譜影像、顯示器色彩調校、智慧搜尋飛彈…等，由於應用領域廣
泛，因此產業價值非常可觀。 
目前超頻譜影像裝置在靜態影像方面如欲取得二維影像，必須將整個裝置或其物體移動，
若應用在生醫或是顯示器方面時，將會不利於量測的方便性，再者更換其前端光學影像系
統(量測鏡頭)時，後段光學系統也必須重新設計，依此目的本計畫欲設計一機構，其主要架
構為光學影像系統、可微控之延遲透鏡組、超光譜系統，其中可微控之延遲透鏡組可輔助
超光譜儀達到 2 維影像與光譜的掃描，而光學影像系統可以是顯微鏡，望遠鏡等成像鏡頭，
或者單純是影像輸入，因此可在不移動影像、光學成像系統、超光譜儀系統三者的情況下，
量測影像的頻譜，此將適用各式光學影像系統，而不需再重新設計後端之光學系統。如此
機構之研發不僅是科學領域的前瞻性，也是未來商品化的影像量測標準裝置，也是作為高
解析度超頻譜動態影像的開端。 
 
Abstract 
   
The hyper-spectrum standard image system can obtain all of the optical information of test 
object including spectral information、spatial information、intensity information、color 
information and image information. The system can apply to visible spectrum measurement、
invisible spectrum measurement、objective(animals/plants/minerals) image information、near 
infra- red spectral information、multi-channel fiber spectral camera system、remote induction、
space and remote sensing and biophotonic spectral image…etc. 
Until now, it has some problems to obtain 2D image through hyper-spectrum setup. One 
problem is not convenient to move the hyper-spectrum setup or object in order to obtain 2D 
image. The other problem is, when we change the measuring lens the behind part of optical 
system must redesign.  
The purpose of this project is to design a new hyper-spectrum setup which includes optical 
image system、relay lenses module and hyper-spectrum system. The new hyper-spectrum setup 
can obtain 2D image through only move relay lenses module. The new hyper-spectrum setup is 
more convenient to apply to measurement and the beginning of commercialization.

四、 技術特點說明 
 
主要架構為可微控之延遲透鏡組(Relay Lens)，可輔助超光譜儀(Hyper-Spectrometer)達
到 2 維影像與光譜的掃描。所謂延遲透鏡可將影像從一處轉播放到另一處，而影像及物體
本身的大小關係固定，常應用在來福槍瞄準器，潛望鏡系統，顯微鏡，以及軍用紅外線影
像系統，屬於非成像(Afocal System)系統。此機構另一端亦可與光學元件連結，因此可在不

微鏡，望遠鏡等，或者單純是影像輸入。而由於此一架構在搜索現有光譜影像相關的專利
與研究論文中，並無依此方式來達到光譜影像的掃描，因此相關的專利正在撰寫申請中。

基文影像公司(Given Imaging)的第一代膠囊內視鏡因為無法控制方向，且受到一秒兩張
的拍攝速率等條件的限制，使得膠囊內視鏡成效最卓越的部分是在小腸病症的診斷方面。
這是因為小腸部位的管徑面積狹小，膠囊可藉由腸道蠕動而自然通過，因此拍攝出來的影
像資訊較齊全。在第一代小腸用膠囊內視鏡發展數年後，基文影像公司另外推出第二項震
驚全球的食道用攝影膠囊（Pill CamTM ESO），這項產品也在 2004 年 10 月通過美國食品
藥物管理局的認證獲准上市。 
膠囊內視鏡的使用相當簡單，患者只需把它當做藥丸般吞入，等待 6 到 8 小時之後自
動排出，在不影響日常生活作息，也沒有不舒適感的情況下，進行腸胃道的內視鏡檢查。
檢查過程除了容易施行，病人活動不會受到太大限制外，由於儀器敏感度高，影像清晰良
好，使得消化道檢查不僅跨入新的境界，也能彌補傳統內視鏡檢查在小腸部分產生死角的
問題，使得消化道檢查更為完整。 
膠囊內視鏡的使用並非目前檢查的常態，它最重要的功能在於提供了傳統內視鏡無法
克服的小腸部分的病症檢查，因此醫學上最常運用於針對小腸道出血原因的鑑別診斷。以
往利用胃鏡和大腸鏡若無法找到出血點，往往造成治療上的困擾，膠囊內視鏡這時候就成
為最不具侵襲性的診察利器，醫師可藉由所獲得的影像，從血跡、血塊等蛛絲馬跡中尋找
病灶，進而進行治療。其他對於小腸腫瘤、腸道發炎等疾病，也具有相當的診斷價值。  
不同於傳統胃鏡或大腸鏡可來回觀看同一部位，且可做局部切片，小腸膠囊內視鏡只能藉
由腸子的自然蠕動通過人體，有時難以判斷是否真的有腫瘤存在，或只是由腸子蠕動收縮
所引起的皺摺。 
因此截至目前為止，膠囊內視鏡尚無法全面取代傳統的電子內視鏡，只有在傳統內視
鏡無法找出病因或病灶時，才會考慮採用。在使用方面的禁忌，若是病人有腸道阻塞或腸
道狹窄等問題，則有可能使膠囊卡在人體內，這時醫師就不會建議使用。通常在使用膠囊
內視鏡前，已對患者以傳統內視鏡進行過檢查，再由醫師判斷是否適用。 
膠囊內視鏡系統由影像接收器、膠囊內視鏡組成。膠囊內之微攝影機以每秒約 2 畫面之速
度，拍攝前方 140 度之消化道影像，取得生化資訊，拍攝畫面以 RF 微波信號傳出，由病
患配戴之接收器天線取得並壓縮存於接收器記憶體內，直到膠囊排出體外，全程約拍攝 5-6
萬張影像畫面。檢測療程完成後，電腦將接收器記憶體內影像資料讀取，並處理為連續畫
面供醫師診斷病變。檢測完成後，電腦將接收器記憶體內影像資料讀取，配合設計作影像
閱覽程式中病理影像辨識功能，輔助醫生作病理的判斷，及病歷資料的整理，更進一步提
供消化系統各樣的病理資訊，以達到最完整的診斷。
