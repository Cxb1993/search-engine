when the plants are not initially at rest.  
    The first-year project deals with the initial value compensation 
for the integral feedback of a PID controller, in which the integral 
feedback is initially set to a value so that the subsequent response 
has zero overshoot. In the proposed scheme, referred to as Zero-
Overshooting PID (ZOPID), the initial value that should be assigned 
to the integral feedback is found to be a nonlinear function of the 
plant’s initial state. This result is different from those obtained in 
the previous initial-value-compensation schemes that constrain the 
controller’s initial state to be a linear function of the plant’s initial 
state. For validations, the proposed scheme is applied to the Layer-
Jump Control and the One-Track- Jump Control of an optical disk 
drive.  
    The second-year project introduces the sliding-mode technique in 
order to improve the robustness of the proposed compensation 
scheme. More specifically, the standard Integral Variable Structure 
Control (IVSC) is redesigned with double integral feedback. One 
integrator’s initial value is assigned to have a sliding mode exist 
throughout an entire response, enhancing performance robustness. 
The other integrator’s initial state is assigned so as to guarantee 
nonovershooting responses. This proposed scheme is referred to as 
Zero-Overshooting IVSC (ZOIVSC). The ZOIVSC is applied to 
positioning control of a linear motion stage. Since all the schemes 
are practically applied to physical mechatronic systems, the 
graduate students participating in this project can receive advanced 
training in Mechatronics and System Integration. 
 
II 
 
 
 
Design and Application of Initial Integral-Value Assignment for 
Nonovershooting Responses 
 
 
ABSTRACT 
This project presents controller designs for nonovershooting responses by compens- ating 
for the initial value of integral feedback. Previous studies on nonovershooting responses assume 
that the plants are initially at rest. However, in most “mode-switching control systems” where 
various controllers are activated interchangeably, the plants have nonzero initial conditions when 
some controller is just activated. For “mode- switching control systems”, the results on 
nonovershooting responses have not been found so far. Distinct from the previous researches, the 
method presented in this project ensures nonovershooting responses even when the plants are not 
initially at rest.  
The first-year project deals with the initial value compensation for the integral feedback of a 
PID controller, in which the integral feedback is initially set to a value so that the subsequent 
response has zero overshoot. In the proposed scheme, referred to as Zero-Overshooting PID 
(ZOPID), the initial value that should be assigned to the integral feedback is found to be a 
nonlinear function of the plant’s initial state. This result is different from those obtained in the 
previous initial-value-compensation schemes that constrain the controller’s initial state to be a 
linear function of the plant’s initial state. For validations, the proposed scheme is applied to the 
Layer-Jump Control and the One-Track- Jump Control of an optical disk drive.  
The second-year project introduces the sliding-mode technique in order to improve the 
robustness of the proposed compensation scheme. More specifically, the standard Integral 
Variable Structure Control (IVSC) is redesigned with double integral feedback. One integrator’s 
initial value is assigned to have a sliding mode exist throughout an entire response, enhancing 
performance robustness. The other integrator’s initial state is assigned so as to guarantee 
nonovershooting responses. This proposed scheme is referred to as Zero-Overshooting IVSC 
(ZOIVSC). The ZOIVSC is applied to positioning control of a linear motion stage. Since all the 
schemes are practically applied to physical mechatronic systems, the graduate students 
participating in this project can receive advanced training in Mechatronics and System 
Integration. 
Keywords: integral control, initial value compensation, nonovershooting response, sliding mode, 
optical disk drive, linear motion stage.

同的碟片轉角下，因切入閉迴路控制時的初
始狀態不同，並無法有效抑制超越量產生，
因此有必要提出一個具高度強健行的控制
方法來解決超越量的問題。在此，使用
ZOPID，其應用方式則是利用在 Matsushita
跨單軌後，進行閉迴路控制時使用之。 
 
◆「ZOPID 於光碟機之 One-Track-Jump 
Control」之結果三：不同碟片轉角下之
ZOPID 控制 
由於碟片旋轉的過程會產生徑向偏擺
（runout）的干擾，不同的碟片轉角有不同
的偏擺干擾量，此結果為針對不同的碟片轉
動角度位置進行單軌跳躍控制，比較 PID 與
ZOPID 性能。 
接著分別以相同控制器參數進行於不
同碟片位置之跨單軌實驗，碟片位置的認定
則如同先前所述，利用自製的 FG 訊號計數
器，分別給予碟片位置 0、1、2、3、4 與 5
的位置編號（碟片轉一圈會出現 6 個 FG 訊
號脈波）。 
z 碟片位置 FG=1 時，加入 M-ZOPID 控制
並不會使原先的響應有更差的表現，如圖
三十六所示（其中 Zero O.S. PID 代表修
改型零超越量 PID 控制，即 M-ZOPID）。 
z 碟片位置 FG=1 時，系統雖僅有微小超越
量，但透過 M-ZOPID 控制可以抑制超越
量產生，如圖三十七所示。 
z 碟片位置 FG=2 時，M-ZOPID 控制明顯
消除超越量，但穩態響應受碟片擺動干擾
影響，如圖三十八所示。 
z 碟片位置 FG=3 時，此時系統響應出現較
大超越量，M-ZOPID 控制仍能有效消
除，如圖三十九所示。 
z 碟片位置 FG=4 時，系統響應依舊有較大
超越量，M-ZOPID 控制透過初始值補
償，使系統超越量明顯減少，如圖四十所
示。 
z 碟片位置 FG=5 時，因超越量較大，受限
於控制量飽和，造成 M-ZOPID 控制仍有
一些超越量未補好，但仍能有效使超越量
減少，如圖四十一所示。

六、計畫成果自評 
本 計 畫 針 對 半 回 授 控 制 系 統
（semi-feedback control systems）與多回授
控制器系統（systems with multiple feedback 
controllers）提出零超越量之積分器初始值
補償方法與應用。第一年針對使用 PID 控制
器之系統，提出積分器初始值補償方法，使
系統響應具零超越量；並將此方法應用於光
碟機之跳層控制（Layer-Jump Control）與單
軌跳躍控制（One-Track-Jump Control）。第
一年度「零超越量 PID 控制（ZOPID）」預
定完成之工作項目如下： 
1. Layer-Jump Control （I）： SONY 跳層機
制之實現。 
2. Layer-Jump Control （II）： 閉迴路聚焦
系統鑑別。 
3. Layer-Jump Control （III）： ZOPID 之實
作驗證、與高增益 PID 控制比較。 
4. Layer-Jump Control （IV）： 再加快跳層
速度之控制比較。 
5. One-Track-Jump Control （I）： 閉迴路循
軌系統鑑別。 
6. One-Track-Jump Control （II）： Matsushita
單軌跳躍控制實現。 
7. One-Track-Jump Control （III）： 不同碟
片轉角下之 ZOPID 控制。 
8. One-Track-Jump Control （IV）： 不同碟
片轉速下之 ZOPID 控制。 
與上一節「六、結果與討論」內容比對，可
知研究內容與原計畫相符且達成預期目
標。第一年計畫成果的學術或應用價值如
下： 
1. 由學理證明，本計畫提出之「零超越量
PID 控制」（ZOPID）與 M-ZOPID 設計方法
可保證受控體(1)在任何初始狀態下皆具有
零超越量之響應。此外，此初始值補償設計
方式容易，無需變更 PID 控制架構，即可使
系統具有無超越量響應。 
2. 在跳層控制方面，本計畫透過光碟機聚
焦伺服系統之實驗驗證，確認 M-ZOPID 可
改善 SONY 跳層機制切入閉迴路控制後，
發生的超越量問題，因此 M-ZOPID 的確有
到達預期之效果。 
3. 在跨單軌控制方面，本計畫透過光碟機
循軌伺服系統之實驗驗證，確認 M-ZOPID
改善Matsushita跨單軌機制切入閉迴路控制
後會發生的超越量問題，增加光碟機在跨單
軌後鎖軌的穩定性。藉由本計畫提出之
M-ZOPID 控制的幫助，亦可加快光碟機跨
單軌的速度，增強光碟機的性能。 
第二年針對參數變化較大或外界干擾
劇烈之系統，使用積分型可變結構控制器
（IVSC），以使系統有較佳之強健性；提出
積分器初始值補償方法，使系統在參數變化
與干擾存在之情況下仍然保證響應之超越

Abstract—This paper investigates the design and control of a 
novel inverted pendulum. First, the structure of the inverted 
pendulum is presented, and its mathematical model is derived 
using Euler-Lagrange’s equations. Using a transformed model, 
a nonlinear control scheme is applied to this new inverted 
pendulum. The non-linear control scheme is based on the 
well-known backstepping design approach to obtain a nonlinear 
controller that is suitable for both regulation and tracking tasks. 
Furthermore, the control scheme is practically implemented 
and compared in this inverted pendulum system in order to 
investigate its effectiveness. 
I. INTRODUCTION 
NVERTED pendulums have been widely employed to 
investigate the feasibility of various control strategies 
because of their virtually unstable, complex nonlinear 
dynamics. Different styles of inverted pendulums have been 
presented in the literature, and they represent a major class of 
underactuated mechanical systems whose minimum number 
of independent coordinates is larger than the number of 
independent actuating forces corresponding to these 
coordinates. Reference [1] provides an overview of various 
inverted pendulums and their control problems. Some 
popular schemes of inverted pendulums are car-pole 
pendulum, ball-and-beam system, Pendubot, Acrobot, Furuta 
pendulum, and inertia wheel pendulum. 
In [2], a linearized model is employed as a basis for 
designing linear state feedback controllers based on LQ and 
LQG optimization procedures. In [3], normal forms for 
underactuated mechanical systems, which allows popular 
design approaches, such as backstepping design, to be 
applied to the underactuated mechanical systems.  The 
proposed controller in [4] was designed assuming that all 
joints’ positions and velocities are small. The conventional 
integrator backstepping design (CBD) [5, 6] is a systematic 
approach to designing controllers that stabilize a vast class of 
nonlinear systems having a cascade or triangular structure. 
The CBD is characterized by a step-by-step procedure that 
recursively designs a virtual control for a scalar state

V. CONCLUSION 
This paper presents the design and the control of a novel 
weight-balanced inverted pendulum. After deriving its 
equations of motion, this system model is transformed in 
strict feedback form that allows backstepping design to be 
employed. The backstepping design is presented in this paper. 
Experimental studies have been conducted to show the 
effectiveness of the presented backstepping design.
