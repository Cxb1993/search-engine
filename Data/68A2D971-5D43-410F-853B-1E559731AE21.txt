中文摘要 
本實驗室在中科院長期的支持下，完成了新一代 Ka 頻段分佈作用放大器之研發。中科院也已
將此技術應用於國防系統中。為更切合實際應用之需求，寬頻微波管研發工作被列為重點研究項
目。在前期計畫中，我們完成三維模擬計算模型的建立與驗證，並且完成了電子光學系統設計與同
步電壓分析。利用此三維模擬軟體建立熱測模型，建立了頻寬分析與電子轉換效率分析之能力。延
續前期計畫之主要目標，本計畫將繼續針對 Ka 頻段寬頻微波管的物理特性與結構設計做為研究主
軸，主要工作重點可分為：(1) 優化微波作用段，使增益可達 25dB；(2) 完成寬頻微波管工程設
計；(3) 電子束於收集極中三維的行進軌跡分析；(4) 建立週期性聚焦系統設計能力，並使電子通
過率達 90%。結合本期與前期計畫之成果，將建立完整的理論分析、數值計算與工程設計資料，進
而大幅縮短寬頻微波管研製時程。 
 
 
關鍵詞：寬頻、微波管

一、前言 
    微波管(microwave tube)為一種真空電子器件，早期主要應用於放大與產生射頻訊號，如今，
隨著輸出功率以及操作頻率的不斷提升，拓展了多元化的用途，如材料燒結之微波源、雷達系統之
訊號源、通訊系統核心元件、電子干擾系統元件、醫學治療、鎮暴武器以及電子加速器中之加速機
制。在固態電子元件尚未成熟的時期，微波管是唯一可用之產生器(generator)，且其可應用的頻
率範圍包含了整個微波頻段。如今，微波管仍主導了高功率產生器的範疇，而較高頻率之應用主要
偏向發展固態電子元件，雖然其頻率與功率有日益增加的趨勢，但其輸出功率仍遠低於微波管，圖
一所示為此兩種微波產生器件之應用分佈圖。 
 
圖一、微波管與固態電子元件之應用分佈圖 
微波管的發展早於第二次世界大戰， 在頻率比較低的操作區間最具代表性的微波管有三極管
(triode)、四極管(tetrode)與 klystrode，亦稱為柵極式微波管(grid tube)。由於過渡時間
(transit time)的限制，柵極式微波管無法應用於頻率高於 1GHz 的範圍。為因應使用上的需求不
斷的提升，進而發展出另兩種類型的微波管，起初磁控管(magnetron)及反射式速調管(reflex 
klystron)為主要的發展項目，此兩種形式的微波管恰好分屬兩大不同類型，一為 M型微波也稱作
場交錯型微波管，最典型的代表即為磁控管。另一為 O型微波管又稱為線型束微波管，主要包含速
調管(klystron)、行波管(traveling wave tube)，其中速調管的概念即是源自於 klystrode 的基
本架構。任何一種型式之微波管都具有相同的操作原理，利用被加速之電子與作用區間之電磁波進
行交互作用，藉由獲取電子的能量，來達到放大電磁波能量之目的。 
Ka 頻段的微波源器件，在製造過程面臨極大技術瓶頸而且操作頻率升高其作用之共振腔縮小亦造
成功率及頻寬方面的限制，正因為如此使得 Ka 頻段的微波放大器價格昂貴，尤其是高功率、寬頻
帶的微波管。以高功率分佈作用放大器(EIA)為例，國際上目前僅有 CPI 的加拿大廠以及非常少數
的國家有從事該類型的研製，可見其技術門檻之高，使得其它研發單位不輕易嚐試。而清華大學物
理系高頻電磁實驗室於六年前完成國內第一支分佈作用放大器的研發，進而讓台灣的微波管研發工
作擠身國際舞台。 
本實驗室與中科院長期合作，致力於新一代微波發射器之研發。在累積了多年的研究成果與經
驗後，於前期合作計畫中完成了 Ka 頻段速調管之研發，此型微波管並已實際應用於裝備上。隨著
使用單位的需求條件增高，促使更新一代微波管的研發必須立即展開，所以我們將利用先前所累積
之經驗與技術進而研發新式作用結構來有效的增加頻寬。 
在前期計畫中，我們完成了新一代 Ka 頻段速調管(EIK)的研製。其中包括了電子槍設計、聚焦
磁系統研發、高頻作用段理論分析及製作、焊接組裝以及熱測。其最大輸出功率可達 930 W，頻寬

分佈作用腔是將多個單一間隙諧振腔，以耦合槽將各腔耦合串聯，形成一個多間隙複合諧振腔體。
以分佈作用腔組成高頻作用段具有的主要優點為功率承載容量大、特徵阻抗高(characteristic 
impedance, R/Q) 及效率好，以下詳述這些優點： 
1. 總承載功率容量大 
分佈作用腔的主要優點是能夠承載高平均功率能量，原因是分佈作用腔內的能量是均分在每一
個間隙上，當分佈作用腔與單一間隙的腔體負載相同的功率時，分佈作用腔內的間隙上所產生的電
場強度，只有單一間隙腔體的1 n倍。因此，分佈作用腔所能承受的總能量將是單一間隙腔體的 2n
倍，其高頻損耗是單間隙諧振腔的1 n倍，以達成高功率微波放大器之目的。 
2. 特徵阻抗值大及效率高 
理論上，在 n個單間隙諧振腔之間的共同壁上開耦合槽而組成的複合諧振腔，它的 R/Q 將是單
腔的 n倍，因為 R/Q 與幾何結構有關。但實際上而言，由於間隙之間的距離要滿足一定的渡越角要
求以配合同步電壓，再加上耦合槽的影響，實際的 R/Q 要比 n倍單腔的 R/Q略小些。分布作用腔的
R/Q 增加，頻寬也會因此而提升。 
另外由於分佈作用腔的電子束和電磁場的作用並非集中在單一間隙上，而是分布在多個間隙
中，因此作用會比較均勻且能量交換比較充分，所以電子轉換效率也比單間隙腔體高。當頻率在毫
米波段以上時，一般的諧振腔由於損耗隨頻率上升而增加及腔體加工的光潔度不易提高，使得共振
腔的特徵阻抗值偏低。而分佈作用腔的結構正好可以彌補毫米波段以上共振腔體的缺失。 
由於具有上述的優點，我們直接使用分佈作用腔體做為本計畫之設計基本架構，取其高 R/Q 的特
性，希望使效率、群聚頻寬能有所提昇。而本計畫中所以採用的分佈作用腔在腔體個數方面將大幅
的增加，以往為了避免過多模式產生而採用較少的腔數。如今，為因應頻寬增加的需求而增加腔體

二、研究目的 
(A)電子光學系統設計 
電子光學系統的設計主要包含兩大部分，第一部份為電子槍主體以及第二部份的聚焦磁系統。
當兩個系統完成設計後，需將兩個系統搭配，並再次進行調整的工作，使電子束由陰極面射出後再
進入聚焦磁場作用區間時達到最小注腰半徑，並藉由磁力維持電子束半徑直到通過高頻作用段為
止。 
1.電子槍模擬結果 
皮爾斯型電子槍是操作在空間電荷限制之下，須遵守 Child-Langmuir Law 所以其電流發射密
度僅與幾何尺寸有關。當幾何尺寸決定後，電壓與電流的關係就須符合 I = p V3/2 的關係式，也因
此在電子槍的模擬中導流係數 p 也就成為一個決定性的參數。而在前期計畫中，電子槍設計的詳
細步驟已有一定的規範。故在本系列計畫中將不深入探討此等細節，本系列的主要目標是利用新型
的三維模擬軟體進行新一代的電子槍的設計，並比對其與 Gun Code 模擬結果之差異。首先，我們
在 Gun Code 中建立計算模型，第一步驟是確認格點數的收斂性，利用分析同一位置的電位來確認
格點數是否夠多。因 Gun Code 為軸對稱形式的模擬軟體，所以其電子束的軌跡為殼層狀，所已經
由收斂測試得到當條數達 70 條以上時即可收斂。依照此步驟，而設計出的電子槍其導流係數為 
0.24 μp 電子束注腰半徑約等於 0.3 mm 。當陽極電壓為 14 kV 時，發射電流為 400 mA，而射程
也可達作用段初始位置，使電子束順利進入聚焦磁場區間，此部份尚須搭配接下來的模擬工作。圖
六為 Gun Code 模擬結果圖。

3.電子光學系統整體模擬結果 
上述的兩部份工作看似獨立的兩個部份，但實際上最終還是需要進行整體的模擬。電子槍負責
電子束的輸出與壓縮電子束，並使其達最小注腰半徑時進入磁場作用區間。當電子束進入磁場作用
區間時，維持電子束半徑的工作就交由聚焦磁系統來進行。聚焦磁力的強弱也有一定的規範，根據
長期開發的經驗，聚焦磁場值不能太小這將會導致電子束發散，但若是聚焦磁力太大，電子束半徑
的擾動也會增大，因此長期所累積的分析結果可知 4000 高斯左右的磁場強度應用於電子通道半徑
為 0.5mm 的結構上是較為恰當的，但還是得透過模擬結果來進行調整。在上述內容中我們利用三套
不同軟體所得之磁場中首先取最低值的結果來進行分析，利用 Gun Code 以及 CST來模擬。如圖十
三，電子束進入作用段區間後半徑保持穩定並且小於 0.5 mm 的限制，大約也是維持在 0.3 mm 左右
的大小。在圖十四中所顯示的為 CST 的模擬結果，最上面所顯示的是磁場方向，中間的是磁場分佈，
而最後則是電子軌跡，我們也可由此結果得知，電子束亦是相當穩定且半徑約等於 0.3 mm，在沒
有與作用段內電磁波交互作用的情況下通過率達 100%。圖十五所示為電子發射軌跡變化圖。

(C) 電子束於收集極中三維的行進軌跡分析 
微波管在操作使用時，從陰極發射之電子經由電子槍的作用形成均勻的電子束，並穿過一系列
之共振腔到達收集極(collector)。而電子束在此過程中經歷的速度調制、群聚、激勵共振腔、再調
制、再群聚、再激勵的反覆過程，電子束的部份能量轉換成高頻能量，剩餘的能量在電子束撞擊收
集極時轉化為熱能，若讓電子束動能降低就能減低撞擊後產生之熱量，以減少因熱效應產生之聚焦
磁系統暫時性消磁或是永久性消磁的可能性。本計畫利用 CST分析電子束在進入收集極之後的軌
跡，電子進入收集極之後因空間電荷力而發散，如預期的大多數之電子撞擊第二收集極之上，此結
果與實驗所得非常吻合。 
在模擬方面我們分為三個部分，第一階段：直接將假設的發射面放置於磁場屏蔽的區域，此一
模擬最為省時且簡單；第二階段：亦是採用假設的發射面，但將發射面置於一與作用段等長之電子
通道前端，並且加上外加磁場，而在此分為一維的簡化磁場與三維的實際磁場兩種，藉此以探討各
項簡化之準確性。而第三階段：將完整的電子槍加入模擬，這其中亦分成一維與三維磁場的兩種模
擬方式。 
第一階段： 
因在電子光學模擬當中，當電子束進入作用段後，受到聚焦磁系統的聚焦，使得電子束半徑均
能保持在半徑 0.3mm左右的大小。故在本階段的模擬工作中，利用一半徑 0.3mm的發射面替代電
子槍，並且將其置於磁場屏蔽的區間。藉此可省去外加磁場，可以大幅的降低運算時間，其結果可
能誤差較大。由圖三十九可知，電子束在進入第二收集極之後才發散的較明顯。 
第二階段： 
在此階段一樣是採用 0.3mm 的電子發射面來替代電子槍，但在此階段中加入了聚焦磁場與電子
通道。我們利用一個與作用段等長的電子通帶來了解電子束在其中運行的狀況。在外加磁場方面，
我們採用兩種不同的方式，第一是利用一維的簡化磁場來進行進算此方法可以省去部分的計算時間
與減低計算資料的大小；第二是直接利用 CST 所計算的三維靜磁場。由圖四十可知，在此計算模型
中，電子完全在轟擊在第二收集極，而且其發散的時間較第一階段的計算結果來得早。而由圖四十
一所示，其結果與圖四十幾乎相同，此結果證明磁場的簡化不會讓計算結果有太大差異，往後可直
接應用，藉此以節省計算時間。 
第三階段： 
第三階段即是將完整的電子光學系統放入，電子束由陰極面發射經聚焦極壓縮後進入作用段，
此時經過聚焦磁場的控制，最後進入電子收集極。在本階段中亦是採用一維簡化磁場與三維磁場兩
種不同模型進行計算，由圖四十二與四十三可知，電子束亦是完全轟擊於第二級上，且此階段計算
在次驗證利用一維簡化磁場進行進算其誤差非常的小。

(E) 柵極式電子數值分析 
在需求單位的委託之下，本計畫額外增加柵極電子槍的模擬工作。 
在一般皮爾斯型電子槍中(如圖五十所示)，大多利用聚焦電極(Focus Electrode)來控制電子
束之發射與截止。但在極短的時間內無法有效將電子完全截止，故須在陰極表面加上陽極網來增強
快速截止電子之能力。 
 
圖五十、皮爾斯型電子槍簡述圖 
除此之外，利用陽極網來控制電子電流(beam current)最大的好處是陽極網操作電壓可遠小於
陰-陽極電壓(cathode-anode voltage)，這可使速調管之調變器體積及重量縮小。 
    一般來說，柵極可分為控制型柵極以及掩蔽型柵極。控制型柵極可單獨應用於皮爾斯型電子槍
上，而掩蔽型柵極須搭配控制型柵極使用，主要是因為掩蔽型柵極的設計能使控制型柵極的操作電
壓更低，但是非必須之設計，所以在此我們只針對控制型柵極進行理論模擬，其類型可見圖五十一。
