計畫緣由 
根據行政院主計處人口結構調查結果，老
年人口比例正以驚人的速度逐年增長，預
計 2031 年將達到 22%，也就是大約四個
年輕人扶養一個年長者[1] [2] [3] ，因此年
長者的居家與在養老院照養方面的安全、
和外籍看護之間的溝通、應變等相關問題，
值得我們投入更多的力量來關注。例如：
年長者外出迷路時需要一個簡易使用、適
合年長者的返家導引介面，因為目前電子
地圖的操作介面對於年長者而言過於複雜、
不易使用；或是年長者外出若遇到緊急情
況。年長者外出時亦有不知道要向誰求救、
不知道可以前往何處求援的困擾，因此需
要一個在地化的求救功能，可以自動搜尋
鄰近的導護站（如鄰里長服務處、警察局
等），並且自動聯繫求救。再者，由外籍看
護陪伴年長者日間生活的情況已經日漸普
遍，但若是陪伴年長者外出時發生狀況，
外籍看護可能因為語言不通、不熟悉環境
等因素，無法於第一時間選擇對年長者最
有保障的應變措施，因此需要一個重要訊
息翻譯與求救的功能。總合上述緣由，本
系統以Google所提出之Android開放平台
為基礎[4] [5] ，設計一便利機動之行動手
持裝置介面，搭配自行設計之無線藍牙生
理感測傳輸介面，並運用前期計畫所完成
之以 Android 開放平台為基礎之中英文
TTS 系統 (An Android Platform Based 
Chinese/English Text-to-Speech System，計
畫編號 NSC98-2220-E-231-001，以下簡稱
為 ABTTS)，達成一套「以 Android 手機
為基礎之年長者路線引導與照護系統」。本
系統實現年長者專屬設計的 Android 人性
化介面手機，並且可作為與外籍看護溝通
的橋樑，將是一實用方便的行動資訊與年
長者照顧系統。 
 
文獻探討 
本系統主要針對年長族群設計出一
Android人性化介面且必需排除複雜設定、
簡易操作、按鈕與文字清楚易懂等需求之
求救、導航與照護系統。有鑑於上述，我
們列出以下表三款具有代表性的相關產品
進行評估，並且與本系統實現之功能進行
比較。

 外籍看護語音溝通輔助(STFN) 
如圖 13、圖 14 所示為本系統外籍看
護語音溝通輔助功能，本功能透過運用語
音辨識 API (Android Voice Recognition 
API)、Android 內建 TTS API 以及自製中
文 ABTTS API (NSC98 計畫成果) ，實現
中英文語音辨識與雙向翻譯之溝通輔助功
能。年長者透過本系統功能說出中文語音，
系統將語音辨識即翻譯成英文語音；反之，
如外籍看護對本系統說初英文語音，系統
亦可將中文意思以中文語音方式念出，實
現具有語音翻譯功能之外籍看護語音溝通
輔助功能。

中、英文翻譯結果，提供年長者與外
