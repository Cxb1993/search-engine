研究摘要： 
隨著半導體製造技術日益精進，使得積體電路能夠低功率消耗、體積小型化與提
高運算速度等，進而使得電子產品的功能與應用日益豐富及多元並且電子產品得以可攜
帶與長時間使用。行動通訊與資訊產品、消費性電子產品的結合已成為未來的發展趨
勢。根據市場調查，台灣為當今行動消費性電子產品的研發與設計重鎮，本研究計畫合
作廠商體會產業趨勢，整合自身已有的 SOPC/FPGA 與嵌入式系統產品，結合本校研發
能量，提出【全球衛星導航多媒體行動系統設計】產業合作計畫，以增進產學技術交流
與厚植產業技術。 
本計畫主要結合『積體電路設計』、『嵌入式系統』、『無線通訊』與『多媒體』等
當前電子產業技術的主流技術，以進行下列項目的研發： (一) 以 Altera SOPC 為核心
來發展GPS的基頻 (Baseband)接收器，用以接收GPS無線信號並轉換為可為ARM CPU
所接收的 GPS 數位信號，(二) 以 ARM CPU 與 WinCE 5.0 嵌入式作業系統為基礎來發
展特定區域(如校園、風景區與商圈)的『行人用導航軟體』，(三) 以 ARM CPU 與 WinCE 
5.0 嵌入式作業系統為基礎來發展『數位圖片與聲音播放系統』。 
由於全球衛星導航系統提供極大的生活便利性、商機無限，因此如 Nokia、
Panasonic、Mitac、Garmin 等國內外大廠紛紛投入相關產品的研發。本計畫的 GPS 研發
主要是配合廠商需求運用公司產品，自行研發 GPS 軟硬體技術。與其他大廠相比，導
航的地圖資料大多為一般公用的道路，無法顧及到校園、風景區與商圈等特定地區。為
了開發此一特定的利基市場，本計畫的導航地圖資料可以由業者或使用者自行製作，事
先可自行輸入導航地區的路口經緯度座標與道路連結關係，十分便利。 
此外由於 ARM CPU 具有強大的運算功能與低耗電等特性，因此在本研究計畫中
不僅用於『行人用導航軟體』的 GPS 數位信號的擷取與處理，同時也用於『數位圖片
與聲音播放系統』的設計。目前國內外與此一播放系統功能相關的產品有【數位相框】，
但是與一般【數位相框】產品比較，本計畫的『數位圖片與聲音播放系統』具有下列優
點：(一) 研發時間短：在以 WinCE 作業系統為基礎上來開發【數位相框】所需的時間
預計將不超過 30 個工作天，(二) 設計彈性高：一般【數位相框】的功能已經內建在硬
體電路中無法依據客戶或市場需求來加以增減，預期在『數位圖片與聲音播放系統』的
設計中將納入無線網路 Wi-Fi 與藍芽(Bluetooth)通訊功能以透過網路傳送圖片或音樂檔
案，此為一般【數位相框】產品所沒有的功能。 
人才培育成果說明： 
在進行本研究計畫期間計畫主持人指定學生施宏旗與朱文宏等人分別負責硬體平台
設計與驗證、軟體程式設計與模擬與系統整合與功能驗證，期間每月定時與合作廠商研發
人員作研究進度報告同時也反應研究過程中所遇到的困難，此時合作廠商的研發人員則就
其實務經驗提出建議及解決方案，如此產學雙方可從學理的探討與實務經驗的分享達到產
學交流的目的。 
本研究計畫的進行對廠商的研發人員提供專業知識使其不僅知其然更進ㄧ步知其所
以然，增加專業領域的廣度與深度，對其日後研發能力的提升有著積極正面的助益。在另
ㄧ方面，此一研究計畫對技職院校學生不啻增加一個難得的實務參與經驗，從使用微軟公
司的程式開發工具 Visual Basic 2005 以進行【全球衛星導航多媒體行動】系統的設計及功
 3
在 Visual Studio 2005 程式開發工具中並無內建的聲音播放元件，因此我們將自行開發。此
外可以透過程式計時器控制輪流播放各個圖像檔案並可同時同步播放聲音檔，而不會產生
延遲現象。本系統同時也設計電子時鐘、鬧鐘及計時等功能，並且可以設定在特定日期如
生日等播放特定圖像與聲音以促進生活樂趣。此外本系統具有無線連線上網功能可以進行
網頁瀏覽。因此本系統已經涵蓋目前市售『數位相框』的所有功能，由於我們對在 WIN CE
系統下的程式設計已經研發出模組化的設計流程，因此大大提高本系統的功能設計的彈性。
