Comments 
--------------------------------------------------------------------------- 
 
Based on joint geometrical distribution, this paper presents an adaptive Golomb 
code for image coding. Due to the adaptivity, the proposed method improves the 
rate-distortion characteristics of image coding as compared to the original 
Golomb code. 
 
1. It is a good idea to generalize the Golomb code by using the 
joint-probability. This could definitely make the Golomb code more adaptive to 
the versatile image contents, thereby enhancing the coding efficiency. 
2. The improvement of coding efficiency is contributed by the more accurate 
joint geometrical distribution model. It would be better if this paper shows 
some statistical results to validate that the joint geometrical model better 
fits the distortion of DCT based image compression. 
3. This paper will be better if the author compared the proposed method with 
other adaptive entropy coding such as context-adaptive variable length coding 
and context-adaptive binary arithmetic coding.

In theory, using the Huffman code can achieve the optimal 
coding efficiency.  However, we need extra memory to record 
the codeword table.  It is not good for compression.  By con-
trast, as the description in Section 2, when using the Golomb 
code, the codeword table is not required.  Although the 
Golomb coding algorithm approximates the distribution of a 
data by a geometric series and sacrifices the optimization, in 
practice, since the codeword table is saved, it can achieve 
higher compression ratio (i.e., lower data rate) than the Huff-
man code (see our simulations in Section IV).  However, we 
believe that the flexibility and the performance of the Golomb 
code can be further improved.

algorithm, we assume that the data y is near to have a geomet-

V. CONCLUSION 
In this paper, we use the joint probability to improve the 
Golomb coding algorithm.  As the Golomb code, our method 
also has the advantages of being suitable for the data with 
geometric distributions and that no codeword table is required.  
Furthermore, since our proposed algorithm use the informa-
tion of another data to adjust the value of p, it can achieve 
better coding efficiency than the original Golomb code.   
Moreover, in Section IV, we combine the JPEG algorithm 
with the proposed adaptive Golomb code for image compres-
sion.  All simulation results showed that the proposed algo-
rithm outperforms the Huffman coding and the conventional 
Golomb coding algorithms.  Thus, the proposed adaptive 
Golomb code with joint probability has the potential to replace 
the Huffman code in image compression, acoustical signal 
compression, and other data compression applications.
