(中文) 整合 GHSOM 與MARS 於資訊通路分店銷售預測模式之建構
(英文) Sales forecasting for IT chain store with multi-branches using GHSOM and MARS
成果歸屬機構
清雲科技大學 發明人
(創作人)
呂奇傑,李天行
技術說明
(中文) 本研究應用增長式自我組織映射圖(growing hierarchical self-organizing 
map, GHSOM)與多元適應性雲形迴歸 (multivariate adaptive regression 
splines, MARS)建構連鎖資訊通路分店的銷售預測模式。所提預測模式先應用
GHSOM將合作企業的所有分店分成數個集群，集群內各分店的銷售特徵近似，之
後再針對各集群應用MARS建構銷售預測模式。所提方法除了可降低只利用單一預
測模式於所有分店銷售預測所面臨無法符合各分店需求的缺點，也可以減少對每
一個分店均建構專屬預測模式會產生的耗時費力問題。本研究以合作公司的86個
經銷點進行實證研究，並且將所提的整合GHSOM與MARS模式與單純SVR模式的結果
進行比較。實驗結果顯示，在評估不同的集群數之後，將86個經銷點分成6群後
再建構MARS銷售預測模式能有最佳的預測績效，並且將分店分成4-6群的銷售預
測結果均能比單純SVR的預測結果為佳。因此，本計畫所提之整合GHSOM與MARS技
術是一個適用於連鎖資訊通路之銷售預測模式。並且由於列在同一集群的分店代
表銷售型態類似，因此合作公司可以將列在同一集群的分店在行銷或管理應用類
似的方法，可降低管理成本與提高績效。
(英文) In this project a sales forecasting model using growing hierarchical self-organizing map 
(GHSOM) and multivariate adaptive regression and splines (MARS) is proposed. The 
GHSOM is an unsupervised learning technique appropriate for clustering and interpreting 
large volumes of high dimensional data when little to no prior information exists. MARS 
is a procedure for fitting adaptive non-linear regression that uses piece-wise linear basis 
functions to define relationships between a response variable and some set of predictors. 
The proposed model first uses GHSOM to cluster the whole branches into several disjoint 
groups. Then, for each group, MARS is used to construct the forecasting method to 
produce the output. Since the branches belong to the same group will have a similar data 
structure, it is easier for constructing effective forecasting models.
產業別 批發業；零售業；綜合零售業
技術/產品應用範圍 可適用於各種具多分店的連鎖通路的銷售預測，尤其是資訊連鎖通路。

預期效益

個方法可降低存貨風險與提升獲利，並且對各分店進行有效的管理，因此具有頗高的推
廣及運用價值。
