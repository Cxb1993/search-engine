一.中文摘要
本研究已建構一人工足踝關節開發及術
前規劃平台。該人工足踝關節開發及術前規劃
平台具有設計分析模組及足踝關節模擬置換
評估模組。本研究量測台灣地區成年人之脛骨
與距骨的相關數據進行統計，歸納出設計人工
足踝關節之特徵尺寸，得以取得適合台灣地區
成年人脛骨與距骨之通用尺寸進行人工足踝
關節設計與改良，並以有限元素分析法分析人
工足踝關節組裝後所產生之應力，進行設計分
析評估。為評估人工足踝關節置換之合理性，
本研究使用人工足踝關節置換模擬模組，模擬
台灣地區成年人之脛骨及距骨與設計之人工
足踝關節結合後之 3D 數位影像模型，並使用
快速成型機（RP）製做所開發之新型人工足踝
關節原形，與實際個案的足部骨塊 RP 模型進
行置換模擬，並由臨床醫師評估置換之全尺寸
3D 實體模型之合理性。本研究整合電腦輔助
設計與製作工程技術及臨床醫師專業經驗可
確認本開發平台具有設計及改良人工足踝關
節之可行性及實用性。
關鍵詞：全人工踝關節、電腦輔助手術設計、
逆向工程、多媒體
Abstract
This study presents a revolutionary
computer-assisted surgery planning and design
system under multimedia environment for the
replacement ankle joint and surgery. This system
uses the full-scale CAD engineering technique in
designing and developing preoperative planning
modules for the ankle joint replacement surgery.
The planning module not only presents a
real-sized three-dimensional image of the ankle
joint , but it also provides detailed interior
measurements of the ankle joint from various
cutting planes. The multimedia system will
combine the total ankle joint design module. This
study applied multimedia user interface to
integrate diferent software’s function to a 
functional integration surgical planning system.
Using statistics technology this study will define
the dimension of replacement ankle joint for
oriental. Furthermore, for the system being a
computer based with multimedia user interface,
surgeons can utilize it as part of their
preoperative planning and design parameters in
searching for the best and most efficient
operative procedure. This system has databank
that can update and extend, will provide the
clinical cases to different users for experience
learning.
Keywords: ankle joint replacement;
Computer-Aided Surgery; Reverse Engineering;
Multimedia.
二.緣由與目的
傳統的關節融合固定手術雖然能夠減輕
疼痛，但是因為喪失了踝關節的活動度，患者
在行走於不平的地面或跑步的時候會遭遇困
難與不適。另一個顧慮是踝關節固定後，可能
會促使下肢其他鄰近關節承受較大的壓力，而
引發其他關節的疼痛，加速其退化性病變。長
期追蹤的報告顯示，幾乎所有的患者十年左右
就會有明顯的鄰近關節退化。全人工足踝關節
置換因為能夠保留踝關節的生理活動，走起路
來會比較自然；同時，鄰近關節的病變會比傳
統的踝關節融合固定術少得多。
傳統設計人工足踝關節主要係以人體足
踝關節之幾何學來評估設計，可分為脛骨部
位、距骨部位及連結脛距骨之軟骨部位。而尺
4關節尺寸。人工足踝關節主要分為脛骨部
位與距骨部位，人工足踝關節脛骨端之形
狀為一凸字形並在一端有延長之斜面，主
要可替代脛骨含弧及延長部位之受損關節
面；人工足踝關節距骨端之形狀為一半月
形，主要可替代距骨之上半部含弧之受損
關節面，其設計原理為可覆蓋脛、距骨之
整體關節面取代受損失部位，並具備正常
足踝運動之功能[2]。
3. 人工足踝關節之設計尺寸：本研究收集台
灣人足踝關節之特徵值，並將所收集之資
料進行統計，進一步將統計之數值分為 S、
M、L 三種通用尺寸。依照此三種大小設計
人工足踝關節 S、M、L 三種尺寸，使其尺
寸大小符合統計之人體足踝關節之特徵值
數值。
4. 設計人工足踝關節：將資料庫所收集的東
方人足踝關節的相關數據作統計分析，與
市售人工足踝關節的相關數據作交叉分析
比對，找出西方人與東方人在足踝關節部
分上的差異，針對人工足踝關節做設計、
改良，以取得設計人工足踝關節之重要參
數（如距骨關節面曲率與脛、距骨長寬比
等），進行人工足踝關節之設計改良，以符
合台灣地區成年人之特徵與尺寸，其中人
工足踝關節之設計草圖如圖二所示。本研
究設計出 S、M、L 三種人工足踝關節尺寸
大小如圖七所示。其中 S size 適用於脛、
距骨骨架較小之病患，通常以女性病患居
多；M size 為較普遍之尺寸大小，較多病
患適用此種尺寸；L size 為本研究所設計最
大尺寸，適用於脛、距骨骨架較大之病患。

5. 人工足踝關節有限元素分析評估：首先在
3D 影像編輯軟體中，分別將病患之脛、距
骨及人工足踝關節影像模型開啟，接著將
人工足踝關節與病患之脛、距骨損壞部位
進行布林運算，切割出一人工足踝關節可
置放之空間，使人工足踝關節可完全取代
損壞之脛、距骨部位。在進行置換時應特
別注意人體脛、距骨與人工足踝關節特徵
點之匹配，人工足踝關節特徵值 L 應對應
至人體脛骨之寬度與厚度，以及人體距骨
之後端下盤寬度及前後長度；人工足踝關
節特徵值 T1 應對應至人體脛骨之高度；人
工足踝關節特徵值 R 應對應至人體足踝關
節之半徑；人工足踝關節特徵值 L 應對應
至人體足踝關節角度。當在選定人工足踝
關節尺寸大小時，應特別注意特徵值 L、
T1、R、A 之數值，必須選擇與該病患之足
踝關節特徵值符合之尺寸大小。另將設計
好之人工足踝關節進行有限元素分析如圖
三所示，量測人工脛、距骨受力情況以及
