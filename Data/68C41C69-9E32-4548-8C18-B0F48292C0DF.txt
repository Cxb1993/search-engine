一、 計畫背景 
GPU 取代 CPU： 
近年來，電腦中央處理器（CPU）的發展遭遇到很大的瓶頸。受限於散熱問題，CPU 的運
作頻率無法再大幅度地提升，只能夠維持在 3GHz 左右。Intel 與 AMD 都改往多核心方向發展，
以求能夠在不提高運作頻率的前提下，增加 CPU 的運算速度。相較於 CPU 的發展瓶頸，顯示
卡上的繪圖處理器（GPU）技術則是發展迅速。受到家庭娛樂市場（電玩遊戲、高畫質影音播
放）龐大需求的驅動，GPU 的運算速度與功能日新月異。目前，GPU 的運算速度已經超越了
CPU。在 GPU 廠商（NVIDIA）的測試中，GPU 的運算速度是 CPU 的 100 倍以上。在 CPU 廠
商（Intel）於 International Symposium on Computer Architecture 發表的報告中，GPU 只比 CPU
快了 14 倍。從雙方的測試報告中，可以確認一個事實，就是 GPU 的運算速度已經遠遠地超越
了 CPU。至於在多核心技術上，GPU 更是遠遠凌駕在 CPU 之上。目前，核心數最高的 CPU，
是 AMD 的 12 核心 Opteron 處理器。至於 GPU 方面，NVIDIA 已經發售具有 480 運算核心的
GeForce GTX 480 顯示卡，是 CPU 核心數量的 40 倍。 
 
GPU 計算： 
在過去的電腦中，大部分的運算工作是由 CPU 負責執行。只有某些和繪圖相關的特定運
算工作，會交給 GPU 執行。例如電玩遊戲中的 3D 繪圖運算工作，以及藍光與 DVD 視訊的
Mpeg 解壓縮工作。然而，隨著 GPU 的運算速度超越了 CPU，在新的電腦系統中，開始利用
GPU 來取代 CPU 的運算工作。或者是利用 GPU，來執行更大型或是更高速的計算工作，例如： 
1. 科學與工程計算： 
傳統的科學與工程計算，例如 DNA 分析、汽車的空氣動力學與結構強度分析等等，都必須
依靠 CPU 進行運算工作。在此領域中，顯示卡大廠 NVIDIA 推出了 Tesla 計算硬體平台
（http://www.nvidia.com/object/tesla_computing_solutions.html），以及 CUDA 軟體開發環境
（http://developer.nvidia.com/object/gpucomputing.html），利用 GPU 取代傳統的 CPU 運算工
作。使得傳統的科學與工程計算，能夠大幅度地縮短計算時間，加速工程產品的開發速度。 
2. 3D 電玩遊戲： 
在過去的電玩遊戲之中，GPU 的主要工作，是進行 3D 電腦繪圖的相關運算。然而，由於

二、 研究目的與重要性 
流體之著色與可視化： 
完成流體運動的計算模擬之後，最後的工作就是要進行著色或是可視化（visualization）的
工作，讓使用者可以看到三度空間中的流體運動。傳統的電腦繪圖技術，是針對物體的表面，
進行材質貼圖（texturing）、打光（lighting）等著色計算工作。因此，計算所得的流體資料，必
須要先產生出一個流體表面，也就是自由液面（free surface）。然後，再針對這個自由液面，進
行著色工作。在傳統的流體動力學計算中，由於並沒有碎波與浪花等複雜的物理現象，可以非
常簡單地產生出自由液面的外形。因此，傳統的電腦繪圖技術，可以順利地應用在傳統的流體
動力學計算之上。然而，當具有碎波與浪花之時（如圖 06），便無法順利地產生出一個明確的
自由液面。在這種情況之下，傳統的電腦繪圖技術就無法派上用場，需要採用全新的電腦繪圖
技術，以便呈現三度空間中的流場動作。 
 
娛樂產業之瓶頸： 
碎波與浪花等物理現象，是娛樂產業所重視的焦點。傳統的 3D 電腦繪圖技術，無法在這
些物理現象上，呈現出精緻且逼真的畫面，使得娛樂產業的發展受到限制。在電影「海角七號」
中，視覺特效是由大腕影像公司（http://www.cg-bulky.com/）所製作。在電影中，「高砂丸」航
行時所產生的波浪，是透過計算模擬所產生。由於傳統的 3D 電腦繪圖技術只能針對自由液面
進行著色，無法處理碎波與浪花等物理現象，所以該公司將船舶的波浪分成許多部分，分別進
行計算模擬，最後再進行合成，如圖 10 所示。然而，這種權宜性的做法，並無法產生出逼真
寫實的波浪畫面。因此在電影之中，並無法出現船舶波浪的局部特寫，而且必須以較為昏暗或
是較為藝術的氣氛，來呈現船舶的航行畫面，如圖 11。 
相同的問題，也出現在另一部知名的電影「艋舺」之中。該電影是以黑幫打鬥為主題，因
此免不了會有血液飛濺的畫面。血液飛濺也是和碎波與浪花類似的物理現象，無法產生出一個
明確的自由液面。因此，該電影改用高速攝影，實際拍攝液體的飛濺畫面，做為電影中的血液
飛濺素材。
